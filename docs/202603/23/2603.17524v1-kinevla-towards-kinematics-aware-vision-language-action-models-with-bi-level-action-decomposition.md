## 摘要
在本文中，我们引入了一项全新的富含运动学信息的视觉-语言-动作（VLA）任务。在该任务中，语言指令在从开始到完成的关键时刻密集地编码了多种运动学属性（如方向、轨迹、朝向和相对位移），这与现有的仅粗略或部分捕捉运动学的动作指令不同，从而支持细粒度和个性化的操作。在这种设定下，任务目标保持不变，而执行轨迹必须适应指令级的运动学规范。为了应对这一挑战，我们提出了 KineVLA，这是一个视觉-语言-动作框架，它通过双层动作表示和双层推理标记（reasoning tokens）将目标层级的不变性与运动学层级的可变性显式解耦，这些标记作为显式的、受监督的中间变量来对齐语言和动作。为了支持这一任务，我们构建了跨越仿真和真实机器人平台的运动学感知 VLA 数据集，其特点是具有指令级的运动学变化和双层标注。在 LIBERO 和 Realman-75 机器人上进行的大量实验表明，KineVLA 在运动学敏感的基准测试中始终优于强大的 VLA 基线模型，实现了更精确、可控且更具泛化性的操作行为。

## Abstract
In this paper, we introduce a novel kinematics-rich vision-language-action (VLA) task, in which language commands densely encode diverse kinematic attributes (such as direction, trajectory, orientation, and relative displacement) from initiation through completion, at key moments, unlike existing action instructions that capture kinematics only coarsely or partially, thereby supporting fine-grained and personalized manipulation. In this setting, where task goals remain invariant while execution trajectories must adapt to instruction-level kinematic specifications. To address this challenge, we propose KineVLA, a vision-language-action framework that explicitly decouples goal-level invariance from kinematics-level variability through a bi-level action representation and bi-level reasoning tokens to serve as explicit, supervised intermediate variables that align language and action. To support this task, we construct the kinematics-aware VLA datasets spanning both simulation and real-world robotic platforms, featuring instruction-level kinematic variations and bi-level annotations. Extensive experiments on LIBERO and a Realman-75 robot demonstrate that KineVLA consistently outperforms strong VLA baselines on kinematics-sensitive benchmarks, achieving more precise, controllable, and generalizable manipulation behaviors.

### 1. 核心问题与研究动机
*   **核心问题**：现有的视觉-语言-动作（VLA）模型（如 OpenVLA, $\pi_0$）主要关注“目标导向”的任务完成，即“做什么”（如：抓取瓶子），但往往忽略了“如何做”的精细运动学规范（如：以特定角度抓取、沿特定轨迹移动、保持特定朝向）。
*   **研究背景**：在现实的人机交互中，语义相同的目标可能需要完全不同的执行轨迹。现有的 VLA 模型通常假设指令的同义改写不影响执行，导致模型对指令中包含的方向、距离、姿态等精细运动学约束不敏感。
*   **研究目标**：提出一种能够理解并执行“富含运动学信息”（Kinematics-rich）指令的框架，实现目标不变但轨迹可变的精准操控。

### 2. 方法论
KineVLA 的核心思想是通过**双层分解**将任务目标（Goal）与运动学细节（Kinematics）进行显式解耦。
*   **双层动作表示（Bi-Level Action Representation）**：
    *   采用残差向量量化变分自编码器（RVQ-VAE）。
    *   **第一层（目标层）**：捕捉低频的全局任务结构和语义目标。
    *   **第二层（运动学层）**：捕捉高频的运动修正、接触细节和精细参数。
    *   **训练策略**：两阶段训练。第一阶段在多任务数据上预训练目标层码本；第二阶段冻结目标层，在运动学感知数据集上训练残差路径的运动学层码本。
*   **双层推理标记（Bi-Level Reasoning Tokens）**：
    *   引入类似思维链（CoT）的生成范式。模型首先生成两段推理文本：一段描述粗粒度目标，一段描述细粒度运动学约束（如轨迹形状、接触时机、目标部位）。
*   **互信息正则化（Mutual Information Regularization）**：
    *   为了确保生成的推理文本与实际动作逻辑一致，引入互信息损失函数 $I(Reasoning; Action | Context)$。通过 InfoNCE 损失拉近配对的推理嵌入与动作嵌入，减少执行漂移。
*   **推理流程**：输入图像和运动学指令 $\rightarrow$ 生成双层推理文本 $\rightarrow$ 生成双层动作 Token $\rightarrow$ 解码为连续控制信号。

### 6. 主要结论与发现
*   **运动学敏感性**：在目标成功率上，各模型表现接近；但在运动学成功率上，KineVLA 显著优于所有基线模型，证明了双层解耦的必要性。
*   **推理与动作的耦合**：引入双层推理 Token 能显著提升模型对复杂指令的遵循能力，而互信息正则化进一步增强了这种一致性。
*   **效率与精度的平衡**：相比于计算昂贵的扩散模型，基于双层 VQ 的架构在保持高精度的同时，具备更快的推理速度。

### 7. 优点（亮点）
*   **任务定义新颖**：首次明确提出了“目标不变、运动学可变”的 VLA 任务设定，贴近真实个性化需求。
*   **架构设计优雅**：通过 RVQ-VAE 的残差结构自然地实现了粗/细粒度动作的解耦，而非简单的多头输出。
*   **可解释性强**：双层推理文本让机器人的“思考过程”可视化，便于用户理解机器人为何产生特定动作。

### 8. 不足与局限
*   **场景局限性**：目前的实验主要集中在桌面操作（Tabletop manipulation），尚未在全身移动操控或更复杂的动态环境中验证。
*   **坐标系依赖**：运动学指令依赖于第三人称相机坐标系（左、右、前、后），如果相机视角发生大幅度变化，模型可能需要重新适应。
*   **数据标注成本**：双层推理文本和精细运动学标注需要大量人工参与，虽然论文提供了数据集，但扩展到更多未知任务时，数据获取成本较高。
