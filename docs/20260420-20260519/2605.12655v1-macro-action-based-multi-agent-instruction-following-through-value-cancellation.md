## Abstract
Multi-agent reinforcement learning (MARL) in real-world use cases may need to adapt to external natural language instructions that interrupt ongoing behavior and conflict with long-horizon objectives. However, conditioning rewards on instructions introduces a fundamental failure mode as Bellman updates couple value estimates across instruction contexts, leading to inconsistent values when instructions interrupt macro-actions. We propose Macro-Action Value Correction for Instruction Compliance (MAVIC), which corrects Bellman backups at instruction boundaries by correcting the incoming instruction objective and restoring the continuation value under the current objective. Unlike reward shaping, MAVIC modifies the bootstrapping target itself, enabling consistent value estimation under stochastic instruction switching within a unified policy. We provide theoretical analysis and an actor-critic implementation, and show that MAVIC achieves high instruction compliance while preserving base task performance in increasingly complex cooperative multi-agent environments.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
提出**MAVIC（Macro-Action Value Correction for Instruction Compliance）**，一种中断感知的价值估计方法，通过在指令边界处修正Bellman备份来实现指令遵从，同时保持基础任务性能。

### 关键问题建模
- 将问题建模为**宏动作-Dec-POMDP（MacDec-POMDP）**，处理长期任务和部分可观测性
- 将人类指令建模为离散指令类集合C，每个类对应语义等价的自然语言命令
- 定义指令增强的MDP：M^C = ⟨I, S×C, A, M, Ω×C, ζ×C, T', R', O', Z', γ⟩

### 价值污染问题分析
- 公式（3）展示：当评估非指令状态(s, c∅)的价值时，期望返回依赖于：
  - 转移到指令状态的概率
  - 智能体在指令下的表现
- 这导致两个问题：
  1. 智能体学会忽略对基础任务次优的指令
  2. 智能体提前改变基础行为以应对未来指令

### MAVIC核心机制
**动态奖励校正**（表2）：
- 当指令保持不变（c = c'）：R' = R_c
- 当指令改变（c ≠ c'）：R' = R_c - γV((s', c')) + γV((s', c))
- 关键思想：用当前指令的延续价值替换新指令的引导值

### 与奖励塑形的关系
- 不同于基于势能的奖励塑形（Ng et al., 1999）
- MAVIC引入**策略依赖的校正**，直接修改引导目标
- 无法表示为固定势函数，而是在Bellman更新层面强制一致性

### 理论保证
- **引理1**：策略Ψ在指令类c下的价值等于在特定指令问题M_c中的价值
- **定理1**：如果在M^C中策略Ψ*是最优的，那么它在每个指令特定问题M_c中也是最优的
- **推论**：MAVIC解耦了指令条件价值函数，同时保持统一策略

## 3. 实验设计：数据集、基准、对比方法

### 实验环境（3个场景）

| 环境 | 描述 | 指令类型 |
|------|------|---------|
| **Box Pushing (BP)** | 两机器人协调推动中央箱子到目标位置 | "Go to small boxes", "Don't push the box" |
| **Warehouse Tool Delivery (WTD)** | 三个异构机器人（两移动+一机械臂）为工人递送工具 | "Get me tool 0-3" |
| **Overcooked (OC)** | 3-4个智能体在7×7网格中协作准备菜肴 | 约束指令、不良请求、直接命令 |

### 对比基线
1. **Naive（朴素方法）**：直接使用指令任务奖励，无价值校正
2. **Switch（切换训练）**：从各指令类独立收集训练数据
3. **Vanilla Mac-IAC**：无指令训练/评估，作为基础任务性能上界

### 评估指标
- **基础任务性能**：无指令环境下的平均返回
- **指令遵从度**：实时指令环境中，已遵从指令数/总指令数

## 6. 论文的主要结论与发现

### 核心发现
1. **MAVIC是唯一能同时实现高基础任务性能和高指令遵从度的方法**
   - OC：Base 122.28（Vanilla 152.31），Compliance 99%
   - BP：Base 288.32（Vanilla 290.42），Compliance 50%（显著优于Naive的5%和Switch的6%）
   - WTD：Base 438.90，Compliance 93%

2. **朴素方法存在明显权衡困境**
   - OC/WH：基础性能大幅下降（如OC从152降至3.62）
   - BP：完全无法遵从指令（仅5%）

3. **切换训练方法不稳定**
   - OC/WH中高方差（标准差49.46和30.11）
   - 可能因每episode任务切换导致训练不稳定

4. **价值校正的必要性取决于任务特性**
   - 当指令仅重新排序现有奖励时（如WTD改变递送顺序），各方法差异不大
   - 当指令引入冲突价值结构时（如OC的约束指令），MAVIC优势显著

5. **分布外泛化良好**
   - 语义相似的新指令达到99%+遵从度
   - 不可行指令导致统一的保守行为

## 7. 优点

### 方法论亮点
1. **问题本质洞察**：精准识别"跨指令类价值污染"这一根本性问题
2. **理论严谨性**：提供引理和定理的完整证明，建立解耦理论基础
3. **修正粒度**：不同于奖励塑形，直接修改Bellman引导目标
4. **统一策略**：保持单一策略处理所有指令，而非为每个指令训练独立策略

### 实验设计亮点
1. **多层次验证**：标准环境 + OOD指令 + 不可行指令测试
2. **公平对比**：与朴素方法、切换训练、上界基线全面对比
3. **统计严谨**：多种子+置信区间+多指标评估

### 实践价值
- 冻结语言编码器避免微调成本
- 可集成到现有宏动作actor-critic方法（如Mac-IAC）
- 通用框架可扩展至非语言条件设置

### 方法论局限

3. **探索未优化**：MAVIC未显式处理稀有或不平衡指令的探索问题

### 理论局限
1. **部分可观测性处理**：用历史条件近似状态条件函数，理论上未完全证明充分性
2. **宏动作持续时间**：假设宏动作执行期间指令可中断，未考虑连续中断的嵌套问题

### 应用范围
1. **离散指令假设**：仅处理离散指令类集合，对开放式语言指令可能不适用
2. **预设宏动作**：依赖预定义的宏动作集合，不适用于需要学习动作层次结构的场景
