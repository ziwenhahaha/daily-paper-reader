## Abstract
On-policy distillation is an efficient alternative to reinforcement learning, offering dense token-level training signals. However, its reliance on a stronger external teacher has driven recent work on on-policy self-distillation, where the same model serves as both teacher and student under different prompt contexts. Yet, existing self-distillation methods largely reduce learning to KL matching toward the context-augmented teacher model. This approach often suffers from training instability and can degrade reasoning performance over time. Moreover, self-distillation from the same model with prompt augmentation lacks the exploratory diversity provided by a genuine external teacher. To address these limitations, we move beyond fixed-teacher KL matching and propose \textbf{P}reference-\textbf{B}ased \textbf{S}elf-\textbf{D}istillation (\textbf{PBSD}), which revisits on-policy self-distillation through a reward-regularized perspective. Instead of directly matching the teacher distribution, we derive a reward-regularized objective whose analytic optimum is a reward-reweighted teacher distribution, yielding a target policy provably superior to the original teacher under this objective. Practically, PBSD optimizes preference gaps between teacher and student samples while maintaining on-policy student sampling. We support this framework with a statistical analysis of the induced preference-learning problem, formally establishing when on policy self-distillation is preferable to learning from an external teacher in our setting. Experiments on mathematical reasoning and tool-use benchmarks across multiple model scales demonstrate that PBSD consistently achieves the strongest average performance among comparable baselines, showing improved training stability over prior self-distillation baselines while preserving token efficiency.

# 论文总结：Preference-Based Self-Distillation: Beyond KL Matching via Reward Regularization

## 1. 核心问题与整体含义（研究动机和背景）

- **On-policy 蒸馏的兴起**：On-policy 蒸馏（OPD）是 LLM 后训练的高效替代方案，提供密集的 token 级训练信号，比传统 RL（如 GRPO）更具 token 效率。
- **现有方法的痛点**：
  - **标准 OPD 依赖外部教师模型**：需要更大的教师模型，增加了计算成本和系统复杂度，且存在词汇表不兼容、输出空间不匹配等问题。
  - **自蒸馏方法退化为 KL 匹配**：现有自蒸馏方法（如 OPSD）将学习简化为直接 KL 匹配到上下文增强的教师分布，存在训练不稳定、推理性能随训练退化的问题（尤其是抑制了模型的认知表达如不确定性、犹豫、自我检查和错误纠正）。
  - **缺乏探索多样性**：自蒸馏中的教师仅通过 prompt 增强构造，与真正的外部强教师相比，缺乏探索多样性和有价值的监督信号。
- **核心洞察**：教师不应被视为需要统一复制的最终目标，而应作为支持分布，结合奖励信息重新分配概率质量。

## 2. 论文提出的方法论

### 核心思想
- 提出 **Preference-Based Self-Distillation (PBSD)**，将自蒸馏重新定义为**奖励正则化优化问题**，而非直接 KL 匹配。
- 通过偏好学习框架优化教师-学生样本间的偏好差距，学习一个**奖励重加权的目标策略**。

### 关键技术细节

#### (1) 奖励正则化目标函数
对于输入 x，优化以下目标：
$$\max_{\pi_\theta(\cdot|x)} \mathbb{E}_{y \sim \pi_\theta(\cdot|x)}[r(x,y)] - \beta D_{KL}(\pi_\theta(\cdot|x) \| \pi_{teach}(\cdot|x))$$
其中：
- $\pi_\theta$ 是学生策略
- $\pi_{teach}(y|x) := \pi(y|x,c)$ 是通过附加上下文 c 构造的教师策略
- $r(x,y)$ 是隐式奖励
- $\beta > 0$ 控制 KL 正则化强度

#### (2) 解析最优解：奖励倾斜的教师分布
通过一阶最优性条件推导，最优策略为：
$$\pi^*(y|x) = \frac{\pi_{teach}(y|x) \exp(r(x,y)/\beta)}{Z(x)}, \quad Z(x) = \sum_y \pi_{teach}(y|x) \exp(r(x,y)/\beta)$$
**关键性质**：该奖励倾斜策略在相同目标函数下**严格优于**原始教师分布（由 Jensen 不等式保证）。

#### (3) 通过偏好学习实例化目标
- **偏好对比构建**：正样本 y+ 来自条件更好的教师，负样本 y- 来自当前 on-policy 采样的学生。
- **偏好边际**：$m_\theta(x, y^+, y^-) = \beta \left[ \log \frac{\pi_\theta(y^+|x)}{\pi_{teach}(y^+|x)} - \log \frac{\pi_\theta(y^-|x)}{\pi_{teach}(y^-|x)} \right]$
- **优化目标**（Bradley-Terry 模型）：
$$\max_\theta \mathbb{E}_{(x,c)\sim D} \mathbb{E}_{y^+ \sim \pi_{teach}(\cdot|x), y^- \sim \pi_\theta(\cdot|x)} \left[ \log \sigma(m_\theta(x, y^+, y^-)) \right]$$
其中 $\sigma(z) = 1/(1+\exp(-z))$ 是 logistic sigmoid 函数。

#### (4) 梯度形式
$$\nabla_\theta \ell_i(\theta) = -\beta \sigma(-m_\theta) \left[ \sum_k \nabla_\theta \log \pi_\theta(y^+_{i,k}|...) - \sum_k \nabla_\theta \log \pi_\theta(y^-_{i,k}|...) \right]$$
梯度下降增加教师生成 token 的似然，同时减少学生生成负样本的似然，权重 $\sigma(-m_\theta)$ 自适应调整。

### 算法流程（PBSD）
1. **输入**：训练集 D（x,c 对），学生策略 $\pi_\theta$，教师策略 $\pi_{teach}(y|x) = \pi(y|x,c)$，温度 $\beta$，学习率 $\eta$，总步数 T。
2. **采样**：从 D 中采样 mini-batch $\{(x_i, c_i)\}$。
3. **生成响应**：对每个 $(x_i, c_i)$：
   - 生成学生响应 $y^-_i \sim \pi_\theta(\cdot|x_i)$
   - 生成教师响应 $y^+_i \sim \pi_{teach}(\cdot|x_i)$
4. **计算损失**：计算 PBSD 边际 $m_\theta$，然后 $L_i = -\log \sigma(m_\theta)$，取平均得到 $L_{PBSD}$。
5. **更新**：$\theta \leftarrow \theta - \eta \nabla_\theta L_{PBSD}$。
6. **教师固定**：训练过程中教师策略固定在初始 checkpoint，避免教师漂移。
