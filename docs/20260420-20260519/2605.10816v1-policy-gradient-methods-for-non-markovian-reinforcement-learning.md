## Abstract
We study policy gradient methods for reinforcement learning in non-Markovian decision processes (NMDPs), where observations and rewards depend on the entire interaction history. To handle this dependence, the agent maintains an internal state that is recursively updated to provide a compact summary of past observations and actions. In contrast to approaches that treat the agent state dynamics as fixed or learn it via predictive objectives, we propose a reward-centric formulation that jointly optimizes the agent state dynamics and the control policy to maximize the expected cumulative reward. To this end, we consider a class of Agent State-Markov (ASM) policies, comprising an agent state dynamics and a control policy that maps the agent state to actions. We establish a novel policy gradient theorem for ASM policies, extending the classical policy gradient results from the Markovian setting to episodic and infinite-horizon discounted NMDPs. Building on this gradient expression, we propose the Agent State-Markov Policy Gradient (ASMPG) algorithm, which leverages the recursive structure of the agent state dynamics for efficient optimization. We establish finite-time and almost sure convergence guarantees, and empirically demonstrate that, on a range of non-Markovian tasks, ASMPG outperforms baselines that learn state representations via predictive objectives.

# 论文详细总结：《Policy Gradient Methods for Non-Markovian Reinforcement Learning》

## 1. 核心问题与整体含义（研究动机和背景）

- **非马尔可夫决策过程（NMDPs）**：传统强化学习基于马尔可夫决策过程（MDP）假设，即奖励和下一个观测仅取决于当前状态和动作。然而在现实场景（如对话系统、推荐系统、医疗治疗）中，奖励和观测依赖于**完整的交互历史**，此时MDP假设失效，过程本质上是非马尔可夫的。
- **历史压缩的挑战**：完整历史随时间线性增长，直接用于决策计算上不可行。已有方法如有限观测窗口、有限状态控制器、预测状态表示（PSR）等存在局限性：要么表示类别固定，要么缺乏端到端奖励优化保证。
- **现有方法的痛点**：当前主流方法倾向于**固定或通过辅助预测目标学习** agent 状态表示（即“信息状态”），然而预测未来观测并非最终目标，直接最大化累积奖励才是 RL 的根本目的。
- **核心研究目标**：如何在 NMDP 中联合优化 agent 状态的递归动态（state dynamics）和控制策略，使得内部状态既能压缩历史，又能直接为最大化期望累积奖励服务。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式与算法

### 2.1 核心思想

- **Agent State-Markov（ASM）策略类**：定义一类策略，其中 agent 维护一个递归更新的内部状态 $S_t$，该状态仅依赖于前一状态 $S_{t-1}$、前一动作用 $A_{t-1}$、当前观测 $O_t$，并通过随机核 $\nu_t$ 更新。控制策略 $\phi_t$ 则基于当前 agent 状态 $S_t$ 选择动作。ASM 策略由 $(\nu, \phi)$ 参数化。
- **奖励驱动（reward-centric）**：与通过辅助预测目标学习状态表示不同，本方法直接以期望累计奖励为优化目标，联合优化 $\nu$（状态动态）和 $\phi$（控制策略）的参数。

### 2.2 关键技术与公式

- **组合核 $\pi_{\theta,t}$**：将状态动态和控制策略融合为单一核：
  \[
  \pi_t(s, a \mid \tilde{s}, \tilde{a}, o) = \nu_t(s \mid \tilde{s}, \tilde{a}, o) \cdot \phi_t(a \mid s)
  \]
  这使得梯度可以分解为两部分：状态更新的梯度 + 策略输出的梯度。
- **策略梯度定理（episodic NMDP）**：对参数化为 $\pi_\theta$ 的 ASM 策略，梯度为：
  \[
  \nabla_\theta J(\theta) = \mathbb{E}_{\pi_\theta, \mu}\left[ \sum_{t=1}^H R_{t:H} \nabla_\theta \log \pi_{\theta,t}(S_t, A_t \mid S_{t-1}, A_{t-1}, O_t) \right]
  \]
  其中 $R_{t:H} = \sum_{\ell=t}^H r_\ell$ 为从 $t$ 到回合结束的回报。
- **策略梯度定理（infinite-horizon discounted NMDP）**：折扣情形下的梯度形式为：
  \[
  \nabla_\theta J_\gamma(\theta) = \mathbb{E}_{\pi_\theta, \mu}\left[ \sum_{t=1}^\infty \gamma^{t-1} R_{\gamma,t:\infty} \nabla_\theta \log \pi_\theta(S_t, A_t \mid S_{t-1}, A_{t-1}, O_t) \right]
  \]
  其中 $R_{\gamma,t:\infty}$ 为折扣累计回报。$1/(1-\gamma)$ 起到有效水平线（horizon）的作用。
- **光滑性（Smoothness）**：在假设 $\nabla_\theta \log \pi_{\theta,t}$ 的范数和 Hessian 范数有界（常数 $G, M$）下，目标函数 $J(\theta)$ 是 $\beta$-光滑的：
  \[
  \beta = \frac{r_{\max} H(H+1)}{6}\left(3M + G^2(2H+1)\right)
  \]
- **收敛性分析**：在 episodic 和 infinite-horizon discounted 两种设定下，均给出了有限时间收敛界（时间平均平方梯度范数为 $O(1/\sqrt{K})$ 量级）以及几乎必然（almost sure）收敛到梯度范数为零的结论。

### 2.3 算法流程（ASMPG）

**Algorithm 1（episodic）**  
1. 初始化参数 $\theta_1$，设置步长序列 $\{\alpha_k\}$。  
2. 对于每个 episode $k$：  
   - 执行当前策略 $\pi_{\theta_k}$，收集轨迹 $\{(O_t^{k}, S_t^{k}, A_t^{k})\}_{t=0}^H$。  
   - 计算梯度估计  
     \[
     \hat{g}_k = \sum_{t=1}^H \left( \sum_{\ell=t}^H r_\ell(O_{1:\ell}^{k}, A_{1:\ell}^{k}) \right) \nabla_\theta \log \pi_{\theta_k,t}(S_t^{k}, A_t^{k} \mid S_{t-1}^{k}, A_{t-1}^{k}, O_t^{k})
     \]  
   - 更新参数 $\theta_{k+1} = \theta_k + \alpha_k \hat{g}_k$。  

**Algorithm 2（infinite-horizon discounted）**  
- 类似结构，梯度估计改为折扣回报形式，实际中通过截断轨迹近似无穷级数。

## 3. 实验设计：数据集、场景、Benchmark 与对比方法

### 3.1 实验环境（5 个非马尔可夫场景）

| 环境 | 描述 | 关键非马尔可夫特性 |
|------|------|-------------------|
| **CheeseMaze** | 部分可观测导航问题（McCallum, 1993），11个潜在状态，7个观测符号，多个位置共享相同观测 | 需要从历史推断潜在位置 |
| **HallwayNavigation** | 确定性部分可观测网格导航（McCallum, 1995），7×4 网格，观察局部墙壁配置 | 相同观测对应不同动作 |
| **HealthcareTreatment** | 治疗规划问题，患者健康状态依赖历史（毒性积累、治疗耐药性），仅观测健康分数 | 隐藏变量（毒性/耐药性）影响转移和奖励 |
| **MachineRepair** | 机器维护问题，机器状态（健康/退化）依赖累积磨损史，仅观测粗粒度健康状况 | 磨损不可观测且随历史演变 |
| **VelocityOnlyCartPole** | 部分可观测 CartPole，仅提供线速度和角速度，位置不可直接观测 | 相同速度对应不同位置/角度组合 |

### 3.2 对比基线

- **AIS-KL**：近似信息状态（Approximate Information State），使用 KL 散度预测目标学习表示。
- **AIS-MMD**：近似信息状态，使用最大均值差异（MMD）预测目标学习表示。
- 所有方法在相同表示维度 $|S|$（agent 状态数）和隐藏层维度 $dh$ 下进行比较，具体取值见 Table 1（CheeseMaze: $|S|=8, dh=128$；HallwayNavigation: $|S|=8, dh=64$；HealthcareTreatment: $|S|=3, dh=64$；MachineRepair: $|S|=8, dh=64$；VelocityOnlyCartPole: $|S|=4, dh=128$）。

### 3.3 评估指标

- **学习曲线**：随训练步数变化的平均奖励（episodic 任务使用每步平均奖励，其他使用每回合平均奖励）。

## 6. 论文的主要结论与发现

1. **ASMPG 在所有 5 个任务上均优于基线**：无论是平均学习曲线还是最佳检查点性能，ASMPG 均显著超过 AIS-KL 和 AIS-MMD。
2. **奖励驱动优化的优势**：直接以累积奖励为目标联合优化状态动态和控制策略，比通过辅助预测目标学习状态表示更有效。
3. **收敛效率**：学习曲线显示 ASMPG 在早期阶段即展现出更快的提升速度，表明端到端奖励优化可更高效地学习有用的状态表示。
4. **理论保证的有效性**：有限时间收敛界和几乎必然收敛结果在实际实验中得到了间接验证（算法能有效学习）。

## 7. 优点

1. **理论创新**：首次将策略梯度定理从标准 MDP 推广到一般 NMDP 的 ASM 策略类，给出了完整的数学推导（见 Appendix B 和 C）。
2. **统一框架**：将状态动态（nu）和控制策略（phi）统一到一个参数化核中，简化了分析和实现。
3. **收敛性分析严格**：提供了有限时间收敛率 $O(1/\sqrt{K})$、高概率界和几乎必然收敛保证，基于标准的光滑性假设。
4. **实验验证充分**：5 个多样化环境，跨 10 个随机种子，充分展示了方法在非马尔可夫任务中的竞争力。
5. **对预测目标的批判性**：明确指出预测未来观测仅为中间目标，直接最大化奖励更符合 RL 本质。

## 8. 不足与局限

1. **高方差问题**：ASMPG 基于 REINFORCE 型估计器，存在固有高方差，论文也承认这点（如导航任务中的波动）。缺乏方差缩减技术（如 baseline、actor-critic）的探索。
2. **表示容量限制**：实验使用的离散状态空间 $|S|$ 较小（3-8），对高维连续状态或观测空间的适用性未知。
3. **消融实验不足**：未系统研究状态动态与控制策略协同优化的边际贡献，也未对比不同状态空间大小对性能的影响。
4. **仅离散空间**：实验环境均为离散观测/动作空间，未涉及图像等高维观测的连续空间场景。
5. **缺乏全局最优性**：收敛保证仅到梯度范数为零的驻点，未讨论陷入局部最优的可能性，与近期 Markovian 设定下的全局收敛结果（如 Mei et al., 2020）形成对比。
6. **计算开销**：每次更新需 10 条 episode 平均梯度，计算量相对较大，在更复杂场景中的可扩展性待验证。
7. **实验客观性**：虽提供了 10 个种子的平均值，但未给出误差棒或显著性检验，部分图中 ASMPG 的波动可能影响结论可靠性。
