## Abstract
We study policy gradient methods for reinforcement learning in non-Markovian decision processes (NMDPs), where observations and rewards depend on the entire interaction history. To handle this dependence, the agent maintains an internal state that is recursively updated to provide a compact summary of past observations and actions. In contrast to approaches that treat the agent state dynamics as fixed or learn it via predictive objectives, we propose a reward-centric formulation that jointly optimizes the agent state dynamics and the control policy to maximize the expected cumulative reward. To this end, we consider a class of Agent State-Markov (ASM) policies, comprising an agent state dynamics and a control policy that maps the agent state to actions. We establish a novel policy gradient theorem for ASM policies, extending the classical policy gradient results from the Markovian setting to episodic and infinite-horizon discounted NMDPs. Building on this gradient expression, we propose the Agent State-Markov Policy Gradient (ASMPG) algorithm, which leverages the recursive structure of the agent state dynamics for efficient optimization. We establish finite-time and almost sure convergence guarantees, and empirically demonstrate that, on a range of non-Markovian tasks, ASMPG outperforms baselines that learn state representations via predictive objectives.

# 论文总结：非马尔可夫决策过程的策略梯度方法

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **问题领域**：强化学习（RL）在**非马尔可夫决策过程（Non-Markovian Decision Process, NMDP）**中的应用。NMDP 的核心特征是环境的转移概率和奖励函数不仅依赖于当前状态和动作，还依赖于**完整的交互历史**（即所有历史观测和动作）。
- **实际背景**：在许多现实场景中马尔可夫性不成立，例如：
  - 部分可观测环境（POMDP）：智能体只能从历史中推断隐状态；
  - 对话系统：合适的回复依赖于完整对话历史；
  - 推荐系统、医疗治疗、股票市场等具有长期依赖或隐变量驱动的领域。
- **核心挑战**：
  1. 历史空间随时间指数增长，直接使用完整历史在计算上不可行；
  2. 表征学习与控制紧密耦合，需要联合学习状态表示和策略；
  3. 传统方法要么**固定**智能体状态动态（如 Dong et al., 2022），要么通过**辅助预测目标**学习表示（如 AIS-KL、MMD 等近似信息状态方法），而非直接针对奖励目标优化。
- **研究动机**：本文认为预测未来观测只是中间步骤，**直接以最大化累积奖励为目标**来联合优化智能体状态动态和控制策略，是更直接、更本质的方式。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式与算法流程

### 2.1 核心思想

提出**智能体状态-马尔可夫（Agent State-Markov, ASM）策略**类：将历史依赖的决策问题转化为对**递归更新的内部状态**的马尔可夫决策问题。ASM 策略包含两个组件：

- **智能体状态动态（Agent State Dynamics）** $\nu_t$：$S_t \sim \nu_t(\cdot | S_{t-1}, A_{t-1}, O_t)$，根据前一步的动作和当前观测递归更新内部状态 $S_t$；
- **控制策略（Control Policy）** $\phi_t$：$A_t \sim \phi_t(\cdot | S_t)$，基于当前内部状态做出决策。

这两者共同诱导出一个历史依赖的策略 $\phi \circ \nu$，智能体只需维护一个远小于完整历史的内部状态。

### 2.2 关键公式与理论结果

#### 定理 2（情景式 NMDP 的 ASM 策略梯度定理）

$$ \nabla_\theta J(\theta) = \mathbb{E}_{\pi_\theta, \mu} \left[ \sum_{t=1}^H R_{t:H} \nabla_\theta \log \pi_{\theta,t}(S_t, A_t \mid S_{t-1}, A_{t-1}, O_t) \right] $$

其中 $R_{t:H} = \sum_{\ell=t}^H r_\ell$ 是从 $t$ 到回合结束的累积奖励，$\pi_{\theta,t}$ 是参数化的 ASM 核。该形式与 REINFORCE 类似，但适用于非马尔可夫环境。

#### 定理 6（无限时域折扣 NMDP 的 ASM 策略梯度定理）

$$ \nabla_\theta J_\gamma(\theta) = \mathbb{E}_{\pi_\theta, \mu} \left[ \sum_{t=1}^\infty \gamma^{t-1} R_{\gamma,t:\infty} \nabla_\theta \log \pi_\theta(S_t, A_t \mid S_{t-1}, A_{t-1}, O_t) \right] $$

其中 $R_{\gamma,t:\infty} = \sum_{t'=t}^\infty \gamma^{t'-t} r_{t'}$ 是折扣累积回报，$1/(1-\gamma)$ 充当有效 horizon。

#### 关键洞察

由于 $\pi_{\theta,t}$ 可分解为 $ \nu_{\theta,t} \cdot \phi_{\theta,t} $，梯度可进一步写为：

$$ \nabla_\theta \log \pi_{\theta,t}(S_t, A_t \mid \cdots) = \nabla_\theta \log \nu_{\theta,t}(S_t \mid \cdots) + \nabla_\theta \log \phi_{\theta,t}(A_t \mid S_t) $$

这使得**智能体状态动态和控制器可以分别独立更新**。

### 2.3 ASMPG 算法流程

#### 情景式版本（Algorithm 1）

1. 初始化参数 $\theta_1$，设定学习率 $\{\alpha_k\}$；
2. 每轮迭代 $k$：
   - 用当前策略 $\pi_{\theta_k}$ 采样一条完整轨迹；
   - 计算梯度估计：
     $$ \hat{g}_k = \sum_{t=1}^H \left( \sum_{\ell=t}^H r_\ell(o_{1:\ell}^{k}, a_{1:\ell}^{k}) \right) \nabla_\theta \log \pi_{\theta_k,t}(s_t^k, a_t^k \mid s_{t-1}^k, a_{t-1}^k, o_t^k) $$
   - 更新参数：$\theta_{k+1} = \theta_k + \alpha_k \hat{g}_k$。

#### 无限时域折扣版本（Algorithm 2）

结构相同，但使用折扣回报 $R_{\gamma,t:\infty}$，在实际中通过截断轨迹近似（选取使 $\gamma^t$ 可忽略的步数）。

### 2.4 理论保证

- **平滑性引理（Lemma 4）**：目标函数 $J(\theta)$ 是 $\beta$-平滑的，其中 $\beta = \frac{r_{\max} H(H+1)}{6} [3M + G^2(2H+1)]$；
- **有限时间收敛率（Theorem 5）**：时间平均的梯度范数平方以 $\tilde{O}(H^{3.5}/\sqrt{K})$ 速率趋于零（$K$ 为迭代次数）；
- **几乎必然收敛（Theorem 9c）**：当步长满足 $\sum \alpha_k = \infty$ 且 $\sum \alpha_k^2 < \infty$ 时，$\|\nabla_\theta J(\theta_k)\| \to 0$ a.s.；
- **无限时域情形**：收敛率 $\tilde{O}((1-\gamma)^{-3.5}/\sqrt{K})$。

## 3. 实验设计：数据集 / 场景、Benchmark 与对比方法

### 3.1 实验环境（5 个非马尔可夫任务）

- 以上两者均通过辅助预测目标学习状态表示，而非直接优化奖励。

## 6. 论文的主要结论与发现

1. **奖励导向的联合优化优于预测导向的表示学习**：ASMPG 在全部 5 个非马尔可夫环境中均优于 AIS-KL 和 AIS-MMD 基线，支持了直接优化奖励比通过辅助预测目标学习状态表示更有效的假设；
2. **策略梯度定理可以推广到 NMDPs**：通过 ASM 策略框架，可以将经典 REINFORCE 型梯度表达式从马尔可夫设定扩展到非马尔可夫设定，且梯度具有简洁的递归形式；
3. **理论收敛保证**：在标准正则性假设下（梯度有界、Hessian 有界），ASMPG 具有 $\tilde{O}(H^{3.5}/\sqrt{K})$ 的有限时间收敛率和几乎必然收敛到驻点的保证；
4. **状态动态与控制策略的解耦更新**：由于 ASM 策略的因子分解性质，两个组件可分别独立更新，简化了实现；
5. **对话系统示例**：以聊天机器人为典型案例，说明了 ASM 策略如何自然地建模历史依赖的交互（如 Figure 1）。

## 7. 优点

1. **方法论创新**：提出**奖励导向的联合优化**框架，首次将智能体状态动态和控制策略的联合优化纳入策略梯度框架，突破了传统方法中状态动态固定或仅通过辅助目标学习的局限；
2. **理论贡献完整**：提供了从策略梯度定理→平滑性分析→有限时间收敛率→几乎必然收敛的完整理论链条；
3. **通用性强**：同时覆盖**情景式**和**无限时域折扣**两种设定，框架具有一般性；
4. **实验设计规范**：多环境、多种子、公开种子、报告变异性（阴影带 + 箱线图），符合学术规范；
5. **梯度形式简洁**：最终梯度仅依赖于当前步的回报-到-结束部分和策略对数梯度，利用了 ASM 递归结构的优势；
6. **理想状态动态条件清晰**（Appendix D）：给出了 ASM 策略不失最优性的充分条件，增强了理论的可解释性。

## 8. 不足与局限

1. **实验对比范围有限**：
   - 仅与预测型信息状态方法比较，未对比 RNN 策略梯度、自然 actor-critic、有限状态控制器等方法；
   - 缺乏对其他非马尔可夫 RL 方法（如 Reward Machines、Regular Decision Processes）的比较；
2. **缺少消融实验**：
   - 未单独分析智能体状态动态学习或控制策略学习的各自贡献；
   - 未验证不同状态空间大小 $|S|$ 对性能的影响趋势；
3. **方差问题**：REINFORCE 型估计器固有高方差，尤其在导航任务中表现明显，作者也承认这一局限；
4. **无限时域的近似处理**：通过截断轨迹近似无限回报，实际中折扣因子的选择和截断长度可能影响精度；
5. **全局最优性未证明**：收敛到的是驻点而非全局最优，未建立与 NMDP 最优历史依赖策略之间的差距上界；
6. **部分环境描述不详**：HealthcareTreatment 和 MachineRepair 的奖励函数细节在正文中较简略，主要依赖附录；
7. **Actor-Critic 扩展未实现**：论文指出可基于 Q-函数形式的梯度表达式设计 actor-critic 算法降低方差，但未实现，属于未来工作。
