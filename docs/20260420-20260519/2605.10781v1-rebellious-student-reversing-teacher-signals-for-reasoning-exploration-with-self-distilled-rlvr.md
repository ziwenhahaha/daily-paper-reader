## Abstract
Self-distillation has emerged as a powerful framework for post-training LLMs, where a teacher conditioned on extra information guides a student without it, both from the same model. While this guidance is useful when the student has failed, on successful rollouts, the same mechanism instead overwrites the student's choices and suppresses it's own reasoning. Therefore, we propose reading the original self-distillation signal in reverse: when the student succeeds along a path the teacher would not have predicted, these tokens reflect its self-driven reasoning. Building on this, we propose RLRT (RLVR with Reversed Teacher), which augments GRPO by reinforcing these tokens on correct rollouts. We interpret this as a new form of exploration in RLVR: not uniform diversity, but valuable exploration grounded in the student's own success. Across base, instruction-tuned, and thinking-tuned Qwen3 checkpoints, RLRT substantially outperforms self-distillation and exploration-based baselines, establishing information asymmetry as a new, principled design axis for RLVR.

# 论文总结：Rebellious Student: Reversing Teacher Signals for Reasoning Exploration with Self-Distilled RLVR

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：强化学习与可验证奖励（RLVR）已成为大语言模型（LLM）在推理任务上进行后训练的主流范式，但其面临**信用分配瓶颈**——每个轨迹末端只有一个稀疏的标量奖励作为学习信号。
- **自蒸馏的兴起**：为缓解这一问题，自蒸馏利用同一模型的两个视图（学生无特权上下文、教师有额外信息）实现信息不对称，从而将稀疏标量奖励转化为密集的令牌级监督。
- **核心问题**：在**成功的轨迹上**，传统自蒸馏机制反而起到负面作用——即使学生已经正确到达答案，蒸馏仍会用教师的选择覆盖学生的选择，将学生的自主推理压制为简单的模仿，这被称为"优化模糊性"（optimization ambiguity）。
- **核心洞察**：成功轨迹上学生与教师分歧最大的令牌，恰恰是学生**自驱动推理**的体现——学生"违背"了教师预测却仍能达到正确答案，这些令牌值得被强化而非抑制。
- **研究目标**：将自蒸馏信号"反转"使用，在正确轨迹上放大这些自驱动的令牌，将其转化为有价值的探索信号。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式与算法流程

### 2.1 核心思想

- 将师生之间的信息不对称重新解读：不是让学生模仿教师，而是让教师信号帮助定位学生自身的成功推理。
- 在正确轨迹上，识别学生选择与教师预测分歧最大的令牌（信息不对称符号为正），并**放大这些令牌**作为探索信号，而非对齐学生到教师。

### 2.2 关键技术与公式

#### 信息不对称的定义
- **令牌级信息不对称**：$\hat{D}_t(y_t) := \log P_t^S(y_t) - \log P_t^T(y_t)$
- **位置级信息不对称**：$\bar{D}_t = \mathbb{E}_{v \sim P_t^S}[\hat{D}_t(v)] = \text{KL}(P_t^S \parallel P_t^T)$
- 其中 $P_t^S(\cdot) = \pi_\theta(\cdot | h_t)$ 是学生（无特权上下文），$P_t^T(\cdot) = \pi_\theta(\cdot | h_t, c)$ 是教师（有正确解决方案 $c$ 作为条件）。

#### RLRT 的令牌重加权
$$
w^{\text{RLRT}}_t = \exp(\text{sign}(A) \cdot \hat{D}_t) = \left(\frac{P_t^S(y_t)}{P_t^T(y_t)}\right)^{\text{sign}(A)}
$$

- 与 RLSD 的关系：$w^{\text{RLRT}}_t = 1 / w^{\text{RLSD}}_t$，即 RLRT 反转了 RLSD 的权重方向。
- 在正向优势（正确轨迹）上，当 $\hat{D}_t > 0$（学生自驱动的令牌）时，$w^{\text{RLRT}}_t > 1$，放大该令牌。

#### 奖励门控更新
$$
A^{\text{RLRT},(k)}_t = 
\begin{cases}
A^{(k)} \cdot \left[(1 - \lambda) + \lambda \cdot \text{clip}(w^{\text{RLRT}}_t, 1 - \varepsilon_w, 1 + \varepsilon_w)\right] & \text{if } r(y^{(k)}) = 1 \\
A^{(k)} & \text{if } r(y^{(k)}) = 0
\end{cases}
$$

- **关键约束**：反转信号仅在正确轨迹（$r=1$）上生效，失败轨迹保持 vanilla GRPO。
- $\lambda \in [0,1]$ 控制反转信号强度（$\lambda=0$ 退化为 GRPO，$\lambda=1$ 完全反转）。
- $\varepsilon_w$ 控制每令牌优势的扰动上界。

### 2.3 算法流程（RLRT）

1. **采样**：学生策略 $\pi_\theta$ 对 prompt $x$ 采样 $K$ 条轨迹 $\{y^{(k)}\}_{k=1}^K$。
2. **计算奖励与优势**：每条轨迹获得可验证奖励 $r(y^{(k)}) \in \{0,1\}$ 和组标准化的优势 $A^{(k)}$。
3. **分组处理**：对每条轨迹 $k$：
   - 若 $r(y^{(k)}) = 1$（正确）：对每个位置 $t$，计算 $\hat{D}_t$，得到 $w^{\text{RLRT}}_t$，再通过奖励门控更新得到 $A^{\text{RLRT},(k)}_t$。
   - 若 $r(y^{(k)}) = 0$（错误）：$A^{\text{RLRT},(k)}_t = A^{(k)}$（标准 GRPO）。
4. **更新**：使用标准 GRPO surrogate 和 $\{A^{\text{RLRT},(k)}_t\}$ 更新参数 $\theta$。

### 3.4 对比方法
| 方法 | 描述 |
|------|------|
| **GRPO** | DeepSeekMath 的基线 RLVR 方法 |
| **SDPO** | 自蒸馏策略优化，错误轨迹蒸馏正确 rollout 到学生 |

| **RLSD** | 自蒸馏 RLVR，权重 $w^{\text{RLSD}}_t = (P_t^T/P_t^S)^{\text{sign}(A)}$，在正确轨迹上强化教师偏好的令牌 |

## 6. 论文的主要结论与发现

1. **RLRT 显著优于基线**：在所有四个 Qwen3 检查点上，RLRT 均大幅超过 GRPO、SDPO、SRPO、RLSD：
   - Qwen3-4B-Base：avg@16 提升 **+18.0%**
   - Qwen3-8B-Base：avg@16 提升 **+12.0%**
   - Qwen3-4B-Instruct：avg@16 提升 **+3.4%**
   - Qwen3-8B（Thinking）：avg@16 提升 **+2.2%**

2. **信息不对称能定位关键决策点**：
   - 高 $\bar{D}_t$ 位置是"关键分支点"（critical positions），在这些位置注入反射提示后结果翻转率显著更高
   - RLRT 能放大这一效应（最高 flip→R 率从 ~18% 提升至 >40%），而 GRPO 则让该效应消失

3. **RLRT 诱导更本质的分布偏移**：
   - GRPO 和 RLSD 主要重新加权基线已偏好的令牌（~80% top-k 重叠）
   - RLRT 重新组织候选集本身，将基线概率 <10⁻³ 的令牌推入 top-1 的频率是其他方法的 10 倍以上

4. **RLRT 的探索更有效**：
   - 在 AIME24/26 的 pass@k 曲线上，RLRT 从 pass@1 到 pass@256 均占优
   - GRPO+EB 仅在局部引入随机性，跟踪 GRPO 紧密
   - DIVER 改善有限（主要在大 k 时有边际增益）

5. **奖励门控是关键设计**：
   - 去掉门控（RLRT-all）在约 40 步后训练崩溃，响应长度和熵无限增长

6. **信息不对称是 RLVR 的新设计轴**：将师生信息不对称从"对齐目标"重新解读为"探索信号"，提出了一个新的、原则性的设计维度。

## 7. 优点：方法与实验设计的亮点

1. **问题定位精准**：准确识别了自蒸馏在成功轨迹上的反向作用，将"优化模糊性"问题转化为创新机会。

2. **理论基础扎实**：
   - 引入贝叶斯教师模型，推导出 $\hat{D}_t = \log \bar{f}_t^S - \log f(v)$ 的形式化
   - 定理 2 证明 $\bar{D}_t$ 控制位置影响力（$\text{Inf}^S(t)^2 \leq 2\bar{D}_t$），为设计选择提供了理论支撑

3. **方法简洁有效**：RLRT 仅修改 GRPO 的令牌级信用分配，保留 rollout、reward、trust-region 不变，实现成本低。

4. **实验设计全面**：
   - 多模型类型覆盖（Base/Instruct/Thinking）
   - 多维度验证：主结果、消融、因果干预、分布分析、pass@k 曲线
   - 统一的对比框架确保公平性

5. **新视角贡献**：将信息不对称从"对齐"重新解读为"探索"，为 RLVR 社区提供了新的设计原则。

6. **直觉可视化**：通过 explore/exploit 令牌分类（反射、推理、决策等标记）和热力图，将抽象信息不对称直观化。

## 8. 不足与局限

1. **场景单一**：所有实验均在数学推理场景进行，未验证方法在代码生成、自然语言推理、多模态等任务上的泛化性。

2. **自蒸馏框架依赖**：实验基于师生共享参数的 self-distillation 设置，未探索跨模型蒸馏（如用更强模型作为教师）的效果。

3. **特权信息的局限**：教师依赖已知正确解决方案 $c$，在真实场景中可能不可用（论文在 Future Work 中提及可扩展到过程级反馈、部分提示等）。

4. **奖励类型限制**：仅使用二元可验证奖励（正确/错误），未验证在噪声奖励或过程奖励下的表现。

5. **SDPO/SRPO 在 Base 模型上崩溃**：SDPO 在 Base 模型上快速崩溃，论文将其归因于过度抑制 hedged/reflection tokens，但这意味着对比不完整。

6. **超参数敏感性**：最佳 $\varepsilon_w$ 随模型类型变化（Base: 1.0, Instruct: 0.5, Thinking: 0.2），可能需要针对新模型调优。

7. **离线蒸馏未探索**：论文聚焦于 on-policy 蒸馏，off-policy 设置下的行为未被研究。
