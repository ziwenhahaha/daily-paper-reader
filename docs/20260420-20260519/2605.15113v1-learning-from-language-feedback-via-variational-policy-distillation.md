## Abstract
Reinforcement learning from verifiable rewards (RLVR) suffers from sparse outcome signals, creating severe exploration bottlenecks on complex reasoning tasks. Recent on-policy self-distillation methods attempt to address this by utilizing language feedback to generate dense, token-level supervision. However, these approaches rely on a fixed, passive teacher to interpret the feedback. As the student policy improves, the teacher's zero-shot assessment capabilities plateau, ultimately halting further learning. To overcome this, we propose Variational Policy Distillation (VPD), a framework that formalizes learning from language feedback as a Variational Expectation-Maximization (EM) problem. VPD co-evolves both policies: in the E-step, the teacher is actively refined on trajectory outcomes via an adaptive trust-region update, translating textual feedback into a dynamically improved target token distribution. In the M-step, the student internalizes this dense distributional guidance on its own on-policy rollouts. By continuously improving the teacher's ability to extract actionable signals from textual critique, VPD overcomes the limitations of passive distillation. Evaluated across diverse sources of diagnostic feedback on scientific reasoning and code generation tasks, VPD consistently outperforms both standard RLVR and existing self-distillation baselines. Finally, by stress-testing our framework on rigid mathematical reasoning and cold-start regimes, we illuminate the fundamental bounds of feedback-driven self-distillation compared to pure environment-driven RL.

# 论文总结：Learning from Language Feedback via Variational Policy Distillation

## 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：强化学习从可验证奖励（RLVR）中学习（如 GRPO、DAPO 等方法），在复杂推理任务（如数学证明、代码生成、科学推理）上面临**稀疏奖励瓶颈**——二元终局信号（正确/错误）无法提供细粒度的 token 级指导，导致探索效率极低。
- **现有方案的缺陷**：近期的**自蒸馏方法**（SDPO、OPSD 等）尝试通过语言反馈 C（如编译器错误、LLM 批判文本）构造密集监督信号，但其根本问题在于：**teacher 是被动、固定的角色**，仅依赖模型零样本理解反馈的能力。随着 student 提升，teacher 的诊断能力逐渐触及天花板，无法持续提供有意义的梯度，最终导致学习停滞。
- **研究动机**：如何让 teacher 能够**主动进化**，随 student 的提升而持续改进其从文本反馈中提取可操作信号的能力？
- **核心贡献**：提出 **Variational Policy Distillation（VPD）**，将语言反馈学习形式化为**变分期望最大化（Variational EM）**问题，实现 teacher 与 student 的**协同进化**。

## 2. 方法论：核心思想、关键技术细节、公式与算法流程

### 2.1 核心思想

- 将语言反馈学习建模为**变分推断问题**：引入一个以诊断反馈 C 为条件的 teacher 分布 $q_\phi(y|x, C)$ 作为最优策略 $\pi^*$ 的近似后验。
- 采用 **EM 算法**交替优化：
  - **E-step（Teacher Refinement）**：主动训练 teacher 区分成功与失败轨迹，将文本批判转化为高质量的目标 token 分布。
  - **M-step（Student Distillation）**：student 通过 token 级 KL 散度在自身 on-policy rollouts 上向改进后的 teacher 学习。
- **动态参考先验**：E-step 中将参考分布从固定的 $\pi_{\text{ref}}$ 替换为**当前 student 策略 $\pi_\theta$**，形成自适应 trust region，保证 teacher 的目标分布始终在 student 可达的范围内。

### 2.2 关键技术细节

**（1）最优目标分布的推导**

在 KL 正则化的 RLVR 目标下，最优策略具有**奖励倾斜分布**形式：
$$\pi^*(y|x) = \frac{1}{Z(x)}\pi_{\text{ref}}(y|x)\exp\left(\frac{1}{\beta}r(x,y)\right)$$

**（2）E-step 目标**

通过 BCO（Binary Classifier Optimization）进行**非配对偏好优化**，避免因每条轨迹对应独特反馈 C 而无法构造偏好对的问题。核心损失函数：
$$\mathcal{L}_{E\text{-step}}(\phi) = -\mathbb{E}_{y^+}[\log\sigma(\tilde{r}_\phi(x,y^+,C^+)-\delta)] - \mathbb{E}_{y^-}[\log\sigma(-(\tilde{r}_\phi(x,y^-,C^-)-\delta))]$$
其中动态 reward shift $\delta = \frac{1}{2}(\mathbb{E}[\tilde{r}_\phi(x,y^+,C^+)] + \mathbb{E}[\tilde{r}_\phi(x,y^-,C^-)])$。

**（3）动态参考先验的隐式奖励**

将参考分布设为 $\pi_\theta$ 后，隐式奖励为：
$$\tilde{r}_\phi(x,y,C) = \beta\log\frac{q_\phi(y|x,C)}{\pi_\theta(y|x)}$$

这使得 E-step 目标等价于在 trust region $D_{\text{KL}}(q_\phi \| \pi_\theta)$ 约束下最大化期望奖励，保证 teacher 的每次更新都在 student 的可达范围内。

**（4）M-step 目标**

在 student 的 on-policy rollouts 上执行 token 级 KL 蒸馏：
$$\mathcal{L}_{M\text{-step}}(\theta) = \mathbb{E}_{x\sim\mathcal{D},y\sim\pi_\theta(\cdot|x)}\left[\sum_{t=1}^{|y|} D_{\text{KL}}\left(\pi_\theta(\cdot|x,y_{<t}) \| \text{sg}[q_\phi(\cdot|x,C,y_{<t})]\right)\right]$$

**（5）统一权重架构**

Teacher 和 Student **共享同一套网络权重**（$\phi = \theta$），仅通过是否附加反馈 C 来区分两个策略角色，消除了多模型范式的巨大显存开销。

### 2.3 算法流程

1. **On-policy Rollout**：从当前 student $\pi_\theta$ 采样轨迹 $y$；
2. **环境批判**：通过环境或 LLM judge 生成诊断反馈 $C$ 和二元奖励 $r$；
3. **E-step**：将 teacher 权重同步为 student ($\phi \leftarrow \theta$)，计算 $\delta$，执行非配对偏好优化更新 teacher；
4. **M-step**：固定 teacher（stop-gradient），执行 token 级 KL 蒸馏更新 student；
5. 循环迭代。

## 6. 主要结论与发现

1. **VPD 在多种反馈源下均显著优于基线**：
   - LiveCodeBench：Qwen3-8B 达到 **49.62%**（GRPO 45.61%，SDPO 47.33%）；
   - SciKnowEval（Qwen3-8B）：VPD 达 **77.15%**（GRPO 73.11%，SDPO 74.44%）；
   - 自批判场景：VPD 同样显著优于 SDPO。

2. **单阶段混合方法效果有限**：联合损失、优势重塑等机制因尺度失配导致训练不稳定，性能提升微弱甚至下降，说明**解耦 EM 设计的必要性**。

3. **E-step 持续改进 teacher 的判别能力**：随着训练进行，VPD 的 reward margin（正确与错误响应的隐式奖励差）持续增大，而 SDPO 的 margin 快速衰减，验证了 teacher 进化机制的有效性。

4. **反馈驱动蒸馏存在根本性边界**：
   - **冷启动场景**：VPD（63.95%）仍远落后于 GRPO（74.49%），且无法避免最终崩溃；
   - **数学推理**：VPD 延迟了 SDPO 的训练崩溃，但 GRPO（83.8% on Math500）仍是最佳方法。
   - 原因：严格数学推导对中间步骤精确性要求极高，语言反馈中的噪声会误导 student 过度约束探索。

5. **动态参考先验至关重要**：固定先验导致训练极不稳定（表 5：平均性能下降约 6.5 分），验证了自适应 trust region 的设计价值。

6. **最优 E-step 更新频率**：F=5（一对五的比例）提供了 teacher 演化速度与 student 收敛稳定性之间的最佳平衡。

## 7. 优点

1. **理论优雅**：将自蒸馏问题严格形式化为变分 EM，提供了坚实的数学基础和单调收敛保证（Proposition 1）；
2. **核心创新——Teacher 主动进化**：首次提出通过 E-step 主动优化 teacher 的诊断能力，而非依赖固定的零样本理解；
3. **动态 trust region 设计**：通过将参考分布设为 $\pi_\theta$ 而非 $\pi_{\text{ref}}$，从根本上避免了 teacher 目标分布与 student 探索空间脱节的问题；
4. **极高的计算效率**：共享权重架构（$\phi=\theta$）消除了多模型显存开销，E-step 和 M-step 复用同一批 on-policy 数据，无额外采样成本；
5. **多反馈源通用性**：框架天然支持环境验证器、对比兄弟轨迹、LLM 自批判等多种反馈机制；
6. **消融实验系统全面**：验证了 E-step 频率、参考先验类型等关键超参数的影响，论证严密；
7. **揭示了方法边界**：诚实报告了 VPD 在数学推理和冷启动场景下的局限性，提升了论文的可信度。

## 8. 不足与局限

1. **数学推理与冷启动场景中仍落后于纯 RL**：VPD 无法完全克服语言反馈的噪声和不精确性，在严格推理任务上最终仍不如 GRPO；
2. **共享权重限制了 teacher 表征能力**：当 teacher 与 student 严格共享参数时，teacher 的诊断能力在架构上受限于 student 的容量；
3. **E-step 更新频率需人工调优**：最优 F 值需通过实验确定，增加了使用门槛；
4. **实验规模相对有限**：压力测试场景仅在 SciKnowEval 和 DAPO-Math 上进行，缺乏在更大规模模型或更多任务类型上的验证；
5. **非配对偏好优化（BCO）的近似误差**：使用 BCE 上界替代标准 DPO 损失时存在理论差距，$\delta$ 的移动平均估计可能引入噪声；
6. **对反馈 C 的质量仍有依赖**：当反馈本身过于模糊或错误时，E-step 优化的 teacher 仍可能产生误导性目标。
