## Abstract
Offline multi-agent reinforcement learning (MARL) enables policy learning from fixed datasets, but is prone to coordination failure: agents trained on static, off-policy data converge to suboptimal joint behaviours because they cannot co-adapt as their policies change. We introduce CODA (Coordination via On-Policy Diffusion for Multi-Agent Reinforcement Learning), a diffusion-based multi-agent trajectory generator for data augmentation that samples conditioned on the current joint policy, producing synthetic experience which reflects the evolving behaviours of the agents, thereby providing a mechanism for co-adaptation. We find that previous diffusion-based augmentation approaches are insufficient for fostering multi-agent coordination because they produce static augmented datasets that do not evolve as the current joint policy changes during training; CODA resolves this by more closely simulating on-policy learning and is a meaningful step toward coordinated behaviours in the offline setting. CODA is algorithm-agnostic and can be layered onto both model-free and model-based offline reinforcement learning pipelines as an augmentation module. Empirically, CODA not only resolves canonical coordination pathologies in continuous polynomial games but also delivers strong results on the more complex MaMuJoCo continuous-control benchmarks.

## 1. 论文核心问题与整体含义

**研究动机：**
- 离线多智能体强化学习（Offline MARL）旨在从固定数据集中学习协同策略，避免危险的在线交互采样。
- 核心挑战在于**协同失败（Coordination Failure）**：当多个智能体在静态离策略数据上训练时，无法像在线学习那样相互适应（co-adapt），导致各智能体策略收敛到互不兼容的次优联合行为。
- 现有扩散模型增强方法虽然可以生成合成轨迹，但生成的数据集是**静态的**，不随当前联合策略的演化而变化，无法真正解决协同问题。

**研究背景：**
- 论文将现有离线 MARL 方法归类为 **BRUD（Best Response Under Data）** 范式：以 MADDPG、MASAC 为代表，在离线更新时，智能体 i 更新自己的动作来自当前策略，而队友动作从数据集中采样。这导致每个智能体的梯度是对"数据集中的队友动作"而非"队友当前策略"的响应，缺失了协同进化的反馈环。

## 2. 论文方法论

**核心思想：**
- 提出 **CODA（Coordination via On-Policy Diffusion）**，核心思路是让生成的合成轨迹**条件依赖于当前联合策略**，使增强数据分布随策略演化而动态变化，从而近似恢复在线学习中存在的"内生联合策略演化"（endogenous joint policy evolution）机制。

**关键技术细节：**

- **扩散模型骨架**：使用 1D 时间卷积 U-Net 对完整轨迹进行去噪（轨迹级别生成，非自回归），训练于离线联合轨迹数据集，学习条件得分函数 $\nabla_{\hat\tau}\log p_{\pi_{\text{off}}}(\hat\tau;\sigma)$。
- **两类条件化方式**：
  - **策略引导的分类器引导（Classifier Guidance）**：通过重要性采样推导，梯度为 $\nabla_{\hat\tau}\log p(\hat\tau;\sigma) + \lambda\sum_{t=0}^{H-1} abla_{\hat a_t}\log\pi_{\text{curr}}(\hat a_t|\hat o_t)$，从当前策略的梯度方向引导去噪，使生成的轨迹偏向当前策略会产生的高概率动作。
  - **分类器无关引导（CFG）**：当策略可被编码为紧凑条件变量（如任务嵌入）时，直接用 CFG 进行条件化，同样得到策略倾斜分布 $\tilde p_{\pi_{\text{curr}}}(\tau)\propto p_{\pi_{\text{off}}}(\tau)\exp\left(\lambda\sum_t\log\pi_{\text{curr}}(a_t|o_t)\right)$。
- **关键性质**：策略引导项是**联合的**——所有智能体的动作梯度被同时叠加，确保生成的联合轨迹在整个多智能体层面保持一致性，同时各智能体仍维持去中心化策略。
- **实施流程**（Algorithm 1）：采样噪声轨迹 → 去噪预测 → 叠加策略梯度引导 → 执行 Euler 步和二阶校正 → 迭代 Ndiff 步 → 映射回原始约束空间（通过 logit-sigmoid 变换保证数值稳定）。

## 6. 主要结论与发现

1. **离线协同失败的根本原因是缺失内生联合策略演化**：BRUD 方法中每个智能体的梯度依赖数据集统计量（均值、方差）而非队友当前策略，导致梯度场是静态的或非自适应的（多项式游戏中退化为常向量场）。

2. **CODA 成功恢复协同优化动态**：在 Multiplication Game 中，CODA 收敛到最优解 $(1,1)$，而所有基线方法因梯度场恒定且受动作边界约束而收敛到次优解。在 Twin Peaks Game 中，CODA 同样收敛到最优联合策略，基线则塌缩到原点 $(0,0)$。

3. **策略引导确实能"舵动"生成轨迹向当前策略倾斜**：在 MaMuJoCo 上的验证表明，引导系数 $\lambda$ 在适中范围内可显著提升生成轨迹在当前策略下的动作对数似然，呈现预期的峰值-塌缩形态。

4. **CODA 性能收益与数据集支持度正相关**：在 Replay 和 Good 数据集上，CODA 一致优于无条件扩散和基线 MADDPG+BC；在 Medium 数据集上性能接近；在 Poor 数据集上增益减弱甚至不如无条件扩散——因为当策略质量较差时，策略引导可能将样本拖出数据集支持范围。

5. **方法具有算法无关性**：CODA 作为数据增强模块可叠加到任意离线 MARL 算法上。

## 7. 优点

- **问题定位精准**：深刻揭示了离线协同失败的结构性根源——缺失协同适应反馈环，而非仅归咎于分布漂移或外推误差。
- **解决方案简洁优雅**：不修改底层 RL 算法，仅通过策略条件的扩散采样实现数据增强，保留了端到端算法的兼容性。
- **理论基础扎实**：从重要性采样推导出策略引导项，结合 CFG 提供了统一的理论视角；附录中对引导系数的几何收缩性分析（$\lambda\in(0,2)$ 时更新是收缩的）进一步提升了方法的可解释性。
- **实验设计层层递进**：从可控的解析多项式游戏到复杂连续控制环境，从简单对比到引导系数消融，体现了较好的科研逻辑。
