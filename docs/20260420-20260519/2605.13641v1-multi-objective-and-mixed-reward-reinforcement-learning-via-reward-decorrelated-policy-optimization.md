## Abstract
Complex reinforcement learning environments frequently employ multi-task and mixed-reward formulations. In these settings, heterogeneous reward distributions and correlated reward dimensions often destabilize the construction of scalar advantages. To address these challenges, we propose Reward-Decorrelated Policy Optimization (RDPO), a reward-processing method designed to explicitly target both failure modes. RDPO first utilizes Magnitude-Aware Quantile normalization to stabilize prompt-level advantage allocation across binary, fractional, and continuous rewards. It then applies Mahalanobis whitening within each active reward subspace to mitigate correlation redundancy prior to aggregation. When applied during the post-training of LongCat-Flash, RDPO enhances instruction following, writing quality, and robustness to hard prompts while remaining broadly competitive on reasoning and coding evaluations.

# 论文详细总结

## 1. 论文的核心问题与整体含义

### 研究动机与背景

- **多任务强化学习的挑战**：在复杂强化学习环境中，单次训练运行通常包含多种任务类型，每种任务提供不同的奖励信号（如正确性、指令遵循、评分标准满足度、偏好模型分数、响应长度等）
- **核心问题**：将这些异构奖励信号聚合为单一标量优势时存在两大失效模式：
  1. **奖励分布异构性**：不同奖励在尺度、分布形状上差异显著
  2. **奖励维度相关性**：各奖励维度之间存在非平凡的相关性
- **影响**：这些问题导致优势构建不稳定，使得策略更新被少数样本主导，影响训练效果

## 2. 论文提出的方法论

### 核心思想

提出**Reward-Decorrelated Policy Optimization (RDPO)**，通过两阶段奖励处理流水线解决上述问题：
1. **Magnitude-Aware Quantile (MAQ) 归一化**：稳定提示级优势分配
2. **Mahalanobis 白化**：在活动奖励子空间内消除相关冗余

### 关键技术细节

#### 2.2.1 有效信息效率指标 (η_eff)

论文引入诊断指标评估混合奖励聚合质量：
- **η_eff = η_proj × η_corr**
- **η_proj**：衡量聚合方向与等权投影的对齐程度（cos²距离）
- **η_corr**：量化聚合后保留的独立信息量，基于绝对相关矩阵计算

#### 2.2.2 Magnitude-Aware Quantile (MAQ) 归一化

**问题**：Z-score 归一化对非高斯分布（binary、tied、skewed、outliers）敏感，可能导致单个 rollout 主导提示级优势分配

**解决方案步骤**：
1. **对数压缩间隙**：计算相邻 rollouts 间的对数压缩间距：
   ```
   gap_j = log(1 + |r_{j+1} - r_j| / (β × σ_global))
   ```
   其中 σ_global 是全局四分位距 (IQR)，β 控制压缩强度

2. **CDF 分配**：将间隙归一化后按比例分配累积分布函数 (CDF) 位置

3. **逆正态映射**：通过逆 CDF 将值映射到标准正态分布：
   ```
   A(j) = Φ^(-1)(u(j))
   ```

**效果**：降低提示级优势主导，提高有效 rollout 参与度

#### 2.2.3 Mahalanobis 白化

**问题**：MAQ 仅稳定单个奖励维度，无法处理维度间的相关性冗余

**解决方案**：
1. 对 MAQ 归一化后的优势向量 **A(i,j)** 应用白化变换：
   ```
   W(i,j) = Σ^(-1/2) × A(i,j)
   ```
   其中 Σ^(-1/2) 通过特征分解获得

2. **运行协方差估计**：使用指数移动平均 (EMA) 在线估计协方差：
   ```
   Σ_t = (1 - α) × Σ_{t-1} + α × Σ_batch
   ```

3. **子空间白化**：仅对观察到的子空间应用白化，避免引入虚假协方差

#### 2.2.4 最终优势计算

```
A_sum = 1^T × W = 1^T × Σ^(-1/2) × A
```
