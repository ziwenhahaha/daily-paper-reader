## Abstract
Self-distillation (SD) offers a promising path for adapting large language models (LLMs) without relying on stronger external teachers. However, SD in autoregressive LLMs remains challenging because self-generated trajectories are free-form, correctness is task-dependent, and plausible rationales can still provide unstable or unreliable supervision. Existing methods mainly examine isolated design choices, leaving their effectiveness, roles, and interactions unclear. In this paper, we propose UniSD, a unified framework to systematically study self-distillation. UniSD integrates complementary mechanisms that address supervision reliability, representation alignment, and training stability, including multi-teacher agreement, EMA teacher stabilization, token-level contrastive learning, feature matching, and divergence clipping. Across six benchmarks and six models from three model families, UniSD reveals when self-distillation improves over static imitation, which components drive the gains, and how these components interact across tasks. Guided by these insights, we construct UniSDfull, an integrated pipeline that combines complementary components and achieves the strongest overall performance, improving over the base model by +5.4 points and the strongest baseline by +2.8 points. Extensive evaluation highlights self-distillation as a practical and steerable approach for efficient LLM adaptation without stronger external teachers.

# UniSD 论文详细总结

## 1. 论文的核心问题与整体含义（研究动机和背景）

### 研究背景
- 大语言模型（LLMs）需要后训练适应（post-training adaptation）来适配新领域、新任务和部署约束
- 传统方法依赖更强的外部模型作为监督来源（如合成数据生成、强化学习、外部教师蒸馏），存在以下问题：
  - 持续依赖更强模型会主导训练成本
  - 受限于访问、政策或许可
  - 可能传播不良属性（如偏见、隐私敏感内容）

### 核心问题
**能否让 LLMs 通过自我生成的监督信号进行改进，而不依赖更强的外部教师？**

### 自蒸馏（Self-Distillation）在自回归 LLMs 中的三大挑战

| 挑战 | 具体描述 |
|------|----------|
| **开放性生成** | LLM 生成是自由形式的轨迹，而非固定预测目标；同一提示可能有多个正确答案、推理路径或代码实现 |
| **不可靠且不稳定的自监督** | 自推导的监督信号天然具有噪声和不稳定性；在策略轨迹暴露模型自身错误，真实演示可能包含错误标签、弱解释或未明确说明的推理 |
| **缺乏系统性理解** | 现有方法通常孤立地研究自蒸馏策略，哪些因素驱动自我改进、它们如何相互作用尚不清楚 |

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
UniSD 将自蒸馏建模为**可靠性感知的自我校正过程**：学生首先尝试完成一个生成，然后通过多个教师视图的比较和监督进行学习，权重可靠信号并将其知识整合到自身行为中。

### 三大轴心
UniSD 的方法论围绕三个互补轴心组织：

#### 2.1 监督可靠性（Supervision Reliability）
**目标：识别哪些自我推导的信号应该指导学习**

1. **多教师一致性（Multi-Teacher Agreement）**
   - 使用多个辅助教师从不同任务保持视角交叉验证同一学生行为
   - 在两个粒度上估计一致性：
     - **Token 级别**：计算局部不可靠 token
     - **序列级别**：捕获完成的全局不稳定性
   - 辅助教师视图通过上下文变化实例化（检索/随机少样本示例或诱导指令）

2. **Token 级对比学习（Token-Level Contrastive Learning）**
   - 将可靠的正向监督与似是而非的错误替代方案对比
   - 使用边际损失鼓励学生轨迹更接近正确监督而非错误替代

#### 2.2 表征对齐（Representation Alignment）
**目标：在输出分布之外延伸自蒸馏**

- **特征匹配（Feature Matching）**：将学生特征正则化到教师表征，促进学习解决方案的结构一致性
- 匹配最终层隐藏状态，提供表征级约束

#### 2.3 训练稳定性（Training Stability）
**目标：控制学生更新的幅度和平滑度**

1. **EMA 教师（EMA Teacher）**
   - 使用指数移动平均教师提供时间平滑的自我推导目标
   - 更新公式：$\bar{\theta}_n = \beta \bar{\theta}_{n-1} + (1 - \beta)\theta_n$

2. **散度裁剪（Divergence Clipping）**
   - 裁剪每个 token 级散度，防止稀有高散度 token 不成比例地影响优化
   - 对散度设置上限 $\kappa$：$\tilde{D}_t = \min(D_t^{(α)}, \kappa)$

### UniSD 整体目标函数
$$
\mathcal{L} = \mathbb{E}_{x, \hat{y} \sim \pi_\theta(\cdot|x)} \left[ \frac{\sum_t m_t w_t \tilde{D}_t}{\sum_t m_t w_t} + \lambda_{aux} \mathcal{L}_{aux}(\theta; x, \hat{y}, c) \right]
$$

### UniSD\* 集成管道
整合所有组件的完整管道：
- 监督视角：多教师一致性 + Token 级对比学习
- 表征视角：特征匹配
- 优化视角：EMA 教师 + 散度裁剪

## 6. 论文的主要结论与发现

### 6.1 核心发现

#### 发现 1：静态模仿不如 on-policy 学习可靠
- SFT 在格式导向任务（ToolAlpaca +4.4）有效，但在多样推理路径任务上表现不佳
- On-policy 基线（SDFT, GKD）提供更强的起点，但仍有敏感性

#### 发现 2：多教师一致性提高监督可靠性
| 粒度 | 特点 | 性能 |
|------|------|------|
| Token 级别 | 利用局部可靠信号 | ScienceQA 最高 85.2，整体 72.2 |
| 序列级别 | 更保守但更稳定 | 整体 72.5，更稳健 |

#### 发现 3：不同组件在不同任务上具有互补优势
- **EMA**：对工具使用最有效（+16.1），代码任务也受益
- **一致性**：在 ScienceQA 和 HumanEval 领先
- **UniSD\***：在 MBPP 和 GPQA 上获得最大提升

#### 发现 4：组合策略效果最佳
- UniSD\* 整体得分 **73.3**，比原始模型 **+5.4**，比最强基线 GKD **+2.8**
- 在 18 个模型-数据集组合中，15 个提升，2 个持平，1 个下降

### 6.2 分布保留关键发现
- **SFT 导致显著分布漂移**：Qwen2.5-7B 保留困惑度从 1.14 增至 1.68，Gemma-3 从 1.27 增至 3.02
- **UniSD 方法保持接近原始分布**：最佳值在 1.09-1.13 之间
- **UniSD\* 改进准确率的同时减少分布漂移**：Token 级 JSD 从 SFT 的 0.054 降至 0.041

### 6.3 泛化能力
- 跨模型家族泛化：Qwen2.5 (+5.4), Llama-3.1 (+3.1), Gemma-3 (+2.2)
- 跨任务格式泛化：15/18 模型-数据集对改善

UniSD 提供了一个系统、严谨的自蒸馏框架，通过模块化设计解耦了自蒸馏的关键因素，并通过大规模实验验证了各组件的有效性和互补性。其核心贡献在于揭示了"可靠性感知"对于自蒸馏成功的重要性，并提供了可操作的解决方案。UniSD\* 作为集成方案，在多个基准上取得了最优或接近最优的性能，同时保持了良好的分布保留特性，为资源受限场景下的 LLM 高效适应提供了有价值的思路。
