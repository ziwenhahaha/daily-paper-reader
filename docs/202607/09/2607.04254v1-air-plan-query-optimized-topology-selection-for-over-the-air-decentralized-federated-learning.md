## 摘要
空中（OTA）聚合利用无线多址信道的叠加特性，在单个传输时隙内聚合来自多个设备的模型更新，显著降低通信延迟。虽然OTA计算已在集中式联邦学习（FL）中得到广泛研究，但其与去中心化联邦学习（DFL）的集成仍 largely 未被探索，且现有工作中缺乏原则性的通信拓扑选择。我们提出AIRPLAN，一种面向空中去中心化联邦学习（OTA-DFL）的查询优化拓扑选择框架。AIRPLAN建立了OTA-DFL与分布式查询处理之间的形式等价性，使得拓扑选择可被表述为基于成本的查询优化问题。利用保护隐私的Count-Min Sketch统计量，AIRPLAN估计工作负载特征，评估候选拓扑上的图感知成本模型，并选择在满足目标精度SLA的同时最小化训练成本的通信图。在五个图族、三个视觉基准、四个客户端规模和多个SNR设置下的实验表明，AIRPLAN在91.4%的工作负载中与最优拓扑匹配，而引入的开销低于1.8%。我们进一步推导了拓扑感知稀疏化的理论误差界，证明连接良好的拓扑能更好地容忍激进压缩。AIRPLAN引入了一种连接无线联邦学习和分布式查询优化的系统导向视角。

## Abstract
Over-the-air (OTA) aggregation exploits the superposition property of wireless multiple-access channels to aggregate model updates from multiple devices within a single transmission slot, significantly reducing communication latency. While OTA computation has been extensively studied for centralized federated learning (FL), its integration with decentralized federated learning (DFL) remains largely unexplored, and principled communication topology selection is absent from existing work.   We present AIRPLAN, a query-optimized topology selection framework for Over-the-Air Decentralized Federated Learning (OTA-DFL). AIRPLAN establishes a formal equivalence between OTA-DFL and distributed query processing, enabling topology selection to be formulated as a cost-based query optimization problem. Using privacy-preserving Count-Min Sketch statistics, AIRPLAN estimates workload characteristics, evaluates a graph-aware cost model across candidate topologies, and selects the communication graph that minimizes training cost while satisfying a target accuracy SLA.   Experiments across five graph families, three vision benchmarks, four client scales, and multiple SNR settings show that AIRPLAN matches the oracle-optimal topology in 91.4% of workloads while introducing less than 1.8% overhead. We further derive theoretical error bounds for topology-aware sparsification, demonstrating that well-connected topologies better tolerate aggressive compression. AIRPLAN introduces a systems-oriented perspective that bridges wireless federated learning and distributed query optimization.

- **问题背景**：空中（Over-the-Air, OTA）聚合利用无线信道的叠加特性，能在单个时隙内聚合多个设备的梯度更新，从而大幅降低通信延迟。现有OTA研究主要集中在**集中式联邦学习（FL）**，即依赖中心服务器的FedAvg架构。然而，集中式架构存在单点故障、扩展性差、需要信任聚合器等问题。
- **去中心化联邦学习（DFL）** 移除中心服务器，客户端通过点对点通信图交换模型更新，但**通信拓扑的选择**成为一个关键且未被充分研究的设计问题：不同拓扑（如环形图、全连接图）在收敛速度、通信成本、抗噪声能力上差异巨大。
- **核心问题**：在OTA-DFL系统中，如何**自动且原则性地选择最优通信拓扑**，以最小化总训练成本，同时满足用户指定的精度服务等级协议（SLA）？
- **研究意义**：本文首次将OTA-DFL的拓扑选择问题与分布式查询处理中的物理计划选择建立形式等价，从而将数据库领域成熟的查询优化技术（成本模型、统计量收集、自适应重优化）直接引入无线联邦学习，开辟了无线通信与数据库系统交叉的新研究方向。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- **形式等价**：将OTA-DFL的每个聚合轮次视为一个在通信图上执行的**近似分布式SUM查询**。通信图是物理执行计划，图的拉普拉斯矩阵谱隙（spectral gap）对应于查询优化中的**基数估计**（控制执行代价），top-k稀疏化对应于**近似查询处理（AQP）**算子。
- **查询优化框架**：拓扑选择转化为**基于成本的物理计划选择**：给定训练工作负载（客户端数N、数据异质性α、信道SNR、模型维度d），估算每个候选拓扑的总执行代价C_ours(G)，选择满足精度SLA的最小代价拓扑。训练过程中根据模型发散度触发自适应重优化（拓扑切换）。

### 关键技术细节
1. **隐私保护统计量收集**：
   - 使用**Count-Min Sketch（CMS）** 对每个客户端的本地标签分布进行压缩表示（仅需约13KB/客户端），而非直接共享原始标签计数。
   - 通过合并所有客户端的CMS估计全局数据异质性参数ˆα（基于成对KL散度），保护了数据隐私。

2. **统一查询代价模型**：
   \[
   C_{\text{ours}}(G) = T(G) \cdot R(\text{SNR}) \cdot C_{\text{struct}}(G, k)
   \]
   - \(T(G) = \kappa / \lambda_2(L)\)：基于谱隙估计的收敛轮次。
   - \(R(\text{SNR}) = 1 + \alpha_R e^{-\beta_R \cdot \text{SNR}}\)：无线可靠性因子。
   - \(C_{\text{struct}}(G, k) = 2|E| \cdot k\)：每轮传输符号数。
   - 成本模型在实验中预测MAPE仅2.9%~4.8%，且误差导致的拓扑错误选择率仅2.1%。

3. **拓扑-稀疏化联合设计**：
   - 从AQP误差界推导：\(\varepsilon_{\text{eff}}(G,k) \leq \sqrt{1-k/d} / (0.5 + \lambda_2(L))\)。
   - 高连通图（大谱隙）可以容忍更高的稀疏化比率（更小的k/d），实现联合优化。

4. **自适应重优化**：
   - 训练过程中监控平均成对模型距离\(\Delta_t\)，若超出阈值\(\tau_\Delta\)，则升级到更稠密的拓扑。

5. **端到端差分隐私分析**：
   - 在CMS+OTA噪声下，通过定理V.1给出明确的(ϵ,δ)保证，典型参数下总ϵ≤0.13（强实际隐私）。

### 算法流程（Algorithm 1）
- 阶段1：每个客户端计算CMS草图并广播给协调器。
- 阶段2：合并草图，估计工作负载参数（N, ˆα, SNR, d）。
- 阶段3：枚举候选拓扑（Ring, ER, Small-World, Clustered, FC），用代价模型计算C_ours(G)，并预测精度是否满足SLA。
- 阶段4：选择最小代价且满足SLA的拓扑G*，开始训练。
- 阶段5：每轮计算Δ_t，若超过阈值则切换到下一更稠密的拓扑。

## 6. 主要结论与发现

1. **拓扑排序与谱隙一致**：高谱隙的拓扑（全连接、聚类图、小世界）收敛更快，且抗噪声和抗稀疏化能力强。
2. **AIRPLAN表现**：
   - 在91.4%的工作负载中匹配最优拓扑，剩余情况精度差距≤0.3个百分点。
   - CMS统计收集开销低于训练总成本的1.8%。

4. **高连通拓扑优势**：小世界和聚类图占据精度-成本Pareto前沿，在精度接近全连接90%的情况下，成本仅为57%。
5. **鲁棒性**：聚类图在客户端掉线率高达30%时仍保持良好性能，AIRPLAN能自动切换到聚类图以增强鲁棒性。

## 8. 不足与局限

1. **理想化信道假设**：OTA模型假设理想信道反演和功率控制，未考虑实际CSI误差和功率约束；硬件实现差异可能影响结论。

3. **同步通信要求**：OTA需要严格的时间同步（≤1μs），在异构边缘设备上存在掉队者问题，异步OTA变体未被纳入。
4. **仿真验证为主**：所有实验基于模拟AWGN信道，缺乏真实无线测试台（如SDR）验证。
