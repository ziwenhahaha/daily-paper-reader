## 摘要
现有的视觉语言导航方法在异构机器人兼容性、实时性能和导航安全性方面面临重大瓶颈。此外，这些方法难以支持开放词汇语义泛化和多模态任务输入。为应对这些挑战，本文提出了 FSUNav：一种用于快速、安全且通用的零样本目标导向导航的大脑-小脑架构，该架构创新性地将视觉语言模型（VLM）与所提架构相结合。小脑模块是一个高频端到端模块，开发了基于深度强化学习的通用局部规划器，实现了跨异构平台（如人形、四足、轮式机器人）的统一导航，在提高导航效率的同时显著降低了碰撞风险。大脑模块构建了一个三层推理模型，并利用 VLM 建立了端到端的检测与验证机制，实现了无需预定义 ID 的零样本开放词汇目标导航，并提升了仿真和真实环境中的任务成功率。此外，该框架支持多模态输入（如文本、目标描述和图像），进一步增强了泛化性、实时性、安全性和鲁棒性。在 MP3D、HM3D 和 OVON 基准测试上的实验结果表明，FSUNav 在物体导航、实例图像导航和任务导航方面达到了最先进的性能，显著优于现有方法。在多种机器人平台上的真实世界部署进一步验证了其鲁棒性和实际应用价值。

## Abstract
Current vision-language navigation methods face substantial bottlenecks regarding heterogeneous robot compatibility, real-time performance, and navigation safety. Furthermore, they struggle to support open-vocabulary semantic generalization and multimodal task inputs. To address these challenges, this paper proposes FSUNav: a Cerebrum-Cerebellum architecture for fast, safe, and universal zero-shot goal-oriented navigation, which innovatively integrates vision-language models (VLMs) with the proposed architecture. The cerebellum module, a high-frequency end-to-end module, develops a universal local planner based on deep reinforcement learning, enabling unified navigation across heterogeneous platforms (e.g., humanoid, quadruped, wheeled robots) to improve navigation efficiency while significantly reducing collision risk. The cerebrum module constructs a three-layer reasoning model and leverages VLMs to build an end-to-end detection and verification mechanism, enabling zero-shot open-vocabulary goal navigation without predefined IDs and improving task success rates in both simulation and real-world environments. Additionally, the framework supports multimodal inputs (e.g., text, target descriptions, and images), further enhancing generalization, real-time performance, safety, and robustness. Experimental results on MP3D, HM3D, and OVON benchmarks demonstrate that FSUNav achieves state-of-the-art performance on object, instance image, and task navigation, significantly outperforming existing methods. Real-world deployments on diverse robotic platforms further validate its robustness and practical applicability.

视觉语言导航（VLN）旨在让机器人根据自然语言指令在复杂环境中自主导航。然而，现有方法存在四大瓶颈：
*   **异构平台兼容性差**：大多数方法针对特定机器人（如轮式）设计，难以直接迁移至人形或四足机器人。
*   **实时性不足**：依赖大型模型导致推理延迟高，难以应对动态环境。
*   **安全性缺失**：现有研究多关注路径成功率，忽视了避障和碰撞风险。
*   **语义泛化受限**：依赖预定义类别，难以理解开放词汇指令或多模态输入（如图像、描述）。

**FSUNav** 提出了一种仿生学的大脑-小脑架构，旨在通过解耦高层推理与底层控制，实现跨平台、实时、安全且支持开放词汇的零样本导航。

#### 2. 方法论：核心思想与技术细节
该架构分为两个核心模块：
*   **小脑模块（底层执行与控制）**：
    *   **核心思想**：基于深度强化学习（DRL）的通用局部规划器，与具体机器人构型解耦。
    *   **技术细节**：采用“维度可配置”的输入表示，将机器人建模为具有特定长宽的矩形。利用 **PointNet** 编码器处理传感器点云，将机器人自身的物理尺寸和动力学限制（速度、加速度）嵌入特征中。
    *   **训练策略**：采用课程学习（Curriculum Learning），在仿真中让机器人从小空间到复杂空间逐步学习避障，实现零样本跨平台部署。
*   **大脑模块（高层理解与决策）**：
    *   **核心思想**：利用视觉语言模型（VLM）进行语义推理，分为三层：
        1.  **语义层**：将多模态目标（文本、图像）解析为结构化剖面，并进行开放词汇的目标定位。
        2.  **空间层**：构建鸟瞰图（BEV），结合几何边界探索与 VLM 驱动的语义路点生成（预测目标可能出现的方向）。
        3.  **规则层**：协调行为逻辑，通过两阶段验证（全局场景验证+局部实例识别）降低误报，并构建语义场景图。

#### 3. 实验设计
*   **数据集/场景**：
    *   **仿真**：Matterport3D (MP3D) 和 Habitat-Matterport3D (HM3D)。
    *   **任务类型**：物体导航（ON）、实例图像导航（IIN）、文本目标导航（TN）和开放词汇导航（OV）。

*   **真实世界测试**：在 Unitree Go2 四足机器人上部署，测试其在非结构化环境中的避障与目标识别能力。

*   **模型**：使用 **Qwen3-VL-32B-Instruct** 作为核心 VLM，通过 Ollama 部署。

*   **感知**：实机使用 Intel RealSense D455 相机（仅提取 RGB，通过模型估计深度）和 Livox MID-360 激光雷达（用于 BEV 建图和避障）。

#### 5. 实验数量与充分性
*   **实验规模**：在 MP3D 和 HM3D 两个大规模数据集上进行了详尽测试，涵盖了成功率（SR）和路径长度加权成功率（SPL）等核心指标。
*   **充分性**：实验不仅对比了同类零样本方法，还挑战了部分监督学习方法。涵盖了从物体到图像、再到复杂文本描述的多种任务，验证了框架的通用性。
*   **客观性**：通过在不同物理特性的机器人（四足、人形、轮式）上进行跨平台验证（部分结果预告在后续版本），增强了结论的说服力。

#### 6. 主要结论与发现
*   **性能领先**：FSUNav 在所有测试任务中均达到 SOTA。在 HM3D 的文本目标导航（TN）中，成功率（39.8%）几乎是之前最佳方法 UniGoal（20.2%）的两倍。
*   **跨平台通用性**：小脑模块成功实现了无需微调的跨机器人部署，能够自适应不同的物理尺寸和运动限制。
*   **零样本能力**：大脑模块通过 VLM 的强大推理能力，无需针对特定环境训练即可处理从未见过的物体类别。

#### 7. 优点
*   **架构解耦**：大脑负责“想”，小脑负责“走”，这种解耦使得系统既能利用大模型的智能，又能保持底层控制的实时性和安全性。
*   **多模态统一**：一个框架同时支持物体名、图像和自然语言描述作为导航目标。
*   **维度感知**：小脑模块能感知机器人自身尺寸，这在狭窄空间避障中具有显著优势。

#### 8. 不足与局限
*   **计算延迟**：虽然引入了冷却机制减少 VLM 调用，但大模型的推理延迟仍可能影响极高速运动下的决策频率。
