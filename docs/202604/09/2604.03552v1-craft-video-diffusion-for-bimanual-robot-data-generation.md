## 摘要
双臂机器人从演示中学习（Learning from demonstrations）受到现实世界数据成本高昂和视觉多样性有限的根本限制，这制约了策略在不同视角、物体配置和机器人本体（embodiments）之间的鲁棒性。我们提出了 CRAFT（Canny-guided Robot Data Generation using Video Diffusion Transformers），这是一个基于视频扩散的可扩展双臂演示生成框架，它在生成动作标签的同时合成时间上连贯的操作视频。通过将视频扩散模型以从模拟器生成的轨迹中提取的基于边缘的结构线索为条件，CRAFT 能够产生物理上合理的轨迹变化，并支持一个统一的增强流水线，涵盖物体姿态变化、摄像机视角、光照和背景变化、跨本体迁移以及多视图合成。我们利用预训练的视频扩散模型，将模拟视频连同模拟轨迹中的动作标签转化为动作一致的演示。仅从少量现实世界的演示开始，CRAFT 就能生成大量视觉多样且具有照片级真实感的训练数据，从而无需在真实机器人上重新播放演示（Sim2Real）。在模拟和现实世界的双臂任务中，CRAFT 相比现有的增强策略和简单的数据缩放提高了成功率，证明了基于扩散的视频生成可以显著扩展演示的多样性，并提高双臂操作任务的泛化能力。项目网站见：https://craftaug.github.io/

## Abstract
Bimanual robot learning from demonstrations is fundamentally limited by the cost and narrow visual diversity of real-world data, which constrains policy robustness across viewpoints, object configurations, and embodiments. We present Canny-guided Robot Data Generation using Video Diffusion Transformers (CRAFT), a video diffusion-based framework for scalable bimanual demonstration generation that synthesizes temporally coherent manipulation videos while producing action labels. By conditioning video diffusion on edge-based structural cues extracted from simulator-generated trajectories, CRAFT produces physically plausible trajectory variations and supports a unified augmentation pipeline spanning object pose changes, camera viewpoints, lighting and background variations, cross-embodiment transfer, and multi-view synthesis. We leverage a pre-trained video diffusion model to convert simulated videos, along with action labels from the simulation trajectories, into action-consistent demonstrations. Starting from only a few real-world demonstrations, CRAFT generates a large, visually diverse set of photorealistic training data, bypassing the need to replay demonstrations on the real robot (Sim2Real). Across simulated and real-world bimanual tasks, CRAFT improves success rates over existing augmentation strategies and straightforward data scaling, demonstrating that diffusion-based video generation can substantially expand demonstration diversity and improve generalization for dual-arm manipulation tasks. Our project website is available at: https://craftaug.github.io/

*   **核心问题**：双臂机器人操作（Bimanual Manipulation）的模仿学习高度依赖大规模、多样化的演示数据。然而，现实世界中的双臂遥操作数据获取成本极高，且在视角、物体配置、光照及机器人本体（Embodiment）方面的视觉多样性非常有限，导致训练出的策略泛化能力差。
*   **研究背景**：现有的数据增强方法通常只针对单一维度（如仅增强背景或仅做跨本体迁移），缺乏一个统一的框架来同时处理多种视觉变化并生成一致的动作标签。

### 2. 方法论：CRAFT 框架
CRAFT 的核心思想是利用**视频扩散模型（Video Diffusion Transformers）**，结合模拟器生成的结构线索，将低保真的模拟轨迹转化为高保真、带动作标签的真实感视频。
*   **核心流程**：
    1.  **轨迹扩展（Trajectory Expansion）**：通过数字孪生（Digital Twin）将少量真实演示迁移至模拟器（Real2Sim），利用子任务分解和随机化采样生成大量成功的模拟轨迹。
    2.  **Canny 边缘引导**：从模拟视频中提取 Canny 边缘图作为结构控制信号。相比深度图或骨架图，Canny 边缘能更好地平衡“保留机器人/物体结构”与“允许视觉外观变化”之间的关系。
    3.  **视频生成**：使用预训练的视频扩散模型（如 Wan 2.1），以 Canny 边缘视频、真实参考图像和语言指令为条件，合成照片级真实的视频。
    4.  **统一增强流水线**：支持七大维度的增强：物体姿态、光照（通过图像生成模型 Veo3 预处理参考图）、物体颜色、背景、跨本体迁移（通过 FK/IK 映射）、相机视角、以及腕部+第三人称视角的同步生成（通过图像平铺技术）。

    *   **真实世界**：使用双臂 Franka Panda 机器人，任务包括 Lift Roller（抬滚筒）、Place Cans 和 Stack Bowls。

### 6. 主要结论与发现
*   **性能提升**：CRAFT 在所有七个增强维度上均显著优于现有的特定基准方法。
*   **数据缩放效应**：使用 CRAFT 生成的 1000 条合成数据训练的策略，其表现甚至超过了直接在目标机器人上收集的 100 条真实数据，证明了合成数据可以作为昂贵真实数据的有效替代。
*   **结构引导的重要性**：消融实验证明，Canny 边缘引导比直接使用原始模拟图像作为输入效果更好，因为它能有效过滤模拟器的低级细节噪声，同时保留关键的接触几何关系。

### 7. 优点（亮点）
*   **统一性**：将多种零散的增强技术（视角、背景、本体等）整合进一个基于视频扩散的统一框架中。
*   **动作一致性**：生成的视频与模拟器中的动作标签严格对齐，解决了生成数据“有图无动作”的难题。
*   **跨本体能力**：通过 FK/IK 结合视频编辑，实现了从一种机器人（如 xArm7 或 UR5）到另一种机器人（如 Franka）的高质量演示迁移。

### 8. 不足与局限
*   **视觉伪影**：与所有生成模型一样，合成视频可能存在时间不一致或视觉伪影，可能干扰策略学习。
*   **模拟器依赖**：轨迹扩展阶段仍需访问模拟器和物体网格（Mesh），对于难以建模的复杂物体或场景存在局限。
*   **提示词工程**：高质量的生成结果依赖于精细的 Prompt 设计和参考图像的选择。
*   **相机距离限制**：若相机距离机器人过远，Canny 边缘会变得模糊，导致生成的抓取接触点精度下降。
