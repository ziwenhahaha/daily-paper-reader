## 摘要
世界动作模型（World Action Model, WAM）能够联合预测未来的世界状态和动作，与传统模型相比展现出更强的物理操作能力。这种强大的物理交互能力是一把双刃剑：如果忽视安全性，它将直接威胁人身安全、财产安全和环境安全。然而，现有研究对一个关键的安全漏洞关注极少：即 WAM 对越狱攻击的脆弱性。为了填补这一空白，我们定义了“三级安全分类框架”，旨在系统地量化机械臂运动的安全性。此外，我们提出了 JailWAM，这是首个专门针对 WAM 的越狱攻击与评估框架，由三个核心组件组成：(1) 视觉-轨迹映射（Visual-Trajectory Mapping），将异构动作空间统一为视觉轨迹表示，从而实现跨架构的统一评估；(2) 风险判别器（Risk Discriminator），作为一种高召回率的筛选工具，在识别视觉轨迹中的破坏性行为时优化了效率与准确率之间的权衡；(3) 双路径验证策略（Dual-Path Verification Strategy），首先通过基于单图像的视频-动作生成模块进行快速粗筛，随后通过全闭环物理仿真进行高效且全面的验证。此外，我们构建了 JailWAM-Bench，这是一个用于全面评估 WAM 在越狱攻击下安全对齐性能的基准测试。在 RoboTwin 仿真环境中的实验表明，所提框架能有效揭示物理漏洞，在最先进的 LingBot-VA 上实现了 84.2% 的攻击成功率。同时，基于 JailWAM 可以构建鲁棒的防御机制，为设计安全可靠的机器人控制系统提供有效的技术方案。

## Abstract
The World Action Model (WAM) can jointly predict future world states and actions, exhibiting stronger physical manipulation capabilities compared with traditional models. Such powerful physical interaction ability is a double-edged sword: if safety is ignored, it will directly threaten personal safety, property security and environmental safety. However, existing research pays extremely limited attention to the critical security gap: the vulnerability of WAM to jailbreak attacks. To fill this gap, we define the Three-Level Safety Classification Framework to systematically quantify the safety of robotic arm motions. Furthermore, we propose JailWAM, the first dedicated jailbreak attack and evaluation framework for WAM, which consists of three core components: (1) Visual-Trajectory Mapping, which unifies heterogeneous action spaces into visual trajectory representations and enables cross-architectural unified evaluation; (2) Risk Discriminator, which serves as a high-recall screening tool that optimizes the efficiency-accuracy trade-off when identifying destructive behaviors in visual trajectories; (3) Dual-Path Verification Strategy, which first conducts rapid coarse screening via a single-image-based video-action generation module, and then performs efficient and comprehensive verification through full closed-loop physical simulation. In addition, we construct JailWAM-Bench, a benchmark for comprehensively evaluating the safety alignment performance of WAM under jailbreak attacks. Experiments in RoboTwin simulation environment demonstrate that the proposed framework efficiently exposes physical vulnerabilities, achieving an 84.2% attack success rate on the state-of-the-art LingBot-VA. Meanwhile, robust defense mechanisms can be constructed based on JailWAM, providing an effective technical solution for designing safe and reliable robot control systems.

#### 1. 研究动机与背景
随着世界动作模型（World Action Models, WAMs）在机器人操作中的兴起，模型展现出了强大的物理交互能力。然而，这种能力是一把“双刃剑”：WAM 继承了生成式底座模型的脆弱性，容易受到越狱攻击（Jailbreak Attacks）。与传统大语言模型（LLM）或视频生成模型（VGM）仅产生数字领域危害（如恶意文本或视频）不同，WAM 的越狱会直接驱动机械臂执行危险的物理动作，导致人身伤害、设备损坏或环境破坏。目前，针对 WAM 的物理安全评估存在三大挑战：缺乏系统性的安全分级、动作空间异构导致难以统一评估、以及全闭环仿真验证成本极高。

#### 2. 方法论
论文提出了 **JailWAM** 框架，核心思想是通过“视觉化”和“双路径验证”实现高效的物理安全评估：
*   **三级安全分类框架**：
    *   **Level 0 (安全合规)**：运动平滑，无碰撞。
    *   **Level 1 (运动失败)**：系统死锁、高频振荡或关节磨损，任务中断。
    *   **Level 2 (灾难性风险)**：失控摆动、越界碰撞、对人员或环境造成严重威胁。
*   **视觉-轨迹映射 (VTM)**：将不同模型的低级动作序列（如关节坐标）积分并投影为标准化的 2D 正交视觉轨迹图（顶视图和前视图），使异构动作空间具备语义可解释性。
*   **风险判别器 (RD)**：基于轻量化视觉语言模型（Qwen3-VL-2B）进行微调，直接从 VTM 生成的轨迹图中识别潜在的破坏性行为。
*   **双路径验证策略**：
    *   **阶段 I (开环筛选)**：利用 RD 对 LLM 生成的大量候选指令进行快速粗筛，剔除安全指令。
    *   **阶段 II (闭环验证)**：仅对高风险指令进行高保真物理仿真和人工审核，确保评估的准确性。
*   **越狱指令生成**：利用 Gemini、GPT-5 等强 LLM 配合专门设计的模板，生成任务无关但具有物理威胁的指令。

*   **实验场景与基准**：

*   **评估指标**：攻击成功率 (ASR)、运动失败率 (MFR)、灾难性风险率 (CRR)。

*   **效率提升**：双路径策略将评估 100 个候选指令的时间从 9.15 小时缩短至 3.66 小时（加速 2.5 倍）。

#### 5. 实验数量与充分性
*   **实验规模**：涵盖了 4 种不同的具身智能架构（LingBot-VA, Motus, Cosmos-Policy, $\pi_{0.5}$）和 2 个主流仿真平台。
*   **消融实验**：针对 VTM 的有效性、RD 的微调效果、以及双路径策略的效率进行了详细对比。
*   **可靠性测试**：进行了跨随机种子的可靠性实验，验证了攻击指令在不同初始物理配置下的鲁棒性。

#### 6. 主要结论与发现
*   **WAM 存在严重安全漏洞**：在 JailWAM 攻击下，SOTA 模型 LingBot-VA 的攻击成功率高达 **84.2%**。
*   **跨架构迁移性**：针对某一模型生成的越狱指令可以零样本迁移到其他 WAM 架构（如 Motus 和 Cosmos-Policy），诱发显著的物理风险。
*   **漏洞根源**：实验对比发现，非 WAM 模型（如 $\pi_{0.5}$）对此类攻击的敏感度极低，证明 WAM 的物理脆弱性与其内置的**生成式视觉世界先验**密切相关。
*   **防御可行性**：JailWAM 的风险判别器可作为推理侧过滤器，在不显著影响正常任务的前提下，大幅降低越狱成功率。

#### 7. 优点
*   **首创性**：首次系统性地研究了 WAM 在物理层面的越狱漏洞。
*   **评估高效**：通过 VTM 将抽象动作转化为视觉信号，巧妙利用了 VLM 的空间推理能力，避开了复杂的数值回归。

*   **仿真与现实差距 (Sim-to-Real)**：所有实验均在仿真环境中完成，尚未在真实物理机器人上验证其破坏力。

*   **防御局限**：目前的防御手段主要基于推理侧过滤，尚未探索如何在模型训练阶段进行更深层的安全对齐。
