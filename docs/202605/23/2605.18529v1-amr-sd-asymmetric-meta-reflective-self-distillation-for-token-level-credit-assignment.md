## 摘要
大型语言模型（LLMs）在复杂推理中的对齐主要依赖于可验证奖励强化学习（RLVR）。然而，标准算法（如 GRPO）对所有令牌统一使用序列级奖励，导致严重的信用分配瓶颈。虽然在策略自蒸馏尝试通过在特权上下文中调节自教师来解决问题，但直接接触原始 oracle 解往往导致过度调节的教师分布、隐式答案泄露以及训练后期崩溃。为克服这些限制，我们提出了非对称元反思自蒸馏（AMR‑SD）。AMR‑SD 不直接基于原始参考轨迹进行调节，而是插入一个反思瓶颈：它将诊断信号——来自验证器结果、同伴 rollout 或参考反馈——压缩为简洁的自生成苏格拉底式提示和批评。此外，我们引入因果信息增益（CIG），采用非对称 ReLU 门控阈值，将这些反思转化为稀疏、高精度的令牌级优势调制。结合时间退火，该机制在保留基线环境奖励的同时过滤分布噪声。在科学、数学和工具使用基准上的实验表明，AMR‑SD 显著优于现有基线，实现了稳健的长时域稳定性并成功防止后期崩溃。

## Abstract
The alignment of Large Language Models (LLMs) for complex reasoning heavily relies on Reinforcement Learning with Verifiable Rewards (RLVR). However, standard algorithms like GRPO apply sequence-level rewards uniformly to all tokens, creating a severe credit-assignment bottleneck. While on-policy self-distillation attempts to resolve this by conditioning a self-teacher on privileged contexts, direct exposure to raw oracle solutions often induces over-conditioned teacher distributions, implicit answer leakage, and late-stage training collapse. To overcome these limitations, we propose Asymmetric Meta-Reflective Self-Distillation (AMR-SD). Instead of conditioning directly on raw reference traces, AMR-SD inserts a reflection bottleneck: it compresses diagnostic signals -- from verifier outcomes, peer rollouts, or reference feedback -- into concise, self-generated Socratic hints and critiques. Furthermore, we introduce Causal Information Gain (CIG) with an asymmetric, ReLU-gated threshold to translate these reflections into sparse, highly precise token-level advantage modulations. Combined with temporal annealing, this mechanism preserves the base environmental reward while filtering out distributional noise. Experiments across scientific, mathematical, and tool-use benchmarks demonstrate that AMR-SD significantly outperforms existing baselines, achieving robust long-horizon stability and successfully preventing late-stage collapse.

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

### 核心思想
AMR‑SD 通过**插入反思瓶颈**（reflection bottleneck）将诊断信号压缩为自生成的苏格拉底式提示（hint）和批评（critique），并利用**因果信息增益（CIG）**结合**非对称 ReLU 门控阈值**将反思转化为稀疏且精确的 token 级优势调制，同时通过**时间退火**保留基础环境奖励并过滤分布噪声。

### 关键技术细节

| 组件 | 作用 | 核心公式 / 机制 |
|------|------|----------------|
| **Meta‑Reflection（元反思）** | 将 verifier 结果、同伴 rollout 或参考反馈等诊断信号压缩为简洁的提示或批评，避免直接暴露原始 oracle 轨迹 | `ci = f_reflect(x, y_i, sgn(Ai))`，成功 rollout 生成 `<hint>`，失败 rollout 生成 `<critique>`（若存在 verifier-approved 同伴则基于对比生成，否则为空 fallback） |
| **因果信息增益（CIG）** | 衡量教师（具备反思上下文）与学生在每个 token 上的概率差异，识别知识缺陷或过度自信 | `I_t^CIG = log( πθsg(at | st, ci) / πθ(at | st) )`，并通过对称截断 `clip(I_t^CIG, -κ, κ)` 保证数值稳定性（κ=5） |
| **非对称 ReLU 门控阈值（τ）** | 仅在 CIG 绝对值超过阈值 τ 时触发稀疏调制，默认保留基线环境奖励，实现噪声过滤 | `∆t = I{Ai≥0}·λ_eff·max(0, I_t^CIG − τ) + I{Ai<0}·γ_eff·max(0, −I_t^CIG − τ)` |
| **时间退火（Temporal Annealing）** | 防止训练后期自蒸馏信号退化，将优化平滑过渡至纯环境奖励 | `{λ_eff, γ_eff} = {λ, γ} · max(0, 1 − t_global / T_decay)`，T_decay=50 |

### 算法流程（Algorithm 1 要点）
1. 对每个 prompt 采样 G=8 条轨迹，计算组归一化优势 Ai；
2. 对成功 rollout 生成 hint（m_i=1），对失败 rollout 若存在 verifier-approved 同伴则生成 critique（m_i=1），否则 fallback 至 GRPO（m_i=0）；
3. 通过 forced decoding 获取教师条件概率与学生无条件概率，计算 CIG 并截断；
4. 根据非对称 ReLU 门控阈值 τ 计算优势调制量 ∆i,t，并更新 token 级优势 `bAi,t = stop_gradient(Ai · (1 + ∆i,t))`；
5. 使用修改后的 PPO 裁剪代理目标 `LAMR-SD(θ)` 更新策略参数。

## 7. 优点：方法或实验设计上的亮点

1. **信息瓶颈设计精巧**：通过 Meta‑Reflection 将诊断信号压缩为低带宽的自然语言提示/批评，而非直接暴露 oracle 轨迹，有效缓解过度条件化与答案泄露；
2. **稀疏精确的 token 级信用分配**：非对称 ReLU 门控阈值默认保留环境奖励，仅在高置信度分歧时触发稀疏调制，避免连续信号衰减；
3. **理论驱动的超参数选择**：基于 CIG 偏态分布（大量负值）提出 λ > γ（0.2 vs 0.1）的非对称配置，理论与实证相呼应；
4. **时间退火避免退化**：通过线性衰减平滑过渡至纯环境奖励，保证后期训练稳定性；
5. **实验设计全面**：多基模型、多任务域、长时域追踪、消融完整、反思能力量化分析；
6. **揭示反直觉发现**：直接使用 ground-truth 作为条件居然不如自生成反思，为后续研究提供了重要警示。
