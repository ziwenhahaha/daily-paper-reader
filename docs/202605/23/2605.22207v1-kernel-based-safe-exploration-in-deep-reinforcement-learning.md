## 摘要
将深度强化学习算法部署到真实世界时，安全性一直是主要关注点。一个有前景的方向是与策略一起学习一个 barrier function，以确保学习到的策略不会访问不安全区域。barrier 是从状态到实数的函数，它为初始状态赋低值、为不安全状态赋高值，并且在每次转移中期望下降；这样的函数可用于界定到达不安全状态的概率。先前尝试直接从探索数据学习 barrier function，但这需要大量数据，或对系统动力学作出限制。本文展示如何在未知动力学随机系统的深度强化学习过程中，使用核嵌入学习 barrier function。我们的算法 kernel-based safe exploration（KBSE）在探索期间同时学习最优策略和 barrier。barrier 以条件均值嵌入表示并迭代计算，随着更多探索提供更好的概率安全保证。探索算法使用学习到的 barrier function 识别安全违规。一旦发生违规，它会进行干预，将不安全动作修改为安全动作，从而确保探索被限制在能够界定到达不安全状态概率的动作集合中。我们在多个复杂连续控制基准上评估 KBSE。实验结果表明，这一新算法适合合成概率安全的控制策略，并且不会降低奖励积累。

## Abstract
Safety has been a major concern when deploying deep reinforcement learning algorithms in the real world. A promising direction that ensures that the learned policy does not visit unsafe regions is to learn a \emph{barrier function} along with the policy. A barrier is a function from states to reals that assigns low values to the initial states, high values to the unsafe states, and decreases in expectation on each transition; such a function can be used to bound the probability of reaching unsafe states. Previous attempts learned a barrier function directly from exploration data, but this required either large amounts of data or restrictions on the system dynamics. In this paper, we show how kernel embeddings can be used to learn barrier functions during deep reinforcement learning for stochastic systems with unknown dynamics. Our algorithm, \emph{kernel-based safe exploration (KBSE)}, learns an optimal policy and a barrier simultaneously during exploration. The barriers are computed iteratively, represented as conditional mean embeddings, and provide better probabilistic safety guarantees with more exploration. The exploration algorithm uses the learned barrier functions to identify safety violations. In the case of violation, it intervenes to modify the unsafe action to a safe action, thereby ensuring that the exploration is restricted to actions that bound the probability of reaching unsafe states.   We evaluate KBSE on several complex continuous control benchmarks. Experimental results establish our new algorithm to be suitable for synthesizing control policies that are probabilistically safe without degradation in reward accumulation.

### 理论背景
- **屏障函数（Barrier Function）**：一个从状态空间映射到实数的函数，具有以下性质：
  1. 对初始状态赋予低值
  2. 对不安全状态赋予高值
  3. 在每个转移上期望值单调递减（或增加不超过常数 c）
- 这样的函数可以**概率化地保证安全**：根据 Kushner 定理，如果存在非负屏障函数 B，则 $P\{\text{some state in } \omega \text{ is in } S_u\} \leq \frac{\eta + cT}{\nu}$
- **关键挑战**：学习屏障函数的第三个条件（条件期望）需要精确估计 $E[B(s^+)|s]$，这在未知动态下极其困难

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

### 核心思想
利用**核方法**（kernel methods）的表示能力，将屏障函数表示为再生核希尔伯特空间（RKHS）中的函数，通过**条件均值嵌入（Conditional Mean Embedding, CME）**将条件期望转化为内积运算，从而将屏障函数的学习问题转化为线性规划问题。

### 关键技术细节

#### 2.1 核与再生核希尔伯特空间（RKHS）
- 选择 **RBF（径向基函数）核**：$k_S(s_i, s_j) = \exp\left[-\frac{\|s_i - s_j\|^2}{2\sigma^2}\right]$
- RKHS 具有**再生性质**：$\langle f(\cdot), k(s, \cdot) \rangle_H = f(s)$
- 这使得函数值可以通过内积计算

#### 2.2 条件均值嵌入（CME）
CME 将条件概率分布嵌入到 RKHS 中：
$$\mu(p)(\cdot, s, a) = E_{s^+ \sim p(s,a,\cdot)}[k_S(\cdot, s^+)|s, a]$$
利用 CME，条件期望可以写成：
$$E[B(s^+)|s, a] = \langle B, \mu(\cdot, s, a) \rangle_{H_{k_S}}$$

#### 2.3 经验 CME 的计算
给定数据集 $R = (\hat{S}, \hat{A}, \hat{R}, \hat{S}^+)$，经验 CME 为：
$$\hat{\mu}(\cdot, s, a) = k_{\hat{S}\hat{A}}(s, a)^T [K_{\hat{S}\hat{A}} + \lambda N I_N]^{-1} k_+(\cdot)$$
其中 $K_{\hat{S}\hat{A}}$ 是 Gram 矩阵，$\lambda$ 是正则化参数。

#### 2.4 屏障函数的表示与学习
- 将屏障函数表示为核函数的线性组合：
$$B(s) = \sum_{i=1}^N \alpha_i \cdot k_S(s, s_i)$$
- 使用线性回归（带正则化）学习系数 $\alpha$：
$$\min_\alpha \|\alpha \cdot K_{\hat{S}} - Y\|^2 + \lambda \|\alpha\|^2$$
其中 $Y$ 是安全/不安全状态的二值标签。

#### 2.5 屏障函数验证与鲁棒性
- 定义**模糊集合**（ambiguity set）$C_\epsilon = \{\mu \in G \mid \|\mu - \hat{\mu}\|_G \leq \epsilon\}$
- 鲁棒验证条件（Theorem 2）：
$$W(s,a)^T \cdot B(\hat{S}^+) - B(s) \leq c - \epsilon \cdot \sqrt{k_{SA}((s,a),(s,a))} \cdot \bar{B}$$
- 计算常数 $c$：
$$c = \min_{s \in S} \max_{a \in A} [W(s,a)^T \cdot B(\hat{S}^+) - B(s) + \epsilon \cdot \sqrt{k_{SA}((s,a),(s,a))} \cdot \bar{B}]$$

#### 2.6 安全动作修正
当检测到安全违规时：
1. 使用局部线性动态模型 $s_{t+1} = P \cdot s_t + Q \cdot a_t$
2. 求解二次规划找到最近的安全动作：
$$\bar{a}_t = \arg\min_{\bar{a}} \|\bar{a} - a_t\|^2 \quad \text{s.t.} \quad B(s_{t+1}) \leq \nu, \quad s_{t+1} = P \cdot s_t + Q \cdot \bar{a}$$

### 算法流程（KBSE）
1. **初始化**：初始化数据集 $R$、策略 $\pi_\theta$、上界 $\bar{B}$ 和精度 $\epsilon$
2. **初始探索**：使用初始策略收集样本，计算初始屏障函数
3. **主循环**：
   - 采样动作 $a_t = \pi_\theta(s_t)$
   - **安全检查**：若 $s_t \in S_u$，修正动作（使用局部动态 + QP）
   - 执行转移，收集数据
   - 更新策略（off-policy 风格）
   - 每个 episode 结束：更新 $\bar{B}$ 和 $\epsilon$
   - 每个 epoch 结束：重新计算屏障函数（直到满足验证条件）
4. **输出**：策略 $\pi_\theta$ 和屏障函数 $B$，安全概率为 $1 - \delta$，其中 $\delta = \frac{\eta + cT}{\nu}$

## 7. 优点：方法或实验设计上的亮点

### 7.1 方法论亮点

1. **无需系统动态模型**：
   - 完全数据驱动，通过 CME 直接从样本中学习条件期望
   - 适用于未知动态的随机系统

2. **概率化安全保证**：
   - 不是"几乎 surely 安全"，而是允许在给定概率阈值 δ 内的安全违反
   - 这使得 KBSE 可以处理**没有策略能几乎 surely 满足安全约束**的系统

3. **核方法的优势**：
   - RKHS 的再生性质使得条件期望计算为简单的内积
   - RBF 核是 universal kernel，能够近似任意连续函数
   - 理论上有收敛保证（Theorem 3）

4. **鲁棒性设计**：
   - 使用 MMD 模糊集合处理 CME 的估计误差
   - 屏障函数的验证是**鲁棒**的，即在所有可能的真实 CME 下都满足条件

5. **安全动作修正机制**：
   - 当检测到违规时，通过局部动态 + QP 找到"最近"的安全动作
   - 保证学习过程稳定（不会突然跳到完全不同的动作）

### 7.2 实验设计亮点

1. **多层面对比**：
   - 不仅比较 reward，还比较 cost 和 episode length
   - 提供训练过程的安全性分析（安全违规时间分布）

2. **多样化的 benchmarks**：
   - 从 2D 连续控制到 376D 人形机器人
   - 安全约束既有简单单变量，也有复杂的多变量组合

## 8. 不足与局限

### 8.1 方法论局限

1. **计算复杂度**：
   - 核方法需要存储和操作 Gram 矩阵 $K \in \mathbb{R}^{N \times N}$，内存复杂度 $O(N^2)$
   - 虽然论文提到固定 $N$ 可以控制内存，但随着状态维度增加，$N$ 可能需要增大

2. **可扩展性问题**：
   - 论文承认核方法的可扩展性是一个限制
   - 虽然可以处理 Humanoid（376 维），但对于超大规模系统可能有困难
   - 未来方向是使用稀疏 CME

3. **假设条件**：
   - 假设 CME 属于某个向量值 RKHS（Remark 3）
   - 虽然在 RBF 核下通常成立，但需要额外验证

4. **最优性保证**：
   - KBSE **不保证**找到 Problem 1 的最优策略
   - 只保证策略是**概率安全的**（with confidence $1-\zeta$）

4. **超参数敏感性**：
   - 核带宽 $\sigma$、正则化参数 $\lambda$、$\bar{B}$ 等超参数的选择对结果有影响
   - 论文未详细讨论敏感性分析

1. **只考虑安全，不考虑其他时序逻辑约束**（如 STL 规格）
