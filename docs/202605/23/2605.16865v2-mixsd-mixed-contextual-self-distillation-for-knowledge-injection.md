## 摘要
监督微调（SFT）被广泛用于向语言模型注入新知识，但它往往会导致预训练能力（如推理和通用领域表现）下降。我们认为，这种遗忘是因为来自人类或外部系统的微调目标与模型的自回归分布偏离，迫使优化器模仿低概率的 token 序列。为了解决这一问题，我们提出 MixSD，一种无需外部教师的分布对齐知识注入方法。MixSD 并不在固定目标上训练，而是通过混合来自基础模型自身的两个条件分布的 token 动态构建监督：一个是观察到注入事实在上下文中的专家条件，另一个是反映模型原始先验的朴素条件。由此产生的监督序列在保留事实学习信号的同时，更接近基础模型的分布。我们在两个自行构建的合成语料库上评估 MixSD，以研究在受控环境下的事实召回和算术函数获取，并与已建立的开放领域事实问答和知识编辑基准进行比较。在多种模型规模和设置下，MixSD 与 SFT 和 on-policy 自蒸馏基线相比，始终取得更好的记忆‑保留权衡，在保持接近完美的训练准确率的同时，保留高达 100% 的基础模型留出能力，而标准 SFT 仅保留低至 1%。我们进一步表明，MixSD 在基础模型下产生显著更低 NLL 的监督目标，并减少沿 Fisher 敏感参数方向的有害移动。这些结果表明，将监督与模型本机生成分布对齐是一种简单而有效的知识注入原则，能够缓解灾难性遗忘。

## Abstract
Supervised fine-tuning (SFT) is widely used to inject new knowledge into language models, but it often degrades pretrained capabilities such as reasoning and general-domain performance. We argue this forgetting arises because fine-tuning targets from humans or external systems diverge from the model's autoregressive distribution, forcing the optimizer to imitate low-probability token sequences. To address this problem, we propose MixSD, a simple external-teacher-free method for distribution-aligned knowledge injection. Instead of training on fixed targets, MixSD constructs supervision dynamically by mixing tokens from two conditionals of the base model itself: an expert conditional that observes the injected fact in context, and a naive conditional that reflects the model's original prior. The resulting supervision sequences preserve the factual learning signal while remaining substantially closer to the base model's distribution. We evaluate MixSD on two synthetic corpora that we construct to study factual recall and arithmetic function acquisition in a controlled setting, together with established benchmarks for open-domain factual question answering and knowledge editing. Across multiple model scales and settings, MixSD consistently achieves a better memorization-retention trade-off compared to SFT and on-policy self distillation baselines, retaining up to 100% of the base model's held-out capability while maintaining near-perfect training accuracy, whereas standard SFT retains as little as 1%. We further show that MixSD produces substantially lower-NLL supervision targets under the base model and reduces harmful movement along Fisher-sensitive parameter directions. These results suggest that aligning supervision with the model's native generation distribution is a simple and effective principle for knowledge injection that mitigates catastrophic forgetting.

## 1. 论文核心问题与整体含义（研究动机和背景）

- **问题**：监督微调（SFT）被广泛用于向预训练语言模型注入新知识（如专有领域事实、算术函数等），但往往导致模型在推理、指令遵循、通用领域表现等方面出现 **灾难性遗忘（catastrophic forgetting）**。  
- **根本原因**：标准 SFT 使用的是外部（人类或合成系统）编写的目标序列，这些目标与模型自身的自回归分布存在偏差——即模型被迫去模仿 **低概率的 token 序列**，从而产生沿敏感参数方向的有害更新。  
- **研究目标**：在不依赖外部教师的情况下，让监督目标 **与模型本机生成分布对齐**，从而在 **保持新知识记忆** 的同时 **最大限度保留预训练能力**。

## 2. 方法论：核心思想、关键技术细节与算法流程

### 2.1 核心思想
- **自蒸馏 + 分布对齐**：利用基础模型自身生成两类条件分布（expert conditional 与 naive conditional），在 token 级别进行 **Bernoulli 混合**，形成 **分布对齐的监督目标**，从而降低目标序列在基础模型下的负对数似然（NLL）。

### 2.2 关键技术细节
| 步骤 | 说明 |
|------|------|
| **Expert Conditional** | 条件为 `x_i ⊕ prompt ⊕ y*_i`（即注入的事实已放在上下文），从基础模型采样得到 `ṹ⁺_{i,t}`，倾向于以模型自身的表面形式表达正确答案。 |
| **Naive Conditional** | 条件仅为原始 prompt `x_i`，采样得到 `ṹ⁻_{i,t}`，反映模型的先验分布。 |
| **Bernoulli Mixing** | 对每个 token 位 t，以概率 `1‑λ` 选取 `ṹ⁺`，以概率 `λ` 选取 `ṹ⁻`，形成混合目标 `y^{mix}_{i,t}`。 |
| **训练目标** | 标准负对数似然（NLL）损失：`L_MixSD = -E_{(x_i, y*_i)} E_{ṹ⁺, ṹ⁻} Σ_t log p_θ(y^{mix}_{i,t} | x_i, y^{mix}_{i,<t})` |
| **λ 的作用** | λ=0 时等价于纯专家条件；λ 增大则引入更多“先验锚点”，提升保留能力但略牺牲记忆精度。 |

### 2.3 关键公式（文字描述）
- **参考损失** `L_ref = E_{(x,y)∼P_ref}[-log p_θ(y|x)]`：衡量模型在原始预训练分布上的表现。  
- **遗忘度量** `F(D) = L_ref(θ_D) - L_ref(θ⋆)`：微调后相对基准模型的损失增量。  
- **Fisher 对齐比** `R = (Δθ⊤ \hat{F} Δθ) / ‖Δθ‖² tr(\hat{F})/d`：量化更新沿敏感方向的程度，R>1 表示更新集中在高曲率方向（易导致遗忘）。

## 6. 论文主要结论与发现

1. **记忆‑保留权衡显著改善**：  
   - **SFT**：在 KGFACT‑SMALL 上训练准确率接近 100%，但通用基准平均保留率跌至 **1‑14%**（模型越小，遗忘越严重）。  
   - **MixSD (λ≈0.3‑0.5)**：保持 **≈95‑100% 训练准确率**，同时保留 **80‑100%** 的通用能力（如 Qwen3‑8B 在 AIME‑2024 上 32% vs. SFT 的 12%）。  

2. **Fisher 对齐比与遗忘强相关**：  
   - 参数位移幅度（‖Δθ‖²）与遗忘的相关性弱（r≈0.02‑0.34）。  
   - Fisher 对齐比 R 与遗忘显著相关（r≈0.56‑0.82），说明 **更新方向**（而非大小）是灾难性遗忘的主要驱动。  

3. **目标分布的 NLL 大幅下降**：  
   - SFT 目标中 **27‑42%** 的 token 在基础模型下 NLL>5；而 MixSD（λ=0.3）仅 **4‑8%**。  
   - 高 NLL（>8）token 在 SFT 中占 **33‑42%**，在 MixSD（λ=0.7）中降至 **2‑3%**，且仍然保留 **81‑98%** 的关键事实 token（Recall 高）。  

4. **错误模式转变**：  
   - SFT 导致大量 **collapse**（短模板答案）和 **leakage**（生成虚构实体）错误。  
   - MixSD 消除这两类错误，使错误分布恢复为 **genuine reasoning errors**，与基础模型一致。  

5. **跨模型家族的泛化**：  
   - 在 Llama‑3.2‑1B‑Instruct 上，MixSD（λ=0.5）保持 98% 训练准确率且平均保留率提升至 **5.3%**（SFT 仅为 1.4%）。  

6. **知识编辑同样受益**：  
   - MQuAKE 场景下，MixSD（λ=0.3）实现 **93‑99%** 编辑准确率，同时在 4B/8B 模型上保留 **≈70‑76%** 通用能力（SFT 降至 **≈8‑40%**），而 MEMIT 虽保持能力但编辑准确率仅 **53‑70%**。  

## 7. 优点

- **无需外部教师**：利用模型自身生成的双条件 rollouts，计算开销低于传统 on‑policy 蒸馏。  
- **分布对齐的监督目标**：通过 Bernoulli 混合显著降低目标序列在基础模型下的 NLL，直接缓解了导致遗忘的“低概率 token”挑战。  
- **λ 超参数提供可解释的权衡控制**： λ 越大越保守（保留能力强），λ 越小越激进（记忆精度高），便于根据实际需求调优。  
- **系统性的实验验证**：跨 3 种模型规模、5 类数据集、多个 λ 设置、丰富的消融与错误分析，确保结论的可重复性与稳健性。  
- **Fisher 对齐分析**：揭示更新方向是遗忘的主因，为后续防止灾难性遗忘提供了理论指导。  

## 8. 不足与局限

- **λ 需要调优**：虽然 λ=0.3 在多数实验中表现良好，但最优值仍取决于任务特性，可能增加实际部署的调参成本。  
- **模型规模限制**：最大实验模型为 8B，尚未在 70B+ 甚至更大规模模型上验证；大规模模型的行为可能出现新模式。  
- **一次性预处理成本**：生成混合 rollouts 需要额外的离线生成过程（尽管是一次性的），在极大规模数据上仍可能成为瓶颈。  
- **实验主要聚焦合成/受控数据**：虽然有 SimpleQA 与 MQuAKE 真实问答，但整体评估仍偏向人工构造的世界图和算术函数，对更开放、噪声更大的真实场景覆盖有待进一步验证。  
- **KL‑loss 变体表现不佳**：论文的消融显示 NLL 目标比 KL‑based 目标更高效，说明在某些设置下仅靠混合 NLL 可能已经足够；若需要更强的保留效果仍需探索其他辅助损失。
