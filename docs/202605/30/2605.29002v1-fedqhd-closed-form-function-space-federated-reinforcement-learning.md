## Abstract
Federated reinforcement learning enables decentralized agents to collaboratively improve policies or value estimates without exchanging raw trajectories. However, FedAvg-style parameter averaging is not function-space consistent: when clients use heterogeneous encoders or even identical nonlinear networks, averaged parameters need not correspond to the weighted average of client value functions in any common function space. We propose FedQHD, a federated Q-learning method using hyperdimensional (random-feature) state encoders with a linear readout, so that Q-functions are nonlinear in state yet linear in trainable parameters. This linear structure enables closed-form aggregation. With a shared encoder, the function-space consensus update coincides exactly with weighted averaging of local readout matrices. With heterogeneous encoders, the server constructs a global teacher by averaging client Q-values on a shared anchor-state set, and each client compiles this teacher into its local representation via a single ridge projection. We formalize the federation gap -- the error incurred when compiling a federated teacher into a heterogeneous client representation -- relative to a client-specific oracle projection. We show that this gap decomposes into subspace misalignment, anchor-set conditioning, and regularization bias. We further identify the anchor-to-dimension ratio $m \geq D_i$ as the well-conditioned regime in which the gap reduces to a multiple of the encoder heterogeneity floor. On four continuous-state, discrete-action control benchmarks, FedQHD matches or outperforms FedAvg-style baselines and distillation-based alternatives while requiring substantially less computation, and the empirical dependence of the federation gap on encoder dimension matches our theoretical analysis.

## 1. 核心问题与整体含义

### 研究动机

- **联邦强化学习（FedRL）** 允许去中心化智能体在不共享原始轨迹的情况下协作改进策略或价值估计。
- **传统 FedAvg 的函数空间不一致问题**：在参数空间中进行平均并不对应于在值函数空间中进行加权平均，尤其当客户端使用**异构编码器**或非线性网络时，平均后的参数无法对应任何公共函数空间中的值函数加权平均。
- **异构联邦的代数定义缺失**：当客户端使用不同编码器、特征维度或架构时，参数平均在代数上变得无定义。
- **蒸馏方法的缺陷**：现有异构联邦方法依赖知识蒸馏，需要每轮迭代优化、对超参数敏感，且在在线 RL 的非平稳 Bellman 目标下不稳定。

### 核心问题

如何在异构编码器条件下实现**函数空间一致的联邦聚合**，同时避免迭代蒸馏的计算开销和训练不稳定性。

## 2. 方法论

### 核心思想

FedQHD 采用**超维随机特征状态编码器**（Hyperdimensional/Random-Feature Encoder）+ **线性读出**，使 Q 函数在状态上非线性但在可训练参数上线性化。这一线性结构使得：
- 共享编码器下：函数空间共识更新**精确等价**于读出矩阵的加权平均。
- 异构编码器下：通过锚状态集上的教师编译实现近优融合，无需迭代优化。

### 关键技术细节

#### Q 函数表示

每个智能体 $i$ 维护超维编码器 $\Phi_i: S \rightarrow \mathbb{K}^{D_i}$ 和读出矩阵 $W_i \in \mathbb{K}^{D_i \times |A|}$，Q 函数定义为：

$$Q_i(s, a) = \Re\left(\langle \Phi_i(s), w_{i,a} \rangle\right)$$

#### 共享编码器下的闭式聚合

当 $\Phi_i = \Phi$ 时，全局模型精确为：

$$W_{\text{glob}} = \sum_{i=1}^N \pi_i W_i$$

即函数空间共识精确对应参数平均，与 $\mu$ 无关。

#### 异构编码器下的锚基聚合

1. **服务器**：在共享锚状态集 $S_{\text{ref}} = \{s_1, \ldots, s_m\}$ 上聚合客户端 Q 值，构建全局教师：
   $$Q_{\text{glob}}^{\text{ref}} = \sum_{i=1}^N \pi_i Q_{\text{ref}}^i$$

2. **客户端**：通过岭回归将教师编译入本地表示（闭式解）：
   $$W_{\text{glob}}^i = \left(X_i^H X_i + \lambda I_{D_i}\right)^{-1} X_i^H Q_{\text{glob}}^{\text{ref}}$$
   
   其中 $X_i \in \mathbb{K}^{m \times D_i}$ 为锚状态编码矩阵。

#### 联邦差距（Federation Gap）分解

论文定义并分析了编译误差的三重来源：

| 项 | 来源 | 解释 |
|---|---|---|
| **项 I** | 编码器异构性 $\bar{h}_i$ | 子空间主角度错位（$\sin\theta_{ij}$）和表示误差（$\varepsilon_{\text{rep}}$） |
| **项 II** | 锚放大效应 | $\frac{\sqrt{m}}{\sqrt{\gamma_i+\lambda}}\bar{h}_i$（当 $m \geq D_i$ 时渐近常数） |
| **项 III** | 岭收缩 | $\frac{\lambda}{\gamma_i+\lambda}\| \hat{W}_i \|_F$（由正则化引入的偏差） |

#### 良态条件

当锚点数 $m \geq D_i$ 时，锚 Gram 矩阵 $G_i = X_i X_i^H$ 满秩，联邦差距简化为异构性下界的倍数：

$$|\Delta_i(s,a)| \leq O(\bar{h}_i) \quad \text{（当 } m \geq D_i, \lambda \rightarrow 0\text{）}$$

## 6. 主要结论与发现

1. **性能优势**：FedQHD 在 8 个配置中的 5 个上取得最佳非 Oracle 性能，在 LunarLander 同构设置下甚至超越 Oracle QHD。

2. **异构编码器的鲁棒性**：锚基投影保留了客户端表示的几何结构，而 Truncate FedAvg-QHD 在异构设置下性能崩溃（如 CartPole 下降 73%）。

3. **计算效率**：FedQHD 比所有 DQN 方法快，异构设置下比同构设置快 3–12 倍，证明**表示维度是计算成本的主要驱动因素**。

4. **联邦差距的理论验证**：
   - 编译误差随 $D$ 的对数-对数斜率为 $-0.525$（接近理论预测的 $-0.5$）。
   - 当 $m \geq D$ 时，联邦差距稳定在几何下界 $O(D^{-1/2})$。
   - 当 $m < D$ 时，锚特征矩阵秩不足，误差不稳定。

5. **可扩展性**：随着客户端数量 $N$ 增加（1→20），FedQHD 在 CartPole 上获得 +30.6% 收益、LunarLander 上获得 +29.5% 收益，且在较大 $N$ 下无性能退化。

6. **零差距条件**：当所有客户端子空间完全对齐（$\theta_{ij}=0$）、无表示误差、$\lambda \to 0$、且锚特征矩阵满列秩时，联邦差距为零。

## 7. 优点

1. **闭式聚合的优雅性**：利用线性参数化结构，将联邦聚合从迭代优化转化为单步岭回归，消除了蒸馏方法的每轮迭代开销。

2. **异构编码器的原生支持**：通过锚状态集上的 Q 值聚合和岭投影，无需参数形状匹配，在异构表示下仍能有效聚合。

3. **理论深度**：提出并严格分析了联邦差距的三重分解（编码器异构性、锚条件、岭收缩），识别出 $m \geq D_i$ 的良态条件和 $O(D^{-1/2})$ 的几何下界。

4. **可解释性**：各分解项具有清晰的统计学意义，便于诊断和调优。

5. **计算效率显著**：完全避免反向传播，用闭式 TD(0) 更新和岭回归替代，在异构设置下计算成本大幅降低。

6. **实验验证严谨**：多维度消融实验覆盖维度变化、锚数量变化和客户端数量变化，且与理论预测高度一致。

7. **与 FedAvg 的精确连接**：在共享编码器下，FedQHD 精确退化为参数平均，建立了经典联邦学习与异构值函数聚合的理论桥梁。

## 8. 不足与局限

1. **动作空间限制**：目前仅支持离散动作空间，连续控制任务中需要扩展至策略梯度或 Actor-Critic 设置。

2. **编码器假设**：依赖固定随机特征编码器（非学习得到），限制了表达能力；扩展至可学习编码器是重要未来方向。

3. **锚集设计敏感性**：锚数量 $m$ 和分布对性能有显著影响，当 $m < D_i$ 时性能不稳定，需要精心设计。

4. **锚集分布假设**：理论分析假设 $\nu_{\text{ref}} \ll \nu_i$（锚分布覆盖客户端状态支撑），在客户端分布差异较大的实际场景中可能不成立。

5. **正则化调优**：岭正则化参数 $\lambda$ 需要调优，其选择影响编译误差和表示保真度的平衡。

6. **在线学习的收敛性**：理论分析仅针对静态编译误差，未给出完整在线 FedQHD 训练 dynamics 的收敛保证。

7. **通信开销**：虽然计算效率高，但需要传输 $m \times |A|$ 的 Q 值矩阵（锚点 Q 值），在 $m$ 较大时通信成本可能成为瓶颈。

8. **公平性风险**：实验仅使用合成异构性（随机带宽和维度变化），真实场景中的异构性可能更复杂（如不同状态覆盖、不平衡数据量）。

9. **奖励函数敏感性**：未分析不同奖励尺度或稀疏奖励对锚基聚合效果的影响。

10. **拓展方向**：论文明确指出未来方向包括可学习编码器、更丰富的观察空间和 Actor-Critic 设置，表明当前框架尚不完整。
