## 摘要
本文研究了异构且互联的多智能体系统的控制策略综合，这些智能体通过通信网络进行数据交换以最小化整体代价。我们提出了一种分布式编码校正双演员-评论家框架，整合了新型消息传递机制。现有方法假设可以无噪声、无延迟地获取全局或局部状态，忽视了在噪声和延迟的情况下，全局状态仍可随时间逐步重建和细化的事实。与此相反，本文显式地基于网络配置对通信采样异步、延迟和链路噪声进行建模。所提出的消息传递机制刻画时序与信息流，以细化并时移全局状态信息，随后用于逐步修正 Q 网络。双 Q 网络设计缓解了过估计偏差，而共享编码器将演员-评论家网络耦合以捕捉智能体间依赖关系。我们在多个测试案例中评估了所提方法，展示其相较于多种基准的优势，并提供了数值遗憾分析。

## Abstract
This paper studies the synthesis of control policies for heterogeneous and interconnected multi-agent systems that collaborate through data exchange over a communication network to minimize a collective cost. We propose a distributed encoded corrective double actor-critic framework that integrates a novel message-passing mechanism. Existing methods assume noise-free and delay-free access to the global or partial states and overlook the fact that the global states, though noisy and delayed, can be progressively reconstructed and refined over time. In contrast, this work explicitly models communication sampling asynchrony, delay, and link noise based on the network configuration. The proposed message-passing mechanism characterizes timing and information flow to refine and time shift global state information, which is then used to incrementally correct the Q-networks. The double Q-network design mitigates overestimation bias, while the shared encoder coupling the actor-critic networks captures inter-agent dependencies. We evaluate our approach in multiple test cases, demonstrate its effectiveness over various baselines, and provide a numerical regret analysis.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 2.1 系统模型
- **多智能体LTI离散时间动力学**：`xℓ(t+1) = Aℓ xℓ(t) + Bℓ uℓ(t)`，整体系统 `X(t+1) = A X(t) + B U(t)`，其中 `X = col(x1,...,xL)`，`U = -K X` 为全局线性反馈控制律。
- **目标函数**：二次代价 `G = Σℓ gℓ`，`gℓ = Σ_t (xℓ(t)ᵀ Sℓ xℓ(t) + uℓ(t)ᵀ Rℓ uℓ(t))`。

### 2.2 通信网络建模
- **网络图** `G = (L, E)`：静态、连通、无向图。  
- **路径与跳数**：`Rℓ,m` 为从智能体 ℓ 到 m 的路径，`Rℓ,m` 为跳数，`dℓ,m = Rℓ,m - 1` 为相对时间偏移。  
- **链路噪声**：非零均值加性白高斯噪声（NAWGN）`ηℓ,m ~ N(μℓ,m, σ²ℓ,m)`，各链路独立。

### 2.3 消息传递机制（Message-Passing）
- **噪声感知路由（Algorithm 1）**：使用 Dijkstra 变体，在路径成本中综合跳数和噪声方差：  
  `J(R) = Σ_{ei,j∈R} (1 + λ σ²i,j)`，λ 为延迟‑噪声平衡系数。  
- **噪声过滤**  
  1. **均值补偿**：`ȳm(t) = ym(t) - μ_total`，去除累积偏置。  
  2. **指数移动平均（EMA）**：`x̃m(t) = β(t) ȳm(t) + (1-β(t)) x̃m(t-1)`，自适应平滑系数 `β(t) = σ²x(t) / (σ²x(t) + σ²_total(t))`，基于短窗口方差估计。  
- **相对时间对齐（Algorithm 2）**  
  - 每个智能体维护容量为 P 的 FIFO 缓冲区 `Bℓ`，根据 `dℓ,m` 将收到的状态信息时移，实现无需全局时钟的相对同步。  
  - 时移后得到时间对齐的全局状态估计 `ˆXℓ(t)`，用于后续的学习校正。

### 2.4 学习框架（CDNet，Algorithm 3）
- **共享编码器** `Φ(·)`：对所有智能体的细化全局状态 `˜X` 编码为潜在表示 `ϕ`（64维），捕获跨智能体耦合。  
- **个体编码头** `Ψℓ(ϕ)`：为每个智能体生成专属特征 `ψℓ`。  
- **演员网络** `κℓ(ψℓ)`：输出控制增益 `Kℓ`，控制律 `Uℓ(t) = -Kℓ ˜Xℓ(t)`。  
- **双评论家网络** `Qℓ(ψℓ, Kℓ|θℓ)` 与 `Q'ℓ(ψℓ, Kℓ|θ'ℓ)`：采用悲观 Q 值更新 `ˆQ = min(Qℓ, Q'ℓ)` 以抑制过估计。  
- **奖励**：`rℓ(t) = -˜Xℓ(t)ᵀ Sℓ ˜Xℓ(t) - Uℓ(t)ᵀ Rℓ Uℓ(t)`。  
- **目标值** `ρℓ = rℓ(t) + γ ˆQ(ψ'ℓ, K*ℓ)`，`K*ℓ = κℓ(ψ'ℓ)`。  
- **校正阶段**：在缓冲区 `Bℓ` 满后（`Dℓ × Pℓ` 步），重新计算奖励 `r'ℓ` 并存入历史缓冲区 `Hℓ`，使用软更新（低学习率）进行二次校正，以补偿异步和延迟带来的时间不一致。

- **时间对齐机制**：FIFO 缓冲区 + 相对时移（Alg 2）使得即使没有全局时钟，也能在异步采样和多跳延迟下实现全局状态对齐。

- **代码公开**：提供完整的 Python 实现，提升可复现性。

## 8. 不足与局限

- **静态网络假设**：仅考虑时间不变的网络拓扑，未涉及动态拓扑、链路失效或移动网络场景。  
- **标量状态实验**：虽然声称适用于向量状态，但在实验部分仅使用标量状态，难以直接验证向量状态下的性能。  
- **训练时长与算力未详细披露**：缺少每种配置的运行时间、显存使用量等信息，导致难以评估实际计算成本。  
- **仅限于二次代价**：目标函数为二次型，且使用线性反馈控制器，对非线性、非二次成本或更复杂控制结构的泛化性未作探讨。  
- **缺乏真实平台验证**：实验均为仿真（LTI 系统），未在实际机器人、无人机等平台上验证鲁棒性。  
- **噪声模型单一**：仅考虑 NAWGN，未对其他噪声分布（如脉冲噪声、非高斯噪声）进行实验。  
- **软更新的超参数依赖**：校正阶段的软更新学习率（文中未给出具体数值）可能对收敛速度与稳定性产生影响，未经系统调参。  
- **对同步需求的假设**：虽然通过相对时移避免全局时钟，但仍然假设每个链路在同一离散时间步内完成一跳，这在高延迟或非均匀链路时可能不成立。  
- **只考虑了线性反馈**：策略形式为 `U = -K X`，对需要更复杂控制律（如基于观测的策略）场景不适用。  
- **基准方法的局限性**：DST 基线未考虑噪声和延迟，若基准实现不够鲁棒，可能导致不公平的优势。
