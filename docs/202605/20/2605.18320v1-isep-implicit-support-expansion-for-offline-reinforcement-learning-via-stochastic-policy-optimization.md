## 摘要
离线强化学习方法通常施加严格的约束以确保安全性；然而，这种刚性往往会阻止在行为策略的直接支持之外发现最优行为。为解决此问题，我们提出隐式支持扩展通过随机策略优化（ISEP），该方法利用在分布内数据与策略样本之间插值的价值函数来隐式扩展可行的动作支持。该机制“密集化”高回报区域，为策略改进创建可通行的路径，同时在理论上保证有界值误差。然而，针对此扩展支持的优化会形成多模态景观，标准的确定性平均会导致模式崩溃和无效动作。ISEP通过随机动作选择策略来缓解这一问题，策略通过在保守克隆和乐观扩展信号之间随机交替来优化。我们将此框架实例化为ISEP-FM，使用条件流匹配并利用无分类器引导来有效捕捉插值的价值信号。

## Abstract
Offline reinforcement learning methods typically enforce strict constraints to ensure safety; yet this rigidity often prevents the discovery of optimal behaviors outside the immediate support of the behavior policy. To address this, we propose Implicit Support Expansion via stochastic Policy optimization (ISEP), which leverages a value function interpolated between in-distribution data and policy samples to implicitly expand the feasible action support. This mechanism "densifies" high-reward regions, creating a navigable path for policy improvement while theoretically guaranteeing bounded value error. However, optimizing against this expanded support creates a multimodal landscape where standard deterministic averaging leads to mode collapse and invalid actions. ISEP mitigates this via a stochastic action selection strategy, optimizing the policy by stochastically alternating between conservative cloning and optimistic expansion signals. We instantiate this framework as ISEP-FM using Conditional Flow Matching utilizing classifier-free guidance to effectively capture the interpolated value signal.

### 研究动机与背景

离线强化学习（Offline RL）的核心挑战在于**保守性与乐观性的平衡问题**：

- **保守性需求**：离线设置下智能体只能从静态数据集中学习，无法在线交互，因此必须依赖数据集支撑来保证价值估计的有效性
- **乐观性需求**：为了发现超越次优数据的最优行为，智能体需要具备一定的泛化能力，能够在数据分布之外进行探索

现有方法通过以下方式解决这一矛盾，但均存在局限：

1. **限制更新范围**：只使用样本内（in-sample）转换 [4,5,6]
2. **限制动作空间**：将动作限制在行为策略的支持范围内 [7,8,9,10]
3. **惩罚过度估计**：对高估的 critic 值进行惩罚 [11,12]

**问题本质**：这些严格约束在保证安全的同时，**阻碍了对数据分布外稀疏高奖励区域的发现**——即最优动作往往处于数据集的"缝隙"之中。

## 2. 论文提出的方法论

### 核心思想

ISEP 提出一种**隐式支持扩展**机制：不再将数据分布视为刚性边界，而是通过价值函数的插值来"密集化"高价值区域，在保证安全的同时创建策略改进的可行路径。

### 三大核心创新

#### 创新一：插值价值目标（Interpolated Value Objective）

提出混合目标函数来平衡安全与探索：

$$L_V(\psi) = (1-p)E_{(s,a)\sim D}[L^2_\tau(\hat{Q}_\theta(s,a) - V_\psi(s))] + pE_{s\sim D, \hat{a}\sim\pi_\phi(\cdot|s)}[L^2_\tau(\hat{Q}_\theta(s,\hat{a}) - V_\psi(s))]$$

- **参数 p**：控制内部分布（保守）与策略采样（探索）之间的权衡
- **第一项**：确保价值估计锚定在真实转换上
- **第二项**：通过查询策略生成动作的价值，推动价值函数向高价值区域扩展

#### 创新二：随机动作选择（Stochastic Action Selection）

**问题**：对保守动作与乐观动作进行确定性插值会导致"模式崩溃"——在非凸动作空间中，线性平均两个有效动作（从左侧vs右侧绕过障碍）会产生无效/碰撞动作。

**解决方案**：使用伯努利门控（Bernoulli gate）随机交替选择目标分布：

$$L_\pi(\phi) = (1-B) \cdot E_{(s,a)\sim D}[\omega(s,a) \log \pi_\phi(a|s)] + B \cdot E_{s\sim D, \hat{a}\sim\pi_\phi(\cdot|s)}[\omega(s,\hat{a}) \log \pi_\phi(\hat{a}|s)]$$

- 避免穿越低价值"山谷"区域
- 保证每次更新只针对有效模式之一

#### 创新三：流匹配实例化（Flow Matching Instantiation）

由于策略目标变为多模态，需要表达能力更强的生成模型：

- 使用**条件流匹配（CFM）**参数化策略
- 条件标记 o ∈ {∅, 1}：o=∅ 为无条件分布，o=1 为优势加权的改进分布
- 推理时使用**无分类器引导（classifier-free guidance）**进行动态权衡

### 算法流程（Algorithm 1）

```
1: 初始化 policy πφ, Q-functions Qθ, target Q-function Qˆθ, value function Vψ
2: for each iteration do
3:   Sample transition mini-batch B = {(s,a,r,s')} ∼ D
4:   Update Vψ by minimizing Equation 2  // 插值价值目标
5:   Update Qθ by minimizing Equation 3  // 标准Bellman误差
6:   Update policy πφ by minimizing Equation 5  // 随机动作选择
7:   Update target network: Qˆθ ← ρQˆθ + (1-ρ)Qθ
8: end for
```

### 理论安全性保证（Theorem 1）

当插值参数 p 满足以下条件时，价值函数有界：

$$p \leq \frac{\delta_\tau + \delta_{sub}}{V_{max} - V^*(s) + \delta_\tau + \delta_{sub}}$$

这确保了即使使用策略生成的 OOD 动作，价值估计仍被最优值函数上界。

## 6. 论文的主要结论与发现

### 核心发现

1. **插值参数 p 的作用**：中等值（p ∈ {0.3, 0.5}）效果最佳，过小（p=0.0）过于保守，过大（p≥0.9）导致数值不稳定甚至发散

2. **随机动作选择优于确定性插值**：在非凸动作空间中，线性平均两个有效动作会产生无效动作；随机选择策略能够保持有效行为模式

3. **隐式支持扩展机制有效**：通过插值价值函数生成样本，可以克服数据集的密度偏差，将概率质量从次优陷阱转移到稀疏全局最优点

4. **Flow Matching变体（ISEP-FM）表现更优**：在所有 benchmark 上均优于 Gaussian 变体，特别是在稀疏数据场景下

### 性能总结

- **MuJoCo locomotion**：ISEP-FM 总分 **91.5**，显著超越 IQL（77.0）、CQL（74.6）等方法
- **Adroit**：ISEP-FM 总分 **84.3**，大幅领先基线
- **Kitchen**：ISEP-FM 总分 **74.8**，在部分任务上达到最优（如 kitchen-complete 87.5）
