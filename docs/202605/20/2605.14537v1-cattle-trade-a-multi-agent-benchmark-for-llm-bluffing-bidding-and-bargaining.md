## 摘要

我们介绍 Cattle Trade，这是一个用于评估大语言模型（LLM）作为不完全信息、对抗性交互和资源约束下策略推理智能体的多智能体基准。该基准在一个持续 50-60 回合的单一长时程游戏中结合了拍卖、隐藏报价贸易挑战（TC）、议价、虚张声势、对手建模和资源分配。与之前隔离测试这些能力的智能体基准不同，Cattle Trade 评估智能体是否在具有冲突激励的多智能体经济游戏中整合这些能力。基准记录每次竞价、TC 报价、反报价和卡牌选择，实现超越最终分数或胜率的行为分析。我们评估了七个性价比高的语言模型和三个确定性代码智能体，跨越 242 场游戏。策略一致性，特别是支出效率、资源纪律和阶段适应性竞价，与排名的关联比支出量或任何单一子技能更强。两个启发式代码智能体优于大多数测试的 LLM，行为轨迹揭示了反复出现的 LLM 失败模式，包括过度竞价、自我竞价、破产 TC 发起和弱对手状态适应。评估智能体能力需要基准测试来在具有冲突激励、不确定性和经济动态的多智能体环境中测试多种能力的联合部署。

## Abstract
We introduce \textsc{Cattle Trade, a multi-agent benchmark for evaluating large language models (LLMs) as agents in strategic reasoning under imperfect information, adversarial interaction, and resource constraints. The benchmark combines auctions, hidden-offer trade challenges (TCs), bargaining, bluffing, opponent modeling, and resource allocation within a single long-horizon game lasting 50--60 turns. Unlike prior agent benchmarks that test these abilities in isolation, \textsc{Cattle Trade} evaluates whether agents integrate them across a competitive, multi-agent economic game with conflicting incentives. The benchmark logs every bid, TC offer, counteroffer, and card selection, enabling behavioural analysis beyond final scores or win rates. We evaluate seven cost-efficient language models and three deterministic code agents across 242 games. Strategic coherence, in particular spending efficiency, resource discipline, and phase-adaptive bidding, is associated with rank more strongly than spending volume or any single subskill. Two heuristic code agents outperform most tested LLMs, and behavioural traces surface recurring LLM failure modes including overbidding, self-bidding, bankrupt TC initiation, and weak opponent-state adaptation. Evaluating agentic competence requires benchmarks that test the joint deployment of multiple capabilities in multi-agent environments with conflicting incentives, uncertainty, and economic dynamics.

## 1. 核心问题与研究动机

### 问题背景

- 现有基准测试（如 MMLU、HumanEval）仅测量知识储备与代码生成能力，无法评估**不确定性下的策略推理**与**对抗性激励**。

- 在下游智能体应用（谈判、采购、投资组合管理等）中，有限资源下的竞拍、议价、虚张声势、资源管理等能力需要**协调整合**，而非单独存在。

### 核心问题

> **能否在单个长周期、多玩家、激励冲突、信息不对称的经济游戏中，评估 LLM 对多种战略能力的综合集成能力？**

### 研究意义

- 超越静态基准和单轮任务成功，测试模型在**扩展交互中的策略连贯性**。
- 揭示 LLM 在多智能体对抗环境中为何失败（而不仅仅是否失败），指向状态追踪、资源约束推理和对手建模等具体能力缺口。

## 2. 方法论：核心思想与关键技术

### 2.1 任务场景：Cattle Trade（改编自 Kuhhandel 桌游）

| 要素 | 说明 |
|------|------|
| 玩家数 | 3–5 人（实验采用 4 人） |
| 游戏时长 | 50–60 回合 |
| 卡片类型 | 10 种动物 × 各 4 张（40 张）；55 张钱牌（面额 0、10、50、100、200、500） |
| 获胜方式 | 收集完整的四张组（quartet），按乘法公式计分 |

### 2.2 计分公式

$$\text{Score} = \left(\sum_{q \in Q} v_q\right) \times |Q|$$

- $Q$：玩家完成的 quartet 集合，$v_q$： quartet 价值（鸡=10，马=1000）
- **乘法结构**使广度优于集中（如 3 个中等 quartet 胜过 2 个高值 quartet），对手建模至关重要

### 2.3 核心机制

#### （1）拍卖（Auction）

- 活跃玩家翻出动物牌，其他玩家**同时出价**（迭代多轮）
- 拍卖师可选择**接受最高出价**（收钱交卡）或**行使购买权**（向最高出价者付款，保留卡片）
- **超支惩罚**：出价超过持有金额 → 财富暴露 + 拍卖重来 → 二次超支则出局
- 无找零机制：支付 60 用 100 元面额需付出 100

#### （2）交易挑战（Trade Challenge, TC）

- 选择与对手**共享动物类型**的对手，发起挑战
- **暗标报价**：发起方将钱牌面朝下放置（可含 0 值牌用于虚张声势）
- 对方选择：
  - **接受**：拿走暗钱（不看），交出动物
  - **反报价**：同样暗标 → 双方揭示 → 出价高者赢
- 平局重复最多 3 次，之后挑战方自动获胜

#### （3）毛驴牌（Donkey Cards）分配

- 翻出毛驴牌时向所有玩家发放钱牌（50/100/200/500），**注入流动性**，制造阶段性转折

### 2.4 关键设计原则

- **信息不对称**：仅公开牌数，钱牌价值隐藏；超支才暴露财富
- **离散货币 + 无找零**：强制精确资源管理，过度支付代价高昂
- **购买权耦合**：竞拍出价同时影响出价者流动性与对手财富状态
- **暗标虚张声势**：0 值牌可伪装高价值，考验对手风险校准能力

### 2.5 代理架构

- **观察空间**：自然语言描述，包含私人状态（自己的钱/动物）、公开状态（他人动物数/钱牌数）、决策上下文、记忆
- **动作空间**：7 类动作（回合选择、竞拍出价、购买权决策、TC 发起/防御/平局重试、草稿板更新）
- **结构化 JSON 输出**：直接控制暗标钱牌组合（而非由启发式代理选择）
- **错误处理**：多阶段 JSON 回退 + 重试机制；超支被故意允许作为战略错误观察

## 7. 优点

1. **首个整合多元能力的基准**：在同一环境中融合竞拍、暗标交易挑战、虚张声势、离散资源管理，而非孤立地测试任一能力。

2. **丰富的行为诊断**：超越最终分数，提供出价纪律、自我出价率、超支频率、TC 紧绷度、阶段自适应等细粒度指标，直接指向能力缺口。

3. **公平统一的评测设计**：所有 LLM 接收完全相同的中性提示，代码代理基于结构化数据运行，消除提示干扰。

4. **真实游戏改编**：改编自成熟桌游，游戏机制经过实践验证，不完美信息、对抗激励、资源约束均自然融入。

5. **扩展性强**：支持参数化配置（玩家数 2–5、每种动物 2–5 张、游戏时长 20–80+ 回合），可从微型二人对决扩展到全程马拉松。

6. **TrueSkill 贝叶斯排名**：相比 Elo 提供更可靠的技能估计，考虑对手质量而非仅统计胜负。

## 8. 不足与局限

### 8.1 实验覆盖不足

- **未进行消融实验**：未测试记忆模式、推理预算、温度等因素的影响，无法厘清这些因素对结果的贡献。
- **Sonnet 小样本问题**：n=14 导致置信区间显著更宽（σ≈1.6 vs ≈1.0），结论可靠性受限。
- **仅测试成本高效模型**：所有模型均为低推理成本设定，未测试前沿大预算模型，无法判断失败模式是否源于成本限制或基础能力缺陷。

### 8.2 潜在偏差风险

- **对手组合不对称**：弱模型可能因特定组合（如双 TrackerAgent）而被系统性压制，影响跨模型公平比较。
- **座位偏差未充分控制**：论文承认牌堆顺序比座位位置影响更大（跨座位胜率差异小 1–2 个数量级），但未进行系统性牌堆顺序消融。
- **提示设计影响**：部分失败可能反映**提示设计缺陷**而非模型能力局限——代码代理无需自然语言提示即可避免这些错误。

### 8.3 方法论局限

- **结构性数据差距**：代码代理在结构化数据上精确计算，而 LLMs 需解析自然语言观察并跨回合追踪状态；部分"推理错误"可能部分源于数值解析或工作记忆限制。
- **排除元通信层**：当前仅支持结构化动作，不支持自由对话（联盟信号说服、关系管理），排除了人类游戏中常见的元通信策略。
- **单一游戏泛化**：失败模式是否泛化至其他经济互动场景尚不可知。

### 8.4 应用限制

- **仅评估竞拍-议价-虚张声势维度**：其他重要能力（如合作、长期规划、社交推理）未被覆盖。
- **人工基准缺失**：未与人类

- **代码代理优势不明确**：虽然 TrackerAgent 击败多数 LLM，但其优势究竟来自**完美信息追踪**还是**贪心决策框架**或**结构化数据接口**尚不清晰——缺乏对这三者独立贡献的消融。
- **长程推理预算未探索**：Gemini 3 Flash 的优势是否来自更长推理链（4,096 tokens）仍属推测；未测试 Claude 或 GPT 系列在等量推理预算下的表现。

## 9. 综合评价

### 定位与贡献

Cattle Trade 填补了 LLM 多智能体**长程战略交互**评测的空白，其核心理念——用**单一连贯游戏环境**同时测试竞拍、议价、虚张声势、资源管理等多能力集成——是具有独创性的。细粒度行为诊断机制使失败分析超越了"胜/负"的二元判断，直接指向具体的战略能力缺口（出价纪律、自我出价控制、阶段自适应）。

### 核心启示

1. **战略连贯性 > 单点能力**：综合指标 η（资本效率）是最强性能预测因子，说明 LLM 在长程交互中的**资源纪律**比瞬时决策精度更关键。
2. **代码代理的启示**：基于精确信息追踪的简单贪心策略（TrackerAgent）足以超越大多数 LLMs，暗示当前 LLM 在**状态追踪与多步规划**上仍有显著缺陷。
3. **自我出价是关键失败模式**：弱模型陷入"无对手时仍持续加价"的螺旋，反映出缺乏**反事实推理**与**对手存在性感知**。

### 改进方向

- 补充记忆模式与推理预算的消融实验
- 纳入中等成本模型（如 Sonnet 3.5、GPT-4o）作为上界参照
- 加入人工玩家对战基准
- 对代码代理进行三因素（信息追踪、贪心决策、结构化接口）独立消融

**总结**：CATTLE TRADE 提出了一套设计严谨、诊断丰富、具有实际应用价值的战略推理基准，揭示了当前 LLM 在对抗性多智能体环境中的关键能力缺口——尤其是在**长程资源纪律**与**反事实推理**方面。实验规模基本充分，但对推理预算、记忆机制和模型规模的分层分析不足，限制了对其"失败源于何因"（模型能力 vs. 推理预算 vs. 提示设计）的精确归因。尽管如此，该基准为后续多智能体战略推理研究提供了重要的评测基础与方法论参考。
