## Abstract
The alignment of Large Language Models (LLMs) for complex reasoning heavily relies on Reinforcement Learning with Verifiable Rewards (RLVR). However, standard algorithms like GRPO apply sequence-level rewards uniformly to all tokens, creating a severe credit-assignment bottleneck. While on-policy self-distillation attempts to resolve this by conditioning a self-teacher on privileged contexts, direct exposure to raw oracle solutions often induces over-conditioned teacher distributions, implicit answer leakage, and late-stage training collapse. To overcome these limitations, we propose Asymmetric Meta-Reflective Self-Distillation (AMR-SD). Instead of conditioning directly on raw reference traces, AMR-SD inserts a reflection bottleneck: it compresses diagnostic signals -- from verifier outcomes, peer rollouts, or reference feedback -- into concise, self-generated Socratic hints and critiques. Furthermore, we introduce Causal Information Gain (CIG) with an asymmetric, ReLU-gated threshold to translate these reflections into sparse, highly precise token-level advantage modulations. Combined with temporal annealing, this mechanism preserves the base environmental reward while filtering out distributional noise. Experiments across scientific, mathematical, and tool-use benchmarks demonstrate that AMR-SD significantly outperforms existing baselines, achieving robust long-horizon stability and successfully preventing late-stage collapse.

## 1. 核心问题与整体含义（研究动机和背景）

- **任务背景**：大语言模型（LLM）在复杂推理任务中的对齐严重依赖强化学习与可验证奖励（RLVR）。标准的策略梯度算法如 GRPO 通过组内基线归一化估计优势，但将序列级的统一标量优势 $A_i$ 等价地应用于 rollout 中所有 token。
- **核心瓶颈**：这种**信用分配（credit-assignment）**问题导致关键的逻辑推理步骤和表面的措辞选择收到完全相同的奖励信号，阻碍了推理任务的优化。
- **现有尝试的缺陷**：社区转向**On-Policy Self-Distillation**（用模型自身作为教师、condition on 特权信息如 ground-truth 解）。但直接强迫学生匹配条件依赖的教师会引入：
  - **信息不对称**：学生盲目模仿自信推理而无法访问底层上下文。
  - **特权信息泄露**（answer leakage）和内在反思能力退化。
  - **训练后期崩溃**（late-stage collapse）。
  - 更近期的改进（如 RLSD）通过连续缩放连续幅度来锚定环境奖励，但连续缩放导致整体学习信号被系统性压制（因为教师 conditioned on oracle 时对大多数学生生成 token 的概率系统性偏低，导致连续乘子大多小于 1）。
- **核心目标**：在避免上述缺陷的前提下，实现**细粒度的 token 级信用分配**，解锁自蒸馏的全部潜力。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 整体框架：AMR-SD（Asymmetric Meta-Reflective Self-Distillation）

AMR-SD 将语言生成建模为 MDP $(S, A, P, R)$，由三个顺序阶段组成：

1. **Meta-Reflection（苏格拉底式再评分）**；
2. **CIG 量化（Causal Information Gain）**；
3. **非对称调制（Asymmetric Modulation）**。

### 2.2 阶段一：Socratic Rescoring via Meta-Reflection

- **核心思想**：不将原始 oracle 解直接暴露给教师，而是让模型自己生成**反思信号**（hint / critique）作为低带宽瓶颈（low-bandwidth bottleneck）。
- **具体流程**：
  - 给定 prompt $x$、生成的轨迹 $y^i$ 和序列级优势 $A_i$ 的符号：
  - 若 $A_i \geq 0$（成功 rollout）：调用 `GenerateHint` 生成 **hint**，从成功轨迹中提取潜在核心推理原则。
  - 若 $A_i < 0$（失败 rollout）：调用 `GenerateCritique` 生成 **critique**，隔离逻辑/句法错误并以前瞻性方式重述。
  - 若无同级验证通过的同伴 rollout，则 fallback 到标准 GRPO。
- **关键公式**：
  $$c_i = f_{\text{reflect}}(x, y^i, \text{sgn}(A_i)) = \mathbb{I}\{A_i \geq 0\} \cdot c_{\text{hint}} + \mathbb{I}\{A_i < 0\} \cdot c_{\text{critique}}$$
- **效果**：将诊断信息压缩为简洁自然语言引导，防止 token 再评分教师直接看到原始 oracle traces，缓解过度条件化和答案泄露。

### 2.3 阶段二：Causal Information Gain（CIG）

- **定义**：从信息论视角，衡量 context-aware 苏格拉底教师与学生策略之间的逐点 log-似然比：
  $$I^{\text{CIG}}_t = \log \frac{\pi_{\theta}^{sg}(a_t | s_t, c_i)}{\pi_{\theta}(a_t | s_t)}$$
  - $I^{\text{CIG}}_t > 0$：学生低概率、教师高概率 → 知识缺陷信号。
  - $I^{\text{CIG}}_t < 0$：学生过度自信信号。
- **对称截断**（保证 Lipschitz 连续和数值稳定）：
  $$\hat{I}^{\text{CIG}}_t = \text{clip}(I^{\text{CIG}}_t, -\kappa, \kappa), \quad \kappa = 5$$

### 2.4 阶段三：Asymmetric Token-Level Advantage Modulation

- **核心公式**（非对称 ReLU 门控阈值）：
  $$\Delta_t = \mathbb{I}\{A_i \geq 0\} \cdot \lambda_{\text{eff}} \max(0, \hat{I}^{\text{CIG}}_t - \tau) + \mathbb{I}\{A_i < 0\} \cdot \gamma_{\text{eff}} \max(0, -\hat{I}^{\text{CIG}}_t - \tau)$$
  - 成功 rollout 中：仅在 $I^{\text{CIG}}_t > \tau$ 时单调增强奖励（放大教师偏好的 token）。
  - 失败 rollout 中：仅在 $-I^{\text{CIG}}_t > \tau$ 时单调增强惩罚（放大学生错误自信的 token）。
- **精化优势张量**：
  $$\hat{A}_{i,t} = \text{stop\_gradient}(A_i \cdot (1 + \Delta_t))$$
- **默认保留基线**：$\Delta_t$ 默认值为 0，仅在高度自信的分歧被检测到时才触发稀疏干预，从而**过滤分布噪声，同时保持环境奖励完整**。

### 2.5 Temporal Annealing Schedule（时序退火）

- 防止后期过拟合到退化的反射信号和耗尽的教师能力：
  $$\{\lambda_{\text{eff}}, \gamma_{\text{eff}}\} = \{\lambda, \gamma\} \cdot \max\left(0, 1 - \frac{t_{\text{global}}}{T_{\text{decay}}}\right)$$
  - 在训练早期保留密集监督，后期平滑切换到纯净环境奖励。
  - 实验设置 $T_{\text{decay}} = 50$ 步。

### 2.6 优化目标

- 修改的 PPO 裁剪代理目标，将序列优势替换为密集估计器 $\hat{A}_{i,t}$：
  $$\mathcal{L}_{\text{AMR-SD}}(\theta) = \mathbb{E}_{x \sim \mathcal{D}, y \sim \pi_{\theta_{\text{old}}}} \left[\frac{1}{T_i}\sum_{t=1}^{T_i} \mathcal{L}_{\text{clip}}(\theta, t)\right]$$
  - 其中 $\mathcal{L}_{\text{clip}}(\theta, t) = \min\left(\rho_t(\theta)\hat{A}_{i,t}, \text{clip}(\rho_t(\theta), 1-\varepsilon, 1+\varepsilon)\hat{A}_{i,t}\right)$，$\rho_t(\theta) = \frac{\pi_\theta(a_t|s_t)}{\pi_{\theta_{\text{old}}}(a_t|s_t)}$。

## 6. 论文的主要结论与发现

1. **AMR-SD 显著超越基线**：在科学推理（尤其是生物和物理）、工具使用和数学推理的几乎所有 benchmarks 上取得最佳或接近最佳性能。
   - Qwen2.5-7B-Instruct 生物任务：66.4 vs. GRPO 48.2（+18.2pp）vs. RLSD 54.6（+11.8pp）。
   - Qwen3-8B 数学平均：62.7 vs. GRPO 60.3（+2.4pp）vs. RLSD 57.2（+5.5pp）。
   - AIME24 达 69.6（最高），HMMT 达 34.6（最高）。
2. **收敛动态独特**：早期（step 15）AMR-SD 常落后于激进利用早期奖励信号的基线，但到 step 75 时反超或领先。体现了"以稳定换取最终最优"的优化哲学。
3. **长期稳定性与抗崩溃**：RLSD 出现早期激增后崩溃；SDPO 出现严重策略崩溃（被排除于数学和后期 Tool Use 评测）；AMR-SD 在整个训练过程中保持单调提升。
4. **Meta-Reflection 优于直接 oracle 条件**：消融显示移除 Meta-Reflection 后性能从 74.3 降至 70.5（-3.8pp）。直接暴露 ground-truth 作为教师条件会因信息不对称导致 CIG 误校准，反而劣于模型自生成的 Socratic 反思。
5. **CIG 阈值和 ReLU 非对称门控不可或缺**：移除 $\tau$ 降至 72.0（-2.3pp）；完全移除非对称 ReLU 降至 71.3（-3.0pp）。阈值保证稀疏性，过滤分布噪声；非对称门控保证方向感知。
6. **Temporal Annealing 保证后期稳定**：移除后降至 70.6（-3.7pp）。后期反射质量退化和教师能力耗尽时，需要退火到纯净环境奖励。
7. **反思能力保护**：AMR-SD 保持健康的反思标记频率（105.32/条），显著高于 RLSD（73.59），证明 Meta-Reflection 瓶颈保护了模型的内部分析探索能力，而非压缩为短视生成。
8. **CIG 分布不对称**：经验分布呈显著左偏（bulk 在 [-1, 0]），峰值略低于 0，因此需要 $\lambda > \gamma$（实验中 $\lambda = 0.2, \gamma = 0.1$）来平衡稀疏的正向 CIG 和大量的负向 CIG。

## 7. 优点

1. **问题定位精准**：直击 RLVR 中 token 级信用分配瓶颈，而非仅改进全局奖励分配。
2. **信息瓶颈设计巧妙**：Meta-Reflection 通过模型自生成 Socratic hint/critique 而非直接暴露 oracle，将特权信息的泄露风险降至最低，同时保留了 token 级指导。
3. **CIG + ReLU 门控阈值的设计精准**：稀疏、精确的干预机制避免了对环境奖励的系统性稀释，同时过滤分布噪声。
4. **非对称性设计有理论支撑**：基于 CIG 经验分布的不对称性（$\lambda > \gamma$），使奖励放大和惩罚放大量级匹配实际信号分布。
5. **Temporal Annealing 提供长期保障**：动态平衡自蒸馏和纯环境奖励，避免后期崩溃。
6. **实验设计全面**：跨科学、数学、工具使用三大类，覆盖两个基座模型，消融完整，可视化了训练动态和反思行为。
7. **教师零延迟**：教师权重与活跃策略严格同步（$\theta_{sg} = \text{stop\_gradient}(\theta)$），无需额外 EMA 调参，简化实现。

## 8. 不足与局限

1. **依赖反思质量**：当基座模型缺乏足够的内省能力时，生成的 hint/critique 容易模糊或事实性错误，传播误导性监督信号。在非 thinking mode 或较弱模型（如 Qwen2.5-7B-Instruct）上，AMR-SD 无法超越 GRPO 基线。方法最适用于具有强 chain-of-thought 或自我评估能力的模型。
2. **推理开销**：Meta-Reflection 引入额外的自回归生成步骤，每个轨迹额外生成最多 4096（科学）或 16384（数学）tokens。实验表明训练速度约为 GRPO 的 80%。对严重资源受限场景可能构成实际障碍。
3. **数学任务仅在 Qwen3-8B + thinking mode 上验证**：未测试其他模型或非 thinking 配置在数学任务上的表现，覆盖面有局限。
4. **工具使用任务对比不完整**：SDPO 在 Tool Use 任务 step 75 被排除（策略崩溃），导致对比不完整。
5. **CIG 分布分析仅限 SciKnowEval**：数学任务的 CIG 分布特性未单独分析，可能与科学任务存在差异。
6. **长期稳定性的理论保证未给出**：Temporal Annealing 是经验性的，缺乏形式化的收敛性分析。
7. **AMR-SD 对 $\lambda$ 和 $\gamma$ 的敏感性未系统探索**：仅报告了一组超参配置（$\lambda=0.2, \gamma=0.1, \tau$ 未明确数值），对超参选择的鲁棒性评估不足。
