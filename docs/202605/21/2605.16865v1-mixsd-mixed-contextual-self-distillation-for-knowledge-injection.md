## 摘要
监督微调（SFT）被广泛用于向语言模型注入新知识，但它往往会导致预训练能力（如推理和通用领域表现）下降。我们认为，这种遗忘是因为来自人类或外部系统的微调目标与模型的自回归分布偏离，迫使优化器模仿低概率的 token 序列。为了解决这一问题，我们提出 MixSD，一种无需外部教师的分布对齐知识注入方法。MixSD 并不在固定目标上训练，而是通过混合来自基础模型自身的两个条件分布的 token 动态构建监督：一个是观察到注入事实在上下文中的专家条件，另一个是反映模型原始先验的朴素条件。由此产生的监督序列在保留事实学习信号的同时，更接近基础模型的分布。我们在两个自行构建的合成语料库上评估 MixSD，以研究在受控环境下的事实召回和算术函数获取，并与已建立的开放领域事实问答和知识编辑基准进行比较。在多种模型规模和设置下，MixSD 与 SFT 和 on-policy 自蒸馏基线相比，始终取得更好的记忆‑保留权衡，在保持接近完美的训练准确率的同时，保留高达 100% 的基础模型留出能力，而标准 SFT 仅保留低至 1%。我们进一步表明，MixSD 在基础模型下产生显著更低 NLL 的监督目标，并减少沿 Fisher 敏感参数方向的有害移动。这些结果表明，将监督与模型本机生成分布对齐是一种简单而有效的知识注入原则，能够缓解灾难性遗忘。

## Abstract
Supervised fine-tuning (SFT) is widely used to inject new knowledge into language models, but it often degrades pretrained capabilities such as reasoning and general-domain performance. We argue this forgetting arises because fine-tuning targets from humans or external systems diverge from the model's autoregressive distribution, forcing the optimizer to imitate low-probability token sequences. To address this problem, we propose MixSD, a simple external-teacher-free method for distribution-aligned knowledge injection. Instead of training on fixed targets, MixSD constructs supervision dynamically by mixing tokens from two conditionals of the base model itself: an expert conditional that observes the injected fact in context, and a naive conditional that reflects the model's original prior. The resulting supervision sequences preserve the factual learning signal while remaining substantially closer to the base model's distribution. We evaluate MixSD on two synthetic corpora that we construct to study factual recall and arithmetic function acquisition in a controlled setting, together with established benchmarks for open-domain factual question answering and knowledge editing. Across multiple model scales and settings, MixSD consistently achieves a better memorization-retention trade-off compared to SFT and on-policy self distillation baselines, retaining up to 100% of the base model's held-out capability while maintaining near-perfect training accuracy, whereas standard SFT retains as little as 1%. We further show that MixSD produces substantially lower-NLL supervision targets under the base model and reduces harmful movement along Fisher-sensitive parameter directions. These results suggest that aligning supervision with the model's native generation distribution is a simple and effective principle for knowledge injection that mitigates catastrophic forgetting.

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

### 2.1 核心思想

- **MixSD（Mixed Contextual Self-Distillation）**：一种无需外部教师的知识注入微调方法。核心思路是**利用基础模型本身生成两条条件 rollout，并按伯努利采样混合**，构造动态监督目标。
- 关键洞察是**"分布对齐"**：让监督 token 本身处于基础模型高概率区域，减少强迫模型学习低概率 token 序列，从而降低对参数空间的破坏性更新。

### 2.2 关键技术细节

- 给定知识注入语料库 $D = \{(x_i, y^*_i)\}_{i=1}^n$，对每个样本定义两条条件 rollout：
  - **专家条件（Expert Conditional）** $\tilde{y}^+_{i,t} \sim p_{\theta^*}(\cdot | x^+_i, y^{\text{mix}}_{i,<t})$：将 ground-truth 目标 $y^*_i$ 拼接到输入上下文（$x^+_i = x_i \oplus \text{prompt instruction} \oplus y^*_i$），模型据此 rollout，倾向于用自己的表层形式表达正确事实。
  - **朴素条件（Naive Conditional）** $\tilde{y}^-_{i,t} \sim p_{\theta^*}(\cdot | x_i, y^{\text{mix}}_{i,<t})$：仅基于原始 prompt，反映模型的原始先验，不包含新知识。

- **Per-token 伯努利混合**：在每个 token 位置 $t$，监督目标按如下方式采样：

$$y^{\text{mix}}_{i,t} = \begin{cases} \tilde{y}^+_{i,t}, & \text{概率 } 1-\lambda \\ \tilde{y}^-_{i,t}, & \text{概率 } \lambda \end{cases}$$

其中 $\lambda \in [0, 1]$ 是混合率超参数。$\lambda = 0$ 对应纯专家条件监督；$\lambda$ 越大则引入越多朴素 token，将模型锚定在其参考分布上。

- **训练目标**：标准 NLL 损失作用于混合后的 rollout：

$$\mathcal{L}_{\text{MIXSD}}(\theta; \lambda) = -\mathbb{E}_{(x_i, y^*_i) \sim D} \mathbb{E}_{\tilde{y}^+_{i}, \tilde{y}^-_{i}} \sum_t \log p_\theta\left(y^{\text{mix}}_{i,t} \mid x_i, y^{\text{mix}}_{i,<t}\right)$$

- **关键特性**：无需外部教师、无需 KL 散度计算（使用 NLL 损失）、离线生成 rollout 后训练（一次性预处理成本）。

### 2.3 理论动机

- 论文定义了**遗忘量化指标** $F(D)$ 为参数更新后参考损失相对基准的增长；以及**分布对齐指标** $N(D)$ 为监督目标在基础模型下的平均 per-token NLL。
- 假设：**低 NLL 的 token 可以用最小的参数变化学习，而低概率 token 诱导沿破坏先验行为方向的更新**。

## 6. 论文的主要结论与发现

1. **SFT 严重损害预训练能力**：在所有模型和数据集中，标准 SFT 达到接近完美的训练准确率，但将平均领域外性能从基线降低 30–80%，在某些情况下低至 1%。

2. **MixSD 显著改善记忆-保留权衡**：MixSD 在保持高训练准确率的同时，保留绝大部分基础模型能力。最佳配置（$\lambda \approx 0.3$ 或 0.5）可保留接近 100% 的 held-out 能力，同时达到接近完美的训练准确率。

3. **$\lambda$ 提供了遗忘-保留权衡的连续控制**：较小的 $\lambda$ 偏向记忆（专家条件），较大的 $\lambda$ 偏向保留（引入更多朴素 token）；$\lambda = 0.7$ 开始出现记忆退化，但保留依然良好。

4. **Fisher 对齐比位移幅度更能预测遗忘**：Raw 参数位移 $\|\Delta\theta\|_2$ 与遗忘相关性弱（r ≈ 0.02–0.34），而 Fisher 对齐比 $R$ 与遗忘强相关（r ≈ 0.56–0.82）。更新方向（而非大小）是能力退化的主要驱动因素。

5. **MixSD 监督目标具有显著更低的 NLL**：在所有模型规模上，MixSD 目标的 per-token NLL 远低于 SFT ground-truth（均值从约 6.13 降至 1.78–2.59），且高 NLL token 占比从 27–42% 降至 4–8%。

6. **定性 failure mode 分析**：SFT 引入了大量 collapse（短模板答案）和 leakage（生成训练语料中的虚构实体）错误；MixSD 保留了基础模型的 error profile，主要为 genuine reasoning errors（连贯但错误的推理尝试）。

7. **跨模型家族泛化**：在 Llama-3.2-1B-Instruct 上验证了相同趋势，证实方法不局限于 Qwen 架构。

8. **知识编辑场景有效**：在 MQuAKE 上，MixSD 达到 93–99% 编辑准确率同时保留 76–90% 的基础能力（4B/8B），优于 MEMIT（编辑准确率仅 53–70%）和 SFT（严重遗忘）。

9. **模型规模效应**：更大的模型表现出更少的遗忘，说明整合来自不同条件分布的 token 需要足够的现有能力。

---

## 7. 优点

1. **理论直觉清晰且有新意**：将灾难性遗忘归因于"监督目标与模型自回归分布的 mismatch"，而非单纯归咎于优化难度，这一视角启发了新的研究方向。

2. **方法简洁、零额外成本**：无需外部教师、无需在线生成 rollout（离线预处理）、无需复杂的 KL 计算，仅需一个超参数 $\lambda$，即可作为 SFT 的直接替代方案。

3. **实验设计严谨周全**：
   - 覆盖合成+真实多场景
   - 多模型规模、多数据集
   - 包含 Fisher 对齐等理论分析
   - 详尽的 error mode 分类
   - 跨家族架构验证

4. **对实践有直接指导价值**：$\lambda = 0.3$ 在大多数场景中表现稳健，且不增加推理时成本，仅在训练时替换监督目标。

5. **提供了有用的诊断工具**：Fisher 对齐比 $R$ 作为比 raw 位移更优的遗忘预测指标，对理解和诊断微调方法具有参考价值。

6. **合成数据集设计科学**：KGFACT 和 KGFUNC 通过虚构世界图和算术函数，完全隔离了预训练污染问题，提供了可控的研究环境。

---

## 8. 不足与局限

1. **超参数敏感性**：MixSD 引入了一个新超参数 $\lambda$，虽然 λ=0.3 在实验中一致表现良好，但最优值可能因任务不同而变化，增加了使用门槛。

2. **最大模型规模有限**：论文最大测试模型为 8B 参数，对 70B 及以上超大模型的行为尚不确定，未探索 scaling 规律。

3. **预处理成本被低估的潜在风险**：生成混合 rollouts 需要基础模型进行大量前向传播（每样本最多 10 次重试），且需要 verifier 判断正确性。对于超大规模训练集，这些成本可能成为瓶颈。

4. **NLL 损失的固有权衡**：论文选择 NLL 而非 KL 损失以提高训练效率，但 KL 损失可能对某些任务提供更好的分布对齐。消融实验显示 KL 方法在保留能力上略优但在记忆效率上显著差（表 12）。

5. **Synthetic 数据集的外部效度**：KGFACT 和 KGFUNC 完全基于虚构实体和规则，与真实世界知识注入场景存在差异，论文也承认需要在更多真实场景（如持续预训练）中进行验证。

6. **遗忘分析仅限于参数空间**：Fisher 对齐分析仅在参数空间进行，未探索激活空间或 embedding 空间中的分布偏移，可能遗漏了其他维度的退化机制。

7. **SimpleQA 训练准确率相对较低**：MixSD 在 SimpleQA 上的训练准确率仅为 7–11%（远低于 SFT 的 16–19%），说明对真实开放域知识的吸收效率仍有限。

8. **知识编辑场景仅测试了单一数据集**：MQuAKE 可能无法完全代表所有类型的知识编辑场景（如实体重叠度高、多跳推理等）。
