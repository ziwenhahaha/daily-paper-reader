## Abstract
We study reinforcement learning (RL) in multi-dimensional continuous state and action spaces with one-sided feedback, where the agent receives partial observations of the state and obtains reward information for only a subset of the state-action space at each time step. This setting introduces substantial challenges in both learning efficiency and privacy preservation. To address these challenges, we propose POOL, a novel privacy-preserving RL algorithm. We conduct a comprehensive theoretical analysis of POOL, deriving a sample complexity bound that matches the known lower bounds for non-private RL. Here, E_rho denotes the privacy parameter, H is the time horizon, and alpha is the optimality-gap parameter. Our findings show that it is possible to enforce strong privacy guarantees while maintaining high learning efficiency, marking a significant step toward practical, privacy-aware RL in multi-dimensional environments with one-sided feedback.

# 论文总结：Privacy Preserving Reinforcement Learning with One-Sided Feedback

## 1. 核心问题与整体含义

- **研究背景**：强化学习（RL）在多维连续状态和动作空间中面临单侧反馈问题，即智能体仅能观察部分状态-动作对的奖励和转移信息，而非完整的反馈。这在实际应用中非常普遍，例如库存控制中的丢失销售模型，只知道实际需求是否超过库存，但无法观测到超过部分的精确值。

- **核心挑战**：
  - **学习效率**：单侧反馈带来的结构化挑战使得标准 RL 技术效果下降，因为无法完整学习转移动态和估计价值函数。
  - **隐私保护**：许多实际应用涉及敏感数据（如医疗决策、金融交易），需要在学习过程中保护数据隐私。
  - **维度灾难**：连续、多维的状态-动作空间使得离散化方法面临指数级增长的复杂度。

- **研究空白**：现有工作要么只关注表格型 RL 的隐私问题，要么只研究单侧反馈但不考虑隐私。**该论文首次同时解决多维连续 MDP 中的隐私保护和单侧反馈问题**。

## 2. 方法论

### 核心思想

论文提出 **POOL（Privacy-Oriented One-Sided Learning）** 算法，结合三大关键技术创新：

1. **部分离散化策略（Partial Discretization）**：基于 ℓ2 范数将 (w+d) 维状态-动作空间划分为 M 个区域，避免传统网格离散化的指数爆炸问题。

2. **多维分段线性近似（Multi-dimensional Piecewise-Linear Approximation）**：在每个区域内构建 w+d 个线性无关的基向量，通过线性组合逼近任意状态-动作对的 Q 值。

3. **高斯机制（Gaussian Mechanism）**：对状态-动作对的访问计数添加高斯噪声，确保满足 ρ-zCDP（零集中差分隐私）。

### 算法流程

```
输入: 离线数据集 D, 奖励函数 r, 隐私预算 ρ
初始化: 估计转移概率 ePh, eVH+1 = 0

for h = H to 1 do:
    if (sh, ah) < λh (在可观测边界内):
        1. 基于 ℓ2 范数将空间划分为 M 个区域
        2. 对每个区域的基向量计算私有 Q 值:
           eQh(bjm) ← rh(bjm) + ePh · eVh+1(bjm)
        3. 使用分段线性近似:
           Qh(sh, ah) = Σ mj Qh(bjm)
    else:
        Qh(sh, ah) ← Qh(λh) (边界值)
    eπh(·|sh) ← argmax Eah~π[Qh(sh, ah)]
    eVh(sh) ← Eah~eπh[Qh(sh, ah)]

输出: 策略 {eπh}Hh=1
```

### 关键公式

- **样本复杂度上界**：˜O((1+Eρ)H³α⁻²)，其中 Eρ = 4√(H log(4HM²(w+d)^w/δ)/ρ)
- **分段线性近似误差上界**（定理1）：Qh(bh) - Q̂h(bh) ≤ Lh√((w+d)/M) + Lh|w+d - ||λh||₂|

### 隐私保证

算法满足 **ρ-zCDP**（零集中差分隐私），可通过参数映射转换为标准的 (ε,δ)-DP。
