## 摘要
自蒸馏利用同一模型作为教师和学生进行学习，但特权信息选择困难且任务适配性差。AVSD提出将特权信息分解为跨视角共识信号和视角特定残差信号，仅在残差信号与共识方向一致且比例适当时添加，实现动态平衡。在数学竞赛（AIME24/25、HMMT25）和代码生成基准（Codeforces、LiveCodeBench）上验证，Qwen3-8B平均提升3.1%超越最强基线。

## Abstract
Self-distillation enables language models to learn on-policy from their own trajectories by using the same model as both student and teacher, with the teacher being conditioned on privileged information unavailable to the student. Such information can come in different types or views, such as solutions, demonstrations, feedback, or final answers. This setup provides dense token-level feedback without relying on a separate external model, but creates a fundamental asymmetry: the teacher may rely on view-specific information that the student cannot access at inference time. Moreover, the best type of privileged information is often task-dependent, making it difficult to choose a single teacher view. In this work, we address both these challenges jointly by introducing AVSD (Adaptive-View Self-Distillation), a novel method of self-distillation with multiple privileged-information views, which reconstructs token-level supervision by separating stable cross-view consensus from view-specific residual signals. AVSD identifies the consensus signal shared across views, which provides a reliable update direction, and then selectively adds the view-specific residual signal to adjust the update magnitude when it both aligns with the consensus direction and remains proportionate to the consensus signal. Experiments on math competition benchmarks (AIME24, AIME25, and HMMT25) show that AVSD consistently outperforms both single-view self-distillation baselines and GRPO, achieving average Avg@8 gains of 3.1% and 2.2% over the strongest baselines on Qwen3-8B and Qwen3-4B, respectively. Moreover, on code-generation benchmarks (Codeforces, LiveCodeBench v6) using Qwen3-8B, AVSD outperforms the single-view self-distillation baseline by 2.4% on average.

- **研究背景**：大语言模型（LLM）在推理任务（如竞赛级数学和代码生成）上训练时面临监督信号稀疏的问题。强化学习验证奖励（RLVR）虽成为主流范式，但其奖励通常是结果级别的、稀疏的，学习效率有限。蒸馏提供了一种密集 token 级指导，但传统离线蒸馏存在分布不匹配问题。
- **自蒸馏的兴起**：自蒸馏（Self-Distillation）通过将同一模型既作为学生又作为教师，教师使用学生无法获取的特权信息（privileged information）进行条件化，从而提供密集的 token 级反馈，无需外部模型。
- **核心问题**：现有自蒸馏方法存在两个根本性挑战：
  1. **基础不对称**：教师依赖的视图特定（view-specific）信息在推理时学生无法访问，可能导致信息泄露和不泛化的更新方向。
  2. **特权信息类型选择依赖任务**：全解、部分推理链、仅最终答案等不同类型的特权信息在不同任务上表现差异显著，没有单一视图在所有任务上都是最优的（图 1 展示了 Qwen3-4B 上不同视图的表现差异）。
  3. **不可靠的残差信号**：仅依赖某一视图或简单聚合多个视图都可能引入噪声或不稳定的更新方向。
- **核心贡献**：提出 **AVSD（Adaptive-View Self-Distillation）**，一种自适应多视图自蒸馏框架，通过将多个特权视图的信号分解为**跨视图共识（consensus）** 和**视图特定残差（residual）**，仅在残差信号与共识方向一致且比例适当时才将其纳入更新。

## 2. 论文提出的方法论

### 2.1 核心思想

将多个特权视图视为一个教师家族（teacher family），把教师信号分解为两部分：
- **共识信号（Consensus Signal）**：跨所有视图一致的部分，提供可靠的更新方向。
- **残差信号（Residual Signal）**：某一视图强烈支持但其他视图不共享的部分，可能是互补信息，也可能是特权伪影。

AVSD 先保留共识方向，再用门控机制有选择性地添加残差来调整更新幅度。

### 2.2 关键数学定义

- **单视图 reverse-KL 目标**：在学生前缀 $h_t$ 处，学生分布 $p_t(v)$，教师分布 $q_t(v)$，token 级蒸馏优势为 $A_t(v) = \log q_t(v) - \log p_t(v)$。$A_t(v) > 0$ 推动 token，$A_t(v) < 0$ 抑制 token。
- **多视图设置**：设 $M$ 个特权视图 $r^{(1)}, \ldots, r^{(M)}$，每个视图产生一个教师分布 $q_t^{(m)}(v)$，对应优势 $\Delta_t^{(m)}(v) = \log q_t^{(m)}(v) - \log p_t(v)$。
- **几何共识目标（Geometric Consensus Target）**：
  - 未归一化几何分数：$e q_t^G(v) = \left(\prod_{m=1}^M q_t^{(m)}(v)\right)^{1/M} = \exp\left(\frac{1}{M}\sum_{m=1}^M \log q_t^{(m)}(v)\right)$
  - 几何目标 $q_t^G(v)$ 是归一化后的几何均值，**强调跨视图共同支持的 token**，是保守的。
  - 几何优势：$A_t^G(v) = \frac{1}{M}\sum_{m=1}^M \Delta_t^{(m)}(v)$，即平均 signed update。
- **算术边际目标（Arithmetic Marginal Target）**：
  - $q_t^A(v) = \frac{1}{M}\sum_{m=1}^M q_t^{(m)}(v)$（算术均值），是包容的，保留至少一个视图强烈支持的 token。
  - 算术优势：$A_t^A(v) = \log q_t^A(v) - \log p_t(v)$。
- **跨视图残差（Cross-View Residual）**：
  - $J_t(v) = \log q_t^A(v) - \log e q_t^G(v) \geq 0$（由算术-几何均值不等式保证非负）。
  - 几何和算术目标的差距即残差，衡量有多少支持仅来自部分视图。

### 2.3 门控重建机制

设计门 $\lambda_t(v) \in [0, 1]$ 来控制残差的添加：

$$\lambda_t(v) = C_t(v) \cdot R_t(v)$$

其中：
- **对齐组件（Alignment Component）** $C_t(v)$：衡量视图间的方向一致性。$C_t(v) = \frac{|A_t^G(v)|}{\frac{1}{M}\sum_{m=1}^M |\Delta_t^{(m)}(v)| + \epsilon}$，范围 $[0,1]$。当所有视图方向一致时 $C_t(v)$ 高，仅部分视图支持时 $C_t(v)$ 低。
- **幅度组件（Magnitude Component）** $R_t(v)$：确保残差与共识幅度成比例。$R_t(v) = \frac{|A_t^G(v)|}{|A_t^G(v)| + J_t(v) + \epsilon}$。

**最终重建优势**：
$$\tilde{A}_t(v) = A_t^G(v) + \lambda_t(v) \cdot J_t(v)$$

关键性质：**残差只能调整幅度，不能翻转共识方向**，因为 $\lambda_t(v) J_t(v) \leq |A_t^G(v)|$（由门的设计保证）。

### 2.4 训练目标

$$\mathcal{L}_{\text{AVSD}}(\theta) = \mathbb{E}_{(x,r)\sim\mathcal{D},\, y\sim P_{\theta}^S(\cdot|x)} \left[\sum_{t=1}^{|y|} D_{\text{KL}}\left(p_\theta(\cdot|h_t) \parallel \text{sg}[q_t^\star(\cdot)]\right)\right]$$

其中 $q_t^\star$ 是基于几何目标和门控残差重建的教师目标。梯度仅流经学生分布 $p_\theta$，教师分布使用 stop-gradient。

### 2.5 与单视图自蒸馏的对比

AVSD 相比 OPSD 的额外开销仅是在同一学生 rollout 上评估 $M$ 个视图条件的教师前向传播。由于 rollout 生成通常是主要计算瓶颈，额外的教师评估（仅是 prefill 前向传播）开销较小（1.13 倍相对时间）。

## 6. 论文的主要结论与发现

1. **AVSD 在数学和代码推理上始终优于所有基线方法**：

   - DeepSeek-R1-Distill-Qwen-7B：平均提升 2.3% 超越最强自蒸馏基线
   - Qwen3-8B 代码任务：平均提升 2.4% 超越 OPSD

2. **单视图自蒸馏的表现对特权信息类型有强烈依赖**（图 1）：没有单一视图在所有基准上都是最优的，验证了多视图学习的必要性。

3. **消融实验验证了门控重建的关键作用**：共识目标和算术目标单独使用均不如 AVSD（全共识平均落后 1.4%，全算术平均落后 1.7%），证明两者结合的必要性。

## 7. 优点

1. **问题定义精准**：清晰地识别了自蒸馏中特权信息选择困难和基础不对称两大挑战，而非仅聚焦于某一个小问题。
2. **理论驱动的方法设计**：从 reverse-KL 的 token 级优势出发，推导出几何共识目标和算术边际目标的数学性质（几何均值鼓励共识、算术均值保留互补信息），方法论有坚实的理论基础。
3. **门控机制简洁有效**：仅用两个组件（对齐 + 幅度）就同时解决了方向一致性和幅度比例问题，且有严格的数学保证（残差不会翻转共识方向）。
4. **计算开销合理**：相对于 OPSD 仅增加 13% 的每步时间，无需额外的学生 rollout，所有视图共享同一生成轨迹。
5. **实验覆盖全面**：跨多个模型规模（4B~8B）、多个任务领域（数学、代码）、多个基准测试，且包含丰富的消融和缩放分析。
6. **Token 级分析深入**：不仅报告最终 benchmark 分数，还深入分析了训练过程中 token 级信用分配的可靠性，提供了方法有效的机制性解释。
7. **与现有工作的互补性**：AVSD 不替代 GRPO 或混合 RL-蒸馏方法，而是提供了多视图聚合的新视角，可与其他方向结合。

## 8. 不足与局限

1. **任务覆盖有限**：主要验证了数学和代码推理任务，虽然方法公式具有一般性，但未在其他场景（如工具使用 Agent、对话系统、多跳推理问答）上验证。
2. **依赖特权视图质量**：AVSD 的效果依赖于构造的特权视图是否真实包含有用信息。如果所有视图都包含相似的偏见或错误，AVSD 可能无法有效识别。
3. **数据质量依赖**：论文指出噪声标注的解决方案可能导致教师信号不可靠，这继承了自蒸馏方法对训练数据质量的敏感性。
4. **仅使用 reverse-KL 目标**：虽然文中讨论了方法可扩展到其他蒸馏损失（如 forward KL），但理论保证（如残差边界性质）仅在 reverse-KL 下成立。
5. **缺乏统计分析**：所有实验均未报告多次运行的均值/方差或统计显著性检验，难以判断观察到的性能差异是否具有统计可靠性。
