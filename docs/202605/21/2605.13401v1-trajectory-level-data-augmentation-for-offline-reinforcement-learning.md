## 摘要
我们提出了一种面向离线强化学习的数据增强方法，灵感来源于主动定位问题。具体而言，我们的方法使得从有限的次优轨迹中训练离策略模型成为可能。我们引入了一种基于轨迹的增强技术，该技术利用任务结构以及奖励、价值函数与记录策略的数学特性之间的几何关系。在数据采集阶段，我们的增强支持次优的记录策略，从而产生更高质量的数据并提升离线强化学习的性能。我们为这些策略提供了理论依据，并在不同维度的定位任务以及部分可观测条件下通过实验验证了其有效性。

## Abstract
We propose a data augmentation method for offline reinforcement learning, motivated by active positioning problems. Particularly, our approach enables the training of off-policy models from a limited number of suboptimal trajectories. We introduce a trajectory-based augmentation technique that exploits task structure and the geometric relationship between rewards, value functions, and mathematical properties of logging policies. During data collection, our augmentation supports suboptimal logging policies, leading to higher data quality and improved offline reinforcement learning performance. We provide theoretical justification for these strategies and validate them empirically across positioning tasks of varying dimensionality and under partial observability.

## 1. 核心问题与整体含义  

**研究动机**  
- 离线强化学习（Offline RL）只能利用预先采集的静态数据集，核心挑战在于**分布偏移**导致的策略外推误差。  
- 数据质量高度依赖产生数据的**日志策略（logging policy）**，而实际系统中日志策略往往是**确定性、结构化且次优的**（如光学元件对准、机械臂定位等）。  
- 在数据采集过程中随意注入更好的动作会破坏日志策略的内部假设，产生“handoff 问题”，导致后续恢复困难。  

**核心问题**  
如何在保持日志策略控制的前提下，**在数据采集阶段主动提升轨迹质量**，从而生成更高回报的离线数据集，进而提升离线 RL 的表现。  

## 2. 方法论  

### 2.1 核心思想  
- **轨迹层面的数据增强**：在已收集的轨迹中寻找“捷径（shortcut）”，即能够跳过冗余或绕远子轨迹的动作序列。  
- **Shortcut 的定义**（Definition 3.2）：对于状态 \((s,W)\)，若动作 \(a\) 使得  
  \[
  \gamma V^{\pi}(s',W) - V^{\pi}(s,W) \ge \|s' - s_W\|
  \]  
  其中 \(s' = f(s,a,W)\)，则称 \(a\) 为 \(\pi\)‑shortcut。  
- **理论保证**：在满足**距离改进（distance‑improving）**、**价值函数 Lipschitz 连续**以及**线性放置误差（LPE）**三条假设时，累加轨迹中若干步的动作仍然是有益的捷径（Theorem 3.6、Corollary 3.8）。  

### 2.2 关键技术细节  

| 关键概念 | 定义/作用 | 关键公式/条件 |
|---|---|---|
| **距离改进策略** | 轨迹每一步的奖励严格递增，\( \|s_{i+1} - s_W\| < \|s_i - s_W\| \) | - |
| **LPE（Linear Placement‑Errors）** | 累积动作的放置误差不超过动作幅度线性增长的上界 \(L_f\) | \(\|f(s_0,\sum_{i=0}^{k-1}a_i,W) - s_k\| \le L_f\sum_{i=0}^{k-1}\|a_i\|\) |
| **Lipschitz 价值函数** | \(|V(s,W)-V(s',W)| \le L_V\|s-s'\|\) | 保证在小误差下价值不会剧烈下降 |
| **Shortcut 采样准则**（Algorithm 1） | 对每个轨迹 \((o_i,a_i,r_i)\)，计算返回 \(G_i = V^{\pi_\beta}(s_i,W)\)。对任意 pair \((i,j)\)（\(i<j\)），若  
  \(\gamma G_j - G_i + r_{j-1} \ge C \sum_{k=i}^{j-1}\|a_k\|\)  
  则把累计动作 \(\hat a = \sum_{k=i}^{j-1} a_k\) 视为捷径并加入增强集。 | \(C\) 为正则化超参，控制选取的严格程度。 |
| **增强后的数据形式** | \((o_i,\hat a, r_{j-1}, o_j)\)：起点观察、捷径动作、对应的奖励与下一个观察。 | - |
| **Augmentor 训练** | 用离线 RL（CQL/IQL）学习的 Q‑函数 \(Q_\theta\) 来决定在每个状态下是否采用 augmentor 的动作 \(a_\theta\)。若 \(a_\theta\) 满足上述 shortcut 条件，则用它替换日志策略的动作；否则保持日志策略。 | \(\pi_{\text{aug}}(o) = a_\theta(o)\ \text{if shortcut else}\ \pi_\beta(o)\) |

### 2.3 算法流程  

1. **离线版（CQL‑SC）**  
   - 在已有的静态数据集上执行 **Algorithm 1**（Shortcut Sampling），生成若干增强样本。  
   - 使用 CQL 在原始 + 增强数据上训练。  

2. **迭代版（LIFT）**  
   - 初始化日志策略 \(\pi_\beta\) 并收集少量轨迹（如 50 条）得到初始数据集。  
   - 在该数据集上训练 augmentor \(a_\theta\)（使用 CQL）。  
   - **Algorithm 2（LIFT）**：在数据采集循环中，以概率 \(p\) 使用 augmentor 的动作；每次交互后重新设置日志策略（模拟 handoff），将轨迹加入数据集。  
   - 期间若开启 **CQL‑SC**，则在每次训练 augmentor 时同样加入 shortcut 采样。

## 6. 主要结论与发现  

1. **Shortcut 增强能够显著提升离线 RL**  
   - 无论是纯离线的 CQL‑SC 还是迭代的 LIFT‑SC，均在大多数场景下优于未增强的基线。  

2. **LIFT‑SC 在高维、部分可观测条件下优势最明显**  
   - 在 \(d=5\) 的 OPO、OLP、OLT 等任务中，LIFT‑SC 明显超越 SAC+Offline Data、CQL‑GTA、DQL 等方法。  

3. **轨迹质量（Dataset Score）与离线 RL 性能高度相关**  
   - LIFT 产生的轨迹拥有最高的平均回报，说明提升轨迹质量比单纯增加探索更重要。  

4. **简化 handoff 的关键**  
   - 通过 augmentor 在数据收集阶段插入高回报的 shortcut，保持了日志策略的内部状态连续性，避免了传统探索注入导致的低质量轨迹。  

5. **理论假设在实际系统中的鲁棒性**  
   - 对于满足 LPE 的扰动（如 fblend、frot、fscale），Shortcut 识别准确无误；  
   - 在 fregrot（非收缩）情况下，shortcut 增益失效；在 fsqrt（不满足 LPE）时仍能带来提升，但优势不明显。  

## 7. 优点  

- **理论与实践并重**：给出了完整的理论框架（距离改进策略、Lipschitz 价值函数、LPE），并推导出可验证的 shortcut 条件。  
- **模块化、易集成**：Shortcut 采样仅为数据预处理，可在任意离线 RL 算法（CQL、IQL 等）上通过单行代码加入 d3rlpy。  
- **适用于次优日志策略**：不要求日志策略本身是优秀的行为策略，只要求其**距离改进**（逐步向目标逼近）。  
- **兼顾数据收集与策略学习**：LIFT 同时改进了数据集质量和后续学习的策略，形成正向循环。  
- **实验覆盖面广**：涵盖线性/非线性/非连续扰动、直接/图像观察、多种离线基线以及完整的消融实验。  

## 8. 不足与局限  

- **依赖强假设**：Shortcut 的有效性依赖于 **LPE** 与 **Lipschitz 价值函数**，在实际物理系统（如强非线性、摩擦、碰撞）中可能失效。  
- **缺乏真实平台验证**：实验均在**半真实模拟**环境中进行，未在真实机器人或光学设备上验证 sim‑to‑real 迁移。  
- **超参数敏感**：正则化参数 **C** 的选取对 shortcut 的质量影响显著，文中默认设为 0（C=0 表示全接受），但在高扰动（如 fregrot）时需要调优。  
- **对 augmentor 的依赖**：Augmentor 的 Q‑函数需要在早期数据很少的情况下训练，若 Q‑估计偏差大，可能导致错误 shortcut，降低数据质量。  
- **实验资源未披露**：未报告 GPU 型号、训练时长、内存占用等实际算力信息，使得结果的可复现性受限。  
- **仅验证于定位任务**：虽然场景涵盖光学、机器人臂等，但方法的通用性在其他离线 RL 领域（如游戏、推荐）尚未得到验证。  

## 9. 总结  

本文提出的 **LIFT**（Logging Improvement via Fine‑tuned Trajectories）通过在数据采集阶段识别并注入**轨迹级捷径**来提升离线强化学习的数据质量。理论上证明了在满足距离改进、Lipschitz 价值函数和线性放置误差的条件下，捷径能够保证价值的提升；实验上在多维度、部分可观测的主动定位任务中验证了 **LIFT‑SC** 持续优于基线方法。该工作为“如何改进日志策略的数据生成过程”提供了系统化、可操作的方向，值得在更广泛的离线 RL 应用场景中进一步探索与验证。
