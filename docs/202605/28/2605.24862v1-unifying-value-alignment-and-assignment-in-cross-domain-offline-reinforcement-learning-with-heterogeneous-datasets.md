## Abstract
Cross-domain offline reinforcement learning (RL) aims to learn a policy in the target domain with a limited target domain dataset and a source domain dataset that exhibits a dynamics shift. Training directly on the original source dataset typically leads to performance collapse. Recent studies perform data filtering from the perspective of dynamics alignment or value alignment to enable efficient policy transfer. However, these studies are typically validated on single-domain or single-behavior-policy source datasets. In this work, we explore a more general heterogeneous cross-domain offline RL setting, where the source datasets may be collected from multiple source domains by diverse behavior policies. We first uncover a critical yet overlooked issue in this setting: value misassignment. Empirically and theoretically, we demonstrate that value misassignment can undermine value alignment, mislead data filtering toward selecting suboptimal samples, and loosen the suboptimality gap, thereby degrading the agent's performance. To address this issue, we propose V2A, which integrates dynamics alignment, value alignment, and value assignment. V2A first employs temporally-consistent modality representation learning to extract dynamics modalities from the source dataset, followed by modality-aware advantage learning to rectify value alignment. Finally, it adopts a data filtering paradigm to selectively share source data for policy learning. Empirical results show that V2A significantly outperforms strong baseline methods under general heterogeneous cross-domain offline RL settings.

# 论文总结：Unifying Value Alignment and Assignment in Cross-Domain Offline Reinforcement Learning with Heterogeneous Datasets

## 1. 核心问题与整体含义（研究动机和背景）

- **问题领域**：跨域离线强化学习（Cross-Domain Offline RL）。其核心挑战在于：目标域只有少量数据，而源域拥有大量数据，但两者之间的转移动态（transition dynamics）存在差异。
- **现有方法的局限**：早期方法（如 IGDF、OTDF）专注于“动态对齐”（Dynamics Alignment），即筛选出与目标域动态相似的源域样本。近期方法 DVDF 提出了“价值对齐”（Value Alignment），进一步考虑源样本的质量（是否为最优轨迹）。然而，**这些方法都假设源域数据集来自单一环境且由单一行为策略生成**。
- **新挑战（异构数据集）**：在实际场景（如机器人学习）中，源数据集往往来源于**多个具有不同物理参数的环境**（多种动态），且数据由**多样化的行为策略**收集。这种“异构性”（Heterogeneous）会导致一个关键但被忽视的问题：**价值误分配（Value Misassignment）**。
- **问题本质**：当源数据混合了不同动态和不同质量的策略时（如在一个 batch 中混合了专家数据和次优数据），在全局数据集上训练的单一价值函数会错误地评估某些样本的真实价值（优势值），导致数据筛选倾向于选择“次优样本”，进而放宽策略的次优性边界（Suboptimality Gap），严重损害目标域策略性能。

## 2. 论文提出的方法论

论文提出 **V2A (Value Alignment and Assignment)** 框架，其核心理念是**通过解耦动态模态（Modality）来解决价值误分配问题**，从而统一动态对齐、价值对齐和价值分配。

### 2.1 关键技术：时序一致的模态表示学习（Temporally-Consistent Modality Representation Learning）
- **目标**：为每个轨迹提取一个隐式的“动态模态”表示 $z$。
- **问题**：传统变分推断的 ELBO 对每个转移独立编码，导致同一条轨迹内的不同转移可能被赋予不同的 $z$，违背时序一致性。
- **解决方案**：提出 **TC-ELBO（时序一致 ELBO）**。
    - **编码器 $q_\psi$**：在**轨迹级别**运行（基于 LSTM/RNN），输出整个轨迹的潜在表示 $z \sim q_\psi(\cdot|\tau)$。
    - **解码器 $p_\theta$**：在**步级别**运行，基于 $(s_t, a_t, z)$ 预测 $s_{t+1}$。
- **优化流程**：采用 **期望最大化（EM）算法**。E 步优化编码器，M 步优化解码器。

### 2.2 关键技术：模态感知的优势学习（Modality-Aware Advantage Learning）
- **数据重标记**：利用学习到的 $z$ 对源域数据集进行重标记，得到 $D'_{src} = \{(s, a, r, s', z)\}$。
- **优势函数训练**：将动态模态 $z$ 作为 Q 函数和 V 函数的输入（如 $Q(s, a, z)$），使用 Sparse-QL 方案进行预训练。
- **核心思想**：同一个状态动作对 $(s, a)$，如果它们处于不同的动态模态 $z$ 下，其真实价值（优势值）应该不同。V2A 赋予它们差异化的价值，从而实现**正确的价值分配**。

### 2.3 数据筛选与策略学习
- **评分函数**：定义了综合评分函数 $f(s, a, s', z) = \lambda \cdot h(s, a, s') + (1-\lambda) \cdot \text{Norm}(A(s, a, z))$。
    - $h(s, a, s')$：衡量动态对齐程度（可由 IGDF 或 OTDF 计算）。
    - $A(s, a, z)$：衡量样本在特定模态下的质量。
- **筛选范式**：基于 $f$ 值筛选源域数据（选择 top-$\xi$ 分位数），仅将这些高质量且动态一致的样本用于训练 Q 网络和策略（基于 IQL）。

## 6. 论文的主要结论与发现

1. **问题确认**：异构跨域离线 RL 中普遍存在**价值误分配**问题。在混合质量数据集中，传统的全局优势函数会导致价值对齐失效，误导数据筛选选择次优样本。
2. **性能提升显著**：V2A 在几乎所有任务上均大幅超越 DVDF。
    - 基于 IGDF 的 V2A 在 24 个任务中的 20 个上优于 IGDF 和 DVDF，总分提升 **21.4%**。
    - 基于 OTDF 的 V2A 在 24 个任务中的 21 个上优于 OTDF 和 DVDF，总分提升 **22.2%**。
    - 相比之下，DVDF 相比 IGDF/OTDF 的提升仅为 6.8% 和 5.5%。
3. **表示有效性**：学习到的模态表示 $z$ 能有效区分不同的动态类型（如运动学 vs 形态学），且模态感知的优势分布更加尖锐，反映了数据的真实最优性分布。

## 7. 优点

1. **问题定位精准**：首次识别并形式化定义了异构跨域离线 RL 中的**价值误分配（Value Misassignment）**问题，这是一个实际且关键的安全隐患。
2. **理论支撑扎实**：通过 Proposition 4.5 建立了异构数据集下次优性边界（Suboptimality Gap）与动态失调、价值失调之间的定量联系，为方法提供了理论依据。
3. **方法创新性强**：
    - **TC-ELBO**：巧妙地利用 EM 算法和轨迹级编码解决了表示学习的时序一致性问题。
    - **模态解耦**：将动态模态作为条件引入优势学习，从根本上解决了价值误分配。
4. **通用性强**：V2A 是一个框架，可以灵活集成多种动态对齐方法（IGDF 或 OTDF）。

## 8. 不足与局限

1. **额外计算开销**：需要进行“时序一致模态表示学习”和“模态感知优势学习”，这增加了预训练的计算成本和时间。

4. **环境覆盖**：实验验证主要在 MuJoCo 模拟器的运动任务上，尚未在更具挑战性的真实机器人或高维观察空间（如图像）中进行验证。
