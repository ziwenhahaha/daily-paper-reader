## 摘要
语言模型从自身轨迹学习的自蒸馏方法。

## Abstract
Self-distillation enables language models to learn on-policy from their own trajectories by using the same model as both student and teacher, with the teacher being conditioned on privileged information unavailable to the student. Such information can come in different types or views, such as solutions, demonstrations, feedback, or final answers. This setup provides dense token-level feedback without relying on a separate external model, but creates a fundamental asymmetry: the teacher may rely on view-specific information that the student cannot access at inference time. Moreover, the best type of privileged information is often task-dependent, making it difficult to choose a single teacher view. In this work, we address both these challenges jointly by introducing AVSD (Adaptive-View Self-Distillation), a novel method of self-distillation with multiple privileged-information views, which reconstructs token-level supervision by separating stable cross-view consensus from view-specific residual signals. AVSD identifies the consensus signal shared across views, which provides a reliable update direction, and then selectively adds the view-specific residual signal to adjust the update magnitude when it both aligns with the consensus direction and remains proportionate to the consensus signal. Experiments on math competition benchmarks (AIME24, AIME25, and HMMT25) show that AVSD consistently outperforms both single-view self-distillation baselines and GRPO, achieving average Avg@8 gains of 3.1% and 2.2% over the strongest baselines on Qwen3-8B and Qwen3-4B, respectively. Moreover, on code-generation benchmarks (Codeforces, LiveCodeBench v6) using Qwen3-8B, AVSD outperforms the single-view self-distillation baseline by 2.4% on average.

## 1. 论文的核心问题与整体含义（研究动机和背景）

### 研究动机
- **自蒸馏范式**：语言模型使用同一模型作为学生和教师，教师访问学生无法获得的特权信息（如完整解答、部分推理、最终答案等），从而提供密集的 token 级反馈，无需依赖外部教师模型。
- **核心挑战**：存在两个根本性不对称问题：
  1. **信息不对称**：教师依赖的视图特定信息学生无法在推理时访问，可能导致学生模仿教师特有的模式而非真正可泛化的推理。
  2. **视图选择依赖性**：不同类型的特权信息在不同任务上表现差异显著，没有单一视图能在所有任务上都是最优的。

### 背景问题
- 已有研究显示，完整解决方案可能导致教师陷入与学生不一致的思维模式；仅提供最终答案则信息量不足但更灵活。
- Yang et al. (2026) 证明信息不对称下的分布匹配存在不可约的互信息差距。
- Kim et al. (2026) 发现最有信息的特权形式会丢弃帮助模型识别推理错误的重要信号。

### 核心问题
**能否同时利用多个特权视图，构建比任何单一特权教师更好的 token 级在线学习信号？**

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
AVSD 将多个特权视图视为教师家族，将教师信号分离为两部分：
1. **共识信号（Consensus Signal）**：跨视图共享的稳定信号，提供可靠更新方向
2. **残差信号（Residual Signal）**：视图特定的支持，调整更新幅度但需经过门控过滤

### 关键技术细节

#### 问题设定
- 学生模型 $p_\theta$ 仅能访问提示 $x$
- 教师模型条件于特权信息 $r^{(m)}$（共有 M 个视图）
- 在相同学生前缀 $h_t$ 下，每个视图产生教师分布 $q^{(m)}_t(v)$

#### 共识目标（Geometric Consensus Target）
定义**几何均值目标**以捕获跨视图的一致支持：
$$eqG_t(v) = \left(\prod_{m=1}^{M} q^{(m)}_t(v)\right)^{1/M} = exp\left(\frac{1}{M}\sum_{m=1}^{M} log\ q^{(m)}_t(v)\right)$$

对应的共识优势（consensus advantage）：
$$A_G^t(v) = log\ eqG_t(v) - log\ p_t(v) = \frac{1}{M}\sum_{m=1}^{M}\Delta^{(m)}_t(v)$$

其中 $\Delta^{(m)}_t(v) = log\ q^{(m)}_t(v) - log\ p_t(v)$ 是各视图的蒸馏优势。

几何目标强调被所有视图共同支持的 token，但可能过于保守。

#### 边际目标（Arithmetic Marginal Target）
定义**算术均值目标**以捕获至少被一个视图强支持的 token：
$$q_A^t(v) = \frac{1}{M}\sum_{m=1}^{M} q^{(m)}_t(v)$$

对应的边际优势：
$$A_A^t(v) = log\ q_A^t(v) - log\ p_t(v)$$

算术目标保留互补支持，但可能受视图特定伪影影响。

#### 残差定义
$$J_t(v) = log\ q_A^t(v) - log\ eqG_t(v) \geq 0$$

残差测量算术边际中保留但被严格几何共识丢弃的支持。

#### 门控机制（Gated Reconstruction）
重建优势定义为：
$$\tilde{A}_t(v) = A_G^t(v) + \lambda_t(v) J_t(v)$$

其中门控 $\lambda_t(v)$ 由两部分组成：

1. **对齐组件（Alignment Component）**：
$$C_t(v) = \frac{|A_G^t(v)|}{\frac{1}{M}\sum_{m=1}^{M}|\Delta^{(m)}_t(v)| + \epsilon} \in [0, 1]$$
当各视图优势同号或弱支持视图接近中性时，$C_t(v)$ 较高。

2. **幅度组件（Magnitude Component）**：
$$R_t(v) = \frac{|A_G^t(v)|}{|A_G^t(v)| + J_t(v) + \epsilon}$$

3. **最终门控**：
$$\lambda_t(v) = C_t(v) \cdot R_t(v) = \frac{|A_G^t(v)|}{\frac{1}{M}\sum_{m=1}^{M}|\Delta^{(m)}_t(v)| + \epsilon} \cdot \frac{|A_G^t(v)|}{|A_G^t(v)| + J_t(v) + \epsilon}$$

关键属性：$\lambda_t(v) J_t(v) \leq |A_G^t(v)|$，确保残差可调整更新幅度但不能反转共识方向。

#### 训练目标
$$\mathcal{L}_{AVSD}(\theta) = \mathbb{E}_{(x,r)\sim D}\ \mathbb{E}_{y\sim P_\theta^s(\cdot|x)} \left[\sum_{t=1}^{|y|} D_{KL}(p_\theta(\cdot|h_t) \parallel sg[q^\star_t(\cdot)])\right]$$

### 算法流程
1. 采样训练样本和特权视图
2. 从当前学生模型采样 rollout
3. 对每个前缀计算学生分布和各视图教师分布
4. 计算几何共识目标和算术边际目标
5. 计算残差 $J_t(v)$
6. 计算对齐组件和幅度组件得到门控 $\lambda_t(v)$
7. 重建教师目标并计算 KL 损失
8. 更新学生模型参数

## 6. 论文的主要结论与发现

### 核心结论
1. **AVSD 一致性优于基线**：在数学和代码推理任务上，AVSD 持续超越单视图自蒸馏和 GRPO
2. **多视图优于单视图**：最优特权视图因任务而异，AVSD 通过组合多个视图获得更鲁棒的学习信号

### 具体性能提升

| 模型 | 基准对比 | 提升幅度 |
|------|----------|----------|
| Qwen3-4B | vs OPSD（最强基线） | Avg@8 提升 2.2% |
| Qwen3-4B | vs Base Model | Avg@8 提升 5.5% |
| Qwen3-8B | vs GRPO（最强基线） | Avg@8 提升 3.1% |
| Qwen3-8B | vs Base Model | Avg@8 提升 4.8% |
| DeepSeek-R1-7B | vs OPSD | Avg@8 提升 2.3% |
| Qwen3-8B 代码 | vs OPSD | Avg@8 提升 2.4% |

### 关键发现

1. **视图选择依赖任务**：没有单一特权视图在所有数据集上最优
2. **共识+残差组合必要**：消融显示单独共识或算术目标均次优，组合才是最优
3. **视图增加带来增益**：性能随特权视图数量增加而提升（ diminishing returns 出现较晚）
4. **Token 级信号更可靠**：AVSD 将错误信用比例从 OPSD 的 12.3% 降至 2.9%
5. **门控打开率适中**：训练中约 21-27% 的 token 位置使用残差路径，表明方法主要依赖共识信号

## 7. 优点

### 方法设计亮点

1. **理论驱动的信号分离**：
   - 首次将多视图教师信号明确分解为共识成分和残差成分
   - 共识目标通过最小化平均反向 KL 推导，几何解释清晰
   - 算术目标通过平均正向 KL 聚合推导，与经典意见池化理论对应

2. **优雅的门控机制**：
   - 两组件门控（对齐 + 幅度）分别解决方向一致性和幅度比例问题
   - 数学保证残差不能反转共识方向（$\lambda_t(v) J_t(v) \leq |A_G^t(v)|$）
   - 自适应调节，无需人工设定阈值

3. **计算效率高**：
   - 所有视图共享同一学生 rollout，无额外采样成本
   - 仅增加批量前向传播开销（1.13 倍时间）
   - 可并行计算所有视图教师分布

4. **无需额外超参数**：
   - 均匀视图权重，无视图可靠性超参数
   - 仅需设置小的常数 $\epsilon$ 避免除零

### 实验设计亮点

1. **跨领域验证**：数学（推理）和代码（生成）双重验证方法泛化性
2. **多规模模型**：覆盖 4B 到 8B 参数，验证尺度可迁移性
3. **系统消融**：清晰分离共识、残差、门控的各自贡献
4. **Token 级分析**：不仅评估最终指标，还深入分析学习信号质量
