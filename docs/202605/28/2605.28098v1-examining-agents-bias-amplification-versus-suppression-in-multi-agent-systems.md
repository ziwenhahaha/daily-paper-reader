## Abstract
Multi-agent systems are increasingly deployed to support various tasks where agents interact to achieve individual and collective objectives. Although these systems can enhance task performance and decision-making, fairness preservation through bias reduction remains challenging. This study examines how agent-level biases shift and impact system-wide fairness. We use prompts to expose individual agents to group-favoring bias, then assess downstream impacts at the system level. To quantify the impact, we propose Favor Bias Strength (FBS), a zero-centered metric that decomposes bias alteration between favored-group uplift and disfavored-group suppression. Using multiple agent designs, benchmarks, and up-to-date large language models, we show that agents endowed with bias can substantially affect system-wide fairness. Interestingly, when agents are exposed to bias uniformly, the system-wide bias elevates, even exceeding the additive sum of the individual agents' biases. The empirical evidence underscores the criticality of fairness in multi-agent systems, which warrants further analyses and empirical tests.

# 论文总结：多智能体系统中偏差的放大与抑制

## 1. 核心问题与整体含义（研究动机和背景）

- **研究问题**：在大语言模型（LLM）驱动的多智能体系统中，当个体智能体被暴露于偏见指令时，系统层面的公平性会受到怎样的影响？偏见是被放大还是被抑制？
- **研究背景**：多智能体系统已在信贷评估、招聘筛选等高风险任务中部署，通过迭代交互和信息交换可提升任务性能与决策质量。然而，系统公平性的保障面临严峻挑战——现有研究多聚焦于单智能体或训练数据的偏差缓解，难以直接迁移至多智能体交互产生的系统性输出场景。
- **核心动机**：探索个体智能体层面的偏见如何在多智能体交互过程中传播、叠加或削弱，从而揭示系统级公平性变化的内在机制。

## 2. 方法论：核心思想、关键技术细节与公式

### 核心思想
- 通过在选定智能体的提示词（prompt）中植入隐蔽的群体偏袒指令，人为制造受控的偏差暴露环境
- 设计**Favor Bias Strength（FBS）**指标，将系统级偏见解构为"受惠群体提升"与"被贬群体压制"两个方向的分量
- 在三条不同的多智能体决策流水线（E0、E1、E2）上系统评估偏差的传播效应

### 关键技术细节

**偏差暴露机制**：
- 在目标智能体的系统提示词中追加隐蔽指令，如：
  - `pro_female`：指示智能体在边界案例中对女性学生（sex=0）给予更有利的考量，降低其失败概率预测
  - `pro_male`：对男性学生（sex=1）偏袒
- 指令要求保持预测准确性且不得在输出中暴露偏见意图

**FBS 指标公式**：

$$
\text{FBS} = (\underbrace{b_f - r_f}_{\text{FBS}_\text{fav}}) + (\underbrace{r_{\bar{f}} - b_{\bar{f}}}_{\text{FBS}_\text{disfav}})
$$

- $b_f$：无偏基线条件下受惠群体 $f$ 的预测失败率
- $r_f$：偏差暴露条件下受惠群体 $f$ 的预测失败率
- $r_{\bar{f}}$：偏差暴露条件下非受惠群体 $\bar{f}$ 的预测失败率
- $\text{FBS}_\text{fav}$：捕捉受惠群体获得更有利结果的"提升效应"
- $\text{FBS}_\text{disfav}$：捕捉非受惠群体获得更差结果的"压制效应"
- FBS > 0 表示偏差按预期方向转移，FBS < 0 表示反向转移，FBS = 0 表示无偏差诱导偏移

**多智能体决策流水线**：

| 流水线 | 组成 | 说明 |
|--------|------|------|
| **E0** | 仅 Prediction 智能体 | 单智能体基线 |
| **E1** | Explanation 智能体 → Prediction 智能体 | 增加公平性解释生成环节 |
| **E2** | Explanation 智能体 → Prediction 智能体 + 校准 ML 预测器 + Judge 智能体 | 引入ML仲裁机制：LLM与ML预测一致时采用ML结果，不一致时由Judge决断 |

**统计估计**：采用配对自助重采样（paired bootstrap resampling），在测试实例上联合重采样干净与偏差暴露预测结果，重复 10,000 次取均值和标准差。

## 6. 论文主要结论与发现

1. **偏差超加性放大（Super-additive Amplification）**
   - 多智能体协同偏差暴露产生的系统级偏见 > 任一单智能体偏差暴露的效应
   - 例：在 E1 中，Gemini-3 的 FBS 从单智能体暴露的 0.476 上升至双智能体暴露的 0.497；在 E2 中，所有智能体暴露的 FBS（0.366）接近单智能体暴露加和（0.188）的两倍

2. **ML 仲裁有效但脆弱**
   - 校准 ML 预测器 + 分歧仲裁机制使单点偏差暴露的 FBS 平均降低约 74%（对比 E1）
   - 但协同偏差暴露会提升分歧率，导致更多案例进入受污染的 Judge 环节，削弱仲裁效果

3. **偏差效应方向不对称性**
   - pro_female 与 pro_male 效果强弱与数据集内的群体-结果分布高度相关
   - Math 数据集中女性通过率更高 → pro_female 效果更强；Portuguese 数据集中男性通过率更高 → pro_male 效果更强
   - 不对称主要由 FBS_fav（受惠群体提升）驱动，而非 FBS_disfav（被贬群体压制）

4. **模型脆弱性差异显著**
   - Gemini-3 最易受偏差影响，始终产生最高 FBS 值
   - GPT-5.4 对偏差几乎免疫，几乎所有条件下 FBS 接近零
   - DeepSeek-V3.2 和 Qwen-3.6+ 在 AUC 和准确率上持续低于其他模型

## 7. 优点

- **问题切入角度新颖**：率先系统研究多智能体系统中个体偏差如何影响系统级公平性，填补了现有研究空白
- **指标设计精准**：FBS 指标将偏差效应解构为"提升"与"压制"两个方向，能够区分偏差诱导偏移与数据中预存的群体差异，比传统 DP/EO 更具诊断价值
- **实验设计严谨**：多模型（5种）× 多数据集（2种）× 多流水线（3种）× 多注入组合 × 多随机种子，覆盖面广且采用配对自助法估计不确定性
- **流水线索的设计层次清晰**：E0/E1/E2 三级流水线逐层增加复杂度，便于解耦各组件对公平性的贡献与削弱作用
- **实践指导性强**：论文末尾提供了面向从业者的具体建议（如将每次提示暴露视为信任边界、引入非 LLM 校准预测器等），具有实际落地价值

## 8. 不足与局限

- **数据集单一**：仅使用两个学生成绩预测数据集，样本量和任务类型有限，无法直接推断至生产级或高维复杂场景
- **偏差暴露形式简单**：使用单一模板生成偏差指令，未探索动态学习、进化式或更隐蔽的偏差注入方式，可能低估真实攻击面的危害
- **敏感属性受限**：仅考察二元性别属性，未涉及多值或数值型敏感属性，FBS 指标在复杂属性场景下的适用性有待验证
- **算力信息缺失**：未披露硬件配置，影响研究可复现性与资源需求评估
- **仲裁机制的单点防御**：ML+Judge 仲裁虽有效，但在多智能体协同偏差暴露下效果显著衰减，缺乏更鲁棒的对抗性防御设计
- **跨领域泛化存疑**：研究聚焦于学生成绩预测这一特定领域，在信用评估、医疗诊断等其他高风险应用中的偏差放大效应尚未验证
