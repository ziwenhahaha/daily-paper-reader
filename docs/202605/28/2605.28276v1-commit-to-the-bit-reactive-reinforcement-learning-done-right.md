## 摘要
强化学习算法通常在马尔可夫假设下进行分析（和设计）。这不现实，因为在实际中遇到的大多数环境要么是部分可观测的，要么需要函数逼近，这限制了智能体获取非马尔可夫状态特征的能力。我们考虑在有限环境中学习最优反应式策略的问题，该环境具有确定性观测（或者等价地，硬状态聚合）。我们引入了一种新算法——承诺Q学习（Committed Q‑learning），并在一个直观假设——我们称之为重连鲁棒性（rewire‑robustness）——下证明了几乎必然收敛到最优反应式策略。该假设严格弱于先前工作中使用的 $q_\star$‑可实现条件。我们的算法是经典Q学习的一种变体，其中行为策略在进入一个特征时承诺执行单一动作，仅在观测到的特征发生变化时才重新采样动作。我们分析的关键部分在于引入准马尔可夫环境。

## Abstract
Reinforcement learning algorithms are commonly analyzed (and designed) under the Markov assumption. This is unrealistic, as most environments encountered in practice are either partially observable, or require function approximation that restricts the agent to access non-Markovian state features. We consider the problem of learning an optimal reactive policy in a finite environment with deterministic observations (or equivalently, hard state aggregation). We introduce a new algorithm, Committed Q-learning, and prove almost-sure convergence to the optimal reactive policy under an intuitive assumption we call rewire-robustness. This assumption is strictly weaker than the $q_\star$-realizability condition used in prior work. Our algorithm is a variant of classical Q-learning in which the behavior policy commits to a single action upon entering a feature, and only resamples actions when the observed feature changes. A crucial part of our analysis is the introduction of quasi-Markov environments.

- **研究目标**：在有限环境（确定性观测，硬状态聚合）中学习最优的**反应式策略**（reactive policy），即不依赖历史特征、仅根据当前特征映射到动作分布的策略。
- **关键挑战**：在状态聚合场景下，经典Q-learning等方法会失效，因为被聚合的状态可能具有完全不同的值，导致特征价值模糊。例如，走廊环境（corridor environment）中，被聚合的蓝色气泡状态具有不同值，经典方法无法找到最优反应式策略。
- **先前工作的局限**：现有理论结果（如Tsitsiklis & Van Roy, 1996; Majeed & Hutter, 2018）要求 $q_\star$‑可实现条件（optimal value function在每个特征内保持常数），这排除了许多看似简单的环境。

## 2. 论文提出的方法论

### 核心思想
- **承诺Q学习（Committed Q-learning）**：经典Q学习的变体，行为策略在进入一个特征时承诺执行单一动作（选项），仅在观测到的特征发生变化时才重新采样动作。
- **关键洞察**：尽管状态聚合导致价值模糊，但存在一个对动态规划有意义的状态特征值——**入口状态的值**（entrance value）。例如走廊环境中，蓝色特征的正确值是入口状态 x=1 的价值 v⋆(1)，而非所有状态的平均值。

### 关键技术概念

| 概念 | 定义 |
|------|------|
| **准马尔可夫环境**（Quasi-Markov Environment） | 每个特征具有唯一的入口分布（entrance distribution），即 dim ¯ςz ≤ 1，此时马尔可夫性质在特征之间成立。 |
| **重连鲁棒性**（Rewire-robustness） | 对环境进行"重连"（改变入口动态，但保持内部结构和转移概率），最优反应式策略保持不变。 |
| **π-重连**（π-rewiring） | 由行为策略π的平稳分布定义的平均入口行为，用于构建辅助环境。 |

### 算法流程（Algorithm 1）
```
1. 初始化 Q(z, ω) = 0 for all z, ω
2. 从初始状态分布采样开始episode
3. 当前特征为z，当前选项为ω
4. 采样过渡：(u ~ ω, x ~ T'_u(:,x), z' = φ(x))
5. 更新 Q(z, ω) ← Q(z, ω) + α_t * Δ, where Δ = r(z') + max_{ω'} Q(z', ω') - Q(z, ω)
6. if z' = z_⊥: 重启episode
7. else if z' ≠ z: z ← z', ω ~ π(z)  // 仅当特征变化时采样新选项
```

### 理论保证
- 在假设3.1（策略proper）、3.2（行为策略探索）和3.3（步长形式）下，承诺Q学习的迭代几乎必然收敛到Q⋆。
- 收敛性基于Liu et al. (2025)对Borkar-Meyn定理的扩展，适用于马尔可夫噪声。

1. **承诺Q学习收敛性**：在重连鲁棒环境下，承诺Q学习几乎必然收敛到最优反应式策略。
