## 摘要
我们研究情节式强化学习，其中奖励函数和转移函数固定，但每个情节依赖的可行动作集合会在该情节开始时被观测到。性能通过相对于每个情节最优价值的累积遗憾来衡量，即所有情节中最优价值与所用策略价值之差的求和，其中 M^k 表示第 k 个情节中的动作上下文。我们表明，MVP 算法可以自然扩展到这一框架，并具有强理论保证。特别地，对于对抗性上下文，我们建立了 minimax 遗憾界，阶为 √(SAH^3K log L) 的对数修正形式，其中 L 表示可能上下文的数量。该结果意味着对于随机上下文，遗憾界为 √(SAH^3K) 的对数修正形式。我们进一步将随机遗憾保证转化为固定上下文分布下约为 SAH^3/ε^2 的样本复杂度界。此外，我们推导了一个依赖 gap 的遗憾界，其中 Δ_min^p 是次优 (h,s,a) 三元组上的全局 p-trimmed 正 gap 下界。当相关次优性 gap 较大时，该界可以显著优于 minimax 速率。

## Abstract
We study episodic reinforcement learning with fixed reward and transition functions, but with episode-dependent admissible action sets that are observed at the start of each episode. Performance is measured by cumulative regret against the episode-wise optimal value, $\sum_{k=1}^K [V^{*,M^k} - V^{π^k,M^k}]$, where $M^k$ represents the action context in the $k$-th episode. We show that the MVP algorithm naturally extends to this framework and enjoys strong theoretical guarantees. In particular, we establish a minimax regret bound of $\widetilde{O}(\sqrt{SAH^3K\log L})$ for adversarial contexts, where $L$ denotes the number of possible contexts. This result implies a regret bound of $\widetilde{O}(\sqrt{SAH^3K})$ for stochastic contexts. We further translate the stochastic regret guarantee into a sample complexity bound of $\widetilde{O}(SAH^3/ε^2)$ for a fixed context distribution.   In addition, we derive a gap-dependent regret bound of \[ \widetilde O\left( \inf_{p\in [0,1)} \left( \frac{1}{Δ_{\min}^{p}} + pKΔ_{\min}^{p} \right)\log K \cdot \mathrm{poly}(S,A,H) \right), \] where $Δ_{\min}^{p}$ is the global $p$-trimmed positive-gap floor over suboptimal $(h,s,a)$ triples. This bound can substantially improve upon the minimax rate when the relevant suboptimality gaps are large.

## 1. 核心问题与整体含义

**研究背景与动机：**
- 传统强化学习（RL）假设每个episode中的动作空间是固定的，但在许多现实应用中，动作可用性会随episode变化
- 论文研究**上下文动作集强化学习**（Contextual Action-Set RL），其中上下文（Context）在每个episode开始时被观察到，决定该episode的可行动作集 $A_{h,s}(M^k)$
- 典型应用场景包括：
  - 机器人关节受损导致某些动作不可用
  - 推荐系统中商品缺货或不可用
  - 医疗治疗方案中某些药物不可用

**核心问题：**
- 在固定转移核和奖励函数但episode依赖的动作集约束下，如何设计算法最小化累积遗憾
- 基准为episode-wise最优值 $V^{*,M^k}$，遗憾定义为 $\sum_{k=1}^K [V^{*,M^k} - V^{\pi^k,M^k}]$

**与相关工作的区别：**
- 区别于一般的上下文MDP（上下文影响转移和奖励），本文假设转移和奖励跨上下文保持不变，仅动作集受上下文影响

## 2. 方法论：核心思想与技术细节

**核心算法：Contextual MVP（Algorithm 1）**

**关键设计理念：**
- 将MVP算法（Monotonic Value Propagation）自然扩展到上下文动作集设定
- 结合model-based optimistic planning、双倍更新调度、单调Bernstein风格bonus

**关键技术流程：**

1. **模型维护：**
   - 维护经验转移估计 $\bar{P}_{h,s,a}$
   - 全局访问计数 $N_{kh}(s,a)$ 和epoch内计数 $N_h(s,a)$

2. **乐观值函数计算（后向归纳，h=H,...,1）：**
   ```
   Q_h(s,a) ← min{r_h(s,a) + ⟨bar P_{h,s,a}, V_{h+1}⟩ + b_h(s,a), H}
   V_h(s) ← max_{a∈A^k_{h,s}} Q_h(s,a)
   ```

3. **探索bonus（单调Bernstein风格）：**
   ```公式
   b_h(s,a) = c1 * sqrt{V(bbar P_{h,s,a}, V_{h+1}) * log(1/δ') / max{N_h(s,a), 1}} + c2 * H * log(1/δ') / max{N_h(s,a), 1}
   ```
   其中 $c_1 = 460/9$，$c_2 = 544/9$

4. **双倍更新调度：**
   - 当任意$(s,a,h)$的访问计数达到2的幂次时，重置epoch计数器并刷新经验模型
   - 显著减少覆盖数（covering number），是扩展到上下文MDP的关键

**理论保障：**
- **乐观性保证**（Lemma 1）：$V^*,M^k \leq V^k_1(s_1)$
- **遗憾分解**：分解为三个可独立界定的项（Term 1, 2, 3）

## 6. 主要结论与发现

**理论贡献：**

1. **对抗性上下文下的遗憾界：**
   - $\tilde{O}(\min\{\sqrt{SAH^3K \cdot \log L}, KH\})$
   - 其中$L$为可能上下文数

2. **随机上下文下的Minimax最优界：**
   - $\tilde{O}(\min\{\sqrt{SAH^3K}, KH\})$
   - 恢复标准Tabular MDP的最佳已知结果

3. **PAC样本复杂度：**
   - $\tilde{O}(SAH^3/\epsilon^2)$ for fixed context distribution

4. **Gap-dependent regret bound（首创）：**
   - 引入$p$-trimmed positive gap $\Delta_{\min}^p$概念

   - 当gap较大时显著优于minimax界

**实验验证：**
- 算法在自定义benchmark上优于S-UCBVI和UCBVI
