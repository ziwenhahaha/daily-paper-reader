## 摘要
人类反馈通常以偏好而非校准数值奖励的形式出现，这推动了从偏好反馈进行强化学习的研究，也称为从人类反馈进行强化学习（RLHF）。我们对情节式核 MDP 中仅基于偏好的学习进行了严格的理论研究。在每个 episode 中，学习器从共同初始状态部署两个策略，并接收一个单比特二元标签，表示哪条轨迹更受偏好；该偏好由累积（未观测）奖励差上的 Bradley--Terry--Luce 链接建模。在对奖励函数和转移函数作核函数假设的条件下（这是适合理论分析的最一般模型之一），我们发展了针对 episode 结束比较的偏好式价值估计和置信集合。我们证明了以高概率成立的悔值界，其随 episode 数次线性增长，从而意味着学得策略的价值会收敛到最优策略的价值。

## Abstract
Human feedback often arrives as preferences rather than calibrated numeric rewards, motivating reinforcement learning from preferential feedback, also referred to as reinforcement learning from human feedback (RLHF). We present a rigorous theoretical study of preference-only learning in episodic kernel MDPs. In each episode, the learner deploys two policies from a common start state and receives a single binary label indicating which trajectory is preferred, modeled by a Bradley--Terry--Luce link on the difference of cumulative (unobserved) rewards. Under kernel-based assumptions on the reward and transition functions (one of the most general models amenable to theoretical analysis) we develop preference-based value estimation and confidence sets tailored to end-of-episode comparisons. We prove high-probability regret bounds that scale sublinearly in the number of episodes, implying that the value of the learned policy converges to that of the optimal policy.

### 1. 核心问题与整体含义（研究动机与背景）

- **问题域**：强化学习中的偏好反馈学习（Preference-based RL / RLHF）
- **研究动机**：人类反馈通常以偏好（哪个轨迹更好）的形式提供，而非精确的数值奖励。在大语言模型对齐、机器人控制、推荐系统等场景中，偏好反馈比数值评分更可靠、信息量更丰富
- **核心挑战**：每回合仅收到一个二进制的偏好标签（单比特反馈），需要从整个轨迹级信息中推断最优策略，且轨迹级信号不提供直接的步级信度分配
- **已有工作局限**：
  - 表格型 MDP 算法计算上不可扩展

  - 核函数 MDP 在标准标量奖励下已有研究，但偏好反馈场景缺乏理论保证

### 2. 论文方法论

#### 核心思想

提出 **PROSTO 算法**（Preference-based Regularized Optimization via Sampled Trajectory induced Optimism），在核函数 MDP 假设下，仅利用偏好标签实现次线性悔值。

#### 关键技术细节

- **模型假设**：奖励函数和转移算子都属于 RKHS（再生核希尔伯特空间），使用 Matérn 核族
- **偏好建模**：使用 Bradley–Terry–Luce (BTL) 模型，将偏好概率建模为 sigmoid 函数作用于累计奖励之差
- **交互协议**：每回合Learner部署两个策略 π 和 π′，执行后收到一个二进制标签 y_k
- **探索策略**：使用高斯过程（GP）扰动实现轨迹级探索，而非确定性椭圆奖励（后者需要预先知道完整轨迹）

#### 核心算法流程

1. **奖励估计**（式 7）：使用带正则化的核 logistic 回归（KLRR）从偏好反馈中估计奖励向量 θ
2. **探索噪声**（式 8）：从后验 GP 中采样 ε_k，协方差由轨迹差特征构造
3. **价值迭代**（式 9）：从后往前逐层计算 Q 函数估计，包含奖励估计 + 探索奖励 + 转移估计 + 椭圆势
4. **策略选择**：每个时间步贪婪选择最大化 Q 估计的动作

#### 关键理论结果

- **悔值上界**（定理 4.1）：  
  R(K) ≤ eO( κ_Z × max{ H^2.5 K^{1-(β_p-1)/(2β_p(β_p+1))}, H^3 K^{1-β_p/(2(β_p+1))} } )
- 其中 β_p 与核的平滑度相关，κ_Z 是 sigmoid 函数最大逆梯度
- 悔值随回合数 K **次线性增长**，即策略价值收敛到最优

#### 关键技术贡献

1. **RKHS 范数控制**：通过正则化参数 τ 显式调控 KLRR 估计器的 RKHS 范数，从而控制对数覆盖数
