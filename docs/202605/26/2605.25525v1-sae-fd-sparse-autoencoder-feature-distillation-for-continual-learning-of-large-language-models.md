## 摘要
持续学习使大型语言模型能够适应不断演化的任务，而无需从头重新训练，但灾难性遗忘仍然是一个核心障碍。在持续学习方法中，基于正则化的方法被广泛用于约束模型更新并减少遗忘，它们作用于权重空间、梯度空间或输出空间。然而，这些稠密表示空间受到特征叠加的影响，即多个概念被编码在重叠维度中，使得在不阻碍新任务学习的情况下选择性保护先前学到的知识变得困难。为解决这一问题，我们提出 SAE-FD（Sparse Autoencoder Feature Distillation），它将模型表示锚定在预训练稀疏自编码器的稀疏特征空间中；在那里，稠密激活被分解为稀疏的过完备基，从而降低表示纠缠，并以更少的新任务学习干扰实现更有针对性的正则化。在跨三种模型架构的两个持续学习基准上的实验表明，SAE-FD 持续优于现有基于正则化的方法，取得最高 52.70% 的平均准确率，且后向迁移仅为 -0.46。

## Abstract
Continual learning enables large language models to adapt to evolving tasks without retraining from scratch, yet catastrophic forgetting remains a central obstacle. Among continual learning methods, regularization-based approaches are widely used to constrain model updates and reduce forgetting, operating in weight space, gradient space, or output space. However, these dense representation spaces suffer from feature superposition, where multiple concepts are encoded in overlapping dimensions, making it difficult to selectively protect previously learned knowledge without impeding new-task learning. To address this issue, we propose \method (Sparse Autoencoder Feature Distillation), which anchors model representations in the sparse feature space of a pre-trained Sparse Autoencoder, where dense activations are decomposed into a sparse overcomplete basis that reduces representational entanglement, enabling more targeted regularization with less interference to new-task learning. Experiments on two continual learning benchmarks across three model architectures show that \method consistently outperforms existing regularization-based methods, achieving up to 52.70% average accuracy with only -0.46 backward transfer.

## 1. 核心问题与整体含义（研究动机和背景）

大型语言模型（LLM）在持续学习场景中面临**灾难性遗忘**问题：当模型顺序学习新任务时，之前习得的知识会被覆盖和干扰。现有正则化方法（包括权重空间、梯度空间和输出空间的正则化）虽然在多个表征空间中约束模型更新，但均存在一个根本局限——它们运作在**稠密表征空间**中，多个概念被编码在重叠的维度上（即"特征叠加"现象），导致难以有针对性地保护旧知识，同时不干扰新任务学习。

**核心动机**：利用稀疏自编码器（SAE）将稠密激活分解为稀疏过完备基，实现特征解耦，从而在更有效的稀疏特征空间中进行正则化，减少表征纠缠带来的干扰。

## 2. 论文提出的方法论

### 核心思想
SAE-FD 将模型表征锚定在预训练稀疏自编码器的稀疏特征空间，利用特征解耦特性实现更有针对性的知识保护，同时保持新任务学习的可塑性。

### 关键技术细节

**Stage 1：Gated SAE 训练**

- 收集基模型最后一层 MLP 的激活向量，来源包括 AG News、Yelp、BoolQ、MMLU 等多种数据集（每个来源最多 2000 样本）
- Gated SAE 使用门控和幅度双路径结构：
  - `g = σ(Wgate(h - bdec) + bgate)` — 决定哪些特征激活
  - `m = Wenc(h - bdec) + benc` — 计算特征强度
  - `fpre = g ⊙ m`，`f = ReLU(fpre)`
- 训练损失：`L_SAE = ||h - ĥ||² + λℓ1·||f||₁`
- 配置：d=4096 → D=32768（8倍扩展），λℓ1=10⁻³

**Stage 2：任务训练与锚点捕获**

- 使用 LoRA 微调（Rank=8, Alpha=32, Dropout=0.1）
- 训练损失：`Ltask = -Σ log P_M(y_j | x<j)`
- 每个任务结束后，从训练数据中选择 Na=200 个锚点样本，执行前向传播提取最后一层 MLP 输出 Hi ∈ R^(L×d)，存储为 (xi, mi, Hi) 元组

**Stage 3：稀疏感知特征蒸馏**

- 编码当前激活 Ha 与存储的锚点激活 Hacurr 通过冻结的 SAE，使用 pre-ReLU 特征以保证梯度流
- **余弦方向损失**（防止全局特征方向漂移）：
  $$L_{cos} = 1 - \frac{f_{curr}^{pre} \cdot f_{anc}^{pre}}{||f_{curr}^{pre}|| \cdot ||f_{anc}^{pre}||}$$
- **激活特征幅度损失**（仅对锚点中已激活的特征施加 MSE）：
  $$L_{mag} = \frac{1}{|A|} \sum_{k \in A}(f_{curr}^{pre,k} - f_{anc}^{pre,k})^2$$
  其中 A = {k : f_anc^{pre,k} > 0}
- 总蒸馏损失：`L_FD = L_cos + L_mag`
- 训练总损失：`L = L_task + λ · L_FD`

**自适应 λ 机制**

- 目标贡献率 ρ 使得 `ρ = λ·L_FD / (λ·L_FD + L_task)`
- 推导得：`λ* = ρ / (1 - ρ) · L_task / L_FD`
- 通过指数移动平均更新：`λ_{t+1} = (1-α)λ_t + α · clip(λ*, λ_min, λ_max)`
- 默认值：ρ=0.15, α=0.05, λ_min=0.2, λ_max=10.0

## 8. 不足与局限

7. **计算开销**：
   - 1.85× SeqLoRA 仍意味着约翻倍训练时间，资源受限场景成本较高
