## 摘要
面向可达性规约的强化学习（RL）是序贯决策中的基础问题，但其理论保证仍较少被探索。近期一项工作实现了向最优策略的渐近收敛。然而，该方法对收敛动态提供的洞见有限。在本文中，我们提出一种替代方法，能够提供更深入的收敛理论洞见。我们的方法建立在带假设的 PAC 学习之上。PAC 学习能够在有限时间内以高置信度保证近似最优策略，但需要知道最小转移概率等 MDP 内部参数。我们认为，尽管这些参数在 RL 中未知，但它们可以被迭代细化并以越来越高的精度进行估计。通过迭代满足 PAC 条件，我们证明了在极限中可以达到精确最优。标准基准上的实证评估验证了我们关于收敛动态的理论洞见。

## Abstract
Reinforcement learning (RL) for reachability specifications is fundamental in sequential decision-making, yet theoretical guarantees remain less explored. A recent work achieves asymptotic convergence to optimal policies. However, this approach provides limited insight into convergence dynamics. In this work, we present an alternative approach that provides deeper theoretical insights into convergence. Our approach builds on PAC learning with assumptions. PAC learning guarantees near-optimal policies with high confidence in finite time but requires knowing internal MDP parameters like minimum transition probability. We argue that while these parameters are unknown in RL, they can be iteratively refined and estimated with increasing accuracy. By iteratively satisfying PAC conditions, we show that exact optimality can be achieved in the limit. Empirical evaluations on standard benchmarks validate our theoretical insights into convergence dynamics.

## 2. 论文方法论

### 核心思想
利用PAC学习与渐近收敛的互补性：**虽无法直接应用PAC（因参数未知），但可迭代猜测并精化参数**，通过逐步满足PAC条件，在极限下实现精确最优。

### 关键算法流程（Algorithm 1）
算法分为无穷个阶段 k = 1, 2, 3, ...，每阶段设置：
- **pk = 1/2^k**：对最小转移概率的猜测（逐阶段缩小）
- **δk = 1/2^k**：置信误差
- **εk = 1/2^k**：近似因子（精度要求逐阶段提高）

每阶段操作：
1. **模拟**：运行 Nk 次仿真采样，更新状态-动作转移计数器
2. **EC检测与折叠**：检测最大终端组件（MEC）并折叠为超状态，防止BVI不收敛
3. **有界值迭代（BVI）**：基于估计的转移概率计算上下界 L(s) 和 U(s)
4. **策略提取**：基于上界选取最优动作，输出确定性无记忆策略 πk

### 关键技术细节

**转移概率估计**（Appendix B）：
- 使用 Hoeffding 不等式构造下界估计
- 置信宽度 c = √(ln(δP/2) / (-2·#(s,a)))
- 估计值：ˆP(s,a,s') = max(0, #(s,a,s')/#(s,a) - c)

**有界值迭代（BVI）**（Section 3.2）：
- 对非EC状态：L(s) = max_{a} Σ_{s'} ˆP(s,a,s')·L(s')
- 对EC中状态：若EC不含目标则L=0，含目标则L=1
- 迭代直至收敛或达到预设次数

**EC检测**（Algorithm 3-4）：
- 判定标准：状态已在当前轨迹中出现过，且候选集合C在部分模型中形成EC
- 需要 δC-sure：每个stay状态-动作对采样至少 n ≥ ln(δC)/ln(1-p_k) 次

**主要理论结果**（Theorem 3.1-3.3）：
- **Theorem 3.1（PAC保证）**：存在 KPAC，当 k ≥ KPAC 时，πk 为 εk-最优策略（置信度 1-δk）
- **Theorem 3.2（精确最优）**：存在 Kopt ≥ KPAC，当 εk < εdiff（最优与次优无记忆策略的最小间隔）时，πk 为精确最优策略
- **Theorem 3.3（渐近收敛）**：Σ δk < ∞，由 Borel-Cantelli 引理，几乎 surely 仅有限阶段失败，最终所有 πk 均为最优

**εdiff 下界**（Section 4, Theorem 4.1）：
- 引入转移复杂度 D（状态-动作对转移概率分母的最小公倍数）
- 证明：任意两无记忆确定性策略的值向量差 ≥ (2D)^{-2|A|·|S|} 2^{-2|S|} 或为0
