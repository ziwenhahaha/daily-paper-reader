## Abstract
Discounted reinforcement learning is usually presented through Bellman equations on closed Markov decision processes. This paper develops a compositional view: a one-step decision process is treated as an open stochastic component, and infinite-horizon policy evaluation is obtained by closing a contractive feedback loop. The resulting semantics assigns typed Bellman transformers to open components, interprets series and parallel wiring as composition and tensoring of transformers, and interprets feedback as an admissible guarded Banach trace realized by a unique fixed point. This perspective yields three theoretical consequences. First, approximate component equivalence is a contextual congruence for admitted well-typed guarded one-hole contexts: local operator error remains controlled after plugging the component into a surrounding circuit that uses the hole once and whose feedback nodes have certified uniform guardedness. Second, exact and approximate state abstractions become commuting or near-commuting coalgebraic diagrams, giving value-preservation and explicit sup-norm distortion bounds. Third, under monotone $ω$-continuous contract-transformer semantics, safety, risk, and resource specifications can be represented as quantale-valued contracts, where local inductive bounds lift through wiring and feedback by least-fixed-point reasoning. Its central claim is not that all RL morphisms form a global traced monoidal category, but that discounted Bellman evaluation admits a contractive feedback semantics on the admissible class of guarded circuits.

# 论文总结：A Contractive Feedback Semantics for Reinforcement Learning

## 1. 核心问题与整体含义

- **研究动机**：传统强化学习（RL）通过 Bellman 方程在封闭的马尔可夫决策过程（MDP）上定义动态规划，但这种视角隐藏了一个结构性事实——单步转移本质上是**开放组件**，需要将"延续"反馈回去才能得到无限时域价值。
- **背景**：现代决策系统由多个模块（感知、局部控制器、安全监控等）组装而成，但现有理论难以回答"局部替换在任意周围系统中是否有效"这类上下文等价性问题。
- **核心问题**：如何为折扣 Bellman 评估建立**组合语义**，使得开放组件可以像电路一样通过串行、并行和反馈布线组合，并且局部误差能在全局上下文中可控传播？

## 2. 论文提出的方法论

### 2.1 核心思想

- 将**单步决策过程**建模为 `Stoch`（Markov 范畴）中的开放随机组件，具有状态输入/输出接口、奖励通道和延续接口。
- **无限时域价值**被视为闭合收缩反馈环得到的 Banach 不动点，而非仅仅是方程的解。
- 折扣因子 γ ∈ (0, 1) 使得延续映射成为**一致收缩映射**，从而反馈操作被"守卫"（guarded），可以用 Banach 不动点唯一实现。

### 2.2 关键技术细节

- **类型化 Bellman 变换器**：给定开放组件 K: X ⊗ A → Y ⊗ R，固定策略 π_X 后，诱导的 Bellman 变换器为：
  ```
  T_K,π_X : B(Y) → B(X)
  (T_K,π_X V)(x) := E[ρ(r) + γV(y)]
  ```
  延续方向 V 活在输出状态 Y 上被拉回到输入状态 X。

- **串行布线**：对应普通函数复合（T_series = T1 ∘ T2），第二阶段的奖励被额外折扣一次（因子 γ）。
- **并行布线**：在独立假设下，保持可分可加结构，变换器满足 T_⊗(V1 ⊕ V2) = (T1 V1) ⊕ (T2 V2)。
- **反馈追踪**（Guarded Banach Trace）：
  - 反馈映射 f_π : I × B(S) → B(S) × B(S)，定义为 f_π(⋆, V) = (T_M,π V, T_M,π V)。
  - f_π 的反馈分量是 γ-收缩的，因此 f_π 是可允许的守卫 Banach 追踪。
  - 追踪操作输出唯一不动点 V^π。

- **上下文等价性**（Contextual Equivalence）：
  - 定义操作度量 d(T, T') := sup_{V∈B_M(Y)} ‖TV - T'V‖_∞。
  - 线性单孔上下文 C[·] 由语法 `C ::= [·] | T0 ∘ C | C ∘ T0 | C ⊗ T0 | T0 ⊗ C | Tr_Z(D)` 生成。
  - 上下文增益 L(C) 通过结构递归规则计算（合成乘以 Lipschitz 常量，张量取最大值，追踪放大因子 ≤ (1+β_Z)/(1-α_Z)）。
  - 定理：plugging 局部误差 ε 后，全局不动点偏差 ≤ L(C)/(1-κ(C)) · ε。

- **余代数抽象**：
  - MDP 同态 φ : S → bS 要求 reward 保持和转移 pushforward 一致。
  - 精确交换 → 通过算子纠缠证明价值保持（V^π = φ^* bV^bπ）。
  - 近似交换 → sup范数失真界：|V^π - φ^* bV^bπ|_∞ ≤ (ε_r + γV_max ε_P)/(1-γ)。

- **Quantale 契约层**：
  - 价值空间 V(S) = Q^S，Q 是 quantale（如 [0,∞] with ⊗ = +）。
  - 契约变换器 T : V(Y) → V(X) 单调且 ω-连续。
  - least fixed point trace（基于 Kleene 迭代）闭合反馈。
  - 定理：如果局部归纳不等式 T(C) ≤ C 成立，则 lfp(T) ≤ C（正确构造安全保证）。

## 6. 主要结论与发现

1. **Bellman 递归即守卫 Banach 追踪**：折扣 Bellman 评估的标准不动点恰好是闭合延续反馈环的守卫 Banach 追踪结果。
2. **类型化组合性**：串行布线对应函数复合，并行布线保持可分可加结构，反馈布线对应 least/greatest 不动点追踪。
3. **上下文等价性定理**：对于"线性单孔守卫上下文"（孔仅使用一次，所有反馈节点有统一守卫证书），局部操作误差在全局上下文中被明确控制（乘以 L(C)/(1-κ(C))）。
4. **余代数抽象误差传播**：精确交换确保价值保持；近似交换给出显式 sup范数失真界，该误差可进一步通过上下文等价性传播到任意电路。
5. **Quantale 契约提升**：单调 ω-连续契约变换器的局部归纳界可沿串行、并行和反馈布线组合提升，保证全局安全/风险/资源约束。
6. **扩展应用**：信念状态提升（将 POMDP 转为完全可观测 MDP）、模块化 off-policy 评估（真实分解下 Radon–Nikodym 导数的链式分解）、非平稳性追踪。

## 7. 优点

- **理论基础扎实**：严格建立在 Markov 范畴、Banach 不动点理论、Quantale 理论和余代数语义上。
- **视角统一**：将标准 Bellman 评估、组合布线、抽象和契约安全统一在同一个语义框架下。
- **显式误差传播**：所有误差界都是具体的、可计算的，而非存在性陈述。
- **模块化精神**：理论与实际系统（模块化决策系统）紧密结合，抽象误差可沿电路传播。
- **保守但合理的设计**：守卫追踪仅在收缩反馈环上定义，不声称全局 traced monoidal 范畴，避免了过度generalization。

## 8. 不足与局限

- **无实证验证**：纯理论工作，提出的框架缺乏在真实 RL 任务（如 Atari、MuJoCo）上的验证，实用性有待检验。
- **线性上下文限制**：上下文等价性定理仅适用于"孔使用一次"的线性上下文，非线性上下文（复制、合并、删除等）需要额外敏感性界。
- **守卫条件限制**：反馈环必须满足均匀收缩（uniform guardedness），不适用于无折扣、平均奖励或风险敏感设置。
- **独立性假设**：并行布线的可分性结果依赖于子系统间独立（product dynamics + product policy），实际系统中耦合很常见。
- **可扩展性未验证**：虽然理论上是模块化的，但未讨论在高维状态空间或大规模电路上的实际计算复杂度。
