## 摘要
基于大型语言模型（LLM）的多智能体系统依赖优化的协作拓扑结构，以平衡性能与通信成本。然而，现有方法在固有的稳定性-可扩展性权衡中陷入困境，且常常将计算预算与查询难度不匹配。我们提出 ATOM，这是一种自适应框架，通过一种新颖的任务驱动强化学习范式生成预算可控的协作图。受原子结构启发，ATOM 采用核-电子层级结构：在推理过程中保持离线学习得到的稳定协作主干（核），同时动态激活基于查询条件的智能体（电子）。关键在于，一种复杂度感知的预算策略通过估计查询难度来严格调控电子的实例化，从而将资源消耗与任务需求对齐。在六个不同基准上的大量实验表明，ATOM 在保持最先进性能的同时，与强基线相比将令牌效率提升最高达 30%。

## Abstract
Large Language Model (LLM)-based multi-agent systems rely on optimized collaboration topologies to balance performance and communication costs. However, current methods struggle with the inherent stability-extensibility trade-off and often misalign computational budgets with query difficulty. We propose \textsc{ATOM}, an adaptive framework that generates budget-controllable collaboration graphs via a novel task-driven reinforcement learning paradigm. Inspired by atomic structures, \textsc{ATOM} employs a nucleus-electron hierarchy: it maintains a stable, offline-learned collaboration backbone (the nucleus) while dynamically activating query-conditioned agents (electrons) during inference. Crucially, a complexity-aware budgeting strategy aligns resource consumption with task demands by estimating query difficulty to strictly regulate electron instantiation. Extensive experiments across six diverse benchmarks demonstrate that \textsc{ATOM} achieves state-of-the-art performance while improving token efficiency by up to $30\%$ compared to strong baselines.

# ATOM 论文详细总结

## 1. 论文的核心问题与整体含义

### 研究背景
- 大型语言模型（LLM）驱动的多智能体系统（MAS）已成为解决复杂推理任务的重要范式
- MAS 的有效性高度依赖其协作拓扑结构——决定智能体组织方式和信息路由的图结构
- 单智能体因有限的专业知识和推理深度，难以应对复杂问题

### 核心问题
论文识别出当前 MAS 拓扑设计面临的两个根本性挑战：

1. **稳定性-可扩展性权衡（Stability-Extensibility Trade-off）**
   - 静态手绘模板（如 Chain、Star 结构）虽可靠但缺乏灵活性
   - 纯生成式方法需在指数增长的搜索空间中导航，结构易波动

2. **预算错配（Budget Misalignment）**
   - "平均复杂度陷阱"：现有方法无法根据查询难度自适应调整资源
   - 简单查询过度分配资源（浪费），复杂查询资源不足
   - 固定密集连接拓扑对所有任务都效率低下

### 研究动机
需要一个能同时满足**有效性（effective）、高效性（efficient）、可执行性（executable）**的协作拓扑生成框架。

## 2. 论文提出的方法论

### 核心思想：核-电子层级结构（Nucleus-Electron Hierarchy）
受原子结构启发，将 MAS 分为两层：
- **核（Nucleus）**：离线学习得到的稳定协作主干，持久激活
- **电子（Electrons）**：推理时动态激活的查询条件智能体，按需扩展

### 关键技术细节

#### 2.1 离线阶段：核骨干构建

**目标**：在推理前学习紧凑的领域特定核骨干，缩小组合搜索空间

**流程**：
1. **随机边参数化**：在完全连接的超级节点上定义空间边 $E^{(S)}$ 和时序边 $E^{(T)}$ 的可学习 logit $\theta^{(S)}_{i,j}$、$\theta^{(T)}_{i,j}$
2. **结构优化目标**（公式 3）：使用 REINFORCE 算法最大化任务效用 $U$，同时惩罚稀疏性
3. **拓扑凝聚**：收敛后，用**期望度中心性（EDC）**评估每个智能体的结构重要性：
   $$S_{node}(v_i) = \sum_j (\rho^{(S)}_{i,j} + \rho^{(S)}_{j,i})$$
4. **分区**：选择 Top-$K_{nuc}$ 最高 saliency 的智能体构成核 $V_{nuc}$，其余进入电子池 $V_{elec}$

#### 2.2 在线阶段：电子激发

**2.2.1 复杂度感知预算（Complexity-Aware Budgeting）**

- **复合特征向量** $x_q = [\text{WPCA}(f_{enc}(q)) \parallel \Phi(q)]$，其中 $\Phi(q)$ 量化符号和逻辑密度
- **域门控机制**：防止域混淆
- **复杂度评分**（公式 6）：
  $$\bar{C}(q) = \text{MLP}_\theta(x_q \odot g_d)$$
- **预算分配**：$K(q) = \lfloor K_{max} \cdot \bar{C}(q) \rfloor$

**2.2.2 动态激活与实例化**

1. **查询条件状态表示**（公式 8）：通过轻量级门控将任务语义注入智能体特征
2. **兼容性评分**（公式 9）：结合关系分数和语义对齐
3. **约束采样**：使用动态规划精确从条件伯努利分布采样，避免 Top-K 截断导致概率失真
4. **关系拓扑合成**：为活跃对生成双通道（空间+时序）路由 logit

**2.2.3 任务驱动 RL 训练**

- **复合奖励**（公式 11）：$R(G) = r_{task} - \lambda_{cost}(|E_{active}|/|E_{max}|)$
- **DAG 约束采样**：空间边采样需保证无环（防止死锁）
- **结构层次正则化**（公式 12）：双边界损失，确保核中心通信优于电子间通信

## 6. 论文的主要结论与发现

### 核心结论

1. **SOTA 性能**：ATOM 在所有 6 个基准上均达到最优，准确率平均提升 **7.53%**
   - MMLU：72.55%（vs ARG-Designer 67.32%，+5.23%）
   - GSM8K：84.00%（vs ARG-Designer 83.87%）

2. **高 Token 效率**：性能提升的同时，最高节省 **30%** Token 消耗
   - HumanEval：~3.7×10⁵ vs ARG-Designer ~5.3×10⁵

3. **复杂度-资源强相关**：Pearson r = 0.737，证明预算机制有效

4. **强鲁棒性**：Prompt Injection 攻击下性能下降仅 **6.4%**（vs 其他方法 9.7%~25.5%）

### 消融实验关键发现

| 组件 | 影响 |
|------|------|
| 核识别（Nucleus Iden.） | 移除导致 MMLU 下降 3.92% |
| 结构损失（Str. Loss） | 移除导致 MMLU 下降 8.50% |
| 成本惩罚（Cost Penalty） | 移除导致 GSM8K Token 翻倍 |
| 自适应预算（Adaptive Budget） | 固定/随机预算均降低准确率并增加消耗 |

1. **创新性层级架构**
   - 核-电子范式从根本上解决稳定性-可扩展性权衡
   - 受原子结构启发的设计具有物理直观性

   - 首次将查询难度估计与资源分配显式对齐

   - 防止电子通信主导核通信
   - 端到端 RL 训练稳定

### 实验设计亮点

1. **双维度 Pareto 分析**：同时评估准确率和 Token 消耗
2. **全面的基线覆盖**：从单智能体到复杂 MAS 均有对比
3. **对抗鲁棒性验证**：增强方法可信度
4. **案例可视化**：直观展示自适应拓扑行为
