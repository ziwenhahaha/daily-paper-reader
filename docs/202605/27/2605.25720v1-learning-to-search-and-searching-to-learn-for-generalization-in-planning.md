## Abstract
Combinatorial generalization remains a central challenge in Deep Reinforcement Learning (DRL). Classical planning provides a simple yet challenging setting to study this problem through explicit relational descriptions, without requiring learning from perception. In sparse-reward domains, standard RL exploration via real-time search is ineffective, and learning-based planning methods often rely on expert demonstrations, hindsight relabeling, or random walks from the goal state. In contrast, planners rely on best-first search methods such as $\mathrm{A}^\star$ to solve problems from scratch. We propose a self-improving $\mathrm{WA}^\star$ learning framework in combination with a value heuristic represented by a Relational Graph Neural Network: the heuristic guides search, and the resulting search data updates the heuristic via $Q$-learning. This loop yields heuristics that can function as general policies and solve new instances even without search, where DRL otherwise fails, as we show on puzzles such as Sokoban, PushWorld, The Witness, and the 2023 International Planning Competition benchmarks. Notably, we demonstrate strong zero-shot generalization: For example, heuristics trained on Blocksworld instances with fewer than 30 blocks successfully solve instances with 488 blocks without search.

## 1. 核心问题与整体含义

### 研究动机
- **组合泛化（Combinatorial generalization）** 是深度强化学习（DRL）的核心挑战：学到的策略或价值函数需要在新环境中泛化，但训练数据有限
- **稀疏奖励问题**：在目标导向的经典规划中，标准RL的实时搜索（real-time search）无法有效探索，因为奖励极度稀疏
- **现有方法的局限**：基于学习的规划方法往往依赖专家演示、后见重标记（HER）或随机游走，缺乏真正的自主探索能力

### 研究背景
- 经典规划提供了一种理想的研究设置：问题结构明确（通过关系描述），不需从像素级感知中学习
- 规划领域（domain）定义固定的关系词汇和动作模式，实例（instance）在对象数量、初态和目标上变化
- 这种清晰的"领域-实例"分离产生了系统性的分布外泛化挑战

### 核心问题
如何让强化学习在模型已知的稀疏奖励规划任务中，自主学习可泛化到不同规模、状态和目标的策略或启发式？

## 2. 论文提出的方法论

### 核心思想
提出**广义搜索规划框架（GSP）**，构建一个自改进的学习循环：
> 启发式引导搜索 → 搜索产生数据 → 数据更新启发式 → 更强的启发式引导更高效的搜索

### 关键技术：关系图神经网络（Relational GNN）表示 Q 函数

**状态与动作的表示**：
- 规划状态 $s$ 是真原子 $p(\bar{o})$ 的集合
- 目标 $g$ 是文字的合取
- 将状态和目标中涉及的原子合并为关系集 $R_{s,g}$
- 为每个适用的具体动作 $a = A(\bar{o}) \in A(s)$ 引入专门的动作对象 $o_a$，形成关系 $R_A$

**消息传递网络**：
- 每个对象 $o$ 维护嵌入向量 $X_i(o) \in \mathbb{R}^d$
- 通过 $L$ 层消息传递更新：沿着原子交换信息
- 每个原子 $q = p(o_1, ..., o_{ar(p)})$ 产生位置级消息
- 对象聚合所有包含它的原子消息
- 更新函数：$X_{i+1}(o) = X_i(o) + \text{Comb}_U(X_i(o), m_o)$

**Q值输出**：
- 池化所有嵌入得到 $\bar{X}(s,g)$
- 对动作 $a$，其 Q 值为：$Q_\theta(s,a) = \text{MLP}_Q([X_L(o_a) \| \bar{X}(s,g)])$

### 搜索流程（Algorithm 1）

1. **初始化**：从初始状态 $s_0$ 出发，为所有适用动作创建状态-动作对，加入优先级队列 $F$
2. **评分函数**：$f(s,a) = g(s) + w \cdot Q_\theta(s,a)$，其中 $g(s)$ 是沿路径的累积成本
3. **扩展循环**：
   - 弹出最高优先级的 $(s,a)$
   - 生成后继状态 $s' = a(s)$
   - **三种情况**：
     - **非终止**：将新的 $(s',a')$ 加入搜索树和队列
     - **死端**：存储 $(s,a,R_\perp)$ 到重放缓冲区
     - **目标**：回溯解路径，为路径上的每个 $(s,a)$ 分配实际返回下界 $R$
4. **终止条件**：达到目标状态或耗尽扩展预算

### Q-learning 更新

从重放缓冲区采样 $(s,a,R)$，计算 bootstrap 目标：
$$\hat{y}(s,a) = -1 + \max_{a' \in A(s')} Q_\theta(s', a')$$

- **非终止对**：$y = \hat{y}(s,a)$
- **死端对**：$y = R_\perp$
- **目标路径对**：$y = \max\{R, \hat{y}(s,a)\}$（搜索找到的解提供下界）

### 实例选择策略

维护三个实例池，以指数增长权重采样：
- **unsolved**：尚未解决的实例（最高优先级）
- **satisficed**：找到解但非最优的实例
- **solved**：扩展节点数等于解长度的实例（启发式已足够自信）

## 6. 主要结论与发现

### 核心发现

1. **零样本强泛化能力**
   - 在 Blocksworld 上：训练实例少于 30 个方块，却能解决 488 个方块的测试实例
   - Sokoban (10×10)：GSP WA⋆ 解决所有 1000 个测试实例，平均仅 207 次扩展

2. **搜索比实时搜索更有效**
   - GSP π 在多个领域优于 LIFTED HER：
     - Sokoban：681 vs 309（覆盖率）
     - Blocksworld：100% vs 98%
   - WA⋆ 引导的搜索大幅减少扩展次数（相比无启发式搜索）

3. **作为启发式 vs 作为策略的权衡**
   - Q 函数作为**局部动作排序机制**强于作为**全局启发式**
   - 贪心执行在某些领域表现更好（如 Blocksworld 100% vs WA⋆ 79%）
   - WA⋆ 在分支因子极大的领域（如 Childsnack）仍可提升覆盖率

4. **跨领域统一性能**
   - 使用相同架构和超参数，在 IPC 基准、谜题域、PushWorld 上都竞争力强
   - PushWorld Level 0→Level 1 迁移成功（48/63），且扩展次数比 LAMA 少约 30 倍

5. **各组件的贡献**
   - 死端监督影响最小（仅几个 IPC 领域有死端）
   - 解路径下界目标贡献最大
   - 优先级采样策略有助于稳定性

### 次要发现

- AlphaZero（MCTS 驱动）在该设置下泛化不足，可能因为 MCTS 不适合单目标寻路
- Spanner 领域存在"次优贪心策略陷阱"：存在高度可泛化的次优策略，但学最优策略导致训练不稳定
- 1-WL 表达力限制影响 Rovers 和 Satellite 领域（所有学习方法均弱于 LAMA）

2. **关系归纳偏置**
   - R-GNN 天然捕获规划域的关系结构
   - 动作选择在上下文中进行（直接比较同状态的所有动作）
   - 支持不同规模实例的零样本泛化

3. **搜索数据的有效利用**
   - 不仅从成功轨迹学习，还从死端状态学习
   - 搜索找到的解提供有价值的下界，防止 bootstrap 目标下降

4. **灵活的测试模式**
   - 同一模型可作为贪心策略（GSP π）或搜索启发式（GSP WA⋆）
   - 适配不同性质的领域

### 实验设计亮点

1. **大规模基准覆盖**
   - IPC 2023 学习基准是标准学术基准
   - 谜题域和 PushWorld 增加多样性

2. **公平的对比设计**
   - 批量大小调整后比较
   - 经典启发式在相同框架下对比
   - 多次随机种子体现稳健性

## 8. 不足与局限

### 实验覆盖局限

1. **部分领域失败**
   - 24-Puzzle 完全无法解决（0/1000）
   - Floortile 领域表现不佳（20-28% 覆盖率）
   - AlphaZero 基线可能未充分调优

2. **1-WL 表达力限制**
   - Rovers 和 Satellite 领域所有学习方法均弱于 LAMA
   - 这是方法本身的理论局限，非实验问题

3. **批量大小影响**
   - 表3 与表6 的扩展次数比较不完全公平（基线用 b=32，GSP 用 b=1）
   - 虽然做了调整，但未与所有基线在 b=32 下比较

### 偏差风险

1. **训练数据依赖**
   - 依赖于训练实例的分布质量
   - Spanner 领域的"次优陷阱"表明训练目标可能与泛化目标冲突

2. **实例选择策略的隐含假设**
   - "satisficed" 和 "solved" 的定义依赖启发式质量
   - 可能导致在某些领域的选择偏差

### 应用限制

1. **模型必须已知**
   - 适用于经典规划（模型已知），不直接适用于现实世界交互任务
   - 无法处理部分可观测或随机环境

2. **计算开销**
   - 需要大量搜索和训练时间（12小时/实验）
   - 实时应用可能受限

3. **跨域迁移**
   - 训练和测试必须在同一领域内
   - 无法处理领域变化

4. **全局校准不足**
   - Q 函数作为局部排序器优于全局启发式
   - 在大状态空间或高分支因子下可能导致 WA⋆ 失败

### 未来方向建议

- 改进值的跨状态泛化能力
- 结合最佳优先搜索与实时搜索/HER 的互补优势
- 将学习启发式与经典符号启发式（如 hff）结合
