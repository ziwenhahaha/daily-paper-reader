## 摘要
自蒸馏通过在模型自身的生成上进行训练来引导大型语言模型（LLMs）。然而，现有方法要么依赖外部信号来筛选自生成输出（例如正确性过滤、执行反馈和奖励搜索），这些信号成本高昂且在表现最佳的尖端模型上不可用；要么完全跳过筛选，直接在所有原始输出上进行训练，这种方法往往是领域特定的，难以泛化。两者还有一个更深层的弱点：自生成输出将任务相关的能力与其他因素（如风格模式、格式伪影和模型特定的错误）纠缠在一起，从而稀释了针对特定能力的改进信号。本文提出自策略蒸馏（SPD），在不依赖任何外部信号的情况下实现可泛化、选择性的能力提升。具体而言，SPD从模型在正确性定义 token 上的梯度中提取低秩能力子空间，在自生成阶段将键值（KV）激活投影到该子空间，并使用标准的下一个 token 预测损失对得到的原始输出进行微调。在代码生成、数学推理和多选问答等多个任务上的广泛实验表明，SPD在不使用外部信号的前提下，相比最先进的自蒸馏方法提升高达13%，相比预训练基线提升高达16%。值得注意的是，SPD在域外泛化设置下表现出色，性能提升约15%。

## Abstract
Self-distillation bootstraps large language models (LLMs) by training on their own generations. However, existing methods either rely on external signals to curate self-generated outputs (e.g., correctness filtering, execution feedback, and reward search), which are costly and unavailable for the best-performing frontier models, or skip curation entirely and train on all raw outputs, an approach that is often domain-specific and hard to generalize. Both also share a deeper weakness that self-generated outputs entangle task-relevant capability with others, such as stylistic patterns, formatting artifacts, and model-specific errors, diluting the signal for the specific capability one aims to improve. In this paper, we propose Self-Policy Distillation (SPD), which achieves generalizable, capability selective without any external signal. Specifically, SPD extracts a low-rank capability subspace from the model's own gradients on correctness-defining tokens, projects key-value (KV) activations into this subspace during self-generation, and fine-tunes on the resulting raw outputs with standard next-token prediction loss. Through extensive experiments across code generation, mathematical reasoning, and multiple-choice QA, we show that SPD achieves up to 13% improvement over state-of-the-art self-distillation methods without external signals and up to 16% improvement over pre-trained baselines. Notably, SPD demonstrates superior generalizability, achieving 15% better performance under out-of-domain generalization settings.

## 2. 论文方法论

### 核心思想
提出 **Self-Policy Distillation (SPD)**，通过两阶段流程实现无需外部信号的能力选择性自蒸馏：
1. 从模型自身梯度中提取低秩能力子空间
2. 在自生成阶段将 KV 激活投影到该子空间以过滤干扰信息
3. 基于标准语言建模损失微调生成的输出

### 关键技术细节

#### 阶段一：能力子空间提取（Capability Subspace Extraction）

**正确性对齐损失（Correctness-aligned Loss）**：
- 仅在"正确性定义 token"上计算损失，而非整个序列
- 代码生成：使用 assertion 相关 token（5-20 个）
- 数学推理：使用最终数值答案 span（1-3 个 token）
- 多选问答：使用 "Answer:" 后的答案字母（1 个 token）

公式表达：
$$\mathcal{L}_{\text{align}}(q^{(i)}, y^{(i)}) = -\frac{1}{|S^{(i)}|} \sum_{t \in S^{(i)}} \log p_{\theta_{\text{old}}}\left(z_t^{(i)} \mid z_{<t}^{(i)}\right)$$

**梯度收集与 SVD 投影**：
- 对每个目标层 ℓ ∈ L，收集 K 和 V 激活关于对齐损失的梯度
- 构造成梯度矩阵：$G_K^{(ℓ)} \in \mathbb{R}^{M \times d_k}$，$G_V^{(ℓ)} \in \mathbb{R}^{M \times d_v}$
- 对梯度矩阵进行 SVD 分解，保留 top-r 右奇异向量
- 构建投影矩阵：$P_K^{(ℓ)} = V_{K,r}^{(ℓ)}(V_{K,r}^{(ℓ)})^\top$

#### 阶段二：能力选择性蒸馏（Capability-Selective Distillation）

**投影钩子（Projection Hooks）**：
- 在自回归生成时，于目标层插入投影钩子
- 投影操作：$\tilde{K}^{(ℓ)} = K^{(ℓ)}P_K^{(ℓ)}$，$\tilde{V}^{(ℓ)} = V^{(ℓ)}P_V^{(ℓ)}$
- 模型参数不变，仅过滤中间 KV 激活

**自生成与微调**：
- 使用 hooked 模型为每个训练 prompt 生成一个 completion
- 移除所有钩子，使用 LoRA 在生成的 (prompt, completion) 对上微调
- 标准下一 token 预测损失：$\min_{\Delta\theta_{\text{LoRA}}} -\sum_{(q,\hat{y}) \in \mathcal{D}_{\text{corpus}}} \sum_{t=2}^{|z|} \log p_{\theta_{\text{old}}+\Delta\theta_{\text{LoRA}}}\left(z_t \mid z_{<t}\right)$

### 算法流程
```
Phase 1: 能力子空间提取
├── 对每个目标层和激活类型（A ∈ {K, V}）
│   ├── 计算正确性对齐损失
│   ├── 收集 token 级梯度
│   ├── SVD 分解梯度矩阵
│   └── 构建投影矩阵 P_A^{(ℓ)}

Phase 2: 能力选择性蒸馏
├── 应用投影钩子
├── 为每个 prompt 生成 completion
├── 构建自生成语料库
├── 移除钩子
└── LoRA 微调
```

5. **数据质量改善**：SPD 生成的数据在微调前就已经比基线更集中、更有针对性

## 7. 优点
