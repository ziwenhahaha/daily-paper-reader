## Abstract
Moving beyond simple scalar rewards toward richer world feedback is a natural path to more scalable RL post-training. On-policy self-distillation (OPSD) is a promising recent approach that uses arbitrary feedback as learning signal, yet its reliability compared to established methods, such as GRPO, remains unclear. We identify a strikingly consistent linear correlation between the initial student-self-teacher performance gap and the final performance improvement in OPSD. This relationship holds across context types and model families, providing a powerful predictive law for anticipating the outcome of an OPSD configuration without running the full training procedure. Interestingly, we show that this linear predictability holds with model scale, suggesting a potential basis for new empirical scaling laws on larger models with stronger in-context learning capabilities. In essence, our findings show that OPSD performance can be predicted and tuned before training, offering a principled way to incorporate world feedback as a first-class component of the post-training pipeline.

### 1. 核心问题与整体含义（研究动机与背景）

- **背景**：强化学习后训练（如 GRPO 等 RLVR 方法）是大语言模型提升推理能力的关键组件，但传统 RLVR 依赖简单的标量奖励（二值结果），难以利用丰富的世界反馈（如代码运行报错、单元测试反馈、环境交互信息等）。
- **问题**：On-Policy Self-Distillation（OPSD）是一种新兴方法，使用任意反馈作为学习信号，但与 GRPO 等成熟方法相比，其可靠性尚不明确。OPSD 需要特权上下文（privileged context）来构建自教师（self-teacher），但如何提前评估不同上下文配置的效果、避免昂贵的完整训练，是一个关键难题。
- **核心发现**：作者发现了一个显著且一致的**线性定律**：初始的学生-自教师性能差距（student–self-teacher gap）与最终学生的性能提升之间存在强线性相关。该定律可跨上下文类型和模型家族泛化，从而实现对 OPSD 配置效果的提前预测，无需运行完整训练流程。

### 2. 方法论：核心思想、关键技术细节与算法流程

#### 核心思想

- OPSD 的核心是将模型自身作为教师（self-teacher），通过特权上下文 `c` 激活更强的行为表现，再将这种能力蒸馏到学生模型中。
- 自教师与学生共享权重（通过 EMA 缓慢跟随学生），但通过 `c` 的条件化，利用上下文学习（in-context learning）在 token 级别产生密集监督信号，而非依赖外部奖励模型。

#### 关键公式

OPSD 目标函数为基于学生采样轨迹的逐 token 逆 KL 散度：

$$L_{\text{OPSD}}(\theta; c) = \mathbb{E}_{x \sim \mathcal{D},\, y \sim \pi_\theta(\cdot|x)} \left[ \frac{1}{|y|} \sum_{t} \text{KL}\left( \pi_\theta(\cdot | y_{<t}, x),\ \text{stopgrad}\left[ \bar{\pi}_\theta(\cdot | y_{<t}, x, c) \right] \right) \right]$$

- `πθ` 为学生策略，`π̄θ` 为自教师（共享权重，EMA 更新）
- `stopgrad` 阻断教师侧的梯度回传
- 特权上下文 `c` 可以包含世界反馈、同伴解决方案、专家前言等多种信息源

#### 特权上下文配置

论文研究了六种不同的特权上下文构造方式：

| 配置名称 | 内容描述 |
|---|---|
| None（对照） | 无特权信息 |
| Expert Preamble | DS/算法清单、时间复杂度表、模式规则、边缘情况、9步方法论 |
| Feedback | 来自失败尝试的环境反馈 |
| LLM Peer Hints | 从同伴 rollout 中提取的正确解决方案提示 |
| Own Solution + Feedback | 若当前 rollout 正确则提供自身解答；否则提供环境反馈 |
| Peer Solution + Feedback | 提供同伴正确解答 + 环境反馈 |

### 3. 实验设计：数据集、场景与对比

- **数据集**：LiveCodeBench v6（2025 年 2–5 月的编程题目），用于评估代码生成与推理能力。
- **评估指标**：mean@4 pass rate（每个题目采样 4 次，统计通过率）。
- **模型家族**：
  - **Qwen3 系列**：`0.6B`、`1.7B`、`4B`、`8B`（非思考模式）
  - **Olmo-3-7B-Instruct**
- **实验场景**：
  - 在两个模型家族上分别测试六种特权上下文配置，验证线性预测定律的泛化性
  - 在 Qwen3 全系列（4 种规模）上固定使用 "Peer Solution + Feedback" 配置，验证定律的尺度不变性
- **对比方法**：实验主要聚焦于 OPSD 本身在不同上下文配置下的对比，未与 GRPO 等外部强化学习方法进行直接性能对比。

### 6. 主要结论与发现

1. **线性预测定律**：初始的学生-自教师性能差距与最终性能提升之间存在强线性相关（Qwen3-8B: Pearson=0.974, Spearman=0.886; Olmo-3-7B: Pearson=0.998, Spearman=1.0）。
2. **跨类型泛化**：该线性关系在所有六种不同性质的特权上下文配置下均成立，说明差距本身是一个足够统计量，不依赖具体的反馈机制。
3. **尺度不变性**：线性预测定律在 Qwen3 全系列（0.6B–8B）上保持一致，表明该规律可推广至更大规模模型。
4. **收敛动态**：训练过程中学生验证准确率稳步接近自教师水平，且自教师能力本身也随训练提升（与前人 OPSD 研究一致）。
5. **实践意义**：可在训练前通过测量初始差距筛选最优特权上下文配置，避免昂贵的后训练试错。

### 7. 优点

- **创新性地提出预测定律**：首次揭示了 OPSD 中初始性能差距与最终改进之间的可量化线性关系，为 OPSD 配置选择提供了低成本预测工具。
- **理论与实践结合**：不仅报告相关性，还通过留一交叉验证证明了预测的泛化能力和可靠性（R² 和 RMSE 指标优异）。
- **实验设计严谨**：跨模型家族（Qwen3、Olmo3）、跨规模（0.6B–8B）、跨上下文类型（6 种不同信息源）的多维度验证，大幅提升了结论的可信度。
- **工具化潜力**：提供了开源代码（GitHub: opsd-predictive-law），便于社区复现和进一步研究。
- **对 OPSD 领域的重要补充**：为此前依赖试错的 OPSD 配置设计提供了原则性的指导框架，有望成为 OPSD 实践的标准步骤。

### 8. 不足与局限

- **基准任务单一**：实验仅在 LiveCodeBench（代码任务）上进行，结论在其他任务（如数学推理、多跳问答等）上的泛化性未经验证。
- **缺乏与 RLVR 方法的直接对比**：论文主要聚焦于 OPSD 内部的上下文配置对比，未与 GRPO 等成熟 RLVR 方法进行性能对比，难以判断 OPSD 的相对优劣。
- **模型规模受限**：虽然验证了 0.6B–8B 的尺度不变性，但更大模型（如 14B 以上）的行为未纳入（作者明确指出需要单独调参），限制了结论向超大模型的外推。
- **超参数固定**：所有实验使用统一的超参数（学习率 1e-6、50 步训练等），未探索不同超参数设置下预测定律的鲁棒性。
- **长时训练效果未知**：仅训练 50 步，线性预测定律在更长训练周期下是否仍然成立尚不清楚。
- **特权上下文设计的局限**：六种配置虽覆盖了不同信息源，但覆盖范围有限，且对何种上下文类型在何种场景下最优缺乏深入的理论解释。
