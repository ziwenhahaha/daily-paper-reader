## Abstract
The Policy-Space Response Oracles (PSRO) framework scales equilibrium computation to large zero-sum games by iteratively expanding a restricted strategy set using deep reinforcement learning (DRL). A central challenge is to construct, under limited computational budgets, a small strategy population whose induced game well approximates the full game. Existing PSRO variants typically expand the population using best responses to meta-strategies computed from restricted-game payoffs, which can lead to inefficient expansions that provide limited global improvement. We propose to guide population expansion by directly evaluating the post-expansion population quality. Specifically, we adopt Population Exploitability (PE) to measure how well a restricted strategy set represents the full game, and introduce a two-phase exploration--selection framework that explicitly minimizes PE during expansion. We instantiate this framework as Global PSRO, a practical DRL-based algorithm that efficiently generates candidate responses and estimates PE via parameter-sharing conditional neural networks. Experiments across multiple two-player zero-sum games show that Global PSRO achieves lower exploitability and approximates Nash equilibria with significantly fewer policy iterations than prior PSRO methods.

- **研究问题**：在大规模两人零和博弈中，如何在有限计算预算下高效构建一个小型策略种群（restricted strategy set），使其诱导的受限博弈能够良好近似完整博弈，从而实现均衡计算。

- **背景**：
  - PSRO（Policy-Space Response Oracles）框架通过迭代扩展受限策略集并使用深度强化学习（DRL）训练最佳响应（BR）来计算均衡。
  - 现有PSRO变体通常基于受限博弈的收益矩阵计算元策略（meta-strategy），然后训练针对该元策略的最佳响应来扩展种群。
  - 这种方法的问题是：受限博弈仅是完整博弈的局部近似，针对其均衡的BR可能局部有效但对全局改进贡献有限，甚至在最坏情况下需要添加几乎整个策略空间才能收敛。

- **核心洞察**：论文指出，基于受限博弈信息的种群扩展是结构性的低效根源，必须引入全局信息来引导扩展方向。

### 2. 论文提出的方法论

#### 2.1 核心思想

- **优化目标**：直接以扩展后种群的**Population Exploitability（PE）**最小化为准则选择新策略。
  - PE定义为：$PE(\Pi^r; G) = \min_{\sigma \in \Delta(\Pi^r)} \epsilon(\sigma)$，即在种群$\Pi^r$上能取得的最小可利用性。
  - 选择策略：$\pi^* \in \arg\min_{\pi \in B(\Delta(\Pi^r))} PE(\Pi^r \cup \{\pi\}; G)$

- **两阶段探索-选择框架**：
  - **探索阶段（Exploration）**：生成多样化的候选响应集合。
  - **选择阶段（Selection）**：评估每个候选扩展后的PE，选择使PE最小化的候选。

#### 2.2 关键技术细节

- **探索阶段**：
  - 构建元策略池 $S_t = \{\sigma_k\}_{k=1}^K$，包含基线RGB-MSS产生的元策略 + 从Dirichlet(1)均匀采样的混合策略。
  - 训练单一条件探索器 $\pi_\theta(a|s, \sigma)$，通过参数共享同时生成针对多个$\sigma_k$的BR响应。
  - 目标函数：$\max_\theta J(\theta) = \frac{1}{K}\sum_{k=1}^K U(\pi_\theta(\cdot|\sigma_k), \sigma_k)$

- **选择阶段**：
  - 使用RM-BR（Regret Minimization + Best Response）程序估计每个候选$\Pi^+_k = \Pi^r \cup \{\pi_k\}$的PE。
  - 采用条件评估器 $\beta_\psi(a|s, \sigma)$ 进行参数共享的BR计算。
  - **正则化评分**：$cPE_{reg,k} = (1-\hat{p}_k)[cPE(\Pi^r; G) - U(\beta_k, \rho_r)] + cPE_k$
    - 其中$\hat{p}_k = \rho_k(\pi_k)$表示新策略在混合中的概率质量，避免新策略获得极低概率时的误导性评估。
  - 选择：$k^* \in \arg\min_k cPE_{reg,k}$

- **扩展规则**：每轮将选中的候选响应$\pi_{k^*}$及其对应的PE评估BR $\beta_{k^*}$ 同时加入种群。

#### 2.3 理论分析

- **定理3.2**：证明了任何基于受限博弈的MSS（RGB-MSS）都存在最坏情况行为——可能需要纳入几乎所有纯策略才能收敛。
- **定理3.3**：展示了即使对于使某RGB-MSS表现最差的博弈，存在一种实例特定的MSS可以在$S+2$次迭代内达到均衡。
- **定理4.2**：证明了所提框架在精确预言机下保持与基线RGB-MSS相同的有限收敛保证。
- **定理4.3**：给出了针对敌对博弈族的期望迭代复杂度上界：$E[T^*] \leq \min\{2 + \frac{2S}{1-(1-c)^{K-1}}, N-1\}$。

### 6. 主要结论与发现

- **Global PSRO在几乎所有游戏中实现了更低的PE**，优于所有基线PSRO变体。
- **迭代效率显著提升**：在匹配的环境交互预算下，Global PSRO能够用更少的策略迭代逼近纳什均衡。
- **在大规模博弈中优势更突出**：如Goofspiel (13 cards)，最终PE约为0.046，而多数基线方法仍在0.13-0.37之间。
- **参数共享的有效性**：与NeuPL相比，Global PSRO表现更优，说明改进主要来自选择机制而非参数共享。
- **各组件均有贡献**：消融实验确认探索、多候选评估和PE正则化都是性能提升的关键因素。
- **多样性可进一步改善**：在Leduc Poker等游戏中，融入多样性目标的Global PSRO表现更好。

### 8. 不足与局限

- **适用范围有限**：论文聚焦于两人零和博弈，多人一般和博弈的扩展存在根本性挑战（PE不再具有鞍点结构）。
- **PE估计误差**：尽管有正则化机制，近似PE仍可能误导选择，特别是在早期迭代中。
- **计算开销**：选择阶段需要额外训练RM-BR评估器，相比标准PSRO增加了一定计算成本。
- **假设条件**：理论分析基于精确预言机假设，实际DRL训练中近似BR可能影响性能。
- **游戏类型**：主要测试了扑克类和Goofspiel，未涵盖实时策略游戏等更复杂的博弈场景。
- **超参数敏感性**：探索池大小K、Dirichlet采样参数等可能需要针对不同游戏调优。
