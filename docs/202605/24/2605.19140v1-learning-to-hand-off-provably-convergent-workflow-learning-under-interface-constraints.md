## 摘要
我们研究一种工作流学习设置，其中专业化智能体通过共享产物交接控制权，每个智能体只观察该产物的局部函数和自身私有状态，且没有集中式学习器访问联合轨迹。这正是跨组织、供应商或信任边界的多智能体 LLM 流水线的运行状态。我们将这一状态形式化为接口约束半马尔可夫决策过程（IC-SMDP），其决策时刻发生在交接时间，并设计 IC-Q，这是一种异步去中心化 Q-learning 算法，其中每次交接时的跨智能体协调恰好是一个标量。主要结果是神经 IC-Q 的有限样本界，该界在随机 option-duration 折扣下分解为三个可独立控制的误差来源：神经函数近似误差、接口表示差距和混合时间残差。建立该界需要将近似信息状态（AIS）框架从单智能体原始步 MDP 提升到多智能体 SMDP，并控制随机持续时间下的马尔可夫噪声，而这些在先前工作中都尚未完成。据我们所知，这是去中心化部分可观测条件下神经 Q-learning 的首个有限样本保证。四个实验，包括验证界逐项成立的受控合成 IC-SMDP、多 LLM 数学推理、多智能体路由和多智能体 CPU 编程，表明 IC-Q 在没有任何智能体观察联合轨迹的情况下匹配集中式 oracle，并且三个误差来源都按界所预测的对应轴缩放。

## Abstract
We study workflow learning in a setting where specialized agents hand off control through a shared artifact, each agent observes only a local function of that artifact and its own private state, and no centralized learner accesses joint trajectories -- the operating regime of multi-agent LLM pipelines that span organizational, vendor, or trust boundaries. We formalize this regime as an interface-constrained semi-Markov decision process (IC-SMDP), whose decision epochs occur at handoff times, and design IC-$Q$, an asynchronous decentralized $Q$-learning algorithm in which cross-agent coordination at every handoff is exactly one scalar. Our main result is a finite-sample bound for neural IC-$Q$ that decomposes into three independently controllable error sources: neural function-approximation error, interface representation gap, and a mixing-time residual, under the random option-duration discount. Establishing this bound requires lifting the approximate information state (AIS) framework from single-agent primitive-step MDPs to multi-agent SMDPs and controlling Markovian noise under random duration, neither of which has been done in prior work. To our knowledge this is the first finite-sample guarantee for neural $Q$-learning under decentralized partial observability. Four experiments: a controlled synthetic IC-SMDP that validates the bound term-by-term, multi-LLM mathematical reasoning, multi-agent routing, and multi-agent CPU programming, show that IC-$Q$ matches a centralized oracle without any agent observing joint trajectories, with each of the three error sources scaling along its corresponding axis as the bound predicts.

## 1. 核心问题与整体含义

**研究动机：**
- 现代多智能体 LLM 系统（如 MetaGPT、AgentCoder、AutoGen）通过流水线式架构协调专业化智能体（规划者、编码者、测试者等）
- 当这些流水线跨越组织、供应商或信任边界时，**中心化假设失效**：无任何方持有联合轨迹，各智能体仅能看到自己接收到的共享产物（artifact）及其私有状态

**核心问题：**
- 如何在这种**去中心化接口约束**环境下学习最优工作流结构（哪个智能体先行动、何时传递控制、传递何种信息）
- 现有理论框架（Dec-POMDP、CTDE、Options、AIS）在该设置下至少违反四个条件之一：并发动作假设、联合轨迹访问要求、全局状态观测假设、缺乏有限样本保证

## 2. 方法论

**核心思想：**
- 提出**接口约束半马尔可夫决策过程（IC-SMDP）**形式化建模该问题
- 将决策时刻定义在**交接时间点**，每个智能体调用视为时间扩展选项（option）
- 设计 **IC-Q** 算法实现异步去中心化 Q 学习，每次跨智能体协调仅需一个标量

**关键技术细节：**

| 组件 | 说明 |
|------|------|
| **状态三层结构** | 全局隐状态 $x_t$（无智能体观测）→ 接口状态 $m_t$（跨智能体传递的唯一信息通道）→ 私有状态 $\ell^{(i)}_t$（智能体 i 私有） |
| **接口约束 (IC1/IC2)** | (IC1) 通道限制：交接时后继智能体仅接收 $m_{t+1}$；(IC2) 表示限制：智能体决策基于 $\phi_i(m_t, \ell^{(i)}_t)$ 而非直接观测 $m_t$ |
| **动作结构** | 每个智能体选择局部动作 $a_t$ 和后继智能体 $c_{t+1}$（包含 STOP 选项） |
| **随机持续时间** | 选项折扣为 $\gamma^{\tau_{k+1}}$，其中 $\tau_{k+1} = t_{k+1} - t_k$ 为随机变量 |

**算法流程（IC-Q）：**
1. **值传递**：在交接时刻 $t_{k+1}$，后继智能体 $c_1$（刚激活）计算 $b_{k+1} = \max_{c_2 \in C} \hat{Q}_{c_1}(o^{(k+1)}_{t_{k+1}}, c_2)$ 并传回给前驱智能体
2. **更新目标**：前驱智能体使用 $y_k = R_k + \gamma^{\tau_{k+1}} b_{k+1}$ 作为回归目标更新本地 Q 函数
3. **跨边界信息**：每次交接仅传递三个标量 $(b_{k+1}, \tau_{k+1}, R_k)$，不暴露内部参数、隐状态或本地观测
4. **可适应智能体**：额外维护动作选择 Q 函数 $Q_{\alpha}^i$，与后继的路由 Q 函数 $Q_{\beta}$ 通过双时间尺度耦合更新

**有限样本边界（Theorem 1）：**
$$E\left[\|Q(\cdot; \theta_T) - Q^{lat}_*\|_2^2\right] \leq 2\alpha_Q^2 + 6\varepsilon_{app} + 6\varepsilon_0 + 6\lambda_{max}(\Sigma_\mu) \cdot \frac{C_0(1 + t_{mix})\log(T+1)}{T}$$

其中三项独立可控误差源：
- **接口表示差距**：$\alpha_Q(\varepsilon_\phi, \delta_\phi, \bar\gamma) = \varepsilon_\phi + \frac{\bar\gamma L_Q \delta_\phi}{1-\bar\gamma}$
- **神经近似误差**：$\varepsilon_{app} + \varepsilon_0$（函数逼近精度）
- **混合时间残差**：$O(\frac{t_{mix}\log T}{T})$（样本复杂度）

## 6. 主要结论与发现

### 理论贡献
1. **首个有限样本保证**：在去中心化部分可观测设置下，神经 Q 学习具有非渐近收敛边界
2. **误差三分解**：可解释的三项独立可控误差源，便于系统设计者权衡
3. **IC-SMDP → AIS lifting**：将 AIS 框架从单智能体原始步 MDP 扩展到多智能体 SMDP，处理随机折扣

### 实证发现
1. **匹配中心化 oracle**：IC-Q 在无需联合轨迹观测的情况下达到预定义最佳工作流性能
2. **自适应数据集特性**：
   - 难题（MathQA、MMLU-HS）：IC-Q 启用 checker，选择 Comb-E
   - 简单题（MMLU-Elem）：IC-Q 收敛至简化 CoT（checker 反而引入错误）
3. **误差源可分离**：三项误差沿各自对应轴线缩放，与理论预测一致
4. **跨设置泛化**：路由实验验证精确 AIS 场景，CPU 编程实验扩展至可适应智能体
