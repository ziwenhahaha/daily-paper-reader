## 摘要
大型语言模型（LLMs）在复杂推理中的对齐主要依赖于可验证奖励强化学习（RLVR）。然而，标准算法（如 GRPO）对所有令牌统一使用序列级奖励，导致严重的信用分配瓶颈。虽然在策略自蒸馏尝试通过在特权上下文中调节自教师来解决问题，但直接接触原始 oracle 解往往导致过度调节的教师分布、隐式答案泄露以及训练后期崩溃。为克服这些限制，我们提出了非对称元反思自蒸馏（AMR‑SD）。AMR‑SD 不直接基于原始参考轨迹进行调节，而是插入一个反思瓶颈：它将诊断信号——来自验证器结果、同伴 rollout 或参考反馈——压缩为简洁的自生成苏格拉底式提示和批评。此外，我们引入因果信息增益（CIG），采用非对称 ReLU 门控阈值，将这些反思转化为稀疏、高精度的令牌级优势调制。结合时间退火，该机制在保留基线环境奖励的同时过滤分布噪声。在科学、数学和工具使用基准上的实验表明，AMR‑SD 显著优于现有基线，实现了稳健的长时域稳定性并成功防止后期崩溃。

## Abstract
The alignment of Large Language Models (LLMs) for complex reasoning heavily relies on Reinforcement Learning with Verifiable Rewards (RLVR). However, standard algorithms like GRPO apply sequence-level rewards uniformly to all tokens, creating a severe credit-assignment bottleneck. While on-policy self-distillation attempts to resolve this by conditioning a self-teacher on privileged contexts, direct exposure to raw oracle solutions often induces over-conditioned teacher distributions, implicit answer leakage, and late-stage training collapse. To overcome these limitations, we propose Asymmetric Meta-Reflective Self-Distillation (AMR-SD). Instead of conditioning directly on raw reference traces, AMR-SD inserts a reflection bottleneck: it compresses diagnostic signals -- from verifier outcomes, peer rollouts, or reference feedback -- into concise, self-generated Socratic hints and critiques. Furthermore, we introduce Causal Information Gain (CIG) with an asymmetric, ReLU-gated threshold to translate these reflections into sparse, highly precise token-level advantage modulations. Combined with temporal annealing, this mechanism preserves the base environmental reward while filtering out distributional noise. Experiments across scientific, mathematical, and tool-use benchmarks demonstrate that AMR-SD significantly outperforms existing baselines, achieving robust long-horizon stability and successfully preventing late-stage collapse.

- **核心痛点**：标准算法如 **GRPO** 对同一条 rollout 中的所有 token 施加统一的序列级优势标量，导致严重的 **信用分配瓶颈**——关键逻辑推导与表面风格选择收到完全相同的奖励信号，难以优化细粒度推理。
- **现有方案的缺陷**：
  - 离线 SFT 提供密集监督但存在严重的训练-测试分布偏移。
  - On-Policy 自蒸馏（让模型自身作为教师、 conditioning 在特权信息如 ground-truth 上）存在三大问题：①特权信息泄露、②教师分布过度条件化、③训练后期崩溃。
  - 近期方法（如 RLSD）通过锚定环境奖励做连续幅度调整来稳定训练，但连续乘数大多小于 1，压制了主要奖励信号，且直接用 ground-truth 作为特权信息效率低下。
- **核心动机**：在保留奖励锚定原则的前提下，用**非对称 ReLU 门控阈值**替代连续缩放，通过**稀疏、精准的 token 级干预**实现细粒度信用分配，同时避免直接暴露 oracle 答案带来的信息泄露和后期崩溃。

## 2. 论文提出的方法论

### 2.1 核心思想

AMR-SD 由三个顺序执行的阶段组成：**Socratic 重评分（元反思）→ CIG 量化 → 非对称调制**，最终将稀疏的 token 级优势张量注入 PPO 目标函数。

### 2.2 关键技术细节

#### (1) Socratic Rescoring via Meta-Reflection

- 让当前 student 策略本身（经 stop-gradient）转变为 **条件式 Socratic 评估器**，而非直接暴露 raw oracle 答案。
- 定义离散映射函数：

$$c_i = f_{\text{reflect}}(x, y_i, \text{sgn}(A_i)) = \mathbb{I}\{A_i \ge 0\} \cdot c_{\text{hint}} + \mathbb{I}\{A_i < 0\} \cdot c_{\text{critique}}$$

- **成功轨迹（$A_i \ge 0$）** → 生成 `<hint>`：从 verifier 批准的同组 peer rollouts 中提取成功推理的核心原则，不重复答案路径。
- **失败轨迹（$A_i < 0$）** → 生成 `<critique>`：定位逻辑/句法分歧，以正向重框（positive-framing）的方式陈述正确原则，**不直接描述错误**。
- 反思生成作为**低带宽瓶颈**，阻止 raw oracle trace 进入 token 重评分教师，同时保留关键的 token 级引导信号。

#### (2) Causal Information Gain（CIG）

- 从信息论角度量化每个 token 的细粒度贡献：

$$ICIG_t = \log \frac{\pi_{\theta}^{sg}(a_t | s_t, c_i)}{\pi_{\theta}(a_t | s_t)}$$

- 正值 $ICIG_t > 0$ 表示 student 概率低但 Socratic 教师概率高 → 知识缺陷；负值表示 student 过度自信。
- 为保证梯度 Lipschitz 连续性和数值稳定，对 $ICIG_t$ 做对称截断：

$$\tilde{ICIG}_t = \text{clip}(ICIG_t, -\kappa, \kappa), \quad \kappa = 5$$

#### (3) Asymmetric Token-Level Advantage Modulation

- 引入 **ReLU 门控的非对称阈值调节**：

$$\Delta_t = \mathbb{I}\{A_i \ge 0\} \cdot \lambda_{\text{eff}} \max(0, \tilde{ICIG}_t - \tau) + \mathbb{I}\{A_i < 0\} \cdot \gamma_{\text{eff}} \max(0, -\tilde{ICIG}_t - tau)$$

- **奖励放大**（成功轨迹）：仅当 $\tilde{ICIG}_t > \tau$ 时激活，放大 student 低于教师置信度的 token。
- **惩罚放大**（失败轨迹）：仅当 $-\tilde{ICIG}_t > \tau$ 时激活，惩罚 student 过度自信的 token。
- 阈值 $\tau$ 使机制扮演**严格门卫**角色，默认保留基线环境奖励不变，仅在检测到高度自信分歧时才触发针对性调整，从而过滤掉平凡的分布噪声。
- 最终 token 级优势张量：

$$\tilde{A}_{i,t} = \text{stop\_gradient}(A_i \cdot (1 + \Delta_t))$$

- 经验上 CIG 分布呈显著左偏（大部分值在 $[-1, 0]$），故设置 $\lambda = 0.2 > \gamma = 0.1$ 以补偿稀缺的正 CIG token。

#### (4) Temporal Annealing Schedule

- 为防止后期反思质量退化与教师能力天花板，使用线性时序衰减将优化平滑转移至纯环境奖励：

$$\{\lambda_{\text{eff}}, \gamma_{\text{eff}}\} = \{\lambda, \gamma\} \cdot \max\left(0, 1 - \frac{t_{\text{global}}}{T_{\text{decay}}}\right), \quad T_{\text{decay}} = 50$$

#### (5) 优化目标

- 用密集张量 $\tilde{A}_{i,t}$ 替换标准 PPO 目标中的轨迹优势：

$$\mathcal{L}_{\text{AMR-SD}}(\theta) = \mathbb{E}_{x \sim D, y \sim \pi_{\theta}^{\text{old}}} \left[ \frac{1}{T_i} \sum_{t=1}^{T_i} \mathcal{L}_{\text{clip}}(\theta, t) \right]$$

$$\mathcal{L}_{\text{clip}}(\theta, t) = \min\left( \rho_t(\theta) \tilde{A}_{i,t}, \text{clip}(\rho_t(\theta), 1-\varepsilon, 1+\varepsilon) \tilde{A}_{i,t} \right)$$

## 6. 论文的主要结论与发现

1. **性能提升显著**：AMR-SD 在科学推理的 Biology 任务上领先 GRPO 最多 **18.2 个百分点**（Qwen2.5-7B），在数学 AIME24/25/HMMT 等最具挑战性 benchmark 上均取得最高分（平均 62.7 vs. GRPO 的 60.3）。

2. **长期稳定性优越**：基线方法（尤其是 SDPO 和 RLSD）在训练后期出现性能回退或崩溃；AMR-SD 在 step 75 及更晚（数学 150 步）的检查点均保持稳定提升，训练 reward 持续增长。

3. **RLSD 后期崩溃的根因**：连续幅度缩放在早期未能过滤分布噪声，使策略陷入过拟合的脆弱状态；一旦蒸馏信号衰减，腐败的策略无法在纯环境奖励下维持优化。

4. **Meta-Reflection 优于直接 oracle conditioning**：消融显示直接用 ground-truth 作为教师条件，性能反而比模型自生成 Socratic 反思低 **3.8 分**，原因是 oracle conditioning 产生了信息不对称，导致 CIG 估计膨胀和错误校准。

5. **CIG 门控稀疏性的必要**：无 ReLU 阈值时低幅度噪声污染梯度；无 ReLU 门本身则允许反方向信号抵消intended 调制；两者共同保障噪声抗性和方向感知。

6. **反思质量在推理中的作用**：反思标记分析表明，RLSD 将标记从 117.57 压缩至 73.59（过度条件化导致探索压缩），而 AMR-SD 维持 105.32 的健康水平，实现"恰到好处"的平衡。

## 7. 优点

1. **信息瓶颈设计精巧**：通过让模型自生成 `<hint>` / `<critique>` 而非直接暴露 oracle 答案，有效阻断了特权信息泄露，同时保留了 token 级指导信号——这是一个反直觉但实验验证有效的设计。

2. **CIG + ReLU 阈值的稀疏调制**：不稀释全局奖励，而是作为严格门卫，仅在高置信度分歧点触发稀疏干预，过滤平凡分布噪声，同时保留 base environmental reward 的完整性。

3. **时序退火防止后期崩溃**：线性衰减机制平滑切换至纯环境奖励，避免了教师能力天花板和反思质量退化导致的过拟合风险。

4. **实验覆盖全面**：横跨科学推理（4 学科）、数学推理（4 个竞赛 benchmark）、工具使用三大场景，基线对比充分，包含完整的消融分析和动态可视化。

5. **对 CIG 分布的深入分析**：通过 50,000 tokens 的大规模采样揭示了 CIG 左偏分布的理论根源，并据此合理设置 $\lambda > \gamma$，展示了从观察→分析→调参的完整闭环。

## 8. 不足与局限

1. **对反思质量强依赖**：当基础模型缺乏足够的内省能力时（如 enable_thinking=False 的 Qwen2.5-7B），生成的 hints 和 critiques 模糊或事实错误，会传播错误监督信号，导致无法超越 GRPO 基线。论文坦承这一限制。

2. **推理额外开销**：Meta-Reflection 阶段需要额外一次自回归生成（约额外 20% wall-clock 时间），在严重资源受限的场景下可能不实用。

3. **消融实验覆盖不足**：消融仅在 Qwen3-8B + 混合 SciKnowEval 上进行，未验证各组件在数学推理或工具使用任务上的独立贡献，也缺乏 out-of-domain 消融评测。

4. **超参数敏感性**：$\lambda$、$\gamma$、$\tau$、$\kappa$ 等超参数基于经验性的 CIG 分布选择，当前设置（$\lambda=0.2, \gamma=0.1, \tau$ 未明确给出具体数值）缺乏系统性的自动调参策略。

5. **教师-学生同步假设**：Socratic 评估器 $\pi_{\theta}^{sg}$ 与 active policy $\theta$ 在每步严格同步（无 EMA 平滑），虽然简化了超参调优，但可能对训练曲线的平滑性带来额外敏感性。

6. **泛化性验证有限**：数学推理仅在 Qwen3-8B + thinking mode 上验证，未在其他模型架构（如 Llama、Mistral）或非推理任务上测试。
