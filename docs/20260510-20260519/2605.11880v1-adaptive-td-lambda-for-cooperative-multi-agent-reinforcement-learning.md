## Abstract
TD($λ$) in value-based MARL algorithms or the Temporal Difference critic learning in Actor-Critic-based (AC-based) algorithms synergistically integrate elements from Monte-Carlo simulation and Q function bootstrapping via dynamic programming, which effectively addresses the inherent bias-variance trade-off in value estimation. Based on that, some recent works link the adaptive $λ$ value to the policy distribution in the single-agent reinforcement learning area. However, because of the large joint action space from multiple number of agents, and the limited transition data in Multi-agent Reinforcement Learning, the policy distribution is infeasible to be calculated statistically. To solve the policy distribution calculation problem in MARL settings, we employ a parametric likelihood-free density ratio estimator with two replay buffers instead of calculating statistically. The two replay buffers of different sizes store the historical trajectories that represent the data distribution of the past and current policies correspondingly. Based on the estimator, we assign Adaptive TD($λ$), \textbf{ATD($λ$)}, values to state-action pairs based on their likelihood under the stationary distribution of the current policy. We apply the proposed method on two competitive baseline methods, QMIX for value-based algorithms, and MAPPO for AC-based algorithms, over SMAC benchmarks and Gfootball academy scenarios, and demonstrate consistently competitive or superior performance compared to other baseline approaches with static $λ$ values.

# Adaptive TD-Lambda for Cooperative Multi-agent Reinforcement Learning 论文详细总结

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：在合作式多智能体强化学习（MARL）中，TD(λ) 方法通过平衡蒙特卡洛模拟和 Q 函数 bootstrapping 来解决价值估计中的偏差-方差权衡问题。然而，如何自适应地选择 λ 值（即 λ 应当多大程度依赖蒙特卡洛回报 vs. bootstrapped Q 值）是一个关键挑战。
- **背景痛点**：
  - TD 学习面临过估计偏差（来自函数近似），而蒙特卡洛方法偏差小但方差大。
  - 在单智能体 RL 中，已有小部分工作将自适应 λ 与策略分布关联起来，但**多智能体环境下，联合动作空间庞大且转移数据有限，导致策略分布无法直接统计计算**。
  - λ 作为超参数，合适值随训练过程动态变化（训练初期适合较大 λ 以加速收敛，后期适合较小 λ 以避免使用过时的蒙特卡洛回报）。
- **研究目标**：提出一种在 MARL 场景下**无需显式建模策略分布即可计算自适应 λ** 的方法——ATD(λ)。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 2.1 核心思想

- **核心洞察**：利用**似然比（density ratio）**来衡量某个状态-动作对由"当前策略"生成的概率（即 on-policy 程度），以此作为该转移的 λ 值。
- **关键思路**：不直接计算策略分布 $d_\pi(s,a)$，而是采用**无似然（likelihood-free）的密度比估计器**，通过比较两个不同缓冲区的数据分布来估算密度比。

### 2.2 关键技术细节

- **双缓冲区架构**：
  - **小型 on-policy 缓冲区** $D_{on}$：仅存储最新轨迹（刷新快，数据更接近当前策略）。
  - **大型 off-policy 缓冲区** $D_{off}$：来自原始 MARL 算法的经验回放缓冲区（存储历史轨迹，混合了不同策略产生的数据）。
  - 两缓冲区容量比推荐为 **50:1**（off-policy:on-policy）。

- **似然比估计网络 ω_φ(s,a)**：
  - 输入：状态-动作对 $(s,a)$。
  - 结构：MLP + GRU（编码历史观测，适配 CTDE 框架）。
  - 输出：经 Sigmoid 激活的 $(0,1)$ 范围值，作为 $\lambda$ 值和重要性权重。

- **基于 f-散度的密度比估计（定理 1）**：
  - 定义密度比 $\omega(s,a) := d_\pi(s,a)/d_D(s,a)$。
  - 通过变分形式估计密度比，最大化下界：
    $$\max_{\omega_\phi} \left( \mathbb{E}_P[f'(\omega_\phi(s,a))] - \mathbb{E}_Q[f^* (f'(\omega_\phi(s,a)))] \right)$$
  - 文中选用与 **Jensen-Shannon 散度** 对称的 $f$ 函数，推导为**二元交叉熵损失**：
    $$L_\omega(\phi) = \text{BCE}_{s,a \sim D_{off}}(\omega_\phi(s,a), 0) + \text{BCE}_{s,a \sim D_{on}}(\omega_\phi(s,a), 1)$$

- **ATD 目标值计算**：
  - TD 目标替换为：
    $$R_\omega(s,a)_t = r(s,a) + \gamma [\omega(s,a) R_\omega(s',a')_{t+1} + (1 - \omega(s,a)) \max_{a'} Q_{tot}(s',a'|\theta^-)]$$
  - 即 $\lambda = \omega(s,a)$ 控制蒙特卡洛与 bootstrapping 的混合比例。

- **λ 值缓存机制**：
  - 为保证目标值在 $\theta^-$ 更新间隔内稳定，将 $\lambda$ 值缓存到 transition 中，随 $\theta^-$ 更新频率一起刷新。

### 2.3 算法流程（伪代码要点）

1. 初始化各网络（utility、mixing、ATD）和双缓冲区。
2. 交互环境，收集新 transition。
3. 将新轨迹存入 $D_{on}$，溢出部分存入 $D_{off}$。
4. 从 $D_{off}$ 采样训练批次，计算每个转移的 $\lambda = \omega_\phi(s,a)$。
5. 通过 ATD 目标值计算 TD 损失，更新 utility 和 mixing 网络。
6. 到达目标网络更新频率时：更新 $\theta^- = \theta$；从双缓冲区采样更新 $\omega_\phi$ 网络。

## 6. 论文的主要结论与发现

1. **ATD(λ) 在 Value-based 和 AC-based 方法上均有效**：
   - 在 QMIX 上集成 ATD，在 SMAC 的 hard 和 super-hard 场景中表现优于或持平固定 λ 的 QMIX。
   - 在 MAPPO 上集成 ATD（改进 critic 训练），在 Gfootball 的 6 个 Academy 场景中显著或略优。

2. **适应性 λ 提升收敛速度与最终性能**：
   - 训练初期，较大 λ 加速价值函数拟合；后期，较小 λ 避免使用过时蒙特卡洛回报。
   - ATD 相比固定 λ 在多数场景更快收敛且最终胜率更高。

3. **插件式设计，易于集成**：
   - 现有 MARL 代码库已有 off-policy 缓冲区（value-based）或 on-policy 缓冲区（policy-based），ATD 可作为轻量插件添加。
   - 兼容性测试表明 ATD 可扩展到 QPLEX、OW-QMIX 等方法。

4. **推荐配置**：
   - on/off-policy 缓冲区容量比 **50:1**。
   - λ 缓存频率与目标网络更新频率一致。

5. **局限性观察**：
   - 在高随机性环境（如 **SMACv2**）中，ATD 提升有限或无明显提升（转移重复性低导致密度比快速衰减至 0）。
   - 固定 λ 方法（尤其是 TD(0））在 SMACv2 上表现也一般，说明该环境本身对 TD(λ) 方法挑战较大。

1. **问题定位精准**：针对 MARL 中"策略分布难统计计算"这一核心障碍，提出无需显式建模的似然比估计方案，填补了自适应 λ 在多智能体场景下的研究空白。

3. **插件式设计，工程友好**：
   - 只需添加一个 ATD 网络和一个小缓冲区，改动小，易于集成到现有 MARL 代码库（QMIX、MAPPO 等）。

4. **双缓冲区策略简洁有效**：
   - 利用新旧轨迹的自然分布差异，无需额外重要性采样或 Retrace 级别的修正复杂度。

5. **实验全面且有深度**：

   - 对比方法包括当前主流 MARL 算法和多种固定 λ 策略（TD(0)、TD(1)、Importance Sampling、Retrace）。

6. **单智能体兼容性验证**：1s_vs_1z 场景说明 ATD 泛化到单智能体 RL 同样有效。

7. **可解释性**：λ 值随训练动态调整，可通过学习曲线观察 λ 演化趋势，有助于理解训练过程。

   - 论文未明确 GPU 数量，大规模复现可能受限于算力。

3. **超参数敏感性**：
   - 50:1 缓冲区容量比为经验推荐，虽有消融实验，但不同任务/环境下最优比可能不同。
   - λ 缓存频率需匹配目标网络更新频率，调参不当时方差增大（图 11a 显示低频和高频均导致性能下降）。

4. **环境覆盖有限**：
   - 主要在 SMAC 和 Gfootball 上验证，未在其他 MARL 基准（如 Multi-Agent MuJoCo、PettingZoo）上测试。
   - 稀疏奖励场景（Gfootball）中价值-based 方法表现差，ATD 对其提升有限。

5. **密度比估计的近似误差**：
   - 使用变分下界估计密度比，假设 $f$ 函数为 JS 散度，实际密度比估计存在误差，可能导致 λ 值偏离理论最优。

6. **on-policy 缓冲区刷新策略未深入探讨**：
   - 快速刷新 on-policy 缓冲区可能引入偏差（如旧轨迹未充分淘汰），但论文未系统分析不同刷新策略的影响。

7. **对非合作场景未验证**：
   - 方法针对完全合作场景（Dec-POMDP）设计，在竞争或混合场景中的适用性未知。
