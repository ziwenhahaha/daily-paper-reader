## Abstract
Coordinating multiple robots in shared environments requires generating feasible trajectories for each agent while accounting for interactions among agents. Centralized planning approaches become difficult to scale as the number of robots increases, while decentralized approaches that allow each agent to plan independently do not inherently account for inter-agent interactions. This paper presents a framework for coordinated multi-robot motion planning that combines decentralized generative trajectory planning with multi-agent reinforcement learning (MARL)-based coordination. Each robot independently generates candidate trajectories using a diffusion model trained on single-agent motion data, leveraging the generative model's ability to produce feasible and diverse trajectories. To reduce conflicts between agents, a centralized value function trained via MARL guides the reverse diffusion process through gradient-based steering, enabling interaction-aware trajectory generation without centralized joint planning or retraining of the generative model. This guidance follows an exponential tilting formulation, in which the value function biases the denoising distribution toward trajectories with higher expected multi-agent return. The framework is evaluated in a simulated maze environment with four mobile robots. Experimental results show that the proposed value-guided diffusion planning reduces the inter-agent interference rate from 55.4% to 41.8%, demonstrating that coordination can be effectively achieved while preserving the scalability of decentralized trajectory generation. These results suggest that MARL-based value guidance can effectively introduce coordination into decentralized generative planners without requiring a fully joint multi-robot model.

- **核心问题**：在共享环境中协调多个机器人运动时，如何生成可行轨迹，同时兼顾智能体间的交互，避免冲突和碰撞。
- **研究背景**：
  - 多机器人系统在工业制造等场景中需求日益增长，需要协调运动以实现并行任务执行和灵活系统操作。
  - **集中式规划**虽然能显式建模智能体间交互，但随机器人数量增加，联joint规划空间呈指数增长，计算复杂度激增，可扩展性差。
  - **去中心式规划**允许每个智能体基于本地信息独立规划，扩展性好，但独立生成的轨迹并不天然考虑智能体间交互，容易产生冲突。
  - 生成式机器学习（尤其是扩散模型）能够从数据中学习可行运动分布，生成多样且不确定性的轨迹，为单智能体轨迹生成提供了新思路，但在多智能体场景下仍缺乏协调机制。
- **研究空白**：如何在保留去中心化可扩展性的同时，无需重新训练生成模型或进行集中式联joint规划，就能有效引入智能体间协调。

### 2. 论文方法论：核心思想、关键技术细节、算法流程

#### 2.1 核心思想

将**去中心化生成式轨迹规划**（单智能体扩散模型）与**多智能体强化学习（MARL）集中式价值函数**相结合，通过价值梯度引导扩散采样过程，实现无需联joint建模或重训练的智能体间协调。

#### 2.2 关键技术细节

- **单智能体策略学习（构建轨迹分布）**：
  - 将单智能体导航建模为马尔可夫决策过程（MDP），用**PPO算法**学习策略 $\pi_\theta$，最大化累积奖励 $J(\theta) = \mathbb{E}_{\pi_\theta}\left[\sum_{t=0}^H \gamma^t r(x_t, a_t)\right]$。
  - 收集多样化的轨迹数据集 $D = \{\tau^{(k)}\}_{k=1}^K$，作为扩散模型训练数据。
  - 目标不是产生单一最优轨迹，而是诱导覆盖可行运动空间的轨迹分布。

- **单智能体轨迹建模（扩散模型）**：
  - 采用**条件扩散模型**，学习轨迹分布 $p_\theta(\tau)$，通过迭代去噪过程生成轨迹。
  - 前向扩散：$q(\tau_t | \tau_{t-1}) = \mathcal{N}(\tau_t; \sqrt{1-\beta_t}\tau_{t-1}, \beta_t I)$，逐步注入高斯噪声。
  - 反向过程使用噪声预测参数化 $\epsilon_\theta(\tau_t, t, c)$，训练目标为最小化均方误差 $\mathcal{L}_{\text{traj-gen}}(\theta) = \mathbb{E}_{\tau_0, \epsilon, t}[\|\epsilon - \epsilon_\theta(\tau_t, t, c)\|^2]$。
  - 推理时，从高斯噪声开始，通过迭代去噪并结合起点-终点边界条件（conditioning）生成可行轨迹。

- **去中心化多智能体轨迹生成**：
  - 每个智能体 $i$ 独立采样 $\tau^i \sim p_\theta(\tau^i | c^i)$，联joint分布为 $p_\theta(X) = \prod_{i=1}^N p_\theta(\tau^i | c^i)$。
  - 保持去中心化可扩展性，计算复杂度随智能体数量线性增长，无需联joint优化或通信。

- **集中式价值函数（MARL协调信号）**：
  - 定义集中式价值函数 $V_\phi(X) \approx \mathbb{E}\left[\sum_{t=0}^H \gamma^t r_{\text{joint}}(X_t)\right]$，评估联joint轨迹的协调质量。
  - 使用**MAPPO**（Multi-Agent PPO）在集中式训练-去中心化执行框架下训练批评器，最小化 $\mathcal{L}_{\text{coord-value}}(\phi) = \mathbb{E}_X[(V_\phi(X) - \hat{R}(X))^2]$。
  - 价值函数基于全局状态 $s = [o_1 \| o_2 \| \cdots \| o_N]$，捕获智能体间交互效应。

- **价值引导的多智能体规划（指数倾斜公式）**：
  - 引导分布定义为 $\tilde{p}_\theta(X) \propto p_\theta(X) \exp(\lambda V_\phi(X))$，其中 $\lambda$ 为引导尺度。
  - 梯度形式为 $\nabla_X \log \tilde{p}_\theta(X) = \nabla_X \log p_\theta(X) + \lambda \nabla_X V_\phi(X)$。
  - 修正的反向转移（每智能体 $i$）：$\tau^{i}_{t-1} \sim \mathcal{N}(\mu_i^t + \lambda \Sigma_t \nabla_i, \Sigma_t)$，其中 $\mu_i^t = \mu_\theta(\tau_t^i, t, c_i)$ 为去噪均值，$\nabla_i$ 为价值梯度的智能体分量。
  - 在每个去噪步中，通过**集中式梯度** $\nabla_X V_\phi(X_t)$ 跨智能体传播协调信息，无需显式通信。

#### 2.3 算法流程（Algorithm 1）

1. 初始化 $N$ 个智能体的轨迹为噪声 $\tau^i_T \sim \mathcal{N}(0, I)$，并通过修复（inpainting）强制边界条件。
2. 迭代 $t = T, ..., 1$：
   - 对每个智能体 $i$ 计算去噪均值 $\mu_i^t$。
   - 聚合所有智能体的当前轨迹 $X_t$。
   - 计算集中式价值梯度 $\nabla_X V_\phi(X_t)$。
   - 对每个智能体执行导向反向采样 $\tau^i_{t-1} \sim \mathcal{N}(\mu_i^t + \lambda \Sigma_t \nabla_i, \Sigma_t)$，并强制边界条件。
3. 返回最终联joint轨迹 $X_0$。

### 6. 论文主要结论与发现

- **智能体间干扰率显著降低**：无引导扩散的干扰率为55.4%，MARL引导后降至41.8%，绝对降低13.6个百分点，证实价值引导能有效减少智能体间冲突。
- **定性轨迹差异**：无引导时智能体沿走廊紧密聚集，轨迹频繁重叠；有引导时智能体保持更大空间分离，遵循相似全局路线。
- **存在权衡**：引导后障碍物干扰率从4.6%升至7.6%，表明减少智能体间干扰可能以增加墙壁接近为代价，源于MARL批评器在开放空间训练、缺乏障碍物信息。
- **方法可行性**：证实MARL价值引导能向去中心化生成规划器引入协调，无需联joint轨迹建模或重训练扩散模型。

### 7. 优点

- **架构创新**：提出将单智能体扩散模型与集中式MARL价值函数解耦结合，通过推理时价值引导实现协调，避免重训练，保留去中心化可扩展性。
- **方法优雅**：指数倾斜公式提供了原则性的机制，将协调目标（价值函数）转化为扩散采样中的梯度偏置，数学形式简洁且有理论依据。
- **无需联joint规划**：每个智能体独立生成轨迹，仅通过共享的价值函数梯度隐式协调，无需通信或联joint优化，适合大规模系统。
- **通用性**：框架中单智能体扩散模型和MARL价值函数可分别独立训练和替换，具有较好通用性。

### 8. 不足与局限

- **实验覆盖不足**：
  - 仅在单一迷宫场景、4个智能体配置下验证，未评估更大规模智能体群体或不同环境复杂度下的表现。

  - 未进行充分的消融实验（如引导尺度 $\lambda$ 的敏感性分析、去噪步数对性能的影响等）。

- **权衡问题**：
  - MARL批评器在开放空间训练，缺少障碍物先验，导致引导后障碍物干扰率上升，揭示了训练环境与目标环境不匹配的问题。
  - 引导可能引入轨迹振荡等局部修正副作用，影响轨迹平滑性。

- **偏差风险**：
  - 扩散模型和价值函数分别在迷宫中（单智能体）和开放空间（多智能体）中训练，存在域差距（domain gap），可能导致在迷宫环境中引导效果次优。
  - 集中式价值函数可能过拟合特定多智能体行为模式，泛化能力未充分验证。

- **应用限制**：
  - 仅在仿真环境中验证，未在真实物理多机器人系统上测试，实际部署可行性未知。
  - 方法假设智能体可获得全局状态用于价值函数评估，可能不适用于完全分布式传感器设置。
  - 缺乏实时性分析，扩散模型的迭代去噪过程计算成本较高，实时应用可能面临挑战。

- **缺乏泛化验证**：未验证方法对不同起点-终点配置、不同地图布局或动态障碍物的鲁棒性。
