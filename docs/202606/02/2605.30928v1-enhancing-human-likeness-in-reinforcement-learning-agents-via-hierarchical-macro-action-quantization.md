## Abstract
Human-like agents are a long-standing goal of artificial intelligence. Despite strong performance, most reinforcement learning (RL) agents remain reward-driven and often exhibit behaviors that differ from humans, limiting interpretability and reliability. In this work, we introduce a novel human-like RL framework that predicts action sequences closely aligned with human behaviors while maximizing rewards. Specifically, we encode human demonstrations into macro actions using a hierarchical macro action quantization approach (termed HiMAQ) consisting of two successive levels of vector quantization. The lower quantization level maps input actions to fine-grained subaction clusters, while the higher quantization level aggregates these subaction clusters into action clusters. Extensive evaluations on the D4RL benchmarks show that our hierarchical approach outperforms the non-hierarchical baseline (MAQ), achieving better human-likeness scores while maintaining comparable or better success rates than previous RL agents. The improvements generalize across integrations with various RL algorithms, namely IQL, SAC, and RLPD.

**背景技术**：现有方法如 HRC（人类类 receding horizon control）和 MAQ（宏动作量化）将人类轨迹片段编码为离散码本，但 MAQ 的单层代码本将每个宏动作视为单一原子原型，无法捕捉宏动作内部的完整变化（如抓取动作的接近、对齐、控制接触等细腻动态）。

## 2. 论文提出的方法论

### 核心思想

提出 **HiMAQ（Hierarchical Macro Action Quantization）**，一种两层向量量化框架，从人类演示中学习更丰富的人类运动表示：

- **低层量化**：将编码的轨迹映射到细粒度**子动作原型（subaction prototypes）**
- **高层量化**：将子动作聚类聚合为更粗粒度的**动作原型（action prototypes）**

### 关键技术细节

1. **输入**：状态 $s_t$ 和宏动作 $m_t = (a_t, a_{t+1}, ..., a_{t+H-1})$（H 步动作序列）
2. **编码器**：将输入映射到潜在嵌入 $e_t$
3. **第一层量化**：$q_Z^t = z_{j^*}$，其中 $j^* = \arg\min_j \|e_t - z_j\|^2$（从子动作码本 Z 中选择最近原型）
4. **第二层量化**：$q_A^t = a_{i^*}$，其中 $i^* = \arg\min_i \|q_Z^t - a_i\|^2$（从动作码本 A 中选择最近原型）
5. **解码器**：根据 $q_A^t$ 和 $s_t$ 重构宏动作 $\tilde{m}_t$

### 训练损失

总损失包含重建损失、承诺损失和代码本损失：

$$L = L_{recon} + \lambda_{commit}(L_{commit}^Z + L_{commit}^A) + \lambda_{codebook}(L_{codebook}^Z + L_{codebook}^A)$$

其中：
- $L_{recon} = \|m_t - \tilde{m}_t\|^2$
- 承诺损失推动编码嵌入向代码本靠近
- 代码本损失更新代码本参数（使用 stop-gradient 操作防止梯度回传）

实验中设置 $\lambda_{commit} = 0.25$，$\lambda_{codebook} = 1.0$。

## 6. 论文的主要结论与发现

### 定量轨迹评估

1. **IQL 基线**：HiMAQ+IQL 将平均 DTW_s 从 0.63 提升至 0.65，成功率从 0.39 提升至 0.56（在 Hammer 任务上从 0.00 提升至 0.87）
2. **SAC 基线**：HiMAQ+SAC 将平均 DTW_s 从 0.56 提升至 0.63，成功率从 0.28 提升至 0.29
3. **RLPD 基线**：HiMAQ+RLPD 将平均 DTW_s 从 0.56 提升至 0.62，成功率从 0.52 提升至 0.56

### 人类评估

1. **图灵测试**：HiMAQ+RLPD 达到 **43%** 胜率（评估者误认为人类），在所有方法中最高
2. **排名测试**：HiMAQ 变体在类人度上持续优于对应 MAQ 变体；HiMAQ+RLPD（54%）最接近人类基线（71%）

### 关键发现

- 分层量化能够捕捉宏动作内的细腻变化，提升轨迹相似性而不牺牲成功率
- 宏动作长度 H=9 在 RLPD 中达到最佳平衡，更长的序列（≥10）收益递减

1. **创新的分层量化架构**：通过两层向量量化同时捕获细粒度子动作和粗粒度动作模式，比单层 MAQ 更丰富地表示人类运动
2. **通用性**：可与多种 RL 算法（IQL、SAC、RLPD）集成，适用范围广

4. **显著的性能提升**：在 Hammer 任务上实现从 0% 到 87% 成功率的突破性提升
5. **公平对比**：与 MAQ 在相同设置下比较，确保贡献归因清晰
