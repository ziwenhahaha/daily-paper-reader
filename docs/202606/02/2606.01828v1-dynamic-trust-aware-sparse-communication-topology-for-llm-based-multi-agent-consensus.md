## 摘要
大型语言模型驱动的多智能体系统通过多轮商议、角色专长化和交叉验证提高复杂推理任务的可靠性。然而，现有的多智能体辩论和协作框架通常采用全连接通信方式，导致消息数量、令牌成本和端到端延迟随智能体数量近似呈二次增长；虽然固定稀疏拓扑能够降低开销，但它们无法根据不同的任务实例或中间推理状态调整通信关系，因而要么保留低价值交互，要么丢失关键的错误纠正信息。为解决此问题，本文提出 DySCo（动态稀疏共识），一种动态信任感知的稀疏共识机制。在每一轮推理中，DySCo 基于智能体可靠性、答案分歧和任务相关性评估通信边的价值，并在预算约束下选取少量高价值边进行消息交换；随后通过动态信任权重聚合不同智能体的答案，并在共识趋于稳定时提前终止讨论。该机制将通用广播替换为按需通信，从而在保留关键交叉验证信息的同时降低通信开销。本文进一步给出了通信复杂度和共识稳定性的分析，并在数学推理、逻辑推理和事实问答任务上评估了 DySCo 的性能。

## Abstract
Large language model-driven multi-agent systems enhance the reliability of complex reasoning tasks through multi-round deliberation, role specialization, and cross-validation. However, existing multi-agent debate and collaboration frameworks typically adopt fully connected communication, causing the number of messages, token costs, and end-to-end latency to grow approximately quadratically with the number of agents; although fixed sparse topologies reduce overhead, they cannot adapt communication relationships to different task instances or intermediate reasoning states, making them prone either to preserving low-value interactions or to losing critical error-correction information. To address this problem, this paper proposes DySCo (Dynamic Sparse Consensus), a dynamic trust-aware sparse consensus mechanism. In each round of reasoning, DySCo estimates the value of communication edges based on agent reliability, answer divergence, and task relevance, and selects a small number of high-value edges for message exchange under budget constraints; it then aggregates the answers of different agents through dynamic trust weights and terminates the discussion early once consensus stabilizes. This mechanism replaces universal broadcasting with on-demand communication, thereby reducing communication overhead while preserving essential cross-validation information. We further present analyses of communication complexity and consensus stability, and evaluate the performance of DySCo on mathematical reasoning, logical reasoning, and factual question-answering tasks.

## 2. 方法论：核心思想、关键技术细节与算法流程

### 2.1 核心思想
- **动态稀疏共识（DySCo）**：将通信拓扑从固定结构转变为 **随任务状态和推理过程自适应的决策变量**。  
- **关键假设**：通信边的价值并不相等——高置信度、历史可靠性强且与当前智能体答案分歧明显的邻居是最有价值的通信伙伴。  

### 2.2 关键技术细节

| 模块 | 作用 | 关键公式 / 机制 |
|------|------|----------------|
| **个体初始化** | 每智能体独立求解任务，产生初始答案 *y⁰ᵢ*、推理轨迹 *r⁰ᵢ*、置信度 *c⁰ᵢ* | `(y⁰ᵢ, r⁰ᵢ, c⁰ᵢ) = LLM_i(P_solve(x))` |
| **动态边选择** | 计算每条有向边 (j→i) 的通信价值得分 `s_t_ij`，并在上界 *k* 下选取 Top‑K 高分邻居 | `s_t_ij = α·T_t_j + β·c_t_j + γ·D_t_ij + δ·H_ij(x) - η·L_t_j`<br>`N_t_i = TopK_{j≠i}(s_t_ij, k)` |
| **消息压缩 & 批判生成** | 只发送压缩后的结构化消息（答案、关键支撑理由、可能的反例或修订建议），控制令牌长度 | `m_t_{j→i} = LLM_j(P_critique(...))` |
| **信任感知的状态更新** | 接收邻居的加权反馈（高信任权重的意见更具说服力），决定保留、部分修订或完全改写答案并更新置信度 | `(y_{t+1}^i, r_{t+1}^i, c_{t+1}^i) = LLM_i(P_revise(...))` |
| **信任权重更新** | 综合外部验证分数 *Vᵢ* 与当前共识一致性 *Aᵢ*，对历史信任进行平滑更新 | `T_{t+1}^i = λ·T_t^i + (1-λ)(ρ·V_t^i + (1-ρ)·A_t^i)` |
| **共识聚合 & 早停** | 采用信任加权的投票得到最终答案 `ŷ`；若加权共识熵低于阈值 ε，或连续两轮答案不变且置信度提升不足，则提前终止通信 | `ŷ = argmax_y Σ_i T_R^i·c_R^i·I[y_R^i = y]` |

- **通信复杂度**：全连接 → `O(R n² ℓ)`（ℓ 为平均消息长度）；DySCo → `O(R n k ℓ_c)`（ℓ_c 为压缩后消息长度），**k ≪ n**，显著降低令牌开销。  

- **共识稳定性抽象**：将离散答案映射为置信向量 `p_t^i ∈ Δ|Y|`，在忽略 LLM 非线性扰动 `ξ_t^i` 的情况下，更新过程等价于时变加权平均。若 **任意连续 B 轮的联合图强连通且非零权重 ≥ μ**，则在扰动有界且均值为零的假设下，置信向量在期望意义上收敛到一个共同邻域（Proposition 1）。

## 6. 主要结论与发现

1. **令牌消耗显著降低**  
   - 与 Dense MAD 相比，DySCo 在 GSM8K、LogiQA、StrategyQA 上分别降低约 **70%** 令牌消耗和约 **50%** 延迟。  

2. **准确率提升或持平**  
   - DySCo 在三个数据集上均取得最高准确率（如 GSM8K 84.3% vs Dense MAD 82.0%），说明动态稀疏通信保留了关键的错误纠正信息。  

3. **早期共识收敛**  
   - 通过信任加权的意见聚合，DySCo 在第二轮即达到低共识熵（0.28），而 Dense MAD 需要三轮才能接近（0.40），触发早停机制后进一步削减不必要的通信。  

4. **可扩展性优势**  
   - 令牌消耗随智能体数目呈 **线性增长**（O(Rnk)），而 Dense MAD 为二次增长；在 n=12 时，DySCo 准确率提升至 86.8%，而 Dense MAD 已出现 plateau。

### 6. 主要结论与发现（续）

- **去除信任权重或答案分歧度均导致显著性能下降**，尤其在逻辑推理任务（LogiQA）上，准确率分别下降 3.2 pp 和 2.8 pp。这表明仅依赖历史可靠性或仅考虑局部答案差异都不足以维持跨智能体的纠错能力，两者协同才能发挥最大效用。  
- **消息压缩与早停机制对效率的贡献突出**：去掉消息压缩后，每轮平均令牌数上升约 42 %，但准确率仅提升 0.3 pp；去掉早停后，系统往往执行满 R=3 轮，导致 25 % 的不必要延迟。  
- **综合对比，DySCo 在效率‑准确性帕累托前沿上占据最佳位置**：在令牌预算受限的真实部署场景（如 API 计费或延迟敏感业务），DySCo 能够以约 30 % 的令牌消耗达到与全连接辩论相近的准确率（差距 ≤ 1 pp），是当前多智能体系统中性价比最高的方案。

## 7. 关键洞见与启示

| 洞见 | 具体表现 |
|------|----------|
| **通信拓扑是可学习的资源** | 将拓扑选择建模为运行时决策，使得系统能够在不同任务阶段自适应地聚焦关键信息源，避免“一刀切”的固定稀疏结构导致的盲区。 |
| **信任是一种动态加权信号** | 通过结合外部验证分数和内部共识一致性，信任权重能够快速捕捉智能体在特定任务上的可靠性变化，从而在后续轮次中为价值最高的交互提供指导。 |
| **消息压缩是降低开销的关键** | 仅传递答案与关键理由（或反例）即可满足大多数协同需求，过度冗余的完整推理链会显著增加令牌成本，却对最终共识贡献有限。 |
| **早期共识收敛不等同于放弃多样性** | 通过在第二轮即触发低熵共识，系统仍保留足够的首轮多样化视角，避免因过早收敛导致的错误锁定。 |

## 9. 总结

- **研究意义**：DySCo 将通信拓扑从被动固定的超参数提升为可自适应优化的决策变量，为大语言模型多智能体系统提供了一种兼顾效率与可靠性的新范式。  
- **核心贡献**：  
  1. 动态稀疏通信机制，基于信任度、答案分歧度、任务相关性等多维度评分，选取最具价值的交互边。  
  2. 信任感知的消息压缩与状态更新，保证关键纠错信息不遗漏。  
  3. 早停机制，基于信任加权共识熵实现提前收敛，显著削减不必要的通信。  
  4. 理论分析，证明了在强连通非零权重的通信图下，系统置信向量在期望意义上收敛。  
- **实验验证**：在数学（GSM8K）、逻辑（LogiQA）与事实问答（StrategyQA）三大任务上，DySCo 均实现了令牌消耗下降约 70 %，延迟降低约 50 %，而准确率保持或略优于全连接辩论。  
- **实践价值**：DySCo 能够在保持高准确率的前提下，大幅降低 API 调用成本与响应时延，适用于商业对话系统、实时推理辅助、教育评估等资源敏感场景。  

> **一句话概括**：DySCo 通过在运行时动态选取高价值通信边、压缩消息并结合信任感知聚合，在保持甚至提升多智能体推理准确率的同时，实现了令牌与时延的显著削减，为大规模 LLM 多智能体系统提供了高效、可扩展的合作框架。
