## 摘要
在这项工作中，我们提出了一个框架，将多智能体强化学习（MARL）与基于模型的控制相结合，以在协作多智能体任务中实现安全、动力学可行的动作。多智能体强化学习能够在长时域的规划中从离散不可微的奖励中学习协作策略，而模型预测控制则在短时域的快速重规划框架中提供鲁棒且安全、动力学可行的动作。我们提出了一种算法，将参与‑批评（actor‑critic）模型预测控制扩展到多智能体强化学习中，我们称之为多智能体参与‑批评模型预测控制（MA‑AC‑MPC）。我们通过将其应用于多智能体追捕‑逃避场景来展示该算法的能力。具体而言，我们比较了逃避团队使用MA‑AC‑MPC模型和多层感知器模型（MA‑AC‑MLP）的策略。追捕团队采用增强比例导引，因为它被公认为先进的对抗控制律。我们还提供了一个异构环境的示例，其中无人机与全向轮式 rover 合作，以实现可重复且成功的着陆，在硬件上 MA‑AC‑MPC 达到 100% 成功率，而 MA‑AC‑MLP 仅为 60%。我们通过硬件实验验证了所提出的 MA‑AC‑MPC 算法在两种环境中的鲁棒性。

## Abstract
In this work, we propose a framework that combines multi-agent reinforcement learning (MARL) with model-based control to achieve safe, dynamically feasible actions in cooperative multi-agent tasks. Multi-agent reinforcement learning provides the advantage of learning cooperative policies for multi-agent teams from discrete non-differentiable rewards in a long planning horizon. Model-predictive control is robust and offers safe, dynamically feasible actions in a fast replanning framework for short horizons. We propose an algorithm that extends actor-critic model predictive control for MARL which we refer to as multi-agent actor-critic model predictive control (MA-AC-MPC). We demonstrate the capabilities of this algorithm by applying it to a multi-agent pursuit-evasion scenario. Specifically, we compare the evader team's strategy using the MA-AC-MPC model and a multi-layer perceptron model (MA-AC-MLP). The pursuer team uses augmented proportional navigation as it is accepted as an advanced adversarial control law. We also provide an example with a heterogeneous environment where a drone and omni-wheeled rover cooperate to achieve repeatable and successful landing with 100% success rate in hardware for MA-AC-MPC compared to 60% for MA-AC-MLP. We demonstrate the robustness of the proposed MA-AC-MPC algorithm in hardware for both environments.

### 2.1 核心思想
- **MA‑AC‑MPC（Multi‑Agent Actor‑Critic Model Predictive Control）**：将 **AC‑MPC**（Actor‑Critic Model Predictive Control）从单智能体扩展到多智能体协同场景。
- **层次化Actor**：第一层为**代价网络**（Cost Network），由神经网络根据观察输出MPC的可学习参数（参考轨迹、权重矩阵）；第二层为**可微分MPC**（Differentiable MPC），基于这些参数求解最优控制，生成可行的动作。
- **集中式训练 / 分布式执行**（CTDE）：在训练时使用全局状态构建中心价值函数（Critic），执行时每个智能体仅依赖本地观察和共享的MPC模块。

## 6. 论文的主要结论与发现

1. **样本效率显著提升**  
   - MA‑AC‑MPC 在仅 2 M 步即达到 70% 逃避胜率，而同等规模的 MA‑AC‑MLP 需要 4 M 步才能接近。  
   - 在无人机‑rover 着陆任务中，MA‑AC‑MPC 在训练收敛前的采样次数更少。

2. **鲁棒性优于纯学习型策略**  
   - 质量扫频显示 MA‑AC‑MPC 在质量偏离训练值 ±10 g 时仍保持 70‑90% 胜率，而 MA‑AC‑MLP 性能急剧下降（最大下降约 27%）。  
   - 这表明将低层动力学交给 MPC 可显著降低策略对模型失配的敏感性。

3. **硬件实现 100% 成功**  
   - 无人机‑rover 着陆实验中，MA‑AC‑MPC 在 5 次硬件试验中全部成功（误差 ≤ 0.075 m），MA‑AC‑MLP 仅有 3/5 成功（两次严重偏离）。  
   - 轨迹对比（Crazyflow / CrazySim / 真实硬件）表明 MA‑AC‑MPC 的仿真‑硬件迁移误差更小。

4. **训练代价更高，但推理代价可接受**  
   - MA‑AC‑MPC 每步推理约 1 ms，虽比 MLP（0.12 ms）慢约 8‑10 倍，仍满足 50 Hz 实时控制需求。  
   - 训练时间显著高于 MLP（162 h vs 18 h），但一次性离线训练成本可接受。

5. **课程学习是实现协同行为的关键**  
   - 直接随机初始位置导致逃避者无法学会协作诱导碰撞；采用 10 级递增难度的课程后，模型能够逐步掌握协同策略。

## 9. 未来研究方向

1. **GPU 加速的 MPC 求解**：将非线性 MPC 部署在 GPU 上（如使用 CUDA 加速的 OSQP 或 Proximal Methods），可显著降低训练时间和推理延迟。
2. **学习非二次代价函数**：探索基于神经网络直接输出非线性代价或约束惩罚项，增强对复杂奖励结构的建模能力。
3. **多智能体通信学习**：在 CTDE 框架下引入可学习的通信协议（如 CommNet、TarMAC），让智能体在执行时共享部分信息，改善协同效率。
4. **大规模验证**：在更大规模（如 5 vs 5、10 vs 10）的追捕‑逃避或协作搬运任务中评估样本效率和鲁棒性。
5. **安全约束集成**：将碰撞avoidance、输入饱和等硬约束硬编码到 MPC 中，而非仅通过代价塑形实现，提高安全保证。
6. **自适应课程学习**：结合元学习（Meta‑Learning）或自动课程学习（AutoCL）自动调整难度曲线，减少人工调参负担。
