## Abstract
Knowledge Distillation (KD) is a powerful tool for model compression, yet the precise mechanisms by which student models acquire feature representations remain underexplored. In this work, we analyze student feature learning using the Interaction Tensor framework. Our analysis reveals that effective KD acts as a regularizer that prunes low-frequency, sample-specific features, encouraging the student to rely on a compact set of highly reusable features. Crucially, we observe that the dataset-level confusion matrix contains structural information analogous to the teacher's "Dark Knowledge." Leveraging this insight, we propose Confusion Distillation (CD), a teacher-free self-distillation method that utilizes the model's own evolving confusion patterns as dynamic soft targets. CD achieves competitive performance on ResNet-34 and ResNet-50 for CIFAR-100, outperforming existing self-distillation methods like CS-KD and PS-KD by 1.2% while offering a computationally efficient alternative to standard KD.

## 2. 论文提出的方法论

### 核心思想

#### 2.1 Interaction Tensor 分析框架
- 采用 Jiang 等人提出的 Interaction Tensor 框架，将模型、数据、特征三者的交互关系解耦。
- 构建二进制张量 Ω ∈ {0,1}^(M×N×T)，其中 Ω_m,n,t = 1 表示模型 m 使用特征簇 t 来分类数据样本 n。
- 分析流程：
  1. 从倒数第二层提取特征，投影到 top-50 主成分
  2. 基于跨模型相关性将特征聚类为共同簇
  3. 通过阈值处理生成二进制激活张量

#### 2.2 Confusion Distillation（混淆蒸馏）方法

**核心假设**：混淆矩阵中蕴含的类间相似性结构与教师模型的软输出具有相似的监督作用。

**方法流程**：

1. **初始化阶段（Early Training）**：
   - 使用标准交叉熵损失 L_CE 进行训练
   - 原因：早期预测分布不稳定，直接使用混淆矩阵会引入噪声

2. **混淆比率更新（Confusion Ratio Update）**：
   - 切换点 t_switch 后开始使用混淆信息
   - 初始化为平滑矩阵 S：
     ```
     S_ij = 1 - ε + ε/K  (i = j，对角线)
     S_ij = ε/K           (i ≠ j)
     ```
   - 采用指数移动平均（EMA）更新：
     ```
     Q̃(t_switch - 1) = S
     Q̃(t) = μ · Q̃(t-1) + (1 - μ) · Q(t)
     ```
     其中 μ = 0.9 为动量系数

3. **损失函数**：
   - 软目标损失（基于 KL 散度）：
     ```
     L_soft^(t+1) = KL(Q̃(t)_k,· ∥ p_T) · T²
     ```
   - 总损失：
     ```
     L = α · L_soft + β · L_CE
     ```
     其中 α:β = 0.7:0.3 为最佳配比

4. **训练调度**：
   - 采用阶段交替策略：硬目标阶段（仅 L_CE）与软目标阶段（αL_soft + βL_CE）交替
   - 最佳调度比例：3:3:6:3:15（300 epochs 总训练时长）

## 6. 论文的主要结论与发现

### 核心发现 1：KD 是特征层面的正则化器
- **基线模型特征分布**：呈现长尾分布，大量低频特征（仅在少量样本中出现）
- **教师模型特征分布**：低频特征显著减少，学习特征总数更少（264 vs 531），特征激活频率更高
- **学生模型表现**：学习特征数（281）和激活频率与教师高度相似
- **解释**：教师软目标引导学生关注高频、可复用的特征，而非记忆样本特定模式

### 核心发现 2：学生采用更紧凑的特征利用策略
- **特征效率**：基线模型平均使用 ~71 个特征/样本，教师 ~59 个，学生 ~60 个
- **置信度-特征关系（CIFAR-100）**：高置信度样本反而激活更多特征（与 CIFAR-10 相反）
- **原因**：CIFAR-100 细粒度分类需要多特征协同排除竞争类（Evidence Accumulation）
- **学生策略**：对困难样本增加特征组合以积累证据，对简单样本用更少特征达到高置信度

### 核心发现 3：混淆矩阵蕴含"暗知识"
- **结构相似性**：
  - 教师平均 softmax 矩阵与基线混淆比率的余弦相似度 ~0.76
  - 皮尔逊相关系数 ~0.85
  - Jaccard 相似度 ~0.38
- **含义**：混淆矩阵不完全复制教师输出，但捕捉了粗粒度的类间相似性结构
- **本质**：混淆信息不是随机的错误记录，而是有意义的类间关系信号

### 核心发现 4：CD 方法的有效性
- **性能提升**：在 ResNet-34/50 上比 CS-KD 和 PS-KD 高出 1.2%
- **训练效率**：200 epochs 即可达到接近 300 epochs PS-KD 的性能
- **特征层面效果**：
  - CD 模型总特征数减少（336→330）
  - 平均激活频率提高（2724→3021）
  - 共享特征激活频率更高
- **机制**：通过混淆结构引导模型学习有用的特征子集，抑制无用特征
