## Abstract
To improve the real-world applicability of reinforcement learning (RL), the field of adversarially robust RL studies how to train agents under adversarial environment perturbations. In this setting, a protagonist agent optimizes a policy under environmental perturbations from an adversary, resulting in a zero-sum Markov game. When adversarially robust RL is combined with model-based RL, the adversary can target a learned transition model instead of the training environment. Extending this idea, this work introduces post-hoc robustification of deep RL agents at inference time. By using the learned model in combination with a trained nominal policy, our approach performs a robust policy improvement step. The goal is to improve robustness without any additional training of neural networks. Specifically, we utilize model-predictive control under adversarial rollouts, which are approximated via projected gradient descent within a bounded uncertainty set. Furthermore, these offline rollouts are performed while considering and mitigating out-of-distribution issues. The proposed methodology is validated by demonstrating significant improvements in robustness when the algorithm is evaluated in perturbed Gymnasium MuJoCo environments, while considering the computational limitations of the post-hoc inference setting.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究背景**：强化学习（RL）在现实世界部署时面临环境变化（如sim2real或sim2sim场景），需要具备对抗环境扰动的鲁棒性。
- **现有方法局限**：传统对抗鲁棒RL通常在训练阶段引入对抗扰动，要求修改训练流程，可能导致性能下降或训练不稳定。
- **核心问题**：能否在不修改已训练好的基于模型的强化学习（MBRL）智能体的情况下，仅在推理阶段提升其鲁棒性？
- **研究动机**：提出"后-hoc鲁棒化"（post-hoc robustification）概念，允许在推理时定义不确定性集合和鲁棒性-最优性权衡，无需重新训练神经网络。

## 2. 论文提出的方法论

### 核心思想

- 在推理时利用预训练的学习模型和原始策略，结合模型预测控制（MPC）执行"鲁棒策略改进步"。
- 核心思路：使用MPC在学习的转移模型上执行对抗性rollout，通过投影梯度下降（PGD）在有界不确定性集合内近似生成最坏情况扰动。

### 关键技术细节

- **OTC不确定性集合**：使用最优传输代价（Optimal Transport Cost）定义不确定性集合，允许直接在状态空间定义扰动半径，而非概率向量。
- **对抗状态扰动**：采用归一化PGD迭代更新状态扰动 $\delta$，最小化学习值函数 $V^{\pi}_{\psi}(s' + \delta)$：
  - 每次迭代：$\delta \leftarrow \text{Proj}_{\|\delta\|_2 \leq r_2}\left(\delta - \alpha \frac{\nabla_{s'} V^{\pi}_{\psi}(s' + \delta)}{\|\nabla_{s'} V^{\pi}_{\psi}(s' + \delta)\|_2}\right)$
  - 投影函数将扰动限制在L2球内。
- **认知不确定性减少**：为防止模型过度利用（OOD问题），使用集成方差截断rollout深度：
  - 计算集成不一致度：$\text{epistemic}(s,a) = \max_{i,j} \|p_{\theta_i}(s,a) - p_{\theta_j}(s,a)\|_2$
  - 当不一致度超过阈值 $h$ 时截断rollout。

### 算法流程

1. 在每个推理时间步 $t$，给定当前状态 $s_t$
2. 使用MPPI（模型预测路径积分控制）在状态 $s_t$ 附近优化局部策略分布 $\pi^{MPC}_{s_t}$
3. 在模型rollout中，对每步预测的下一个状态 $s'$ 添加对抗扰动 $\delta$（通过PGD）
4. 使用集成方差检测并截断可能导致OOD的rollout
5. 输出优化后的动作 $a_t \sim \pi^{MPC}_{s_t}$

## 6. 论文的主要结论与发现

- **鲁棒性提升**：提出的方法显著减少了在环境参数扰动下（质量、摩擦、阻尼等）的性能衰减，验证了推理时鲁棒化的有效性。
- **Rollout深度影响**：对于Hopper-v4，增加rollout深度N能持续提升鲁棒性，直到模型误差累积过度（N>70后崩溃）。
- **不确定性半径影响**：存在稳定的性能区间（r=0.045~0.075），过大的r会导致性能衰减。
- **OOD Mitigation必要性**：消融实验显示，无认知不确定性减少时，性能甚至低于基线，确认了处理分布外问题的关键性。
- **推理时间可控**：即使在较复杂的Hopper上，每步推理仅需0.09~0.15秒（GPU环境），适合推理时部署。

## 7. 优点

- **无需重新训练**：在推理阶段提升鲁棒性，不影响原始训练流程，避免了对抗训练带来的性能下降或训练不稳定。
- **灵活的不确定性定义**：使用OTC不确定性集合，可直接在状态空间定义扰动半径，具有更好的可解释性。
- **与现有MBRL框架兼容**：基于MBPO和MPPI，修改最小，易于集成。
- **计算效率**：使用JAX实现，支持多种GPU上的高效推理，满足推理时计算资源限制。
- **系统性消融验证**：通过多组消融实验分离了各组件（对抗扰动、OOD mitigation）的贡献。
