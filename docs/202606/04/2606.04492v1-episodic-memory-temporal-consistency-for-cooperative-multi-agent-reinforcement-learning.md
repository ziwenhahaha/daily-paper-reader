## Abstract
Cooperative Multi-Agent Reinforcement Learning (MARL) frequently suffers from severe reward sparsity and exploration bottlenecks. While episodic memory mechanisms mitigate these issues by reusing high-return trajectories, they often trap agents in local optima due to unconstrained incentive distribution and semantic representation collapse. To address this, we propose Episodic Memory Temporal Consistency (EMTC), a framework that robustly constructs and selectively leverages historical experiences. EMTC introduces two synergistic components: (1) a Temporally Consistent Semantic Embedder that integrates contrastive learning with time-conditioned state reconstruction, preventing representation collapse and enabling precise memory retrieval; and (2) a Temporal Consistency Gating Mechanism that dynamically modulates episodic incentives based on temporal consistency error. This adaptive gate filters misleading signals from pseudo-successful trajectories, effectively mitigating Q-value overestimation. We provide theoretical guarantees, establishing a strict error bound that directly links the observable temporal consistency error to the underlying trajectory optimality and representation quality. Extensive evaluations on the SMAC and GRF benchmarks demonstrate that EMTC consistently outperforms state-of-the-art baselines. Notably, compared to the strongest episodic baseline, EMTC achieves absolute win-rate improvements of up to 24% in super-hard SMAC scenarios and an average improvement of 28% across GRF tasks.

## 1. 核心问题与整体含义

**研究背景：**
- 合作式多智能体强化学习（MARL）在机器人协同控制、自动驾驶车队协同、通信网络优化等领域展现出了巨大潜力
- 然而，现实应用中面临两大核心挑战：**部分可观测性**和**复杂的协调动态**，导致严重的奖励稀疏问题和探索瓶颈

**核心问题：**
传统的情景记忆机制虽然在一定程度上复用高回报轨迹加速探索，但存在两个耦合的系统性缺陷：
1. **表示崩溃（Representation Collapse）**：标准编码器过度强调时序序列建模，压制了空间语义的判别性，导致关键的有害状态被错误映射到成功经验集群，破坏了基于距离的记忆检索机制
2. **价值过估计（Value Overestimation）**：检索歧义导致伪成功轨迹的返回值被盲目传播为内在奖励，造成Q值虚高，将策略锁死在局部最优

## 2. 论文提出的方法论

### 核心思想
EMTC（Episodic Memory Temporal Consistency）提出**时序一致性语义嵌入器**与**时序一致性门控机制**双组件协同的框架，将记忆检索与激励分配统一建模，通过时序一致性约束从根本上解决上述耦合问题。

### 关键技术细节

**（1）时序一致性语义嵌入器（TCSE）**

采用**双目标学习框架**结合状态嵌入：
- **条件去噪自编码器（dCAE）损失**：
  $$L_{dCAE} = \|H_t - f^H_\psi(f_\phi(s_t, t))\|^2 + \lambda_{rcon}\|s_t - f^s_\psi(f_\phi(s_t, t))\|^2$$
  其中 $t$ 为条件时间步，$H_t$ 为历史最高回报

- **对比学习损失（InfoNCE）**：
  $$L_{cl} = -\mathbb{E}_{(s_i,s_j)\sim P}\left[\log\frac{\exp(sim(z_i,z_j)/\tau_{cl})}{\sum_{k=1}^N\exp(sim(z_i,z_k)/\tau_{cl})}\right]$$
  正样本对来自相同时刻不同增强视图的相同状态，负样本来自不同时刻或不同状态

- **数据增强策略**：为1D MARL状态向量设计了语义保持的增强方法——随机掩码（Random Masking）和高斯噪声（Gaussian Noise），避免破坏实体的索引对齐

**（2）时序一致性门控机制（TCGM）**

基于Bellman最优性准则动态调节内在奖励：
- **时序一致性误差**：
  $$\Delta_{mem}(s,a) = |r(s,a) + \gamma H(s') - H(s)|$$
- **一致性系数（Gaussian门控）**：
  $$\beta(\Delta_{mem}) = \exp\left(-\frac{\Delta_{mem}^2}{2\tau^2}\right)$$
  高误差轨迹被大幅衰减（$\beta \to 0$），低误差轨迹保持完整激励（$\beta \to 1$）

- **一致性感知内在奖励**：
  $$\tilde{r}_p(s,a) = \beta(\Delta_{mem}) \cdot \gamma\frac{N_\xi(s')}{N_{call}(s')}\left(H(s') - \max_{a'}Q_{\theta^-}(s',a')\right)$$

- **温度参数退火策略**：
  $$\tau(t) = \max\left(\tau_{end}, \tau_{start} - \frac{\tau_{start}-\tau_{end}}{T_{decay}} \cdot t\right)$$
  其中 $\tau_{start}=1.2$, $\tau_{end}=0.6$，实现早期宽松探索、后期严格过滤的双阶段调节

### 理论保证
论文提供了严格的误差界证明：
$$\Delta_{mem} \leq \Delta_{opt} + (1+\gamma)(\epsilon_{rep} + \epsilon_{store})$$
其中 $\alpha = (1+\gamma)(\epsilon_{rep} + \epsilon_{store})$ 为表示误差与存储陈旧性导致的聚合噪声，TCSE通过L-Lipschitz连续性假设将 $\epsilon_{rep}$ 限制在 $L\delta$ 以内。

### 算法流程
1. **记忆构建**：周期性运行Algorithm 2更新情景记忆缓冲区，包含前向索引链（idx_next）保存时序因果关系
2. **时序一致性评估**：对每条新收集轨迹计算 $\Delta_{mem}$ 和 $\beta$，存入经验回放缓冲区
3. **策略优化**：使用门控TD损失 $L_\beta^\theta$ 更新联合Q网络
4. **嵌入器定期更新**：每$T_{emb}=1K$步运行Algorithm 1优化TCSE参数，并重新投影所有历史状态的嵌入，防止陈旧表示

## 8. 不足与局限

1. **温度退火启发式**：$\tau$的线性退火策略（$T_{decay}$）仍是经验设定，可能需要环境特定调优，缺乏自适应机制

2. **Lipschitz连续性假设**：理论保障依赖于$V^*$在潜在空间$L$-Lipschitz连续的假设，但标准深度网络并不天然满足此性质；高随机性稀疏奖励环境下可能存在局部不连续性

3. **计算资源依赖**：实验需A100级别GPU，对于资源受限场景可能存在部署障碍

4. **情景记忆容量限制**：1M记录容量在高维长时间任务中可能面临存储瓶颈

5. **仅验证合作场景**：论文聚焦完全合作MARL，未探索竞争性或混合博弈场景的适用性

6. **部分可观测性处理**：虽提及部分可观测性，但实验评估主要在可获取全局状态的SMAC/GRF默认设置下进行

7. **未来方向**：论文指出可探索基于滚动方差的自适应$\tau$调度，以及将Lipschitz正则化显式融入语义嵌入器
