## Abstract
We present a differentiable optimization framework for multi-agent coordination. An input is decomposed into overlapping local views, each processed by an agent that solves a convex subproblem parameterized by a neural encoder. Agents coordinate through the Alternating Direction Method of Multipliers (ADMM) with inter-agent constraints specified by a cellular sheaf. The sheaf specifies which aspects of neighboring solutions must agree, allowing for heterogeneous notions of global consensus. Backpropagating through the unrolled optimization jointly trains all components of the multi-agent system. We evaluate on maze pathfinding, image classification, and Sudoku, where agents with individually insufficient local views learn to coordinate to produce correct global outputs. On MNIST, the local-view decomposition yields improved robustness to distribution shifts relative to a standard CNN. On Sudoku, the optimization-derived structure yields markedly higher solve rates than parameter-matched MPNN baselines. Finally, the ADMM structure exposes distinct primal, consensus, and dual state variables, opening the coordination dynamics to direct analysis and intervention -- a property unavailable in standard message-passing architectures.

- **核心问题**：传统神经网络采用单体式架构，将输入作为统一整体处理。然而自然界中的智能往往来自具有有限局部视野的多个智能体之间的协调。该论文研究如何在每个智能体只能观察到输入的局部 patch 的情况下，通过学习实现全局协调。
- **研究背景**：受自然界群体智能启发，结合分布式优化（ADMM）和细胞丛理论（cellular sheaf），提出一种可端到端训练的多智能体协调框架。
- **核心挑战**：每个智能体的局部信息不足以独立完成任务，必须通过通信达成全局一致。

## 2. 方法论：核心思想、关键技术细节、算法流程

### 核心思想
- 将任务建模为约束优化问题，通过 ADMM 分解为局部子问题，由细胞丛（cellular sheaf）定义智能体间的协调约束。
- 整个流程完全可微分，可通过反向传播联合训练编码器、ADMM 层和解码器。

### 关键技术细节

**（1）输入分解与智能体设置**
- 输入图像/数据被分解为 $N$ 个重叠的局部视图，每个视图对应一个智能体。
- 智能体构成图结构 $G=(V,E)$，边定义通信拓扑。

**（2）ADMM 三步迭代**
- **x-update（局部优化）**：每个智能体独立求解局部凸子问题
  $$x_i^{k+1} = \arg\min_{x_i} f_i(x_i) + \frac{\rho}{2}\|x_i - z_i^k + u_i^k\|^2$$
  其中 $f_i$ 由编码器参数化，支持二次型、二次型+ℓ1 等形式。

- **z-update（丛扩散/共识）**：通过 sheaf Laplacian 执行共识约束
  $$z^{k+1} = \arg\min_z \frac{\rho}{2}\|x^{k+1} - z + u^k\|^2 \quad \text{s.t.} \ Fz = 0$$
  $F$ 为 coboundary 矩阵，$L_F = F^\top F$ 为 sheaf Laplacian。

- **u-update（对偶累积）**：
  $$u^{k+1} = u^k + x^{k+1} - z^{k+1}$$

**（3）细胞丛约束**
- 限制映射（restriction maps）$F_{i\to e}$ 定义相邻智能体在哪些投影维度上必须达成一致。
- 支持硬约束（$Fz=0$）或软惩罚（$\frac{\gamma}{2}\|Fz\|^2$）。

**（4）编码器与解码器**
- 编码器：共享 MLP，将局部视图映射为局部目标函数的参数（$Q_i, q_i, \lambda_i$ 等）。
- 解码器：将最终状态 $x_i^K$ 和局部视图映射为局部预测，聚合为全局输出。

### 算法流程概述
1. 输入分解为局部视图 → 编码器生成优化参数
2. 循环 $K$ 次 ADMM 迭代：x-update（并行）→ z-update（丛扩散 $T$ 步）→ u-update
3. 解码器生成预测 → 与真值计算损失 → 反向传播更新所有组件

## 6. 主要结论与发现

- **Sheaf-ADMM 有效性**：在三项任务上均成功实现全局协调，各智能体通过局部信息不足的输入协作产生正确输出。
- **显著优于 MPNN 基线**：Sudoku 解决率达 92.6%（参数匹配 MPNN 仅 10.7%），Maze 在极低维度（$d_v=10$）下仍达 99.9% 解决率。
- **局部视图分解的优势**：MNIST 上对分布偏移（padding、patch dropout、噪声）表现出更强的鲁棒性，远超标准 CNN。
- **尺寸泛化能力**：迷宫模型训练于 19×19，可零样本泛化至约 2 倍分辨率（39×39）。
- **组件重要性**：
  - ℓ1 正则化在 Maze 和 Sudoku 中关键（二次型+ℓ1 解决率 99.8%/92.5% vs 仅二次型 38.9%/15.8%）
  - 共轭梯度法（CG）显著优于梯度下降（MNIST 98.5% vs 52.2%）
  - 限制映射的 LoRA 调制对 Maze 任务至关重要（8.9% → 99.8%）
- **协调动力学可解释**：ADMM 的三状态变量（$x, z, u$）允许直接分析残差分布和对偶累积过程。

## 7. 优点

- **理论扎实**：基于 ADMM 和细胞丛理论的严谨数学框架，具有收敛保证和可解释性。
- **可微分优化**：通过展开 ADMM 迭代实现端到端训练，所有组件联合优化。
- **异构共识**：细胞丛允许不同智能体只在特定投影空间达成一致，而非全状态匹配，更灵活。
- **稀疏通信**：仅邻居间通信，计算高效，适合大规模部署。
- **状态可解释**：显式的原始变量 $x$、共识变量 $z$、对偶变量 $u$ 分离，便于分析和干预（优于黑箱 MPNN）。
- **强泛化性能**：局部编码+共享权重实现尺寸/分布外泛化。
- **通用架构**：适用于空间网格和非空间图结构（如数独约束图）。

## 8. 不足与局限

- **任务可分解性假设**：核心归纳偏置是任务可分解为重叠局部子问题，当全局预测依赖非局部统计或无法表示为低维重叠变量时可能失败。
- **图结构匹配风险**：若智能体图未反映真实依赖结构（重叠太少或错位），协调效果受限。
- **限制映射容量瓶颈**：静态共享映射在高维任务中可能需要较大 $d_v, d_e$ 才能有效（表 5 显示 $d_v$ 必须显著大于 $d_e$）。
- **迭代深度限制**：性能随 ADMM 迭代数 $K$ 先升后降，深层展开存在梯度退化风险。
- **凸性约束**：潜在迭代为凸优化，虽有理论保证但可能限制表达能力；编码器/解码器处理非凸性。
- **算力信息缺失**：未报告硬件配置和训练时长，难以评估实际部署可行性。
- **对比基线有限**：主要与 MPNN 比较，未与更多先进方法（如 Transformer、Graph Transformer）对比。
