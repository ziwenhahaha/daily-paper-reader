## 摘要
为大型语言模型配备显式技能已成为一种有前景的范式，使自主智能体能够解决复杂任务。智能体技能本质上可分为用于广泛认知迁移的通用技能和用于动态执行的任务特定技能。然而，现有的基于技能的强化学习（RL）方法通常迫使我们在两种极端之间做出刚性选择：完全外部化会产生难以承受的上下文开销，而完全内化则面临过拟合和知识冲突的风险。为解决这一困境，我们提出Skill0.5，这是一种新型智能体RL框架，通过将通用技能内化与任务特定技能利用相结合来明确区分技能处理方式。Skill0.5由一个动态的难度感知路由器驱动，将任务流式分配到不同的掌握层级以应用定制化优化策略：对于困难任务，它通过特权蒸馏将通用技能内化以构建认知基础，而对于简单任务则使用诊断探针来惩罚捷径并强制执行任务特定技能利用。在ALFWorld和WebShop上的实验表明，Skill0.5优于基于记忆和基于技能的RL基线方法，在分布内和分布外场景中均实现了性能提升。

## Abstract
Equipping large language models with explicit skills has emerged as a promising paradigm for enabling autonomous agents to solve complex tasks. Agent skills can be inherently divided into general skills for broad cognitive transfer and task-specific skills for dynamic execution. However, existing skill-based reinforcement learning (RL) methods typically force a rigid choice between full externalization, which incurs prohibitive context overhead, and full internalization, which risks overfitting and knowledge conflicts. To address this dilemma, we propose Skill0.5, a novel agentic RL framework that explicitly differentiates skill treatments by combining general skill internalization with task-specific skill utilization. Driven by a dynamic, difficulty-aware router, Skill0.5 streams tasks into distinct mastery tiers to apply tailored optimization strategies: it internalizes general skills via privileged distillation to build a cognitive foundation for hard tasks, while using diagnostic probing on easy tasks to penalize shortcuts and enforce specific skill utilization. Experiments on ALFWorld and WebShop demonstrate that Skill0.5 outperforms both memory-based and skill-based RL baselines, yielding performance improvements across both in-distribution and out-of-distribution scenarios.

## 2. 论文提出的方法论

### 核心思想
Skill0.5提出了一种统一的智能体RL框架，通过**难度感知路由器**动态地将任务分配到三个不同的掌握层级，并为每个层级应用定制的优化策略，实现通用技能内化与任务特定技能利用的联合优化。

### 关键技术细节

#### Phase-1：难度感知路由
- **任务分桶**：基于经验任务通过率（pass rate）将任务分配到不同层级：
  - **Hard层**：通过率 $p_i = 0$（完全失败）
  - **Medium层**：$0 < p_i \leq \eta_t$（存在部分成功但低于动态阈值）
  - **Easy层**：$p_i > \eta_t$（高通过率）
- **动态阈值**：使用跨步滑动窗口平均计算 $\eta_t = \frac{1}{\min(W,t)}\sum_{j=t-\min(W,t)+1}^{t}\bar{p}_j$，更鲁棒地处理批次内有限任务量。

#### Phase-2：分层优化

**Hard层 - 通用技能内化（特权蒸馏）**：
- 使用**特权提示** $c_{priv}^t = S_G \cup K_t(x_i)$（包含通用技能）指导重新采样成功的教师轨迹 $\tau^{(2)}$。
- 通过令牌级**Jensen-Shannon散度（JSD）**将教师行为蒸馏到学生策略（仅有标准提示 $c_{std}^t$），使模型无需显式依赖通用技能上下文即可执行基础推理。
- 损失函数：$L_{hard} = \frac{1}{|T|}\sum_{\tau \in T}\frac{1}{|\tau|}\sum_{k=1}^{|\tau|}JSD(\pi_\theta^{(t,k)},\pi_\theta^{(s,k)})$

**Medium层 - 能力强化（标准GRPO）**：
- 使用Phase-1采样的标准轨迹，通过组内标准化优势进行标准GRPO优化。
- 优势计算：$A_i^{(g)} = \frac{R_i^{(g)} - mean(R_i)}{std(R_i)}$

**Easy层 - 反捷径利用（对比诊断探测）**：
- 在无技能提示 $c_{none}^t = \emptyset$ 下采样诊断轨迹，获得基线通过率 $p_{none}^i$。
- 计算**利用增益** $u_i = p_i - p_{none}^i$，衡量技能对成功的因果贡献。
- 构建任务级利用优势 $A_{ui} = \frac{u_i - \bar{u}_t}{\sigma_u}$，与轨迹级优势组合形成复合优势 $\hat{A}_i^{(g)} = A_i^{(g)} + A_{ui}$。
- 通过负偏移惩罚捷径学习行为（当 $u_i < \bar{u}_t$ 时），强制模型忠实利用任务特定技能。

### 总体优化目标
$L = L_{hard} + L_{medium} + L_{easy}$

### 推理阶段
仅使用任务特定技能 $c_t = K_t(x)$，通用技能已内化到模型参数中。
