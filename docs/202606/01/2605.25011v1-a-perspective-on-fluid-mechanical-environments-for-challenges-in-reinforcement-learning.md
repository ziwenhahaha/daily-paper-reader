## Abstract
We consider the challenge of developing agents that efficiently interact with high-dimensional, evolving environments, towards a view of practical reinforcement learning (RL) agents interacting with open worlds, of which they witness and affect only a small part. We argue that canonical fluid mechanics problems, and their simulations, present a compelling testbed for the development of such methods. These problems arise in nonlinear instabilities, where small disturbances can grow to transform the dynamics of a system. Nonlinear instabilities represent several open scientific challenges with industrial applications -- the droplet breakup of a liquid jet, mixing at an interface between two fluids, and the appearance of unusually tall rogue waves in the ocean. In these settings, agents may leverage preserved representations across the changing dynamics to learn efficiently.   We present two problem descriptions of agents interacting with a fluid mechanical environment, and describe the state and action spaces, and reward functions, for these agents. For these examples, we specify the aspects of the environment which are nonstationary and the preserved invariances. We note Dedalus and JAX-CFD as open-source simulators that can be used for the development of reinforcement learning methods (Burns et al., 2016; Kochkov et al., 2021)) We demonstrate the use of Dedalus for environment generation by creating RL agents that learn to navigate in a stationary environment that is simulated using Dedalus. This sets the stage for future development of RL agents that learn to meaningfully interact with simulated environments that represent scientific challenges in natural and industrial flows.

## 2. 方法论

### 核心思想
利用流体不稳定性中**不变守恒律**（Navier-Stokes 方程）与**高度变化的流场结构**之间的对比，构建 RL 智能体需要学习"跨动态迁移"的挑战场景。智能体可利用已知物理约束来指导行为策略的泛化。

### 关键物理模型
Navier-Stokes 方程（不可压缩流）：

$$\nabla \cdot \mathbf{u} = 0 \quad \text{（质量守恒）}$$
$$\mathbf{u}_t + (\mathbf{u} \cdot \nabla)\mathbf{u} = -\frac{\nabla p}{\rho} + \nu\nabla^2\mathbf{u} \quad \text{（动量守恒）}$$

其中 $\mathbf{u}$ 为速度场，$p$ 为压力，$\rho$ 为密度，$\nu$ 为粘度系数。

### 两个问题描述

#### （1）过渡管道流中的智能体
| 要素 | 描述 |
|------|------|
| **环境** | 圆管中充分发展的抛物线速度剖面（$u = U_0(1-(r/R)^2)\mathbf{e}_z$），通过注入扰动触发向湍流的转变 |
| **状态空间** | $S \equiv [\mathbf{u}, p]$（三维速度和压力场） |
| **动作空间** | $A \equiv \dot{\theta}$（移动障碍物的方向变化率），障碍物为直径 $\delta < R$ 的刚性小球 |
| **奖励函数** | $r = \|\mathbf{u} - U_0(1-(r/R)^2)\mathbf{e}_z\|^2$（在给定时间范围 $T$ 内最大化对平均流剖面的扰动） |
| **非平稳性来源** | 扰动引发流场结构的高度变化（向湍流转变） |
| **保持的不变性** | Navier-Stokes 方程描述的质量和动量守恒 |

#### （2）演化的细胞流中的智能体
| 要素 | 描述 |
|------|------|
| **环境** | 二维细胞流：$\omega = -2U_0\cos(x)\cos(y)\exp(-2\nu t)$，其中 $\omega = \nabla \times \mathbf{u}$ |
| **状态空间** | $S \equiv [\omega, \theta]$（涡量场 + 粒子朝向） |
| **动作空间** | $A \equiv \dot{\theta}$（改变朝向） |
| **奖励函数** | $R \equiv y(s') - y(s)$（沿 $y$ 方向的位移增量，即向上游动） |
| **非平稳性来源** | 指数衰减因子 $\exp(-2\nu t)$ 导致流场随时间演化 |
| **保持的不变性** | Navier-Stokes 方程、流场的周期性结构、粘度等材料属性 |

## 6. 主要结论与发现

1. **流体不稳定性作为 RL 测试平台**：非线性不稳定性在流体中普遍存在，且具备高度变化的流场结构与守恒定律（不变表示）并存的特征，非常适合用来研究智能体如何在非平稳环境中高效学习。

2. **非平稳环境的挑战性**：微小扰动即可触发流场结构的根本性转变，智能体需在状态空间高度变化的情况下做出决策，远超传统 RL 中平稳马尔可夫环境的假设。

3. **可利用的不变性**：Navier-Stokes 方程、流场的周期性、粘度等物理约束在动态转变中保持不变，智能体可利用这些先验知识构建更稳健的策略。

4. **仿真工具的可行性**：Dedalus 可用于生成符合物理规律的 RL 训练环境，Q-learning 在简单离散场景下可学到有意义的导航策略。

5. **策略随环境结构演化**：学习到的游动策略随背景流场相对强度的变化而改变，表明智能体能够感知并利用流场的结构性信息。

6. **未来方向**：基于 Dedalus 构建更复杂的非平稳流体环境；探索学习代理模型作为高速仿真器。

## 8. 不足与局限

1. **实验规模有限**：
   - 仅使用表格 Q-learning，未尝试现代深度 RL 算法（如 PPO、SAC、TD3 等）。
   - 训练回合数较少（1000 回合），难以评估算法在复杂环境中的可扩展性。
   - 缺乏系统性超参数敏感性分析。

2. **环境覆盖不足**：
   - 论文提出的两个核心问题（非平稳管道流和细胞流）实际上**未在实验部分实现**，实验仅验证了一个等效的静止细胞流场景。
   - 缺乏对非线性不稳定性中真实物理过程（如湍流转变、混合层增长）的 RL 交互实验。

3. **对比基线薄弱**：仅与朴素策略（固定方向游动）对比，未与业界标准 RL 方法或其他领域（如控制理论中的 PID、MPC）进行对比。

4. **算力信息缺失**：未明确报告 GPU 型号、数量、训练时长等关键资源信息，难以评估方法的可复现性和计算成本。

5. **仿真器选择局限**：Dedalus 使用光谱方法，在涉及复杂几何边界（如弯曲管道、非结构化网格）时可能面临局限性；JAX-CFD 的适用范围也有待进一步验证。

6. **奖励函数设计单一**：两个问题的奖励函数均为简单的扰动最大化或位移差，未涉及多目标优化（如能效、稳定性约束等）。

7. **缺乏真实环境验证**：所有实验均在仿真环境中进行，未涉及真实流体实验数据的迁移或 sim-to-real 迁移的探讨。

8. **泛化性论证不足**：未系统验证智能体在训练域外（out-of-distribution）的非平稳流场中的泛化能力，而这是实际应用的核心需求。
