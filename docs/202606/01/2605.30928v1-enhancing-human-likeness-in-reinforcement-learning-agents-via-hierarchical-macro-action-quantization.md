## 摘要
类人智能体一直是人工智能的长期目标。尽管表现出色，大多数强化学习（RL）智能体仍然是奖励驱动的，且常常表现出与人类不同的行为，限制了可解释性和可靠性。在本工作中，我们提出了一种新颖的类人强化学习框架，该框架在最大化奖励的同时，预测与人类行为高度一致的动作序列。具体而言，我们使用一种层级宏动作量化方法（称为 HiMAQ）将人类演示编码为宏动作，该方法包含两个连续的向量量化层。较低层的量化将输入动作映射到细粒度子动作簇，而较高层的量化将这些子动作簇聚合成动作簇。在 D4RL 基准上进行的广泛评估表明，我们的层级方法优于非层级基线（MAQ），在保持与先前强化学习智能体相当或更好的成功率的同时，实现了更高的类人相似度分数。这些改进在各种强化学习算法的集成中得以推广，包括 IQL、SAC 和 RLPD。

## Abstract
Human-like agents are a long-standing goal of artificial intelligence. Despite strong performance, most reinforcement learning (RL) agents remain reward-driven and often exhibit behaviors that differ from humans, limiting interpretability and reliability. In this work, we introduce a novel human-like RL framework that predicts action sequences closely aligned with human behaviors while maximizing rewards. Specifically, we encode human demonstrations into macro actions using a hierarchical macro action quantization approach (termed HiMAQ) consisting of two successive levels of vector quantization. The lower quantization level maps input actions to fine-grained subaction clusters, while the higher quantization level aggregates these subaction clusters into action clusters. Extensive evaluations on the D4RL benchmarks show that our hierarchical approach outperforms the non-hierarchical baseline (MAQ), achieving better human-likeness scores while maintaining comparable or better success rates than previous RL agents. The improvements generalize across integrations with various RL algorithms, namely IQL, SAC, and RLPD.

## 2. 论文提出的方法论

**核心思想：**
提出 HiMAQ（Hierarchical Macro Action Quantization，层级宏动作量化），通过两级向量量化层次对人类演示进行编码：
- **低层量化**：将输入动作映射到细粒度子动作簇（subaction prototypes）
- **高层量化**：将这些子动作簇聚合成粗粒度动作簇（action prototypes）

**关键技术细节：**

1. **两级码本结构**：
   - 子动作码本 $Z = \{z_j\}_{j=1}^{\alpha K}$，包含 $\alpha K$ 个子动作原型
   - 动作码本 $A = \{a_i\}_{i=1}^{K}$，包含 $K$ 个动作原型
   - $\alpha$ 为控制子动作码本大小的比例参数

2. **两阶段量化流程**：
   - **第一阶段**：$q_t^Z = z_{j^*}$，其中 $j^* = \arg\min_j \|e_t - z_j\|_2$
   - **第二阶段**：$q_t^A = a_{i^*}$，其中 $i^* = \arg\min_i \|q_t^Z - a_i\|_2$

3. **训练损失函数**：
   - 重建损失：$L_{recon} = \|m_t - \hat{m}_t\|^2$
   - 承诺损失：$L_{commit}^Z = \|e_t - sg[q_t^Z]\|^2$，$L_{commit}^A = \|q_t^Z - sg[q_t^A]\|^2$
   - 码本损失：$L_{codebook}^Z = \|sg[e_t] - q_t^Z\|^2$，$L_{codebook}^A = \|sg[q_t^Z] - q_t^A\|^2$
   - 总损失：$L = L_{recon} + \lambda_{commit}(L_{commit}^Z + L_{commit}^A) + lambda_{codebook}(L_{codebook}^Z + L_{codebook}^A)$
   - 实验设置：$\lambda_{commit} = 0.25$，$\lambda_{codebook} = 1.0$

4. **RL策略执行**：
   - 策略 $\pi_\theta$ 以当前状态 $s_t$ 为输入，输出子动作码本上的分布
   - 采样获得 $q_t^Z$，据此检索 $q_t^A$，最后解码生成宏动作

## 6. 主要结论与发现

1. **轨迹相似度显著提升**：HiMAQ 在 DTW 和 WD 指标上均优于 MAQ
   - IQL：平均 DTW_s 从 0.63 提升到 0.65
   - SAC：平均 DTW_s 从 0.56 提升到 0.63
   - RLPD：平均 DTW_s 从 0.56 提升到 0.62

2. **任务成功率保持或提升**：
   - HiMAQ+IQL 在 Hammer 任务上从 0.00 提升到 0.87
   - HiMAQ+SAC 成功率从 0.28 提升到 0.29
   - HiMAQ+RLPD 成功率从 0.52 提升到 0.56

3. **人类感知验证**：
   - 图灵测试：HiMAQ+RLPD 达到 43% 欺骗率（最佳）
   - 排序测试：HiMAQ+RLPD 获得 54% 胜率，仅次于真实人类（71%）

4. **宏动作长度 H=9 为最优**：性能从 H=1 到 H=9 提升，之后趋于平稳或下降

5. **泛化性验证**：改进在三种不同 RL 算法中均保持一致
