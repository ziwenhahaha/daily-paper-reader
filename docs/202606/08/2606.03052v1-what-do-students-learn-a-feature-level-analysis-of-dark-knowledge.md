## 摘要
知识蒸馏（KD）是一种强大的模型压缩工具，然而学生模型获取特征表示的具体机制仍缺乏深入研究。本文在交互张量框架下对学生特征学习进行分析。我们的分析表明，有效的KD充当一种正则化器，剪枝低频、样本特定的特征，鼓励学生依赖一组紧凑且高度可复用的特征。关键发现是，数据集层面的混淆矩阵蕴含与教师模型的“暗知识”相似的结构信息。基于此，我们提出混淆蒸馏（CD），一种无教师的自蒸馏方法，利用模型自身不断演化的混淆模式作为动态软目标。CD在ResNet-34和ResNet-50的CIFAR-100任务上取得竞争力的性能，较现有的自蒸馏方法如CS-KD和PS-KD提升了1.2%，同时在计算上比标准KD更为高效。

## Abstract
Knowledge Distillation (KD) is a powerful tool for model compression, yet the precise mechanisms by which student models acquire feature representations remain underexplored. In this work, we analyze student feature learning using the Interaction Tensor framework. Our analysis reveals that effective KD acts as a regularizer that prunes low-frequency, sample-specific features, encouraging the student to rely on a compact set of highly reusable features. Crucially, we observe that the dataset-level confusion matrix contains structural information analogous to the teacher's "Dark Knowledge." Leveraging this insight, we propose Confusion Distillation (CD), a teacher-free self-distillation method that utilizes the model's own evolving confusion patterns as dynamic soft targets. CD achieves competitive performance on ResNet-34 and ResNet-50 for CIFAR-100, outperforming existing self-distillation methods like CS-KD and PS-KD by 1.2% while offering a computationally efficient alternative to standard KD.

## 2. 方法论

### 2.1 特征分析框架——交互张量（Interaction Tensor）

作者采用 Jiang 等人提出的交互张量框架来分析模型、数据与特征的三元交互。该方法的核心步骤如下：

1. **特征提取**：将模型 penultimate 层的输出投影到前 50 个主成分上。
2. **特征聚类**：基于跨模型相关性将特征聚类为共享簇，以处理随机初始化差异。
3. **张量构建**：通过阈值化特征激活，构建二进制张量 Ω ∈ {0,1}^(M×N×T)，其中 Ω_{m,n,t}=1 表示模型 m 利用特征簇 t 对数据样本 n 进行分类。

### 2.2 混淆蒸馏（Confusion Distillation, CD）

#### 核心思想

教师模型的软输出包含类间相似性结构（暗知识），而学生模型自身的混淆矩阵同样编码了类似的结构——即哪些类别容易被混淆。因此，无需外部教师，模型可以利用自身的混淆模式作为软目标来引导特征学习。

#### 关键技术细节

**① 混淆比矩阵定义**

对于真实类别 j，混淆比矩阵 Q 的行向量 Q_{j,:} 表示模型对类别 j 的样本在各类别的预测分布：
$$Q_{j,i} = \frac{1}{|D_j|} \sum_{n \in D_j} \mathbb{I}(\hat{y}_n = i), \quad i, j = 1, \ldots, K$$

**② 指数移动平均（EMA）平滑**

为避免早期训练中混淆比的不稳定，采用 EMA 策略维护平滑的混淆矩阵：
$$\tilde{Q}^{(t)} = \mu \tilde{Q}^{(t-1)} + (1 - \mu) Q^{(t)}, \quad \mu = 0.9$$

初始平滑矩阵 S 采用类似标签平滑的定义：
$$S_{ij} = \begin{cases} 1 - \epsilon + \frac{\epsilon}{K}, & i = j \\ \frac{\epsilon}{K}, & i \neq j \end{cases}$$

**③ 损失函数**

在过渡阶段后，总损失结合软标签损失与硬标签损失：
$$L = \alpha \cdot L_{\text{soft}} + \beta \cdot L_{\text{CE}}$$

其中软标签损失定义为：
$$L_{\text{soft}} = \text{KL}(\tilde{Q}^{(t)}_{k,:} \parallel p_T) \cdot T^2$$

训练策略采用分阶段切换：早期仅使用交叉熵损失，后期在硬标签与基于混淆的软标签之间交替切换，最优配置为软:硬 = 0.7:0.3，切换比例为 3:3:6:3:15（总计 300 epochs）。

## 6. 主要结论与发现

### 6.1 特征层面的洞察

1. **特征频率分布**：Baseline 模型依赖大量低频（样本特定）特征，而 KD 训练的学生模型显著抑制低频特征，学习更少但激活频率更高的特征集。特征数量：|F_baseline| = 531 vs |F_student| = 281 vs |F_teacher| = 264。

2. **特征-置信度关系**：CIFAR-100 的细粒度分类要求多个特征协同累积证据（Evidence Accumulation），高置信度样本激活更多特征。KD 使学生模型在高置信区域更高效（使用更少特征），在低置信区域利用更多特征缓解不确定性。

3. **共享特征分析**：学生与教师共享更多特征（170 vs 132），表明 KD 效果源于特征对齐而非学习全新特征。

### 6.2 混淆矩阵的结构意义

- 教师平均 softmax 矩阵与学生混淆比矩阵的**余弦相似度 ≈ 0.76**，**皮尔逊相关系数 ≈ 0.85**，**Jaccard 相似度 ≈ 0.38**，表明两者捕获了粗粒度的类间相似性结构。
- 混淆矩阵并非随机噪声，而是可作为类间关系的有效软目标。

### 6.3 CD 的效果

- CD 减少了总激活特征数（336 → 330），抑制低频特征，提高特征复用效率（平均激活频率：2724 → 3021）。
- **性能提升**：在 ResNet-34/50 上较 CS-KD 和 PS-KD 提升约 **1.2%**，且在 200 epochs 下即可达到与 PS-KD 300 epochs 相近的性能。

## 7. 优点

1. **特征层面的创新分析**：首次利用交互张量框架定量分析 KD 对学生特征表示的影响，揭示了"特征正则化"机制，填补了输出层分析之外的空白。

2. **重新定义混淆矩阵的角色**：将混淆矩阵从评估工具提升为知识蒸馏信号，提供了"利用自身不确定性来改进自身"的全新视角。

3. **计算效率**：无需训练教师模型，CD 作为自蒸馏方法在计算成本上远低于标准 KD，同时保持竞争力性能。

4. **可复现性强**：提供了清晰的算法流程（EMA 平滑、阶段切换策略）和最优超参数配置，便于社区复现。

5. **实验覆盖广泛**：涵盖多种架构、多组超参数配置，并与多种自蒸馏方法进行公平对比。

## 8. 不足与局限

1. **数据集单一**：实验仅在 CIFAR-100 上进行，缺乏对更大规模数据集（如 ImageNet）或真实应用场景的验证，方法的泛化性有待考察。

2. **架构局限**：主要使用 ResNet 系列，对其他架构（如 ViT、MLP-Mixer）的适用性未经验证。

3. **噪声混淆的风险**：论文也指出过长的混淆训练阶段会导致性能下降，因为错误混淆信号会被强化。这意味着 CD 对超参数（切换时机、持续时长）敏感，需要仔细调优。

4. **缺乏理论支撑**：虽然实验观察丰富，但缺少对"为什么混淆矩阵能编码类间相似性"的理论分析或形式化证明。

5. **性能提升幅度有限**：相比 baseline 提升约 1-2%，虽然优于部分自蒸馏方法，但未与完整 KD（带真实教师）进行直接对比。

6. **评估指标单一**：仅使用 Top-1/5 准确率，未考察模型校准、泛化误差等深度学习社区关心的其他指标。
