## 摘要
迭代式生成建模技术，如流匹配，为复杂的离线强化学习（RL）行为建模提供了强有力的工具。本工作提出了一种基于先前数据的离策略强化学习算法，通过训练流策略实现。我们的思路从“扩展”马尔可夫决策过程（MDP）框架出发，将每一次流细化步骤视为MDP中的单独动作。为在该框架下实现离策略RL，我们采用两种技术：通过“逆向”流生成虚拟on-policy轨迹，使该框架兼容先前数据；以及采用偏差-方差缩减技术来缓解离策略RL中的时域诅咒。我们将得到的算法称为逆向Q学习（RQL）。相对于以往的基于流的RL方法，RQL具有多项优势：不受时间反向传播的影响，能够更好地利用已学习的价值函数，并直接训练完整且富有表现力的流策略。通过在50个具有挑战性的模拟机器人任务上的实验，我们表明RQL在平均离线RL性能上优于最先进的基于流的离线强化学习算法。

## Abstract
Iterative generative modeling techniques, such as flow matching, provide powerful tools to model complex behaviors for effective offline reinforcement learning (RL). In this work, we propose a new off-policy RL algorithm that trains a flow policy based on prior data. Our idea starts from the "expanded" Markov decision process (MDP) framework, which treats individual flow refinement steps as separate actions in an MDP. To enable off-policy RL within this framework, we apply two techniques: we generate virtual on-policy trajectories (by "reversing" flows) to make this framework compatible with prior data, and we apply a bias-and-variance reduction technique to mitigate the curse of horizon in off-policy RL. We call the resulting algorithm Reversal Q-learning (RQL). RQL has several advantages over previous flow-based RL methods: it does not suffer from backpropagation through time, makes better use of the learned value function, and directly trains the full, expressive flow policy. Through our experiments on 50 challenging simulated robotic tasks, we show that RQL leads to the best average offline RL performance compared to state-of-the-art flow-based offline RL algorithms.

## 2. 论文提出的方法论

### 核心思想
利用**扩展MDP（Expanded MDP）**框架，将流策略的每个细化步骤视为独立的MDP动作，然后通过**流逆转（Flow Reversal）**生成虚拟轨迹，并应用**多步返回（Multi-step Returns）**缩短有效时域。

### 关键技术细节

#### 2.1 扩展MDP框架
- **增强状态空间**：$\tilde{S} = S \times R^d \times \{0, 1, \ldots, F-1\}$，包含原始状态 $s$、部分生成的动作 $x$、以及流时间步 $f$
- **扩展转换**：每$F$步查询一次原始环境，其他时间仅更新部分生成的动作
- **扩展奖励**：仅在每个流轨迹结束时给予奖励（$f = F-1$时）

#### 2.2 流逆转生成虚拟轨迹
对于数据集中的每个转移 $(s, a, r, s')$，通过逆转流ODE计算虚拟轨迹：
$$\frac{d}{df}\theta(s, x, f) = -v(s, \theta(s, x, f), f)$$
使用Euler法求解：
$$x_{f-1} \leftarrow x_f - v(s, x_f, f)$$
从 $x_F = a$ 反向计算到 $x_0$，得到完整的扩展MDP转移序列。

#### 2.3 无偏多步返回
标准TD学习的问题：时域扩展 $T \times F$ 倍导致偏差累积。

**解决方案**：由于虚拟轨迹是确定性且on-policy的，多步返回是无偏且零方差的：
$$L(Q) = \mathbb{E}_{\tilde{\tau}}\left[\left(Q((s, x_f, f), a_f) - r - \gamma \max_{a'}\bar{Q}((s', x'_0, 0), a')\right)^2\right]$$

该技术将有效价值时域从 $T \times F$ 减少到平均 $T$。

#### 2.4 期望回归估计最大值
使用IVL-like损失避免OOD查询：
$$L(V) = \mathbb{E}_{\tilde{\tau}}\left[\ell_\kappa^{\frac{1}{2}}\left(V(s, x_f, f) - (r + \gamma V(s', x'_0, 0))\right)\right]$$
其中 $\ell_\kappa^{\frac{1}{2}}(x) = |\kappa - \mathbb{I}(x > 0)|x^2$ 是期望损失。

#### 2.5 策略学习
DDPG风格的策略更新 + 行为正则化：
$$L(v) = -\mathbb{E}_{\tilde{\tau}}[V(s, x_f + v(s, x_f, f), f+1)] + \alpha \mathcal{L}_{BC}(v)$$

## 7. 优点

### 方法创新
1. **避免BPTT**：通过扩展MDP框架，将迭代生成转化为MDP步骤，天然规避了梯度通过时间反向传播的问题
2. **充分利用价值梯度**：相比零阶方法（加权回归、拒绝采样），RQL使用一阶信息，更高效
3. **无偏多步返回**：利用流的确定性和逆转性质，实现零方差的价值估计
4. **简洁实现**：无需复杂的层次结构或蒸馏流程

### 实验设计
1. **大规模基准**：50个任务覆盖多种机器人操作场景
2. **全面对比**：18种基线方法，包括同一框架的变体（BDPO）和消融版本（TFQL）
3. **细致消融**：明确验证每个组件的贡献

### 实用性
