## Abstract
In this paper, we study the power allocation for an integrated sensing and communication (ISAC) system which tracks a mobile target. We first model the problem as a Markov decision process, and then tackle it with a soft actor-critic (SAC) based deep reinforcement learning (DRL) approach. We also combine a Dirichlet policy, which naturally produces normalized continuous actions under random target motion. To exploit different features of sensing and communication operations, we carefully design a reward function such that the system can dynamically control power allocation to conserve resources. The simulation results demonstrate that the proposed scheme enhances tracking performance compared to other baselines while sustaining communication performance.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 2.1 核心思想

将 ISAC 系统的功率分配问题建模为马尔可夫决策过程（MDP），采用基于软演员-评论家（SAC）的深度强化学习（DRL）方法进行求解，并结合 Dirichlet 策略以自然产生归一化的连续动作。

### 2.2 系统模型

- **场景设置**：基站在配备 $M$ 天元的均匀线性阵列下，同时服务于单天线用户并跟踪移动目标。
- **发射信号模型**（第 $n$ 时隙）：$x_n = \sqrt{p_{c,n}} u c_n + \sqrt{p_{s,n}} w_{t,n} s_n$，其中 $p_{c,n}$、$p_{s,n}$ 分别为通信和感知功率，满足 $p_{c,n} + p_{s,n} \leq P_{\text{max}}$。
- **通信模型**：接收信号为 $y_{c,n} = \sqrt{p_{c,n}} h_c^H u c_n + \sqrt{p_{s,n}} h_c^H w_{t,n} s_n + n_{c,n}$，可达通信速率为 $C_n = \log_2\left(1 + \frac{p_{c,n} |h_c^H u|^2}{p_{s,n} |h_c^H w_{t,n}|^2 + \sigma_c^2}\right)$。
- **感知模型**：通过空间-时间匹配滤波器处理回波信号，SNR 表达式为 $\text{SNR}_n = \frac{p_{s,n} \kappa^4 \gamma_{t,n} \gamma_{r,n} \sigma_\alpha^2}{M \sigma_s^2}$。

### 2.3 目标跟踪模型

- **运动模型**：目标状态向量 $\xi_n = [x_n, y_n, \dot{x}_n, \dot{y}_n]^T$，状态转移方程为 $\xi_n = F_n \xi_{\hat{n}} + \omega_{\hat{n}}$，其中 $F_n$ 由有效感知间隔 $g_n T_s$ 决定。
- **测量模型**：非线性测量函数 $h(\xi_n) = [d_n, v_n, \theta_n]^T$ 用于从回波中估计距离、径向速度和角度。
- **PCRB（后验克拉美-罗界）**：作为跟踪可靠性的度量指标，用于衡量估计精度的下界。

### 2.4 优化问题

- **目标函数**：$\max_{p_{c,n}, p_{s,n}} C_n$
- **约束条件**：
  - $p_{c,n} > 0$，$p_{s,n} \geq 0$
  - $p_{c,n} + p_{s,n} = P_{\text{max}}$
  - $\Gamma_n \leq \bar{\Gamma}$（PCRB 阈值约束）
- **关键挑战**：实际 PCRB 仅在获取当前测量后才可用，但功率分配必须提前确定。

### 2.5 SAC-based DRL 功率分配算法

#### 2.5.1 状态（Observation）

$s_n = [\hat{\Gamma}_n, p_{s,\hat{n}}, \Gamma_n^p, \bar{p}, g_n, |h_c^H u|^2, n]$，包含最近一次 ISAC 模式的实际 PCRB、感知功率、当前预测 PCRB、平均功率分配、感知间隔和通信信道增益等信息。

#### 2.5.2 动作（Action）

$a_n = \left[\frac{p_{c,n}}{P_{\text{max}}}, \frac{p_{s,n}}{P_{\text{max}}}\right]$，定义为功率分配比例，基于 simplex 约束的归一化动作空间。

#### 2.5.3 奖励函数

$$
r_n = \begin{cases}
\omega_c (1 - \exp(-0.1 C_n)) - \omega_T g_n, & \text{if } \hat{\Gamma}_n \leq \bar{\Gamma} \\
\omega_s (\bar{\Gamma} - \hat{\Gamma}_n) - \omega_T g_n, & \text{otherwise}
\end{cases}
$$

- 当 PCRB 约束满足时，给予通信速率相关奖励（采用指数变换以稳定训练）。
- 当 PCRB 约束不满足时，施加 PCRB 惩罚。
- 引入感知间隔惩罚项 $\omega_T g_n$ 以抑制过度跳过感知。

#### 2.5.4 Dirichlet 策略

采用 Dirichlet 分布参数化的随机策略：
$$\pi_\phi(a_n|s_n) = \frac{1}{B(z_\phi(s_n))} \prod_{i=1}^N a_{z_i}(s_n)^{-1}$$
- 该策略自然生成 simplex 约束内的连续动作，无需额外的变换或雅可比校正。
- 策略梯度估计在重参数化技巧下是无偏的。

#### 2.5.5 双 Q 网络训练

- **评论家网络损失函数**：$J_{Q(\theta_i)} = E_{(s_n,a_n)\sim D}\left[\frac{1}{2}(Q_{\theta_i}(s_n, a_n) - \hat{Q}(s_n, a_n))^2\right]$
- **目标 Q 值**：$\hat{Q}(s_n, a_n) = r_n + \gamma E_{a_{n+1}\sim\pi_\phi(\cdot|s_{n+1})}[ \min_{i=1,2} Q_{\bar{\theta}_i}(s_{n+1}, a_{n+1}) - \beta \log \pi_\phi(a_{n+1}|s_{n+1})]$
- **演员网络损失函数**：$J_\pi(\phi) = E_{s_n\sim D, a_n\sim\pi_\phi}[ \beta \log(\pi_\phi(a_n|s_n)) - \min_{i=1,2} Q_{\theta_i}(s_n, a_n)]$
- **熵温度自动调节**：在训练过程中自动调整 $\beta$ 以维持策略熵接近目标值（设为 -2）。

## 6. 论文的主要结论与发现

1. **PCRB 约束可行性**：所提 SAC-Dirichlet 方案在几乎整个 episode 中持续将 PCRB 保持在阈值以下，而其他方法存在间歇性的违规峰值（CVX-based 方案有约 40% 的不可行时隙）。

3. **通信与感知的平衡**：所提方案能够在维持跟踪精度的同时，逐步提升通信速率；而"SAC with predicted state"方案未能有效平衡二者。
4. **功率分配动态性**：基于 PCRB 信息的设计使智能体能够学习感知与通信性能之间的关系，实现资源高效利用。

## 7. 优点

1. **MDP 建模与 SAC 算法**：将功率分配问题建模为 MDP，采用 SAC 算法处理部分观测环境，支持随机策略探索，有利于应对跟踪可靠性违规。
2. **Dirichlet 策略**：自然产生 simplex 约束内的归一化连续动作，避免了 softmax 或 tanh 变换带来的偏差，策略梯度估计无偏。
3. **精心设计的奖励函数**：通过区分 PCRB 约束满足/不满足两种情况，结合感知间隔惩罚和指数变换的通信奖励，有效平衡了感知与通信需求。
4. **PCRB 作为状态信息**：将最近的 PCRB 纳入状态表示，量化了目标状态估计的可靠性，避免了仅依赖预测状态导致的决策不确定性。
5. **离线训练、在线应用**：智能体在离线环境中学习跨时隙的相关性，在线部署时可直接决策，提高了实用性。
