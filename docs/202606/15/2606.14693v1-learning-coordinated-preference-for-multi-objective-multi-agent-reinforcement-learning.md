## Abstract
Cooperative multi-objective multi-agent reinforcement learning (MOMARL) models team decision making under multiple, potentially conflicting objectives. In this setting, conflicts arise not only across objectives but also across agents with different observations, roles, and contributions. We propose Preference Coordinated Multi-agent Policy Optimization (PCMA), which learns coordinated agent-specific preferences to enable complementary trade-offs among agents. Theoretically, we formulate cooperative MOMARL as a team-optimal game and show that, under suitable conditions, preference diversity can induce team improvement through a first-order improvement decomposition. Experiments on multiple cooperative MOMA environments and a practical traffic-control scenario show that PCMA improves both performance and trade-off coordination.

## 论文总结

### 1. 核心问题与整体含义（研究动机和背景）

- **问题领域**：合作式多目标多智能体强化学习（Cooperative Multi-Objective Multi-Agent RL, MOMARL），建模多个智能体在多个可能冲突的目标下进行团队决策的问题。
- **核心挑战**：
  - 冲突不仅存在于各目标之间，还存在于具有不同观察、角色和贡献的智能体之间。
  - 现有方法使用统一的标量化权重向量，难以捕捉智能体间的偏好差异。
  - 当所有智能体以相同方式权衡目标时，可能产生竞争行为（如两车在无信号交叉口都优先效率导致碰撞，或都优先安全导致僵局）。
- **研究动机**：提出智能体特定的协调偏好学习，使不同智能体能聚焦不同的目标权衡方向，从而减少行为冲突并提升团队整体性能。

### 2. 论文提出的方法论

#### 核心思想
- 将合作式MOMARL建模为**团队最优均衡问题**，目标是学习一个协调的偏好档案（preference profile），其诱导的偏好条件均衡能最大化团队目标。
- 核心创新：**偏好协调机制**，允许智能体通过学习到的差异化偏好沿Pareto前沿分布，而非强制使用相同偏好。

#### 关键技术细节
- **偏好定义**：偏好向量 $p_i \in \Delta^{K-1}$ 定义智能体i的效用 $U_i(\theta; p_i) = J_{team}(\theta) + p_i^\top J_i(\theta)$。
- **随机偏好规划器**：每个智能体配备基于Dirichlet分布的随机偏好规划器 $\phi_\psi(o_i)$，根据局部观察采样偏好，实现条件协调。
- **偏好条件策略**：策略 $\pi_\theta(a_i | o_i, p_i)$ 以偏好为额外输入，支持基于偏好的条件化行为。
- **双 Critic 设计**：
  - 团队 Critic：估计稀疏团队优势 $A_{team}$
  - 个体向量 Critic：估计多目标引导优势 $A_{ind}^i$
- **偏好多样性正则化**：$D_\alpha = \mathbb{E}[D_p]$ 鼓励智能体避免偏好坍缩到同一方向。

#### 算法流程（PCMA）
1. 初始化偏好规划器、演员网络、团队Critic和向量Critic
2. 对每个智能体：基于局部观察采样偏好 $p_i \sim Dir(\phi_\psi(o_i))$，采样动作 $a_i \sim \pi_\theta(\cdot | o_i, p_i)$
3. 执行联合动作，存储经验
4. 估计团队优势和个体向量优势
5. 更新Critic网络（最小化 $L_{critic}$）
6. 更新演员网络（PPO目标，$A_{U_i} = A_{team} + \lambda p_i^\top A_{ind}^i$）
7. 更新偏好规划器（PPO + 多样性正则化 $- \lambda_1 D_\alpha$）

3. **Pareto前沿覆盖**：协调偏好使团队联合rollout返回能够覆盖近似Pareto前沿的不同区域
4. **理论验证**：偏好多样性贡献正的一阶团队改进（定理4.2），均衡跟踪保证偏好优化可逐步进行（定理4.6）
5. **消融发现**：

6. **实际应用验证**：在CARLA交通控制场景中，PCMA改进SAC基线，合作设置下成功率69.6%、碰撞率30.4%、吞吐量1716车辆/小时
