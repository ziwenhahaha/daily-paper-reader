## Abstract
Reinforcement learning with verifiable rewards (RLVR), especially Group Relative Policy Optimization (GRPO), has been widely used to improve reasoning in large language models. However, outcome-level rewards provide only sparse supervision, and group-relative advantages vanish when all sampled trajectories for a prompt are either correct or incorrect. On-Policy Self-Distillation (OPSD) offers dense token-level guidance, but its token preferences are not necessarily aligned with trajectory correctness; empirical diagnostics show that OPSD signals behave differently on correct and incorrect rollouts, with teacher-positive and teacher-negative gap signals exhibiting different noise profiles. These diagnostics are conducted under an OPSD-style privileged teacher context for analysis only, whereas CAST training uses answer-free self-teacher scoring.Motivated by these observations, this work proposes CAST, an answer-free self-distillation method for GRPO-style RLVR. CAST keeps the verifier-grounded GRPO objective, but uses a stop-gradient self-teacher to shape token-level advantages according to trajectory correctness. Unlike prior self-distilled RLVR methods, CAST does not require reference-solution-conditioned teacher scoring, keeps the self-teacher log-probability gap active throughout training, and applies bidirectional local advantage sign reversal: teacher-negative tokens in correct trajectories can receive negative token-level advantages, while teacher-positive tokens in incorrect trajectories can receive bounded positive local advantages. For zero-variance all-correct and all-wrong groups, CAST assigns bounded sign-constrained base advantages, so these otherwise zero-gradient groups can contribute verifier-signed token feedback. Experiments on mathematical reasoning show that CAST improves RLVR training while retaining a lightweight, verifier-grounded trajectory-level objective.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题领域**：大语言模型（LLM）的强化学习训练，特别是基于可验证奖励的强化学习（RLVR）与组相对策略优化（GRPO）。
- **核心痛点**：
  - **稀疏监督**：GRPO 依赖结果级（outcome-level）奖励，仅提供轨迹级别的二元正确/错误信号，缺乏细粒度的 token 级指导。
  - **零方差组失效**：当同一提示的所有采样轨迹全部正确或全部错误时，组内奖励方差为零，归一化优势消失，导致梯度失效，模型无法从这些数据中学习。
  - **现有自蒸馏方法的局限**：OPSD（On-Policy Self-Distillation）虽提供密集 token 级信号，但其教师-学生 gap 信号未考虑轨迹正确性，teacher-positive 和 teacher-negative 信号在不同正确性轨迹上表现差异显著（噪声特征不同），不适合直接作为统一的 token 级监督。

## 2. 论文提出的方法论

### 核心思想
- **CAST**（Non-Privileged Clipped Asymmetric Self-Teaching with Advantage Flipping）是一种**无需特权答案的自蒸馏方法**，在保留验证器锚定（verifier-grounded）GRPO 目标的基础上，引入 answer-free 的 stop-gradient 自教师来根据轨迹正确性塑形 token 级优势。

### 关键技术细节

1. **Answer-Free 自教师评分**：
   - 自教师仅能访问 prompt 和学生生成的响应前缀，**不使用真实答案或参考答案**。
   - 教师-学生 gap 定义为：
     $$g_{i,t} = \log \pi_\phi(y_{i,t} | x, y_{i,<t}) - \log \pi_{\theta_{old}}(y_{i,t} | x, y_{i,<t})$$

2. **零方差组的有界基优势**：
   - **MIXED 组**：使用标准 GRPO 归一化优势 $A_i = \frac{r_i - \bar{r}}{\sigma_r + \epsilon}$
   - **ALL CORRECT 组**：分配固定的正向基优势 $B_i = +b_{correct}$（默认 $b_{correct} = 1$）
   - **ALL WRONG 组**：分配固定的负向基优势 $B_i = -b_{wrong}$（默认 $b_{wrong} = 1$）
   - 作用：防止零方差组成为无效的零梯度组。

3. **正确性感知的 Token 加权与不对称裁剪**：
   - 将 gap 转换为带符号的 token 权重：$z_{i,t} = \exp(\text{sign}(B_i) \cdot g_{i,t})$
   - **不对称裁剪**：
     - 正向基轨迹（$B_i > 0$）：裁剪至 $(\ell^+, u^+) = (0.8, 1.05)$，限制增强幅度
     - 负向基轨迹（$B_i < 0$）：裁剪至 $(\ell^-, u^-) = (0.95, 1.2)$，允许更强的抑制

4. **双向局部优势符号翻转（Bidirectional Advantage Sign Reversal）**：
   - **正确轨迹中的负向翻转**：若 $B_i > 0$ 且 $g_{i,t} < 0$（teacher-negative token），则生成负向 token 级优势 $A^{down}_{i,t}$，抑制局部不良 token（如冗余答案模板）。
   - **错误轨迹中的正向翻转**：若 $B_i < 0$ 且 $g_{i,t} > 0$（teacher-positive token），则生成**有界**的正向局部优势 $A^{up}_{i,t}$，在受较小正向上限约束下给予局部有用的部分推理正向信用。

5. **最终 Token 优势裁剪**：最终优势裁剪至 $[A_{min}, A_{max}] = [-1.2, 1.2]$。

6. **优化目标**：使用 DAPO/GRPO 的裁剪重要性比率目标，token 级优势由 CAST 构造的 $\tilde{A}_{i,t}$ 驱动，梯度仅流经策略比率项，不流经教师-学生 gap 或裁剪操作。

## 6. 论文的主要结论与发现

### 核心结论
1. **CAST 显著优于所有基线方法**：在三个模型规模上，CAST 均取得最强的 Avg@16 和具竞争力的 Pass@16。
2. **零方差组贡献有效信号**：禁用零方差分支后性能明显下降（Qwen3-4B AIME24 从 41.25% 降至 34.38%），证明 ALL CORRECT/ALL WRONG 组的固定基优势设计有效。
3. **双向优势翻转是关键**：移除符号翻转后性能大幅下降（AIME24 从 41.25% 降至 25.83%），说明 correct-path 的局部抑制和 incorrect-path 的局部credit 对于精细化学习至关重要。
4. **Answer-Free 自教师评分足够**：使用参考答案上下文的 ablation 反而略差于 answer-free 设置，证明无需特权教师即可获得有效信号。
5. **不对称裁剪优于对称裁剪**：两种对称设置均弱于默认的不对称配置。
6. **持续的自教师塑形优于衰减**：shaping cutoff 越晚关闭，性能越好，说明教师-gap 应作为持续信号而非早期正则化。

## 7. 优点

1. **方法创新性强**：
   - 首次提出双向局部优势符号翻转机制，突破"正确轨迹所有 token 均为正"的直觉

   - 不对称裁剪策略反映了对 teacher-negative 信号稳定性的实证观察

2. **理论与实践结合紧密**：
   - 从 OPSD token-shift 诊断出发，揭示 teacher-positive/negative 信号在不同正确性轨迹上的噪声差异
   - 诊断结果直接指导设计决策（如bounded翻转受较小正向上限约束）

3. **实验设计严谨**：
   - 多规模验证、完整消融、分布外测试、随机种子敏感性分析一应俱全
   - 消融实验采用控制变量法，每次仅改变一个组件
   - 训练诊断数据（如零方差组占比 51.5%）支撑方法必要性论证

4. **实用性**：
   - 保留 verifier-grounded 的 GRPO 目标，无需额外奖励模型
   - answer-free 自教师设计避免信息泄露
   - 计算开销仅增加约 25%（相比 GRPO）

5. **洞察深刻**：
   - 指出 teacher-negative 信号规模更大、更稳定，适合作为强抑制项
   - 揭示 teacher-positive 信号在错误轨迹中相对分散，不适合作为统一正向监督

## 8. 不足与局限

1. **规模覆盖不足**：
   - 仅在 Qwen3 系列（1.7B-8B）上验证，未测试更大规模模型（如 70B+）
   - 仅使用 LoRA 微调，未在完全参数更新场景下验证

2. **任务范围有限**：
   - 专注于数学推理任务，对代码生成、自然语言推理等 RLVR 适用场景未探索
   - 显式链式思维生成模式未测试

3. **超参数敏感性**：
   - 依赖验证调优的裁剪和 token-gap 超参数
   - 不对称裁剪边界 $(\ell^+, u^+)$、$(\ell^-, u^-)$ 和基优势 $b_{correct}$、$b_{wrong}$ 的默认值可能非最优

4. **推理成本**：
   - CAST 生成的输出平均更长（如 Qwen3-4B：9,779 vs 5,242 tokens），增加推理成本

5. **诊断范围有限**：
   - 机制分析主要在 300 步 Qwen3-4B 设置下进行
   - OPSD token-shift 诊断使用特权教师上下文，与 CAST 实际训练设置不同

6. **零方差分支的解释性**：
   - ALL CORRECT/ALL WRONG 组的固定基优势本质上是有界验证器签名的塑形启发式，而非无偏的组相对优势估计器
   - 其理论性质和收敛保证未深入分析
