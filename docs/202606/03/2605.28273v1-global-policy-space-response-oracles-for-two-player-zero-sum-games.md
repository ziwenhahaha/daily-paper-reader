## 摘要
策略空间响应预言机（PSRO）框架通过使用深度强化学习（DRL）迭代扩展受限策略集合，将均衡计算扩展到大型零和博弈。核心挑战在于在有限计算资源下构建一个规模较小的策略种群，使其诱导的博弈能够良好近似完整博弈。现有PSRO变体通常基于受限博弈收益计算得到的元策略的最佳响应来扩展种群，这可能导致扩展效率低下，全球改进有限。我们提议通过直接评估扩展后种群的质量来引导种群扩展。具体而言，我们采用种群可利用性（PE）来衡量受限策略集合对完整博弈的代表性，并引入两阶段探索—选择框架，在扩展过程中显式最小化PE。我们将该框架实例化为全局PSRO，一种基于DRL的实际算法，通过参数共享的条件神经网络高效生成候选响应并估计PE。在多种双人零和博弈上的实验表明，全局PSRO实现了更低的可利用性，并在显著更少的策略迭代次数下逼近纳什均衡，优于以往的PSRO方法。

## Abstract
The Policy-Space Response Oracles (PSRO) framework scales equilibrium computation to large zero-sum games by iteratively expanding a restricted strategy set using deep reinforcement learning (DRL). A central challenge is to construct, under limited computational budgets, a small strategy population whose induced game well approximates the full game. Existing PSRO variants typically expand the population using best responses to meta-strategies computed from restricted-game payoffs, which can lead to inefficient expansions that provide limited global improvement. We propose to guide population expansion by directly evaluating the post-expansion population quality. Specifically, we adopt Population Exploitability (PE) to measure how well a restricted strategy set represents the full game, and introduce a two-phase exploration--selection framework that explicitly minimizes PE during expansion. We instantiate this framework as Global PSRO, a practical DRL-based algorithm that efficiently generates candidate responses and estimates PE via parameter-sharing conditional neural networks. Experiments across multiple two-player zero-sum games show that Global PSRO achieves lower exploitability and approximates Nash equilibria with significantly fewer policy iterations than prior PSRO methods.

### 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：在大规模零和博弈中计算均衡（如纳什均衡）是一个核心挑战，因为策略空间巨大，难以穷举分析。
- **现有方法局限**：PSRO（Policy-Space Response Oracles）框架通过迭代扩展受限策略集合来逼近均衡，但现有 PSRO 变体（如使用受限博弈 NE、AlphaRank、PRD、Uniform 等元策略求解器 MSS）**仅依赖受限博弈的收益矩阵**来引导种群扩展。这种方式可能导致扩展效率极低——在某些对抗构造的博弈中，可能需要加入几乎整个策略空间才能收敛（如 Theorem 3.2 所证明）。

- **研究目标**：用全局信息引导种群扩展，使每次添加的新策略都能最大化地提升种群对完整博弈的近似质量。

### 2. 论文提出的方法论

#### 2.1 核心思想

- **种群可利用性（Population Exploitability, PE）**：定义 $PE(\Pi_r; G) = \min_{\sigma \in \Delta(\Pi_r)} \epsilon(\sigma)$，衡量受限策略集 $\Pi_r$ 对完整博弈的近似质量——即在种群上能找到的最小可利用性。
- **优化目标**：选择下一个添加的策略 $\pi^*$，使得扩展后的种群 PE 最小化：
  $$\pi^* \in \arg\min_{\pi \in B(\Delta(\Pi_r))} PE(\Pi_r \cup \{\pi\}; G)$$
  这与标准 PSRO 仅响应单一元策略的本质区别在于：**直接评估种群级别的改进**。

#### 2.2 两阶段探索-选择框架

- **阶段一（探索）**：生成多样化的候选响应池。
  - 构建元策略池 $S_t = \{\sigma_k\}_{k=1}^K \subset \Delta(\Pi_r)$，包含基于受限博弈的 MSS 输出（如 Nash 均衡）和从均匀 simplex 采样的混合策略（Dirichlet(1)）。
  - 训练一个**条件探索器** $\pi_\theta(a|s,\sigma)$，通过 DRL 同时针对所有 $\sigma_k$ 训练，最大化总体收益 $J(\theta) = \frac{1}{K}\sum_{k=1}^K U(\pi_\theta(\cdot|\sigma_k), \sigma_k)$。
  - 生成候选集 $\Pi_k^+ = \Pi_r \cup \{\pi_\theta(\cdot|\sigma_k)\}$。

- **阶段二（选择）**：评估每个候选扩展的 PE，选择最优者。
  - 使用 RM-BR（后悔最小化最佳响应）程序估计 $PE(\Pi_k^+; G)$：RM 维护混合策略 $\rho_k$，BR 学习器跟踪 $\rho_k$。
  - 引入**正则化 PE 评分**以应对近似误差：
    $$cPE_{reg}^k = (1 - \hat{p}_k)\left(cPE(\Pi_r; G) - U(\beta_k, \rho_r)\right) + cPE_k$$
    其中 $\hat{p}_k = \rho_k(\pi_k)$，当新策略在混合中占比很小时给予更大惩罚。
  - 选择 $k^* = \arg\min_k cPE_{reg}^k$，将选中的候选响应 $\pi_{k^*}$ 和对应的 PE 评估 BR $\beta_{k^*}$ 同时加入种群：$\Pi_r \leftarrow \Pi_r \cup \{\pi_{k^*}, \beta_{k^*}\}$。

#### 2.3 算法流程（Global PSRO）

详见 Algorithm 1，主要步骤包括：形成元策略池 → 训练条件探索器生成候选响应 → 用条件评估器 $\beta_\psi(a|s,\sigma)$ 配合 RM-BR 估计各候选的 PE → 正则化评分选择 → 更新种群和收益矩阵。

#### 2.4 理论分析

- **Theorem 4.2**：在精确预言机假设下，若某个 RGB-MSS 能保证 PSRO 收敛到 NE，则满足基础包含和保守平局决断条件的两阶段框架也能在有限 PSRO 风格迭代中收敛。
- **Theorem 4.3**：给出了在对抗构造博弈中的非渐进迭代上界 $E[T^*] \leq \min\{2 + \frac{2S}{1-(1-c)^{K-1}}, N-1\}$，表明丰富的探索和 PE 选择可以绕过最坏情况行为，特别是当均衡支持大小 S 较小时效果显著。

### 6. 论文的主要结论与发现

1. **Global PSRO 一致性优于基线**：在 Liar's Dice、Leduc Poker、所有 Goofspiel 变体上，Global PSRO 实现了更低的 PE（更好的均衡逼近），仅在极小的 Kuhn Poker 上略逊于受限博弈 NE（因为受限博弈很快接近完整博弈，额外全局信息增益有限）。
2. **显著减少迭代次数**：理论分析表明，在对抗构造的博弈中，Global PSRO 可以在 ~2-4 次 PSRO 风格迭代内达到 NE，而标准 PSRO 可能需要接近 N 次迭代。
3. **PE 驱动的选择机制是主要改进来源**：消融实验显示，"仅利用"和"随机选择"变体导致性能大幅下降，验证了使用全局 PE 信息进行种群扩展选择的核心贡献。
4. **参数共享使多候选方案可行**：条件神经网络使得同时生成和评估多个候选响应成为可能，显著降低了计算开销。
5. **多样性增强可进一步提升**：在 Global PSRO 的探索阶段加入多样性正则化（KL 散度），在 Leduc Poker 等游戏上取得了额外的性能提升。
6. **正则化 PE 评分有效**：正则化机制缓解了 PE 估计误差带来的选择偏差。

### 8. 不足与局限

- **游戏类型受限**：仅考虑双人零和博弈，未扩展到多人一般和博弈。论文也明确指出，PE 在多人设置下不再具有博弈论上的鞍点结构，计算更困难。
- **PE 估计的计算开销**：即使有参数共享和近似策略，大规模博弈的 PE 估计仍可能成为瓶颈。
- **实验规模**：虽然涵盖了多种博弈，但最大仅到 Goofspiel（13 张牌），在超大规模实际问题（如真实世界策略博弈）上的效果未知。
- **超参数敏感性**：算法引入了多个超参数（K 候选数、RM 步数、多样性权重等），不同游戏可能需要不同调优。
- **收敛速度的实证验证**：理论给出了期望迭代上界，但实际收敛速度在不同类型博弈上的方差可能较大。
- **对手不存在于当前种群时的 BR 近似**：在候选生成阶段，若元策略诱导的 BR 在种群中已存在，则评估 BR 可能引入无关策略，需要进一步考虑去重或重复处理机制。
