## 摘要
在策略自蒸馏（on-policy self-distillation）是指语言模型以特权上下文为条件来监督自身生成，这是一种为稀疏奖励强化学习提供密集监督的有前景方法。事实上，它可以实例化为一种辅助的全词汇表学生-教师反向Kullback-Leibler散度损失。因此，我们提出SDPG，一种自蒸馏策略梯度框架，它结合了组相对验证器优势与归一化标准差、精确的全词汇表在策略自蒸馏，以及参考策略KL正则化。实验结果表明，SDPG在稳定性和性能上均优于RLVR和自蒸馏基线模型。代码可在 https://github.com/lauyikfung/SDPG 获取。

## Abstract
On-policy self-distillation, where a language model conditions on privileged context to supervise its own generations, is a promising source of dense supervision for sparse-reward reinforcement learning. Actually, it can be instantiated as an auxiliary full-vocabulary student-to-teacher reverse Kullback-Leibler divergence loss. We therefore propose SDPG, a self-distilled policy-gradient framework that combines group-relative verifier advantages with normalized standard deviation, exact full-vocabulary on-policy self-distillation, as well as reference-policy KL regularization. Empirically, SDPG improves stability and performance over RLVR and self-distillation baselines. The code is available at https://github.com/lauyikfung/SDPG.

## 1. 论文的核心问题与整体含义

### 研究背景

- **RLVR（可验证奖励强化学习）** 在大语言模型（LLM）的复杂推理任务（如数学、代码生成）中表现优异，已成为后训练推理模型的标准范式
- **核心问题**：RLVR 面临两大局限：
  - **稀疏奖励问题**：序列级奖励在整个生成过程中仅在末尾提供反馈，导致 token 级信用分配不足
  - **训练不稳定性**：训练早期大量负优势（A < 0）会导致模型基础语言能力退化

### 现有方案及其局限

- **Dr.GRPO、DAPO、GSPO** 等方法通过双剪切阈值和序列级优势解决了部分不稳定性问题，但**稀疏性仍未解决**
- **On-Policy 自蒸馏（OPD）** 方法通过教师模型提供密集 token 级监督信号，但：
  - 需要外部大型教师模型，内存负担大
  - 异构教师信号可能损害训练平滑性

### 研究动机

- 自蒸馏（self-distillation）方法中，教师和学生是同一模型在不同上下文条件下的不同分支，可消除对外部教师模型的依赖
- **核心观察**：自蒸馏可形式化为全词汇表学生-教师反向 KL 散度损失，其学生侧梯度具有局部策略梯度形式
- **关键问题**：纯自蒸馏缺乏验证器机制，可能强化局部合理但全局错误的轨迹

## 2. 论文提出的方法论

### 核心思想

SDPG（Self-Distilled Policy Gradient）将**自蒸馏**建模为**全词汇表反向 KL 散度损失**，并与**验证器基础 RLVR** 相结合，形成统一策略梯度框架。

### 关键技术细节

#### （1）问题建模

- **学生分布**（无特权上下文）：$p_t(a) = \pi_\theta(a | x, y_{<t})$
- **教师分布**（有特权上下文 c）：$q_t(a) = \pi_\theta(a | c, x, y_{<t})$
- **全词汇表反向 KL 损失**：$\ell_{OPD}^{i,t}(\theta) = D_{KL}(p_{i,t} \| SG[q_{i,t}]) = \sum_{a \in V} p_{i,t}(a) \log \frac{p_{i,t}(a)}{SG[q_{i,t}(a)]}$

#### （2）梯度恒等式（Proposition 3.1）

- 在固定 rollout 前缀且教师分支 stop-gradient 的条件下，反向 KL 全词汇表 OPD 损失的**学生侧梯度**等价于：
  $$\nabla_\theta L_{OPD,t}(\theta) = -E_{a \sim \bar{p}_t} \left[ Adist_t(a) \nabla_\theta \log p_t(a) \right]$$
  其中 $Adist_t(a) = SG \left[ \bar{D}_t - \log \frac{\bar{p}_t(a)}{\bar{q}_t(a)} \right]$，为中心化 log-ratio token 优势

#### （3）总目标函数

$$L_{SDPG}(\theta) = L_{out}(\theta) + \beta(k) L^+_{OPD}(\theta) + \alpha L_K(\pi_\theta, \pi_{ref})$$

- **$L_{out}$**：基于组相对优势的 REINFORCE 风格回报损失
- **$L^+_{OPD}$**：正优势门控的全词汇表 OPD 损失
- **$L_K$**：参考策略的 KL 正则化（支持 UFKL 和 URKL 两种形式）

#### （4）关键稳定化技术

- **正优势门控（Positive Advantage Gating）**：$m_i = 1[A_{out}^{(i)} > 0]$，仅在验证器认可的轨迹上应用蒸馏信号，避免在错误轨迹上强化局部合理 token
- **β 调度器**：warmup-decay 机制
  - 训练初期：$\beta$ 从 0 逐渐增加，避免早期噪声干扰探索
  - 训练末期：$\beta$ 衰减，逐步移除特权信号，鼓励学生独立探索

#### （5）算法流程（Algorithm 1）

1. 对每个训练批次，采样 prompts 和特权上下文
2. 从当前策略 rollout 生成 G 个响应
3. 计算二进制验证器奖励和组相对优势
4. 对每个响应的每个 token 位置：
   - 计算学生分布 $p_{j,i,t}$ 和特权教师分布 $q_{j,i,t}$
   - 计算全词汇表 OPD 损失
5. 应用正优势门控和 β 调度器
6. 最小化总 SDPG 损失更新策略

2. **训练稳定性**：SDPG 在训练早期即建立准确率优势，且维持较高策略熵，避免 RLSD 中观察到的熵崩溃现象
3. **收敛效率**：SDPG 比 GRPO 提前数百步达到高奖励平台期

### 详细发现

| 指标 | GRPO | RLSD | SDPG-URKL | SDPG-UFKL |
|------|------|------|-----------|-----------|

### 消融实验关键发现

- **移除 OPD（β=0）**：失去早期训练准确率优势，证实特权蒸馏是快速收敛的主要驱动力
- **移除 KL 正则化（α=0）**：准确率略高或持平，但响应长度严重缩短（约 2000 tokens），熵值上升，策略偏离连贯推理模式
- **OPCD 崩溃**：纯自蒸馏（无验证器）在 1.7B 模型上训练至 250 步后准确率急剧下降至 0.02，响应长度崩溃至 300 tokens 以下

## 7. 优点

### 方法创新

1. **理论贡献**：严格证明自蒸馏的全词汇表反向 KL 损失具有等效策略梯度形式，建立了蒸馏与策略优化的深层联系
2. **框架统一性**：将 RLVR 的验证器奖励与 OPD 的密集监督信号有机结合，而非简单叠加
3. **稳定化机制**：正优势门控和 warmup-decay β 调度器有效防止蒸馏信号过约束或欠约束

### 技术优势

4. **精确全词汇表**：使用完整词汇表 KL 而非采样近似，提高梯度估计精度
5. **KL 正则化形式**：支持 UFKL 和 URKL 两种变体，提供灵活的实现选择
6. **无需外部教师**：自蒸馏机制消除对大型外部教师模型的依赖，显著降低内存开销

### 实验设计亮点

7. **公平对比**：与 GRPO、RLSD 等主流方法对比，具有实际参考价值
8. **多规模验证**：在 4B 和 1.7B 两种规模上验证，展示方法的可扩展性
9. **充分消融**：清晰分离各组件贡献，验证设计决策的有效性
