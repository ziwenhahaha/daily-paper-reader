## Abstract
A central goal of large language model (LLM) research is to build agentic systems that can plan, act, and adapt through sustained interaction with dynamic environments. While recent LLM-based agents exhibit impressive contextual reasoning, their long-horizon decision-making remains fragile, often suffering from objective drift, where goals and plans drift over extended interactions. We introduce Multi$^2$, a hierarchical multi-agent decision-making framework that explicitly decomposes agent behavior into complementary roles. A high-level agent (System 1) focuses on context-aware sub-goal generation using supervised fine-tuning (SFT), while a low-level agent (System 2) executes atomic actions through offline-to-online reinforcement learning (RL) in interactive environments. This separation enables stable long-horizon control, mitigates objective drift, and allows efficient adaptation. Across diverse interactive environments, Multi$^2$ consistently outperforms strong agentic baselines, demonstrating improved robustness and coordination in multi-turn interaction. Beyond performance, we introduce and release three hierarchical benchmark datasets, filling a long-standing gap in training and evaluating hierarchical decision-making for LLM-based agents.

## 1. 论文核心问题与整体含义

### 研究动机
大型语言模型（LLM）智能体在长时间交互中面临两大核心挑战：

- **目标漂移（Objective Drift）**：随着交互展开，智能体的目标和计划逐渐偏离原始意图，小的执行错误在多轮交互中累积，导致状态与初始意图逐步失调。
- **令牌效率低下**：智能体依赖不断增长的交互历史来维持意图，导致推理时令牌消耗过高。

### 背景问题
现有层级规划方法虽能通过子目标分解降低规划失败率，但执行层面仍会在长horizon rollouts 中因复合错误和约束密集型动作而崩溃。此外，许多层级智能体通过提示实现角色专业化，随着上下文积累，角色边界会模糊，使智能体易受约束违反和循环影响。

## 2. 论文方法论

### 核心思想
Multi² 提出一种层级多智能体决策框架，将智能体行为显式分解为互补角色：

- **System 1（高层智能体）**：负责上下文感知的子目标生成，通过监督微调（SFT）训练
- **System 2（低层智能体）**：负责原子动作执行，通过离线到在线强化学习（RL）训练

### 关键技术细节

**层次化数据集构建**：
- System 1 数据集 D_{sys1} = { (K_n, {(o_h, g^h)}_{h=1}^H) }，包含任务描述与观察-子目标序列对
- System 2 数据集 D_{sys2} = { (g^{(i)}, {ξ_t^{(i)}}_{t=1}^{M^{(i)}}) }，包含子目标条件下有序的低层转移序列

**System 1 训练（SFT）**：
- 使用标准交叉熵目标：L_φ = -E_{(K_n,o_t,g^h)∼D_{sys1}}[log π_φ(g^h | o_t; K_n)]
- 旨在保持任务级一致性

**System 2 离线训练**：
- Q函数训练（TD误差）：L_ω = E_{(g^h,o_t,a_t,r_t,o_{t+1})∼D_{sys2}}[(r_t + γV_ψ(o_{t+1}; g^h) - Q_ω(o_t, a_t; g^h))^2]
- 值函数训练（expectile regression）：L_ψ = E_{(g^h,o_t,a_t,r_t,o_{t+1})∼D_{sys2}}[L_τ(A(o_t, a_t))]
- 离线策略更新（引入策略锚定优势项）：L_{offline}^θ = -E_{(g^h,o_t,a_t)∼D_{sys2}}[λA(ot, π_off^θ(at|ot;gh);gh) + exp(βA(ot, at;gh)) × log π_off^θ(at|ot;gh)]

**System 2 在线训练**：
- 在线策略更新 + KL正则化：L_{online}^θ = -E_{(g^h,o_t,a_t)∼B_{sys2}}[w_t log π_on^θ(at|ot;gh)] + η E_{(g^h,o_t)∼B_{sys2}}[DKL(π_on^θ ∥ π_off^θ)(o_t;g^h)]
- 权重 w_t = exp(1/α A(o_t, a_t; g^h))，基于优势函数计算
- KL正则化防止策略突变，稳定自改进

**角色专业化适配器**：
- System 1 和 System 2 共享预训练 backbone，但使用独立的 LoRA 适配器
- 推理时只激活对应系统的适配器

## 6. 主要结论与发现

### 核心结论
1. **性能领先**：Multi² 在 ScienceWorld、ALFWorld、TextCraft 三大基准上持续优于所有基线方法，在ID和OOD分割上均取得最高分数

2. **令牌效率优势**：Multi² 在保持最高性能的同时实现更低令牌消耗，归一化令牌效率（ReAct=1.0）达4.48（ID）和13.77（OOD）

3. **目标漂移鲁棒性**：在高难度任务（需更长交互）上，Multi² 性能下降幅度最小，验证其缓解目标漂移的能力

4. **在线适应有效**：从离线策略到在线适应，性能从 60.68%/29.04% 提升至 69.30%/30.12%（ScienceWorld ID/OOD）

5. **角色专业化关键**：消融实验证实 SFT 用于 System 1、RL 用于 System 2 的分配最优，角色专用适配器显著优于共享适配器

### 关键发现
- 子目标错误占总失败比例低（ID: 3.70%, OOD: 5.88%），说明大部分失败源于执行层而非规划层
- 模型规模在OOD分割上影响更显著（1.5B→7B，OOD从48.57%升至64.29%）
- 在线RL在分布偏移下收益更大，OOD分割改善率高于ID分割

## 7. 优点

1. **层次化解耦设计**：显式分离规划与执行，将两个功能作为不同优化问题处理，而非仅通过提示分离

2. **离线到在线RL管线**：结合离线RL的稳定初始化和在线RL的持续自改进，通过KL正则化确保稳定性

3. **策略锚定优势项**：离线目标中引入 critic 引导的偏好项，减少过度模仿，鼓励超越离线轨迹的迁移

4. **角色专用适配器**：参数级解耦而非仅层级提示分离，确保角色边界清晰

5. **全面评估**：多基准、多backbone、ID/OOD双分割、令牌效率分析，实验覆盖完整

6. **资源公开**：发布三个层级基准数据集及预处理代码，支持可复现训练和公平比较
